# LES-120: Quantity stepper input not synced with DNA design system

## Symptom
Ô tăng giảm số lượng (`.wz-qty`) hiển thị spinner arrows mặc định của browser, background/border không đồng bộ với DNA design tokens. Trông lạc lõng so với phần còn lại của UI.

## Root Cause
`input[type=number]` bên trong `.wz-qty` không có CSS reset:
- Thiếu `-webkit-appearance: none` → Chrome/Safari hiển thị spinner arrows
- Thiếu `-moz-appearance: textfield` → Firefox hiển thị spinner arrows
- Không có component CSS `.wz-qty` trong `src/main.css` — hoàn toàn phụ thuộc Tailwind inline classes, không đủ để override browser defaults

## Detection
```css
/* Kiểm tra compiled CSS có chứa: */
.wz-qty input.qty::-webkit-inner-spin-button { -webkit-appearance: none }
/* và */
.wz-qty input.qty { -moz-appearance: textfield }
```

Nếu KHÔNG có → bug.

## Fix
Thêm component CSS vào `src/main.css`:

```css
/* Quantity stepper — DNA design system */
.wz-qty {
  display: inline-flex;
  align-items: center;
  border: 1.5px solid var(--wz-outline-variant, #c4c6d2);
  border-radius: var(--wz-radius);
  background: var(--wz-surface-pure);
  overflow: hidden;
}

.wz-qty__btn {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 2rem;
  height: 2rem;
  background: transparent;
  border: none;
  color: var(--wz-primary);
  cursor: pointer;
  transition: background 0.15s;
  flex-shrink: 0;
}

.wz-qty__btn:hover {
  background: var(--wz-surface-cool);
}

.wz-qty__btn:active {
  background: var(--wz-primary-container);
  color: var(--wz-on-primary-container, #8ba6ed);
}

.wz-qty input.qty {
  width: 2.5rem;
  height: 2rem;
  text-align: center;
  font-size: 0.875rem;
  font-weight: 600;
  font-family: var(--wz-font, inherit);
  color: var(--wz-on-surface);
  background: var(--wz-surface-pure);
  border: none;
  border-left: 1.5px solid var(--wz-outline-variant, #c4c6d2);
  border-right: 1.5px solid var(--wz-outline-variant, #c4c6d2);
  outline: none;
  -moz-appearance: textfield;
}

.wz-qty input.qty::-webkit-outer-spin-button,
.wz-qty input.qty::-webkit-inner-spin-button {
  -webkit-appearance: none;
  margin: 0;
}
```

## Prevention
- Khi tạo component `.wz-qty` mới → luôn thêm CSS component vào `src/main.css`, không chỉ dùng Tailwind inline
- Mọi `input[type=number]` trong design system phải có appearance reset
- Checklist trước bàn giao: inspect quantity stepper → không thấy browser spinner arrows
