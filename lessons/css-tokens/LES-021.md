**Triệu chứng:** Product images hiển thị ở kích thước gốc (1200×1600px), phá vỡ grid/flex containers. Ảnh không bị constrain bởi parent.
**Nguyên nhân:** `tailwind.config.js` có `corePlugins: { preflight: false }`. Tailwind preflight bình thường thêm `img { display: block; max-width: 100%; height: auto; }`. Khi tắt preflight, images là `display: inline` và không có `max-width`.
**Pattern sai:**
```js
// tailwind.config.js
corePlugins: { preflight: false }
// + KHÔNG có image reset trong src/main.css
```
**Phải làm:**
```css
/* src/main.css — thêm ngay sau @tailwind utilities */
img, video, svg {
  display: block;
  max-width: 100%;
  height: auto;
}
```
**Grep check:** `preflight:\s*false` trong tailwind.config.js + thiếu `img.*max-width` trong src/*.css → CRITICAL
