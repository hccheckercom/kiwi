**Pattern sai:** Hardcode array navigation items trực tiếp trong header.php.

**Code sai:**
```php
// ❌ Hardcode — admin không thể thay đổi menu
$nav_items = array(
    home_url( '/bo-suu-tap/' ) => 'Bộ sưu tập',
    home_url( '/phong-khach/' ) => 'Phòng khách',
    home_url( '/phong-ngu/' ) => 'Phòng ngủ',
    home_url( '/ban-an/' ) => 'Bàn ăn',
    home_url( '/uu-dai/' ) => 'Ưu đãi',
);
```

**Code đúng:**
```php
// ✅ Dùng WordPress menu system (admin quản lý từ Appearance → Menus)
wp_nav_menu( [
    'theme_location' => 'primary',
    'container'      => false,
    'menu_class'     => 'flex items-center gap-6',
    'fallback_cb'    => false,
] );

// Hoặc nếu cần custom markup:
$menu_items = wp_get_nav_menu_items( 'primary' );
if ( $menu_items ) :
    foreach ( $menu_items as $item ) : ?>
        <a href="<?php echo esc_url( $item->url ); ?>" 
           class="text-sm font-medium hover:text-primary">
            <?php echo esc_html( $item->title ); ?>
        </a>
    <?php endforeach;
endif;
```

**Lý do:** Blueprint Rule #14 — KHÔNG hardcode nội dung tĩnh. Hardcode nav = admin không thể thay đổi menu từ dashboard. Khi clone theme cho shop khác, phải sửa code thay vì chỉ đổi menu trong admin. Đây là anti-pattern cho Golden Template (clone-ready).

**Grep check:** `\$nav_items\s*=\s*array` hoặc hardcoded Vietnamese URLs trong header.php → HIGH
