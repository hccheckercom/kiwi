## Bad
```tsx
<h1 className="font-bold lg:text-4xl">Tiêu đề</h1>
```
Có lg:text-4xl nhưng không có base text size — mobile dùng default (text-base), quá nhỏ cho heading.

## Good
```tsx
<h1 className="text-2xl md:text-3xl lg:text-4xl font-bold">Tiêu đề</h1>
```

## Why
Heading phải scale từ mobile → desktop. Thiếu base size khiến mobile heading quá nhỏ hoặc phụ thuộc vào browser default.
