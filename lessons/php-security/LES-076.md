## Bad
```php
$wpdb->query("DELETE FROM {$wpdb->prefix}orders WHERE id = $id");
```

## Good
```php
$wpdb->query( $wpdb->prepare(
    "DELETE FROM {$wpdb->prefix}orders WHERE id = %d", $id
) );
```

## Why
SQL query without prepare() — SQL injection. Always use $wpdb->prepare() with placeholders.
