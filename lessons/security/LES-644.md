## Bad
```php
subprocess.run(cmd, shell=True)
```

## Good
```php
subprocess.run(shlex.split(cmd), shell=False)
```

## Why
shell=True passes command through shell interpreter, enabling injection via user input

## Grep
`subprocess\.\w+\(.*shell\s*=\s*True` — in `**/*.py`
