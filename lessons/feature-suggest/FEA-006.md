**Mô tả:**
- Admin page PHP thuần tại `wp-admin/admin.php?page=wezone-voucher-template`
- Hiển thị 3 preview cards trực quan (Ticket/Gradient/Stamp) với inline CSS
- Click card → highlight border accent + checkmark → Save → PUT REST API
- Setting lưu vào `wz_voucher_card_template` WordPress option (giá trị 1/2/3)
- Áp dụng toàn site — tất cả thẻ voucher dùng mẫu đã chọn

**Detect pattern (absence):**
- `packages/wezone-core/src/AdminUI/` thiếu `VoucherTemplateAdminPage.php` → chưa có admin UI
- `get_option('wz_voucher_card_template')` không tồn tại → chưa có setting

**Files cần có:**
- `packages/wezone-core/src/AdminUI/VoucherTemplateAdminPage.php` — admin page class
- `packages/wezone-core/src/AdminUI/AdminPage.php` — `add_submenu_page()` dưới `wezone-admin`
- `packages/wezone-core/src/AdminUI/Plugin.php` — boot `VoucherTemplateAdminPage`
- REST endpoint `GET|PUT /wezone/v1/settings/voucher-template` trong `CouponController.php`

**3 mẫu thẻ voucher (card.php):**
- Template 1 — Ticket: gradient trái + code phải + perforated dashed divider + notch tròn
- Template 2 — Gradient: full gradient + wave SVG pattern + label lớn (text-3xl)
- Template 3 — Stamp: dashed border + badge tròn accent + icon ticket

**Lợi ích:** Admin kiểm soát visual branding thẻ voucher mà không cần code, phù hợp nhiều phong cách shop khác nhau
