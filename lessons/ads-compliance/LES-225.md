## Bad
```tsx
'Responsive 100% trên mọi thiết bị'
'Nền tảng thiết kế web TMĐT #1 Việt Nam'
'Dịch vụ thiết kế web tốt nhất Đắk Lắk'
'Đơn vị thiết kế web đầu tiên có trụ sở tại Tây Nguyên'
```
Absolute superlatives without third-party verification violate Google "Misleading Claims" policy.

## Good
```tsx
// Remove superlative
'Responsive trên mobile, tablet và desktop'

// Add source for ranking claim
'Nền tảng thiết kế web TMĐT phổ biến tại Việt Nam'

// Soften with qualifier
'Một trong những dịch vụ thiết kế web uy tín tại Đắk Lắk'

// If verifiable, cite source
'#1 trên Google cho "thiết kế web Buôn Ma Thuột" (tháng 5/2026)'
```

## Why
Google requires that superlative claims ("best", "#1", "100%") be verifiable by a third-party source. Without verification, these are classified as "unsubstantiated claims." The fix is simple: either (1) remove the superlative, (2) add a verifiable source, or (3) soften to a non-absolute statement. Note: "100%" in CSS context (width: 100%) is fine — only marketing claims are affected.
