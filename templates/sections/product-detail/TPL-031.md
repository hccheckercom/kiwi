# Product Quick View Modal — Lightbox Card

Quick view modal for product preview without leaving the current page.

## Features
- Modal overlay with product details
- Image gallery (thumbnail nav)
- Price, name, short description
- Add to cart button
- Close button + ESC key
- Responsive (full screen on mobile)

            <!-- Right: Details -->

## Usage

Include in footer or layout template. Trigger via JS:

```javascript
// On product card quick view button click
document.querySelectorAll('.quick-view-btn').forEach(btn => {
    btn.addEventListener('click', (e) => {
        const productId = e.target.dataset.productId;
        openQuickView(productId);
    });
});

function openQuickView(productId) {
    // Fetch product data via AJAX
    // Populate modal fields
    // Show modal
    document.getElementById('quick-view-modal').classList.remove('hidden');
}
```

## CSS Classes
- `.quick-view-modal` — modal wrapper (fixed, z-index high)
- `.quick-view-modal__overlay` — dark backdrop
- `.quick-view-modal__content` — white card (centered)

## JavaScript Required
- Fetch product data via AJAX
- Populate modal fields
- Image gallery navigation
- Quantity controls
- Add to cart AJAX
- Close modal (button, overlay click, ESC key)
