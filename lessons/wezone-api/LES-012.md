**Pattern sai:**
```php
// template gọi wz_get_login_url() nhưng inc/wz-shims.php không có function này
<a href="<?php echo esc_url( wz_get_login_url() ); ?>">Đăng nhập</a>
// → PHP Fatal: Call to undefined function wz_get_login_url()
```
**Phải làm:**
```php
// Đảm bảo mọi wz_*() gọi trong template đều có khai báo trong wz-shims.php
if ( ! function_exists( 'wz_get_login_url' ) ) {
    function wz_get_login_url(): string {
        return wp_login_url( get_permalink() );
    }
}
```
**Lý do:** Template gọi function chưa tồn tại → fatal error trên production
**Grep check:** `wz_[a-z_]+\(` trong templates → cross-check với `function wz_` trong wz-shims.php — thiếu = CRITICAL
