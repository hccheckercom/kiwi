## Bad
```html
<!-- Render-blocking script -->
<script src="/heavy-library.js"></script>

<!-- Image without dimensions — causes CLS -->
<img src="/hero.jpg">

<!-- Render-blocking CSS import -->
<style>@import url('https://fonts.googleapis.com/...');</style>
```
Poor Core Web Vitals (LCP > 2.5s, CLS > 0.1, INP > 200ms) directly reduce Google Ads Quality Score and increase CPC.

## Good
```tsx
// Next.js Image with dimensions — prevents CLS
<Image src="/hero.jpg" width={1200} height={630} priority alt="Hero" />

// Async/defer for non-critical scripts
<script src="/analytics.js" defer></script>

// Font preload instead of @import
<link rel="preload" href="/fonts/inter.woff2" as="font" type="font/woff2" crossOrigin="" />
```

## Why
Google confirmed Core Web Vitals affect Ad Rank since 2021. Landing pages with poor performance get: (1) lower Quality Score (1-3 points penalty), (2) higher CPC (up to 2x), (3) lower ad position. Key thresholds: LCP < 2.5s (largest image/text), CLS < 0.1 (no layout shifts), INP < 200ms (responsive to clicks). Use `next/image` with priority for above-fold images, defer non-critical JS, and avoid render-blocking resources.
