## Bad
```php
echo json_encode($data);
```

## Good
```php
echo json_encode($data, JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES);
```

## Why
json_encode() without JSON_UNESCAPED_UNICODE — Vietnamese text becomes \uXXXX sequences. Unreadable in logs.
