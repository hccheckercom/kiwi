## Pattern

Developer dùng WooCommerce JavaScript events, localized objects, hoặc script handles.

## Bad

```javascript
// WooCommerce JS events
jQuery( document.body ).on( 'added_to_cart', function( e, fragments, cart_hash, $button ) {
    // Update mini cart
});

jQuery( document.body ).on( 'removed_from_cart', function() {
    // Update cart count
});

jQuery( document.body ).on( 'updated_cart_totals', function() {
    // Recalculate
});

jQuery( document.body ).on( 'updated_checkout', function() {
    // Refresh payment methods
});

// WooCommerce localized params
var ajaxUrl = wc_add_to_cart_params.ajax_url;
var cartUrl = wc_add_to_cart_params.cart_url;
var nonce   = wc_checkout_params.update_order_review_nonce;

// Trigger WC events
jQuery( document.body ).trigger( 'wc_fragment_refresh' );
jQuery( '.variations_form' ).trigger( 'wc_variation_form' );
```

```php
// Enqueue WC scripts
wp_enqueue_script( 'wc-add-to-cart' );
wp_enqueue_script( 'wc-cart-fragments' );
wp_enqueue_script( 'wc-checkout' );
wp_enqueue_script( 'wc-single-product' );
wp_enqueue_script( 'woocommerce' );
```

## Good

```javascript
// Wezone JS events
document.addEventListener( 'wz:cart:added', function( e ) {
    const { productId, qty, cartCount } = e.detail;
    // Update mini cart
});

document.addEventListener( 'wz:cart:removed', function( e ) {
    const { itemKey, cartCount } = e.detail;
    // Update cart count
});

document.addEventListener( 'wz:cart:updated', function( e ) {
    const { total, items } = e.detail;
    // Recalculate
});

document.addEventListener( 'wz:checkout:updated', function( e ) {
    // Refresh payment methods
});

// Wezone localized data
var ajaxUrl = wzData.ajaxUrl;
var cartUrl = wzData.cartUrl;
var nonce   = wzData.nonce;

// Trigger Wezone events
document.dispatchEvent( new CustomEvent( 'wz:cart:refresh' ) );
document.dispatchEvent( new CustomEvent( 'wz:variation:changed', { detail: { variationId } } ) );
```

```php
// Enqueue Wezone scripts
wp_enqueue_script( 'wz-cart' );
wp_enqueue_script( 'wz-checkout' );
wp_enqueue_script( 'wz-product' );
```

## Why

- WooCommerce JS scripts (`wc-add-to-cart`, `wc-cart-fragments`) không được register → `wp_enqueue_script` fails silently
- Localized objects (`wc_add_to_cart_params`, `wc_checkout_params`) không được `wp_localize_script` → undefined JS error
- jQuery events (`added_to_cart`, `updated_checkout`) không bao giờ được trigger → listeners không chạy
- Wezone dùng vanilla JS CustomEvents với prefix `wz:` — không phụ thuộc jQuery
- Data object: `wzData` (localized qua `wp_localize_script( 'wz-core', 'wzData', [...] )`)
