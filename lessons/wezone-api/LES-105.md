## Bad
```php
define('WZ_API_KEY', get_option('wz_api_key'));
```

## Good
```php
// Don't define constants from DB at file load time
// Instead, use a getter:
function wz_api_key(): string {
    static $key = null;
    if ($key === null) { $key = get_option('wz_api_key', ''); }
    return $key;
}
```

## Why
define() with get_option() at file load time — if DB isn't ready yet, constant = false forever.
