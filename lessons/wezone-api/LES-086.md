## Bad
```php
namespace WeZone\Shipping;
function boot() {
    add_action('init', 'register_routes');
    flush_rewrite_rules();
}
```

## Good
```php
namespace WeZone\Shipping;
function boot() {
    \add_action('init', [self::class, 'register_routes']);
    \flush_rewrite_rules();
}
```

## Why
WordPress functions called without \ prefix in namespace — PHP looks for WeZone\Shipping\add_action() which doesn't exist. Fatal error.
