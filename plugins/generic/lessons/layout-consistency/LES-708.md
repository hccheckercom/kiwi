## Bad
```tsx
// app/shop/page.tsx
<section className="px-4">...</section>
<section className="px-6">...</section>
```
Hai section cùng trang dùng padding khác nhau — visual rhythm bị phá.

## Good
```tsx
<section className="px-4">...</section>
<section className="px-4">...</section>
```

## Why
Container padding phải nhất quán trong cùng page/section group. px-4 là standard cho mobile, px-6/px-8 gây lệch alignment với header/footer.
