**Mô tả:**
- Button toggle dark/light mode với `data-theme-toggle` attribute
- `<html>` element dùng `data-theme="dark|light"` để switch CSS variables
- Persist vào localStorage, respect `prefers-color-scheme` nếu chưa có preference

**Detect pattern (absence):**
- `header.php` hoặc JS files thiếu `data-theme-toggle` → chưa có dark mode toggle
- Không có `data-theme` attribute handling → theme chỉ có 1 mode

**Files cần có:**
- `header.php` — button với `data-theme-toggle` attribute + sun/moon icons
- `assets/js/main.js` — toggle handler: read/write localStorage('theme'), set `data-theme` on `<html>`
- `assets/css/main.css` — `[data-theme="dark"]` block với semantic token overrides

**Từ blueprint:**
- NEVER hardcode colors — always use `var(--color-*)` tokens
- System preference respected — if no localStorage, follow `prefers-color-scheme`
- Transition on toggle — `transition: background-color 200ms, color 200ms`
