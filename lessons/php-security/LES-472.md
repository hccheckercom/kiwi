## Bad
```php
$secret = get_option( 's3_secret_key', '' ); // Plaintext in wp_options
$key = get_option( 's3_access_key', '' );
```

## Good
```php
// Option 1: Use constants (recommended)
$secret = defined( 'WEZONE_S3_SECRET' ) ? WEZONE_S3_SECRET : '';
$key = defined( 'WEZONE_S3_KEY' ) ? WEZONE_S3_KEY : '';

// Option 2: Encrypt before storing
$secret = $this->decrypt( get_option( 's3_secret_key_encrypted', '' ) );
```

## Why
wp_options table is dumped in DB backups and visible to any plugin with DB access. Secrets stored plaintext are exposed. Use constants (wp-config.php) or encrypt before storing.

## Grep
`get_option\([^)]*secret|get_option\([^)]*password|get_option\([^)]*api_key` — in `**/*.php`
