## Bad
```php
cache = {}
def update(key, val):
    cache[key] = val  # race condition
```

## Good
```php
import threading
lock = threading.Lock()
def update(key, val):
    with lock:
        cache[key] = val
```

## Why
Concurrent dict mutation without locking causes race conditions and data corruption
