## Bad
```php
function handle_ajax_request() {
    if (!$valid) {
        wp_die('Invalid request');
    }
}
```

## Good
```php
function handle_ajax_request() {
    if (!$valid) {
        wp_send_json_error(['message' => 'Invalid request'], 400);
    }
}
```

## Why
wp_die() in REST/AJAX handler returns HTML — frontend JS expects JSON, gets parse error.
