## Bad
```tsx
<nav className="hidden">
  <SidebarLinks />
</nav>
```
Element is `hidden` with no responsive show — permanently invisible, likely a mistake.

## Good
```tsx
<nav className="hidden md:block">
  <SidebarLinks />
</nav>
```
Hidden on mobile, shown on desktop — intentional responsive behavior.

## Why
`hidden` without any responsive show class means content is permanently invisible. Either it's dead code (remove it) or a responsive breakpoint is missing. Both are bugs.
