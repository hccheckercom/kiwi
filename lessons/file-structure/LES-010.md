**Pattern sai:**
```css
/*
Theme Name: WeZone Theme
Template: storefront
*/
```
**Phải làm:**
```css
/*
Theme Name: WeZone Theme
/* Không có dòng Template: */
*/
```
**Lý do:** `Template:` trong style.css = WordPress coi đây là child theme → load parent theme CSS/PHP
**Grep check:** `^Template:` — trong style.css → CRITICAL
