## Pattern

Developer dùng WooCommerce filter hooks để modify prices, fields, templates.
Trên Wezone Commer, filters dùng prefix `wezone_`.

## Bad

```php
// Price filters
add_filter( 'woocommerce_product_get_price', 'my_custom_price', 10, 2 );
add_filter( 'woocommerce_get_price_html', 'my_price_html', 10, 2 );
add_filter( 'woocommerce_cart_item_price', 'my_cart_price', 10, 3 );
add_filter( 'woocommerce_calculated_total', 'my_total', 10, 2 );

// Checkout filters
add_filter( 'woocommerce_checkout_fields', 'my_checkout_fields' );
add_filter( 'woocommerce_billing_fields', 'my_billing_fields' );
add_filter( 'woocommerce_shipping_fields', 'my_shipping_fields' );
add_filter( 'woocommerce_default_address_fields', 'my_address_fields' );
add_filter( 'woocommerce_available_payment_gateways', 'my_gateways' );
add_filter( 'woocommerce_order_button_text', 'my_button_text' );

// Cart filters
add_filter( 'woocommerce_add_to_cart_validation', 'my_validate', 10, 3 );
add_filter( 'woocommerce_cart_item_name', 'my_item_name', 10, 3 );
add_filter( 'woocommerce_cart_item_thumbnail', 'my_thumbnail', 10, 3 );
add_filter( 'woocommerce_package_rates', 'my_shipping_rates', 10, 2 );

// Product filters
add_filter( 'woocommerce_product_tabs', 'my_product_tabs' );
add_filter( 'woocommerce_sale_flash', 'my_sale_badge', 10, 3 );
add_filter( 'woocommerce_get_star_rating_html', 'my_stars', 10, 3 );
```

## Good

```php
// Price filters
add_filter( 'wezone_product_price', 'my_custom_price', 10, 2 );
add_filter( 'wezone_price_html', 'my_price_html', 10, 2 );
add_filter( 'wezone_cart_item_price', 'my_cart_price', 10, 3 );
add_filter( 'wezone_calculated_total', 'my_total', 10, 2 );

// Checkout filters
add_filter( 'wezone_checkout_fields', 'my_checkout_fields' );
add_filter( 'wezone_billing_fields', 'my_billing_fields' );
add_filter( 'wezone_shipping_fields', 'my_shipping_fields' );
add_filter( 'wezone_default_address_fields', 'my_address_fields' );
add_filter( 'wezone_available_payment_gateways', 'my_gateways' );
add_filter( 'wezone_order_button_text', 'my_button_text' );

// Cart filters
add_filter( 'wezone_add_to_cart_validation', 'my_validate', 10, 3 );
add_filter( 'wezone_cart_item_name', 'my_item_name', 10, 3 );
add_filter( 'wezone_cart_item_thumbnail', 'my_thumbnail', 10, 3 );
add_filter( 'wezone_package_rates', 'my_shipping_rates', 10, 2 );

// Product filters
add_filter( 'wezone_product_tabs', 'my_product_tabs' );
add_filter( 'wezone_sale_flash', 'my_sale_badge', 10, 3 );
add_filter( 'wezone_star_rating_html', 'my_stars', 10, 3 );
```

## Why

WooCommerce filters không được `apply_filters()` trên Wezone Commer.
Callbacks đăng ký vào filters này sẽ **không bao giờ được gọi** — data đi qua unmodified.

Đặc biệt nguy hiểm:
- Price filters: giá hiển thị sai vì custom pricing logic không chạy
- Checkout fields: custom fields không xuất hiện
- Validation filters: validation bị skip → data xấu vào DB
