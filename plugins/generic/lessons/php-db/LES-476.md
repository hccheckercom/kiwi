## Bad
```php
return $wpdb->get_results( "SELECT * FROM {$table} WHERE status = 'success'" ); // Returns array|null, but caller expects array
```

## Good
```php
$results = $wpdb->get_results( "SELECT * FROM {$table} WHERE status = 'success'" );
return is_array( $results ) ? $results : array();
```

## Why
$wpdb->get_results() returns null on DB failure. Returning directly causes type errors when caller expects array. Guard with is_array() or null coalesce.
