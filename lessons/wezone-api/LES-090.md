## Bad
```php
add_action('init', [$this, 'onInit']);
// But class has no onInit() method
```

## Good
```php
add_action('init', [$this, 'register_routes']);
// Method exists in this class
public function register_routes(): void { ... }
```

## Why
Hook callback references non-existent method — WordPress silently fails. Feature never activates, no error.
