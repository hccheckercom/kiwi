**Triệu chứng:** Scrollbar ngang hiển thị trên mobile ở các section scroll (Flash Sale, Best Sellers, Blog, Recently Viewed). Class `hide-scrollbar` có trong HTML nhưng không có CSS rule nào match.
**Nguyên nhân:** `hide-scrollbar` không phải Tailwind built-in utility. Cần define thủ công trong `src/main.css` hoặc qua plugin.
**Phải làm:**
```css
/* src/main.css */
.hide-scrollbar {
  -ms-overflow-style: none;
  scrollbar-width: none;
}
.hide-scrollbar::-webkit-scrollbar {
  display: none;
}
```
**Grep check:** `hide-scrollbar` trong *.php templates + thiếu trong src/*.css hoặc tailwind.config.js → HIGH
