## Bad
```php
private function normalize( \WP_User $user ): array {
    return array(
        'phone'   => get_user_meta( $user->ID, 'billing_phone', true ),
        'billing' => array(
            'first_name' => get_user_meta( $user->ID, 'billing_first_name', true ),
            'last_name'  => get_user_meta( $user->ID, 'billing_last_name', true ),
            // ... 15 more calls = 18 total queries per user
        ),
    );
}
```

## Good
```php
private function normalize( \WP_User $user ): array {
    $meta = get_user_meta( $user->ID ); // 1 query — returns ALL meta
    $get  = static fn( string $key ) => $meta[ $key ][0] ?? '';

    return array(
        'phone'   => $get( 'billing_phone' ),
        'billing' => array(
            'first_name' => $get( 'billing_first_name' ),
            'last_name'  => $get( 'billing_last_name' ),
            // ...
        ),
    );
}
```

## Why
`get_user_meta($id, $key, true)` fires 1 query per call. In `search()` returning 20 users, that's 18x20=360 queries. `get_user_meta($id)` (no key) fetches all meta in 1 query, then pick from the array.
