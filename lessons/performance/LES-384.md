# Bug: wp_cache_set() Không Có TTL → Redis Phình Vô Hạn

## Vấn đề

`wp_cache_set()` không truyền TTL (tham số thứ 4) → key tồn tại mãi mãi trong Redis → memory tăng liên tục.

**Ví dụ lỗi:**
```php
// ❌ SAI — không có TTL, key tồn tại vĩnh viễn
wp_cache_set( 'wz_products_list', $products, 'wezone' );

// ❌ SAI — TTL = 0 nghĩa là vĩnh viễn
wp_cache_set( 'wz_products_list', $products, 'wezone', 0 );
```

**Timeline:**
```
Day 1:   100 product keys cached = 1MB
Day 30:  Products updated, old keys vẫn còn = 10MB
Day 180: Thousands of orphan keys = 100MB+
```

## Root Cause

1. `wp_cache_set()` default TTL = 0 = no expiration
2. Redis `SET` without `EX`/`PX` → key never expires
3. Không có cleanup mechanism cho orphan keys
4. `WP_REDIS_MAXTTL` chưa set → plugin không enforce TTL

## Fix

### 1. Luôn truyền TTL (BẮT BUỘC)

```php
// ✅ ĐÚNG — có TTL
wp_cache_set( 'wz_products_list', $products, 'wezone', 3600 );  // 1 hour
wp_cache_set( 'wz_settings', $settings, 'wezone', 86400 );      // 24 hours
wp_cache_set( 'wz_user_cart', $cart, 'wezone', 1800 );           // 30 min
```

### 2. Set WP_REDIS_MAXTTL trong wp-config.php

```php
// wp-config.php — safety net: mọi key hết hạn sau 24h dù code quên TTL
define( 'WP_REDIS_MAXTTL', 86400 );
```

### 3. TTL Guide theo loại data

| Data type | TTL | Lý do |
|-----------|-----|-------|
| Product list | 3600 (1h) | Thay đổi hiếm |
| Product detail | 1800 (30m) | Có thể cập nhật giá |
| Cart/session | 1800 (30m) | User-specific, cần fresh |
| Settings | 86400 (24h) | Hầu như không đổi |
| Search results | 600 (10m) | Cần fresh, high variety |
| API responses | 300 (5m) | External data, cần fresh |

## Verification

```bash
# Check wp_cache_set calls thiếu TTL
rg 'wp_cache_set\s*\([^)]+,\s*[^)]+,\s*['"'"'""][^'"'"'""]+['"'"'""\s]*\)' --type php

# Check Redis key TTL
redis-cli --scan --pattern '*wz_*' | while read key; do
    ttl=$(redis-cli TTL "$key")
    echo "$key → TTL: $ttl"
done

# Check keys without TTL (TTL = -1)
redis-cli --scan | while read key; do
    ttl=$(redis-cli TTL "$key")
    [ "$ttl" = "-1" ] && echo "NO TTL: $key"
done
```
