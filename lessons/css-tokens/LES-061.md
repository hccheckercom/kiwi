**Pattern sai:**
```html
<nav aria-label="Breadcrumb">
  <ol class="flex items-center gap-1 ...">
```
**Phải làm:**
Thêm CSS reset global trong `src/main.css`:
```css
nav[aria-label="Breadcrumb"] ol {
  list-style: none;
  padding: 0;
  margin: 0;
}
```
**Lý do:** Khi Tailwind preflight bị disable hoặc theme không dùng `@tailwind base` đầy đủ, browser default `list-style: decimal` vẫn áp dụng cho `<ol>`. Breadcrumb dùng `<ol>` cho semantic nhưng không cần hiển thị số. Fix global trong CSS thay vì thêm `list-none` vào từng template (17+ files).
**Grep check:** `<ol` trong `nav[aria-label="Breadcrumb"]` + kiểm tra `src/main.css` có reset `list-style: none` — nếu thiếu → HIGH
