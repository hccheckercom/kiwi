# Bad Code

```python
def read_config():
    f = open('config.json')
    data = json.load(f)
    f.close()  # Forgot to close if json.load() raises
    return data

def query_database(sql):
    conn = db.connect()
    cursor = conn.cursor()
    cursor.execute(sql)
    result = cursor.fetchall()
    conn.close()  # Never called if execute() raises
    return result

def process_file(path):
    file = open(path)
    try:
        return process(file.read())
    finally:
        file.close()  # Verbose, error-prone
```

# Good Code

```python
def read_config():
    with open('config.json') as f:
        return json.load(f)

def query_database(sql):
    with db.connect() as conn:
        with conn.cursor() as cursor:
            cursor.execute(sql)
            return cursor.fetchall()

# Or nested context managers
def query_database_v2(sql):
    with db.connect() as conn, conn.cursor() as cursor:
        cursor.execute(sql)
        return cursor.fetchall()

def process_file(path):
    with open(path) as file:
        return process(file.read())
```

# Why

**Context managers guarantee cleanup:**
- **Always runs:** `__exit__` called even if exception raised
- **Automatic:** No manual cleanup code needed
- **Composable:** Multiple resources with nested `with`
- **Readable:** Intent is clear

**Without context managers:**
- **Resource leaks:** File handles, connections, locks never released
- **Production issues:** "Too many open files" errors
- **Memory leaks:** Objects held longer than needed
- **Deadlocks:** Locks not released on exception

**Common resources needing context managers:**
- Files: `open()`
- Database connections: `db.connect()`
- Locks: `threading.Lock()`, `asyncio.Lock()`
- Network sockets: `socket.socket()`
- Temporary files: `tempfile.NamedTemporaryFile()`

**Custom context managers:**
```python
from contextlib import contextmanager

@contextmanager
def timer(name):
    start = time.time()
    try:
        yield
    finally:
        print(f"{name}: {time.time() - start:.2f}s")

with timer("process_data"):
    process_data()
```

**Async context managers:**
```python
async with aiohttp.ClientSession() as session:
    async with session.get(url) as response:
        return await response.json()
```
