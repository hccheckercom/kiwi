**Pattern sai:**
```php
// change_password() — không check user existence
public function change_password( WP_REST_Request $request ): WP_REST_Response {
    $user = wp_get_current_user();
    $current_password = $request->get_param('current_password');
    if ( ! wp_check_password( $current_password, $user->user_pass, $user->ID ) ) {
        return new WP_REST_Response( array( 'error' => 'Wrong password' ), 400 );
    }
}
```
**Phải làm:**
```php
$user = wp_get_current_user();
if ( ! $user->exists() ) {
    return new WP_REST_Response( array( 'error' => 'User not found' ), 401 );
}
```
**Lý do:** Race condition — user có thể bị xóa sau khi require_login() pass nhưng trước khi wp_get_current_user() chạy → fatal error khi access $user->user_pass
**Grep check:** `wp_get_current_user()` không có `->exists()` check sau đó → HIGH
