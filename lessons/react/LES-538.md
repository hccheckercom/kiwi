## Bad
```php
// RiskHeatmap.tsx
const response = await axios.post('http://localhost:8000/api/scan', {
  path: 'wezone-plugins',
  severity: 'ALL'
})

// DependencyGraph.tsx
const response = await axios.post('http://localhost:8000/api/plan', {
  path: 'web',
  severity: 'CRITICAL'
})
```

## Good
```php
// config.ts
export const API_BASE_URL = import.meta.env.VITE_API_URL || 'http://localhost:8000'

// RiskHeatmap.tsx
import { API_BASE_URL } from '../config'

const response = await axios.post(`${API_BASE_URL}/api/scan`, {
  path: 'wezone-plugins',
  severity: 'ALL'
})
```

## Why
Hardcoded localhost URLs make it impossible to deploy to production or staging without code changes. Every component that calls the API must be updated manually. Use environment variables (VITE_API_URL, REACT_APP_API_URL, NEXT_PUBLIC_API_URL) to configure API base URL per environment.
