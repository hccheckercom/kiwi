**Mô tả:**
Theme cần helper function để abbreviate số lớn theo quy tắc blueprint:

| Context | Threshold | Display |
|---------|-----------|---------|
| Price | > 999,999,999 | "1.2 tỷ" |
| Stock | > 999 | "999+" |
| Reviews | > 9,999 | "9.9K+" |
| Cart badge | > 99 | "99+" |
| Sold count | > 9,999 | "10K+" |

**Implementation gợi ý:**
```php
function wz_abbreviate_number( int $num, string $context = 'default' ): string {
    switch ( $context ) {
        case 'cart':
            return $num > 99 ? '99+' : (string) $num;
        case 'stock':
            return $num > 999 ? '999+' : number_format( $num );
        case 'sold':
        case 'reviews':
            if ( $num >= 10000 ) {
                return round( $num / 1000, 1 ) . 'K+';
            }
            return number_format( $num );
        case 'price':
            if ( $num >= 1000000000 ) {
                return round( $num / 1000000000, 1 ) . ' tỷ';
            }
            return number_format( $num );
        default:
            if ( $num >= 10000 ) {
                return round( $num / 1000, 1 ) . 'K+';
            }
            return number_format( $num );
    }
}
```
