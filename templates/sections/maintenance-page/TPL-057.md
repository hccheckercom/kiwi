# Maintenance Page — 503 Service Unavailable

Maintenance mode page với countdown timer và email subscription.

## Use Cases
- Scheduled maintenance (trang 14)
- Site under construction
- Coming soon page
- Emergency downtime

## Features
- 🔧 Clear maintenance message
- ⏰ Countdown timer (optional)
- 📧 Email notification signup
- 📱 Mobile-friendly
- 🔍 SEO-friendly (503 status code)
- 🚫 Blocks Googlebot properly (Retry-After header)

## Code

## Data Sources

- `maintenance_reason` — Reason message
- `facebook_url` — Facebook page URL
- `email` — Support email
