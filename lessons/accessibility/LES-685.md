## Bad
```php
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head><?php wp_head(); ?></head>
<body <?php body_class(); ?>>
  <header>
    <nav>...</nav>
  </header>
  <main>
```

## Good
```php
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head><?php wp_head(); ?></head>
<body <?php body_class(); ?>>
  <a href="#main-content" class="sr-only focus:not-sr-only focus:absolute focus:top-4 focus:left-4 focus:z-50 focus:bg-white focus:px-4 focus:py-2">
    Bỏ qua điều hướng
  </a>
  <header>
    <nav>...</nav>
  </header>
  <main id="main-content">
```

## Why
WCAG 2.4.1: skip navigation link bắt buộc. WP themes thường có header phức tạp (logo, menu, search, cart) — keyboard users cần skip link.
