## Pattern sai

```tsx
// Parent div có onClick handler
<div
  onClick={() => handleSelect(item)}
  className="cursor-pointer ..."
>
  <span>{item.name}</span>
  {/* Button con chỉ có stopPropagation — không đủ */}
  <button
    type="button"
    onClick={(e) => {
      e.stopPropagation()  // ❌ Thiếu preventDefault
      handleDelete(item.id)
    }}
  >
    <Trash2 />
  </button>
</div>
```

## Phải làm

```tsx
// Cần CẢ HAI: stopPropagation + preventDefault
<div
  onClick={() => handleSelect(item)}
  className="cursor-pointer ..."
>
  <span>{item.name}</span>
  <button
    type="button"
    onClick={(e) => {
      e.stopPropagation()  // ✅ Ngăn bubble lên parent div
      e.preventDefault()   // ✅ Ngăn default browser behavior
      handleDelete(item.id)
    }}
  >
    <Trash2 />
  </button>
</div>
```

## Dropdown không có click-outside handler

```tsx
// SAI — không đóng được khi click outside
export default function Picker() {
  const [open, setOpen] = useState(false)
  // ❌ Không có useRef + useEffect click-outside
  return (
    <div>
      <button onClick={() => setOpen(!open)}>Open</button>
      {open && <div className="absolute ...">...</div>}
    </div>
  )
}

// ĐÚNG — có click-outside handler
export default function Picker() {
  const [open, setOpen] = useState(false)
  const ref = useRef<HTMLDivElement>(null)

  useEffect(() => {
    const handler = (e: MouseEvent) => {
      if (ref.current && !ref.current.contains(e.target as Node)) {
        setOpen(false)
      }
    }
    if (open) document.addEventListener('mousedown', handler)
    return () => document.removeEventListener('mousedown', handler)
  }, [open])

  return (
    <div ref={ref}>
      <button onClick={() => setOpen(!open)}>Open</button>
      {open && <div className="absolute ...">...</div>}
    </div>
  )
}
```

## Lý do

Khi button con nằm trong `<div onClick>`, click event bubble từ button lên div. `stopPropagation()` ngăn bubble nhưng không ngăn default behavior của browser. Trong một số trường hợp (đặc biệt khi có `confirm()` dialog hoặc form submit), cần cả `preventDefault()` để đảm bảo chỉ handler của button con chạy.

Dropdown không có click-outside handler gây UX bug: user không thể đóng dropdown bằng cách click ra ngoài.
