**Pattern sai:**
```css
box-shadow: 0 4px 16px rgba(249, 115, 22, 0.25);
border: 1px solid rgba(196, 197, 215, 0.3);
```
**Phải làm:**
```css
box-shadow: 0 4px 16px var(--shadow-accent);
border: 1px solid var(--color-border-light);
```
**Lý do:** Khi đổi brand color chỉ cần sửa _tokens.css — hardcode phải sửa toàn bộ theme
**Grep check:** `rgba?\(|#[0-9a-fA-F]{3,8}` — trong *.css ngoài _tokens.css → HIGH
