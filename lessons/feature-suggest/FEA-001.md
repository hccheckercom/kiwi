**Mô tả:**
- Sidebar filter (giá, thuộc tính, tình trạng) gửi AJAX khi user thay đổi bất kỳ input nào
- Product grid cập nhật realtime (debounce 300ms) mà không reload trang
- Sort dropdown cũng trigger AJAX thay vì reload
**Detect pattern (absence):**
- JS file thiếu `wzRunFilter` hoặc `wz_filter_products` → chưa có realtime filter
**Files cần có:**
- `inc/filter.php` — AJAX handler `wz_filter_products`
- `assets/js/main.js` — `wzRunFilter()`, `wzDebouncedFilter()`, `wzGetFilterParams()`
- `template-parts/archive/sidebar-filters.php` — `data-category-id` attribute
**Lợi ích:** UX mượt hơn, giảm page load, user thấy kết quả ngay khi chọn filter
