**Mô tả:**
- Admin tùy chỉnh 3 màu: primary (nền chính), accent (màu nổi bật), text (màu chữ)
- 6 bộ màu preset có sẵn (Mặc định, Đỏ, Xanh lá, Tím, Hồng, Cam)
- Color picker + hex input cho từng màu
- Preview cards cập nhật realtime theo màu đang chọn
- Lưu vào `wz_voucher_colors` WordPress option qua REST API

**Detect pattern (absence):**
- `CouponController.php` thiếu `get_voucher_colors` → chưa có REST endpoint màu
- `card.php` thiếu `wz_voucher_colors` → thẻ dùng màu cứng, không tùy chỉnh được

**Files cần có:**
- `packages/wezone-core/src/AdminUI/CouponController.php` — `get_voucher_colors()`, `update_voucher_colors()` (validate hex #rrggbb)
- `packages/wezone-core/src/AdminUI/Plugin.php` — route `GET|PUT /wezone/v1/settings/voucher-colors`
- `template-parts/voucher/card.php` — đọc `get_option('wz_voucher_colors')`, dùng `$_primary`, `$_accent`, `$_text` trong inline style
- `packages/wezone-core/src/AdminUI/pages/VoucherTemplatePicker.tsx` — color picker section + preset buttons, preview realtime

**Security note:** Validate hex color với regex `/^#[0-9a-fA-F]{6}$/` trước khi lưu — tránh XSS qua inline style injection.

**Lợi ích:** Admin thay đổi màu sắc thẻ voucher phù hợp brand của từng shop mà không cần code
