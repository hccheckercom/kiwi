# LES-113: wp_get_attachment_url() N+1 Trong List Endpoints

## Problem

Khi format danh sách products, mỗi product gọi `wp_get_attachment_url()` cho main image + gallery images:
- Mỗi call = 2 queries (SELECT from posts + SELECT _wp_attached_file from postmeta)
- 20 products × (1 main + 2 gallery) = ~60 queries chỉ cho images
- Với 100 products/page → 300+ queries

## Fix

Batch load tất cả attachment URLs trong 2 queries:

```php
private function batch_load_attachment_urls( array $attachment_ids ): array {
    if ( empty( $attachment_ids ) ) return array();
    $attachment_ids = array_unique( array_filter( $attachment_ids ) );

    global $wpdb;
    $placeholders = implode( ',', array_fill( 0, count( $attachment_ids ), '%d' ) );

    // Query 1: Get attachment posts
    $rows = $wpdb->get_results( $wpdb->prepare(
        "SELECT ID, guid FROM {$wpdb->posts} WHERE ID IN ({$placeholders}) AND post_type = 'attachment'",
        $attachment_ids
    ), ARRAY_A );

    // Query 2: Get _wp_attached_file meta
    $meta_rows = $wpdb->get_results( $wpdb->prepare(
        "SELECT post_id, meta_value FROM {$wpdb->postmeta}
         WHERE post_id IN ({$placeholders}) AND meta_key = '_wp_attached_file'",
        $attachment_ids
    ), ARRAY_A );

    $upload_dir = wp_get_upload_dir();
    $baseurl    = $upload_dir['baseurl'];

    $file_map = [];
    foreach ( $meta_rows as $m ) {
        $file_map[ (int) $m['post_id'] ] = $m['meta_value'];
    }

    $url_map = [];
    foreach ( $rows as $row ) {
        $id   = (int) $row['ID'];
        $file = $file_map[ $id ] ?? '';
        $url_map[ $id ] = $file ? $baseurl . '/' . $file : $row['guid'];
    }
    return $url_map;
}
```

Collect tất cả image IDs trước khi format, rồi pass URL map vào format function.

## Impact

- Queries: ~60 (20 products) → 2 (batch)
- Time: ~200ms → ~5ms

## Detection

Tìm `wp_get_attachment_url` bên trong `array_map`, `foreach` loop trong REST controller.
