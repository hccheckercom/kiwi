**Mô tả:**
- Inline `<script>` trong `<head>` (trước CSS load) detect theme preference
- Prevent FOUC (Flash of Unstyled Content) khi user đã chọn dark mode
- Set `data-theme` attribute TRƯỚC khi browser render bất kỳ content nào

**Detect pattern (absence):**
- `header.php` thiếu inline script với `localStorage.getItem('theme')` hoặc `prefers-color-scheme`
- → Page flash trắng rồi mới chuyển dark, UX kém

**Code mẫu (inline in `<head>`):**
```html
<script>
(function() {
  var t = localStorage.getItem('theme');
  if (!t) t = matchMedia('(prefers-color-scheme:dark)').matches ? 'dark' : 'light';
  document.documentElement.setAttribute('data-theme', t);
})();
</script>
```

**Lợi ích:** User không bị flash trắng khi đã chọn dark mode — UX mượt ngay từ first paint
