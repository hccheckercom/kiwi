# LES-244: Theme Bỏ Qua Banner Layout Setting

## Pattern sai

```php
// template-parts/hero/default.php — chỉ lấy banner[0], không đọc layout
$hero_banners = wz_get_banners( 'home_hero' );
$hero         = ! empty( $hero_banners ) ? $hero_banners[0] : null;
// ... render 1 banner duy nhất
```

```php
// inc/banner-helpers.php — thiếu hoàn toàn wz_get_banner_layout()
// Không có function này → theme không biết admin đang chọn layout gì
```

## Pattern đúng

```php
// inc/banner-helpers.php — phải có helper đọc layout
if ( ! function_exists( 'wz_get_banner_layout' ) ) :
function wz_get_banner_layout(): string {
    return get_option( 'wz_banner_layout', 'classic' );
}
endif;
```

```php
// template-parts/hero/default.php — đọc layout rồi render đúng
$layout       = function_exists( 'wz_get_banner_layout' ) ? wz_get_banner_layout() : 'classic';
$hero_banners = function_exists( 'wz_get_banners' ) ? wz_get_banners( 'home_hero' ) : [];

if ( 'slider' === $layout ) {
    // render tất cả banners với dots navigation + JS auto-rotate
} elseif ( 'mini-left' === $layout ) {
    // render 2 mini banners trái + 1 hero lớn phải
} else {
    // classic: 1 hero + product cards
}
```

## Lý do

Plugin Banner Marketing lưu layout vào `wp_options` key `wz_banner_layout` với 3 giá trị:
- `classic` — 1 hero lớn + product cards
- `slider` — tất cả banners `home_hero` trượt xoay vòng tự động
- `mini-left` — 2 mini banners trái + 1 hero lớn phải

Nếu theme không đọc setting này, admin chọn layout nào trên dashboard cũng không có tác dụng — theme luôn render `$hero_banners[0]`.

## Checklist

- [ ] `inc/banner-helpers.php` có `wz_get_banner_layout()` đọc `get_option('wz_banner_layout', 'classic')`
- [ ] Hero template đọc layout trước khi render: `$layout = wz_get_banner_layout()`
- [ ] Layout `slider`: render tất cả banners, có dots, JS auto-rotate
- [ ] Layout `mini-left`: grid 2 cột (mini-col + main-col)
- [ ] Layout `classic`: fallback mặc định

## Grep check

```

```
