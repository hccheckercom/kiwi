## Bad
```css
@font-face {
    font-family: 'Material Symbols Outlined';
    font-display: block;
    src: url('fonts/MaterialSymbols.woff2') format('woff2');
}
```

## Good
```css
@font-face {
    font-family: 'Material Symbols Outlined';
    font-display: swap;
    src: url('fonts/MaterialSymbols.woff2') format('woff2');
}
```

## Why
`font-display: block` hides text completely until the font loads (Flash of Invisible Text — FOIT), degrading perceived performance and LCP. `font-display: swap` shows fallback text immediately, then swaps when the font is ready. This is required by performance.md for all @font-face declarations.
