## Problem

Registering both `template_redirect` and a custom action hook that both call increment on the same counter leads to double-counting views when both hooks fire for the same product page load.

## Bad

```php
add_action( 'template_redirect', array( $this, 'track_product_view' ) );
add_action( 'wezone/product/viewed', array( $this, 'track_product_view_event' ) );
// Both increment wz_analytics_views_{date} — double count!
```

## Good

```php
add_action( 'template_redirect', array( $this, 'track_product_view' ) );

private array $tracked_views = [];

public function track_product_view(): void {
    $product_id = get_queried_object_id();
    if ( isset( $this->tracked_views[ $product_id ] ) ) {
        return;
    }
    $this->tracked_views[ $product_id ] = true;
    $this->increment_counter( "wz_analytics_views_{$date}" );
}
```

## Why

Inflated view counts corrupt funnel conversion rates (views to cart to checkout). Analytics data becomes unreliable and misleading for business decisions.
