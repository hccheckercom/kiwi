## Bad
```tsx
<div className="flex md:flex-row gap-4">
  <aside>Filters</aside>
  <main>Products</main>
</div>
```
Có md:flex-row nhưng không có base flex-col — mobile items nằm ngang.

## Good
```tsx
<div className="flex flex-col md:flex-row gap-4">
  <aside>Filters</aside>
  <main>Products</main>
</div>
```

## Why
Mobile-first: base phải là flex-col (stack dọc), md:flex-row cho tablet+. Thiếu flex-col trên mobile khiến sidebar và content nằm ngang, bị squeeze.
