**Pattern sai:**
```php
// store-config.php sau clone vẫn giữ giá trị 0 hoặc placeholder
'shipping' => [
    'free_ship_threshold' => 0,   // ← zero = mọi đơn đều free ship
    'fees' => [
        'default' => 0,           // ← zero = phí ship luôn = 0
    ],
],
```
**Phải làm:**
```php
// Cập nhật giá trị thực tế cho shop mới ngay sau clone
'shipping' => [
    'free_ship_threshold' => 2000000,
    'fees' => [
        'default' => 50000,
    ],
],
```
**Lý do:** Template code dùng `wz_config('shipping.free_ship_threshold', 2000000)` — nếu config trả 0, logic `$subtotal >= 0` luôn true → free ship cho mọi đơn. UI hiển thị "Miễn phí vận chuyển đơn từ 0₫" — sai nghiệp vụ.
**Grep check:** `'free_ship_threshold'\s*=>\s*0|'default'\s*=>\s*0` trong store-config.php — zero value cho business field → HIGH
