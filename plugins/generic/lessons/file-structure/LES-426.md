## Vấn đề

Tailwind CSS CDN (https://cdn.tailwindcss.com) là **JavaScript runtime**, KHÔNG phải CSS file.
Khi load bằng `wp_enqueue_style()`, WordPress tạo `<link rel="stylesheet">` → trình duyệt tải
file nhưng **không thực thi JavaScript** → tất cả Tailwind classes không hoạt động → giao diện
chỉ hiển thị text thuần, không có styling.

## Pattern lỗi

```php
// SAI — load JS file bằng stylesheet tag
wp_enqueue_style( 'tailwind', 'https://cdn.tailwindcss.com', array(), '3.4.0' );
// Tạo: <link rel="stylesheet" href="https://cdn.tailwindcss.com">
// → Trình duyệt tải nhưng KHÔNG chạy JS → không có CSS
```

## Fix

```php
// ĐÚNG — load bằng script tag, đặt trong <head> (false = not in footer)
wp_enqueue_script( 'tailwind', 'https://cdn.tailwindcss.com', array(), '3.4.0', false );
// Tạo: <script src="https://cdn.tailwindcss.com"></script>
// → Trình duyệt tải VÀ chạy JS → CSS được generate
```

## Lưu ý thêm

- Tailwind CDN config phải load **TRƯỚC** script CDN (đặt inline script trong `<head>` trước `wp_head()`)
- Tailwind CDN chỉ nên dùng cho development/prototype — production nên compile CSS
- Arbitrary values như `bg-[#006e2f]` cần CDN JavaScript runtime để parse
