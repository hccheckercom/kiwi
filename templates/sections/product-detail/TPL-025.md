# Product Tabs — Description + Specs + Reviews

Tabbed interface for product information: description, specifications, and reviews.

## Features
- Tab navigation (description, specs, reviews)
- Active tab highlighting
- Responsive (horizontal tabs desktop, stacked mobile)
- Material Icons
- Empty state handling

## Code

```php
<?php
$description = $product['description'] ?? '';
$specs = $product['specifications'] ?? [];
$reviews_count = $product['reviews_count'] ?? 0;
?>

<div class="product-tabs">
    <div class="product-tabs__nav">
        <button class="product-tabs__tab active" data-tab="description">
            <span class="material-symbols-outlined">description</span>
            Mô tả
        </button>
        <button class="product-tabs__tab" data-tab="specs">
            <span class="material-symbols-outlined">list_alt</span>
            Thông số
        </button>
        <button class="product-tabs__tab" data-tab="reviews">
            <span class="material-symbols-outlined">star</span>
            Đánh giá (<?php echo esc_html( (string) $reviews_count ); ?>)
        </button>
    </div>

    <div class="product-tabs__content">
        <!-- Description tab -->
        <div class="product-tabs__panel active" data-panel="description">
            <?php if ( $description ) : ?>
                <div class="product-description">
                    <?php echo wp_kses_post( $description ); ?>
                </div>
            <?php else : ?>
                <p class="text-muted">Chưa có mô tả chi tiết.</p>
            <?php endif; ?>
        </div>

        <!-- Specs tab -->
        <div class="product-tabs__panel" data-panel="specs">
            <?php if ( ! empty( $specs ) ) : ?>
                <table class="product-specs">
                    <?php foreach ( $specs as $label => $value ) : ?>
                        <tr>
                            <th><?php echo esc_html( $label ); ?></th>
                            <td><?php echo esc_html( $value ); ?></td>
                        </tr>
                    <?php endforeach; ?>
                </table>
            <?php else : ?>
                <p class="text-muted">Chưa có thông số kỹ thuật.</p>
            <?php endif; ?>
        </div>

        <!-- Reviews tab -->
        <div class="product-tabs__panel" data-panel="reviews">
            <?php if ( $reviews_count > 0 ) : ?>
                <div id="product-reviews-list">
                    <!-- Reviews loaded via AJAX -->
                </div>
            <?php else : ?>
                <p class="text-muted">Chưa có đánh giá nào.</p>
            <?php endif; ?>
        </div>
    </div>
</div>
```

## Usage

```php
// In single-wz_product.php
get_template_part( 'template-parts/product/tabs', null, [
    'product' => $product,
] );
```

## JavaScript Required
- Tab click → switch active tab/panel
- Reviews tab → lazy load reviews via AJAX
