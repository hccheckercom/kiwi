## Bad
```php
public function boot(): void {
    if ( ! $this->should_show() ) {
        return;
    }
    add_action( 'wp_footer', [ $this, 'render' ] );
}
```

## Good
```php
public function boot(): void {
    if ( function_exists( 'wz_config' ) ) {
        wz_config( 'demo-bar' );
    }
    if ( ! $this->should_show() ) {
        return;
    }
    add_action( 'wp_footer', [ $this, 'render' ] );
}
```

## Why
Per CLAUDE.md, all plugin entry points must call wz_config() at the start of boot() to load config. Missing this call means plugin config is not loaded properly.
