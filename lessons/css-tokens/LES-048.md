## Bad
```php
<input type="email" name="email" class="w-full px-4 py-2">
```

## Good
```php
<label for="wz-email">Email</label>
<input type="email" id="wz-email" name="email" aria-required="true" class="w-full px-4 py-2">
```

## Why
Form input without associated label — WCAG 2.1 Level A violation. Screen readers can't identify the field.
