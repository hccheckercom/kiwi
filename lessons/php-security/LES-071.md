## Bad
```php
register_rest_route( 'wezone/v1', '/orders', [
    'methods' => 'POST',
    'permission_callback' => '__return_true',
]);
```

## Good
```php
register_rest_route( 'wezone/v1', '/orders', [
    'methods' => 'POST',
    'permission_callback' => function() {
        return current_user_can( 'edit_posts' );
    },
]);
```

## Why
Write endpoint with __return_true permission — anyone can create/modify data without authentication.
