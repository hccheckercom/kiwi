**Pattern sai:**
```php
// Signature thực tế: wz_get_categories( ?int $parent_id = null ): array
// Nhưng template gọi:
$categories = wz_get_categories( [ 'limit' => 4 ] );
// → PHP Fatal TypeError: Argument #1 must be of type ?int, array given
```
**Phải làm:**
```php
// Gọi không có args, rồi slice kết quả
$categories = function_exists( 'wz_get_categories' ) ? array_slice( wz_get_categories(), 0, 4 ) : [];
```
**Lý do:** `wz_get_categories()` chỉ nhận `?int $parent_id` — không phải array options như WooCommerce `get_terms()`
**Grep check:** `wz_get_categories\s*\(\s*\[` — truyền array = CRITICAL TypeError
