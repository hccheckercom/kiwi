## Bad
```php
$template = $_GET['page'];
include $template . '.php';
```

## Good
```php
$allowed = ['home', 'about', 'contact'];
$page = sanitize_file_name( $_GET['page'] ?? 'home' );
if ( in_array( $page, $allowed, true ) ) {
    include get_template_directory() . '/pages/' . $page . '.php';
}
```

## Why
include/require with variable path — Local File Inclusion (LFI) allows attacker to read arbitrary files.
