**Pattern sai:**
```html
<!-- via-secondary = màu chính đậm, quá rực -->
<div class="bg-gradient-to-r from-[var(--wz-primary)] via-[var(--wz-secondary)] to-[var(--wz-accent)]">
```
**Phải làm:**
```html
<!-- via-secondary-container = pastel nhạt, đúng theo demo -->
<div class="bg-gradient-to-r from-[var(--wz-primary)] via-[var(--wz-secondary-container)] to-[var(--wz-accent)]">
```
**Lý do:** `--wz-secondary` và `--wz-secondary-container` khác nhau rõ rệt về sắc độ. Dùng sai token khiến gradient quá đậm/nhạt so với mockup. Phải đối chiếu demo HTML để chọn đúng variant (-container, -cool, -pure).
**Grep check:** `var\(--wz-(primary|secondary|accent)\)` trong gradient class — cross-check demo HTML xem có dùng đúng variant không → HIGH
