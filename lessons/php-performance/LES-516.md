## Bad
```php
foreach ( $products as $product ) {
    $wpdb->insert( 'wz_products', [
        'name' => $product['name'],
        'price' => $product['price'],
    ] );
}
```

## Good
```php
$rows = array_map( fn( $p ) => [
    $p['name'],
    $p['price'],
    time(),
], $products );

wz_bulk_insert( 'wz_products', $rows, [ '%s', '%f', '%d' ] );
```

## Why
Inserting rows one-by-one in a loop causes N queries. Use wz_bulk_insert() for single-query batch insert (100x faster for large datasets).
