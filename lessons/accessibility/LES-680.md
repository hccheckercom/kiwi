## Bad
```tsx
export default function Layout({ children }) {
  return (
    <div>
      <div className="nav">...</div>
      <div>{children}</div>
      <div className="footer">...</div>
    </div>
  )
}
```

## Good
```tsx
export default function Layout({ children }) {
  return (
    <div>
      <nav>...</nav>
      <main>{children}</main>
      <footer>...</footer>
    </div>
  )
}
```

## Why
Screen readers dùng landmark roles (main, nav, header, footer) để skip navigation. Thiếu landmarks khiến user phải tab qua toàn bộ page content.
