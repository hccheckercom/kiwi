## Bad
```tsx
import { RichTextEditor } from '@/components/RichTextEditor'
// 150KB component loaded on every page, even if user never opens editor
```

## Good
```tsx
import dynamic from 'next/dynamic'
const RichTextEditor = dynamic(() => import('@/components/RichTextEditor'), {
  loading: () => <div className="h-48 animate-pulse bg-gray-100 rounded" />
})
```

## Why
Heavy components (editors, charts, maps, calendars) add hundreds of KB to the initial bundle. Using `next/dynamic` or `React.lazy` loads them only when rendered — reducing Time to Interactive (TTI) on initial page load.
