## Pattern

Nút yêu thích (wishlist heart icon) trong template single-product hoặc product-card tồn tại trong HTML nhưng **không hoạt động** vì thiếu contract với JavaScript:

1. Thiếu class `.wz-wishlist-btn` (JS listener dùng `e.target.closest('.wz-wishlist-btn')`)
2. Thiếu `data-product-id` attribute (JS cần để gửi AJAX)
3. Thiếu active state tracking (`aria-pressed`, class `wz-wishlist-active`)

## Bad

```php
<button class="p-3 border rounded" aria-label="Thêm vào yêu thích">
    <span class="material-symbols-outlined">favorite_border</span>
</button>
```

## Good

```php
<?php $in_wl = is_user_logged_in() && in_array( $product_id, (array) get_user_meta( get_current_user_id(), 'wz_wishlist', true ), false ); ?>
<button type="button"
        class="wz-wishlist-btn p-3 border rounded<?php echo $in_wl ? ' wz-wishlist-active' : ''; ?>"
        data-product-id="<?php echo esc_attr( (string) $product_id ); ?>"
        aria-label="<?php echo $in_wl ? 'Xóa khỏi yêu thích' : 'Thêm vào yêu thích'; ?>"
        aria-pressed="<?php echo $in_wl ? 'true' : 'false'; ?>">
    <span class="material-symbols-outlined" <?php echo $in_wl ? 'style="font-variation-settings:\'FILL\' 1"' : ''; ?>>
        <?php echo $in_wl ? 'favorite' : 'favorite_border'; ?>
    </span>
</button>
```

## Checklist

- [ ] Button has class `wz-wishlist-btn`
- [ ] Button has `data-product-id` with valid product ID
- [ ] AJAX handler `wz_toggle_wishlist` registered (either in theme `inc/wishlist.php` or via `wezone-wishlist` plugin)
- [ ] JS file `wishlist.js` enqueued with nonce localized
- [ ] CSS includes `.wz-wishlist-active` state (filled heart + red color)
- [ ] Initial state reflects user's current wishlist (server-side check)

## Root cause

Theme cloning hoặc code mới tạo button UI cho yêu thích nhưng quên contract với JS handler. Button chỉ là decoration — click không có phản hồi.

## Scan rule

```
file: template-parts/**/*.php, wezone-templates/**/*.php
pattern: favorite_border
absence: wz-wishlist-btn
message: "Nút yêu thích thiếu class wz-wishlist-btn và data-product-id"
```
