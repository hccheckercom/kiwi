## Bad
```php
// BreadcrumbList only on checkout and archive — product pages have NO breadcrumb schema
function wezone_theme_checkout_structured_data(): void {
    if ( strpos( $uri, '/checkout' ) === false && strpos( $uri, '/thank-you' ) === false ) return;
    // ...
}
function wezone_theme_archive_breadcrumb(): void {
    if ( ! is_archive() && ! is_search() ) return;
    // ...
}
// Product pages: NO BreadcrumbList JSON-LD!
```

## Good
```php
function wezone_theme_product_breadcrumb(): void {
    if ( ! function_exists( 'wz_is_product_page' ) || ! wz_is_product_page() ) return;

    $product = get_query_var( 'wz_tpl_product' );
    if ( ! is_array( $product ) ) return;

    $items = [
        ['@type' => 'ListItem', 'position' => 1, 'name' => 'Trang chủ', 'item' => home_url('/')],
    ];

    $cat = $product['category_name'] ?? '';
    $cat_url = $product['category_url'] ?? '';
    if ( $cat && $cat_url ) {
        $items[] = ['@type' => 'ListItem', 'position' => 2, 'name' => $cat, 'item' => $cat_url];
        $items[] = ['@type' => 'ListItem', 'position' => 3, 'name' => $product['name'] ?? ''];
    } else {
        $items[] = ['@type' => 'ListItem', 'position' => 2, 'name' => $product['name'] ?? ''];
    }

    $ld = ['@context' => 'https://schema.org', '@type' => 'BreadcrumbList', 'itemListElement' => $items];
    printf('<script type="application/ld+json">%s</script>', wp_json_encode($ld, JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES));
}
add_action( 'wp_head', 'wezone_theme_product_breadcrumb', 100 );
```

## Why
Blueprint `seo.md` requires BreadcrumbList on ALL pages except homepage. Product pages are the most important for SEO — missing breadcrumb schema means Google won't show the category trail in SERPs, reducing CTR and losing category context signals.
