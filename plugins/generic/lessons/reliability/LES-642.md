## Bad
```php
assert isinstance(user_id, int), "must be int"
```

## Good
```php
if not isinstance(user_id, int):
    raise TypeError("user_id must be int")
```

## Why
assert statements are removed when Python runs with -O flag. Use if/raise for validation
