## Bad
```php
$wpdb->query('DROP TABLE wp_orders');  // in a helper function
```

## Good
```php
// Destructive operations ONLY in migration files
// migrations/001_drop_legacy_orders.php
$wpdb->query('DROP TABLE IF EXISTS wp_orders');
```

## Why
DROP/TRUNCATE outside migration file — accidental data loss. Destructive DB ops must be in versioned migrations.
