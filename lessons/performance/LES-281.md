## Bad
```php
<img src="<?php echo esc_url( $image ); ?>" alt="<?php echo esc_attr( $name ); ?>"
    style="width:100%;height:100%;object-fit:cover;" loading="lazy" decoding="async" />
```

## Good
```php
<img src="<?php echo esc_url( $image ); ?>" alt="<?php echo esc_attr( $name ); ?>"
    width="400" height="400" style="width:100%;aspect-ratio:1/1;object-fit:cover;" loading="lazy" decoding="async" />
```

## Why
Images without explicit width/height attributes cause Cumulative Layout Shift (CLS) because the browser cannot reserve space before the image loads. This directly impacts Core Web Vitals scores. All `<img>` tags must have width+height or CSS aspect-ratio to prevent layout shifts.
