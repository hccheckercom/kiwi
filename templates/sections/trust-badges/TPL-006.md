## Preview

Bar ngang full-width, nền trắng, border trên + dưới. 4 items ngang hàng (2 cột mobile, 4 desktop). Mỗi item: icon Material Symbols bên trái + title bold + description nhỏ bên phải.

## Files

| File | Role |
|------|------|
| template-parts/home/trust-badges.php | Template chính |

## Code — PHP

```php
<?php
$badges = array(
    array( 'icon' => 'forest',         'title' => __( 'Gỗ tự nhiên chuẩn FSC', 'wezone-haven' ),  'desc' => __( 'Bền vững & Thân thiện với môi trường', 'wezone-haven' ) ),
    array( 'icon' => 'verified_user',  'title' => __( 'Bảo hành 36 tháng', 'wezone-haven' ),       'desc' => __( 'Cam kết chất lượng dài lâu', 'wezone-haven' ) ),
    array( 'icon' => 'local_shipping', 'title' => __( 'Giao lắp tận nhà', 'wezone-haven' ),        'desc' => __( 'Nhanh chóng & Tận tâm', 'wezone-haven' ) ),
    array( 'icon' => 'autorenew',      'title' => __( 'Đổi trả 30 ngày', 'wezone-haven' ),         'desc' => __( 'Miễn phí & Dễ dàng', 'wezone-haven' ) ),
);
?>

<section style="width:100%;background:#fff;border-top:1px solid var(--wz-border,#E8E0D5);border-bottom:1px solid var(--wz-border,#E8E0D5);padding:28px 24px;">
    <div style="max-width:1280px;margin:0 auto;display:grid;grid-template-columns:repeat(2,1fr);gap:20px;" class="haven-trust-grid">
        <?php foreach ( $badges as $badge ) : ?>
            <div style="display:flex;align-items:center;gap:14px;">
                <span class="material-symbols-outlined" aria-hidden="true" style="font-size:28px;color:var(--wz-primary,#8B7355);flex-shrink:0;"><?php echo esc_html( $badge['icon'] ); ?></span>
                <div>
                    <div style="font-size:13px;font-weight:600;color:var(--wz-text,#3D2B1F);margin-bottom:2px;"><?php echo esc_html( $badge['title'] ); ?></div>
                    <div style="font-size:12px;color:var(--wz-text-muted,#8B7355);line-height:1.4;"><?php echo esc_html( $badge['desc'] ); ?></div>
                </div>
            </div>
        <?php endforeach; ?>
    </div>
</section>

<style>
@media (min-width: 768px) { .haven-trust-grid { grid-template-columns: repeat(4, 1fr) !important; } }
</style>
```

## Usage Notes

- Badges array cần customize theo ngành hàng
- Icons: dùng Material Symbols Outlined (cần enqueue font)
- Thường đặt giữa hero và products, hoặc trước footer
- Không cần JS
