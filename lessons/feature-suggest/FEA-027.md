**Mô tả:**
Blueprint yêu cầu xử lý 6 loại error state. Theme hiện có 404 page và order-failed page, nhưng thiếu:
- Network offline detection + cached data display
- 500 server error page
- Timeout handling với retry button

**Error states cần thiết:**
| Error | Message | Action |
|-------|---------|--------|
| Network offline | "Mất kết nối mạng" | Show cached + retry |
| 500 Server Error | "Đã xảy ra lỗi, vui lòng thử lại" | Retry + contact |
| Timeout | "Tải quá lâu, vui lòng thử lại" | Retry |

**Implementation gợi ý (JS):**
```js
window.addEventListener('offline', () => {
    document.body.classList.add('is-offline');
    // Show offline banner
});
window.addEventListener('online', () => {
    document.body.classList.remove('is-offline');
    // Hide banner, retry failed requests
});
```
