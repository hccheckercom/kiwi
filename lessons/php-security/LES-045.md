## Bad
```php
$name = $_GET['name'];
$wpdb->query("SELECT * FROM users WHERE name = '$name'");
```

## Good
```php
$name = sanitize_text_field( wp_unslash( $_GET['name'] ?? '' ) );
```

## Why
Raw superglobal access without sanitization — SQL injection, XSS, and other injection risks.
