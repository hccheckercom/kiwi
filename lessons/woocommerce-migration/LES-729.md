## Pattern

Developer dùng WooCommerce product/order query functions. Trên Wezone Commer phải dùng `wz_*`.

## Bad

```php
// Product
$product = wc_get_product( $id );
$price   = $product->get_price();
$name    = $product->get_name();
$sku     = $product->get_sku();
$stock   = $product->get_stock_quantity();
$image   = $product->get_image();

// Product query
$products = wc_get_products( array(
    'status'   => 'publish',
    'limit'    => 20,
    'category' => array( 'shoes' ),
    'orderby'  => 'date',
) );

// Order
$order  = wc_get_order( $order_id );
$status = $order->get_status();
$total  = $order->get_total();
$items  = $order->get_items();

// Order query
$orders = wc_get_orders( array(
    'customer_id' => $user_id,
    'status'      => 'completed',
    'limit'       => 10,
) );

// Lookup
$id = wc_get_product_id_by_sku( 'SKU-001' );
```

## Good

```php
// Product
$product = wz_get_product( $id );
$price   = $product['price'];
$name    = $product['name'];
$sku     = $product['sku'];
$stock   = $product['stock_qty'];
$image   = $product['image'];

// Product query
$products = wz_get_products( array(
    'status'      => 'publish',
    'limit'       => 20,
    'category_id' => $cat_id,
    'orderby'     => 'date',
) );

// Order
$order  = wz_get_order( $order_id );
$status = $order['status'];
$total  = $order['total'];
$items  = $order['items'];

// Order query
$orders = wz_get_orders( array(
    'customer_id' => $user_id,
    'status'      => 'completed',
    'limit'       => 10,
) );

// Lookup
$id = wz_get_product_by_sku( 'SKU-001' );
```

## Why

- WooCommerce trả về object (`WC_Product`, `WC_Order`) với getter methods
- Wezone Commer trả về **array** với key access: `$product['key']` không phải `$product->get_key()`
- `wc_get_products()` dùng WC_Product_Query — không tồn tại trên Wezone
- Category query: WooCommerce dùng taxonomy slug, Wezone dùng `category_id` (integer)
