## Bad
```php
<section class="bg-white py-8">
  <div class="container mx-auto">
    <?php get_template_part('template-parts/home/flash-sale'); ?>
  </div>
</section>
```

## Good
```php
<section class="bg-white dark:bg-gray-900 py-8">
  <div class="container mx-auto">
    <?php get_template_part('template-parts/home/flash-sale'); ?>
  </div>
</section>
```

## Why
WP theme sections dùng bg-white phải có dark variant. Nếu theme hỗ trợ dark mode, mọi surface color phải paired.
