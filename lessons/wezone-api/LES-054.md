**Pattern sai:**
```php
// wezone-templates/account/order-detail.php
$order_id = (int) get_query_var( 'wz_tpl_order_id', 0 );
// → luôn = 0 vì Router bind vào 'wz_order_id', không phải 'wz_tpl_order_id'
// → $order = null → redirect về /my-account/orders/
```
**Phải làm:**
```php
$order_id = (int) get_query_var( 'wz_order_id', 0 );
```
**Lý do:** Router.php đăng ký rewrite rule `^my-account/orders/([0-9]+)/?$` → `wz_order_id=$matches[1]` và add_query_vars() đăng ký `wz_order_id`. TemplateLoader KHÔNG dùng prefix `wz_tpl_` cho order_id — chỉ dùng `wz_order_id` trực tiếp. Nhầm lẫn với pattern `wz_tpl_*` của TemplateLoader::set() (LES-040) gây ra bug này.
**Grep check:** `get_query_var.*wz_tpl_order_id` trong wezone-templates/ — phải đổi thành `wz_order_id` → HIGH
