## Bad
```php
<img src="product.jpg" loading="lazy" class="w-full">
```

## Good
```php
<img src="product.jpg" loading="lazy" width="400" height="400" class="w-full h-auto">
```

## Why
Lazy-loaded images without explicit width/height cause CLS (Cumulative Layout Shift). Browser can't reserve space until image loads.
