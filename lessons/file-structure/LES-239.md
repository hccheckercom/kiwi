## Bad
```bash
# Post-deploy script tạo subsite không guard → exit 1 nếu đã tồn tại
wp site create --slug=trunganh --title="Trung Anh" --email="admin@example.com"
```

## Good
```bash
# Guard bằng wp site list trước, hoặc dùng || true
if ! wp site list --field=path | grep -q "^/trunganh/"; then
    wp site create --slug=trunganh --title="Trung Anh" --email="admin@example.com"
else
    echo "  SKIP: /trunganh/ already exists"
fi
```

## Why
`wp site create` trả về exit code 1 nếu subsite đã tồn tại, làm toàn bộ post-deploy script fail dù atomic swap và nginx reload đã thành công. Deploy báo failed nhưng thực ra live code đã được swap đúng. Gây nhầm lẫn khi đọc deploy log và có thể trigger rollback không cần thiết nếu script có auto-rollback.

## Grep check
Tìm `wp site create` trong `scripts/` → nếu không có `|| true` hoặc `wp site list` guard trước đó → HIGH
