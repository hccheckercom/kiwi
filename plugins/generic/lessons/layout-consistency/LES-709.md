## Bad
```tsx
// app/shop/danh-muc/page.tsx — grid-cols-2
<div className="grid grid-cols-2 gap-4">

// app/shop/tim-kiem/page.tsx — grid-cols-3
<div className="grid grid-cols-3 gap-4">
```
Cùng hiển thị product listing nhưng số cột khác nhau.

## Good
```tsx
// Cả hai dùng cùng grid structure
<div className="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
```

## Why
Product listing phải hiển thị nhất quán số cột trên cùng breakpoint. User navigate từ category → search thấy layout thay đổi gây confusing.
