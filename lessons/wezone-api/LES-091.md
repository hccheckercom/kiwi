## Bad
```php
public function boot(): void {
    $this->register_hooks();
}
```

## Good
```php
public function boot(): void {
    if ( function_exists('wz_config') ) {
        wz_config('plugin-name');
    }
    $this->register_hooks();
}
```

## Why
Plugin boot() without wz_config() — config never loads, all settings return null. Silent misconfiguration.
