# LES-118: Missing body margin reset causes fullwidth gap

## Symptom
Header, footer, announcement bar hoặc bất kỳ element fullwidth nào có khoảng trắng ~8px 2 bên mép viewport. Nền màu không tràn edge-to-edge.

## Root Cause
Browser default stylesheet: `body { margin: 8px }`. Khi Tailwind compile thiếu preflight (hoặc preflight bị disable/strip), body giữ margin mặc định → mọi element con đều bị thu hẹp.

## Detection
```css
/* Kiểm tra compiled CSS có chứa: */
body{margin:0}
/* hoặc */
body{margin:0;padding:0}
```

Nếu KHÔNG có → bug.

## Fix
Thêm vào `src/main.css` (trước hoặc sau @tailwind directives):
```css
body {
  margin: 0;
}
```

Và thêm `body{margin:0}` vào đầu file compiled `assets/css/tailwind.css`.

## Prevention
- Khi tạo theme mới, luôn verify preflight có trong compiled output
- Nếu dùng `corePlugins: { preflight: false }` trong tailwind.config → phải tự thêm body reset
- Checklist trước bàn giao: mở DevTools → inspect body → confirm margin: 0
