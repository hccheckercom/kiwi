## Bad

```python
def process_data(raw):
    import json
    import re
    return json.loads(re.sub(r'\s+', ' ', raw))
```

## Good

```python
import json
import re

def process_data(raw):
    return json.loads(re.sub(r'\s+', ' ', raw))
```

## Why

Top-level imports are loaded once at module init and cached. Nested imports add per-call overhead and hide dependencies. Exceptions: lazy imports to avoid circular dependencies (mark with `# lazy` or `# avoid circular`), and optional heavy dependencies (`# optional`).

## Grep

```bash
rg '^\s{4,}import\s+\w' --type py -g '!test_*.py'
```
