## Bad
```php
os.chmod(filepath, 0o777)
```

## Good
```php
os.chmod(filepath, 0o644)  # owner rw, group/other r
```

## Why
World-writable files can be modified by any user on the system

## Grep
`chmod.*0o?777` — in `**/*.py`
