## Bad
```tsx
// FilterSidebar.tsx — luôn hiện
<aside className="w-64 border-r p-4">
  <h3>Bộ lọc</h3>
</aside>
```

## Good
```tsx
<aside className="hidden lg:block w-64 border-r p-4">
  <h3>Bộ lọc</h3>
</aside>
```

## Why
Sidebar filter trên mobile chiếm quá nhiều không gian. Nên hidden trên mobile, dùng drawer/bottom sheet thay thế. Desktop mới hiện inline sidebar.
