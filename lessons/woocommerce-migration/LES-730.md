## Pattern

Developer tạo instance hoặc extend WooCommerce classes. Wezone Commer không có các class này.

## Bad

```php
// Type hints
function process( WC_Product $product ): void { }
function handle( WC_Order $order ): void { }

// instanceof checks
if ( $product instanceof WC_Product_Variable ) {
    $variations = $product->get_available_variations();
}

// Extending WC classes
class My_Gateway extends WC_Payment_Gateway {
    public function process_payment( $order_id ) { }
}

class My_Shipping extends WC_Shipping_Method {
    public function calculate_shipping( $package = array() ) { }
}

class My_Email extends WC_Email {
    public function get_content_html() { }
}

// Direct instantiation
$product  = new WC_Product_Simple( $id );
$customer = new WC_Customer( $user_id );
$coupon   = new WC_Coupon( 'SAVE10' );
```

## Good

```php
// Type hints — dùng array
function process( array $product ): void { }
function handle( array $order ): void { }

// Type checks — dùng product_type field
if ( 'variable' === $product['product_type'] ) {
    $variations = wz_get_variations( $product['id'] );
}

// Payment gateway — implement interface Wezone
class My_Gateway implements Wz_Payment_Gateway_Interface {
    public function process_payment( int $order_id ): array { }
}

// Shipping — implement interface Wezone
class My_Shipping implements Wz_Shipping_Method_Interface {
    public function calculate( array $package ): array { }
}

// Email — dùng wz_send_email()
wz_send_email( 'order_confirmed', $order_id, array(
    'template' => 'emails/order-confirmed.php',
) );

// Lookup — dùng wz_* functions
$product  = wz_get_product( $id );
$customer = wz_get_customer( $user_id );
$coupon   = wz_get_coupon( 'SAVE10' );
```

## Why

Wezone Commer kiến trúc khác WooCommerce:
- **Products/Orders**: array-based, không phải OOP objects
- **Payment gateways**: implement `Wz_Payment_Gateway_Interface` thay vì extend `WC_Payment_Gateway`
- **Shipping**: implement `Wz_Shipping_Method_Interface`
- **Emails**: dùng `wz_send_email()` template system
- **Coupons**: `wz_get_coupon()` trả array, không phải WC_Coupon object

Danh sách WC classes phổ biến cần thay:
| WC Class | Wezone Equivalent |
|---|---|
| `WC_Product` | `array` via `wz_get_product()` |
| `WC_Product_Simple` | `$product['product_type'] === 'simple'` |
| `WC_Product_Variable` | `$product['product_type'] === 'variable'` |
| `WC_Product_Variation` | `wz_get_variation( $id )` |
| `WC_Order` | `array` via `wz_get_order()` |
| `WC_Cart` | `wz_cart_*()` functions |
| `WC_Customer` | `array` via `wz_get_customer()` |
| `WC_Coupon` | `array` via `wz_get_coupon()` |
| `WC_Payment_Gateway` | `Wz_Payment_Gateway_Interface` |
| `WC_Shipping_Method` | `Wz_Shipping_Method_Interface` |
| `WC_Email` | `wz_send_email()` |
| `WC_Tax` | `wz_calculate_tax()` |
| `WC_Data_Store` | Direct DB via `$wpdb` + helpers |
