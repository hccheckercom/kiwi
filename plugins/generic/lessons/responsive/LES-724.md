## Bad
```tsx
<section className="py-4 lg:py-16">
```
Nhảy từ py-4 (16px) lên py-16 (64px) — 4x increase, quá lớn.

## Good
```tsx
<section className="py-4 md:py-8 lg:py-16">
```

## Why
Spacing jumps nên smooth: double mỗi breakpoint (4→8→16). Jump quá lớn (4→16) gây visual jarring trên tablet.
