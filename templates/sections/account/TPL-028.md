# Account Profile Edit — Form with Validation

Profile edit page with avatar upload, form fields, and validation.

## Features
- Avatar upload with preview
- Form fields (name, email, phone, birthday)
- Client-side validation
- Success/error messages
- Save button with loading state

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

if ( ! is_user_logged_in() ) {
    wp_safe_redirect( home_url( '/my-account/login/' ) );
    exit;
}

get_header();

$user = wp_get_current_user();
$user_meta = get_user_meta( $user->ID );
$avatar_url = get_avatar_url( $user->ID, [ 'size' => 120 ] );

set_query_var( 'wz_tpl_account_page', 'profile' );
?>

<div class="bg-[var(--wz-surface)] min-h-screen">
<div class="max-w-[1280px] mx-auto px-5 md:px-[20px] py-4">

    <div class="flex flex-col lg:flex-row lg:gap-6">
        <?php include get_stylesheet_directory() . '/template-parts/account/sidebar.php'; ?>

        <div class="flex-1 min-w-0 mt-4 lg:mt-0">
            <div class="bg-[var(--wz-surface-pure)] rounded-[var(--wz-radius-card)] p-5 md:p-6 shadow-[0_4px_24px_rgba(27,58,122,0.08)]">
                <h1 class="text-xl font-bold text-[var(--wz-primary)] mb-6">Thông tin cá nhân</h1>

                <form id="profile-form" class="space-y-5">
                    <?php wp_nonce_field( 'wz_update_profile', 'nonce' ); ?>

                    <!-- Avatar -->
                    <div class="flex items-center gap-4">
                        <img id="avatar-preview" src="<?php echo esc_url( $avatar_url ); ?>" alt="Avatar" class="w-20 h-20 rounded-full object-cover">
                        <div>
                            <label for="avatar-upload" class="inline-block bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] px-4 py-2 rounded-[var(--wz-radius)] text-sm font-semibold cursor-pointer hover:bg-[var(--wz-primary)]/10 transition">
                                Đổi ảnh đại diện
                            </label>
                            <input type="file" id="avatar-upload" name="avatar" accept="image/*" class="hidden">
                            <p class="text-xs text-[var(--wz-outline)] mt-1">JPG, PNG tối đa 2MB</p>
                        </div>
                    </div>

                    <!-- Name -->
                    <div>
                        <label for="display_name" class="block text-sm font-semibold text-[var(--wz-on-surface)] mb-2">
                            Họ và tên <span class="text-[var(--wz-error)]">*</span>
                        </label>
                        <input type="text" id="display_name" name="display_name" required
                               value="<?php echo esc_attr( $user->display_name ); ?>"
                               class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-4 py-2.5 text-sm focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                    </div>

                    <!-- Email -->
                    <div>
                        <label for="user_email" class="block text-sm font-semibold text-[var(--wz-on-surface)] mb-2">
                            Email <span class="text-[var(--wz-error)]">*</span>
                        </label>
                        <input type="email" id="user_email" name="user_email" required
                               value="<?php echo esc_attr( $user->user_email ); ?>"
                               class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-4 py-2.5 text-sm focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                    </div>

                    <!-- Phone -->
                    <div>
                        <label for="phone" class="block text-sm font-semibold text-[var(--wz-on-surface)] mb-2">
                            Số điện thoại
                        </label>
                        <input type="tel" id="phone" name="phone"
                               value="<?php echo esc_attr( $user_meta['phone'][0] ?? '' ); ?>"
                               class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-4 py-2.5 text-sm focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                    </div>

                    <!-- Birthday -->
                    <div>
                        <label for="birthday" class="block text-sm font-semibold text-[var(--wz-on-surface)] mb-2">
                            Ngày sinh
                        </label>
                        <input type="date" id="birthday" name="birthday"
                               value="<?php echo esc_attr( $user_meta['birthday'][0] ?? '' ); ?>"
                               class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-4 py-2.5 text-sm focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                    </div>

                    <!-- Messages -->
                    <div id="profile-message" class="hidden p-4 rounded-[var(--wz-radius)] text-sm"></div>

                    <!-- Submit -->
                    <button type="submit" id="profile-submit"
                            class="w-full md:w-auto bg-[var(--wz-primary)] text-white px-8 py-3 rounded-[var(--wz-radius)] font-semibold hover:opacity-90 transition disabled:opacity-50">
                        Lưu thay đổi
                    </button>
                </form>
            </div>
        </div>
    </div>

</div>
</div>

<?php get_footer(); ?>
```

## Usage

Create as `wezone-templates/account/profile.php`

## JavaScript Required
- Avatar upload preview
- Form validation
- AJAX submit
- Success/error message display
