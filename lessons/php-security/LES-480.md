## Bad
```php
$host = sanitize_text_field( wp_unslash( $_SERVER['HTTP_HOST'] ?? '' ) );
if ( str_contains( $host, 'wp.wezone.vn' ) ) {
    return true; // BYPASS: Host: evil.com:wp.wezone.vn
}
```

## Good
```php
$host = sanitize_text_field( wp_unslash( $_SERVER['HTTP_HOST'] ?? '' ) );
$parsed = parse_url( 'http://' . $host );
$hostname = $parsed['host'] ?? '';
if ( $hostname === 'wp.wezone.vn' || str_ends_with( $hostname, '.wp.wezone.vn' ) ) {
    return true;
}
```
