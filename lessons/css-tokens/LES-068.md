## Bad
```html
<!-- tailwind.config.js có preflight: false -->
<!-- Input w-full + px-4 + border tràn ra ngoài container -->
<input class="w-full px-4 py-2.5 border ...">
```

## Good
```css
/* src/main.css */
input, select, textarea, button { box-sizing: border-box; }
```

## Why
Khi `preflight: false`, Tailwind không inject `box-sizing: border-box`. Form elements mặc định `content-box`, nên `w-full` + padding + border > 100% → input tràn.

## Grep
`preflight: false` + `w-full px-` trên input + thiếu `box-sizing: border-box` trong CSS
