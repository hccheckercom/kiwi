## Bad
```php
document.body.classList.add('overflow-hidden');
// Modal opens... but never removes the class on close
```

## Good
```php
document.body.classList.add('overflow-hidden');
modal.addEventListener('close', () => {
  document.body.classList.remove('overflow-hidden');
});
```

## Why
Adding overflow-hidden without cleanup — user can never scroll again after closing modal. Permanent scroll lock.
