## Bad
```php
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

## Good
```php
if __name__ == "__main__":
    import uvicorn
    import socket
    
    port = 8000
    try:
        # Check if port is already in use
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.bind(('0.0.0.0', port))
        sock.close()
    except OSError:
        print(f"Port {port} already in use. Kill existing process or use different port.")
        sys.exit(1)
    
    uvicorn.run(app, host="0.0.0.0", port=port)
```

## Why
When uvicorn.run() is called on a port already in use, it crashes with "errno 10048: only one usage of each socket address is normally permitted". This happens when previous process wasn't killed properly or multiple instances are started. Solution: check port availability before starting server, or use uvicorn CLI with --reload which handles this better.
