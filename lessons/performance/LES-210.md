## Bad
```ts
import _ from 'lodash'
const sorted = _.sortBy(products, 'price')
// Imports entire lodash (71KB gzipped) for one function
```

## Good
```ts
import sortBy from 'lodash/sortBy'
const sorted = sortBy(products, 'price')
// Only imports sortBy (2KB gzipped)
```
Or use native:
```ts
const sorted = [...products].sort((a, b) => a.price - b.price)
```

## Why
Barrel imports pull entire libraries into the client bundle. On slow 3G connections, every KB matters — 70KB extra = 1+ second added to page load. Use path imports or native alternatives for single-function usage.
