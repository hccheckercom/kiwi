## Pattern

Developer dùng WooCommerce action hooks. Trên Wezone Commer, tất cả hooks dùng prefix `wezone_`.

## Bad

```php
// Shop loop
add_action( 'woocommerce_before_shop_loop', 'my_before_loop' );
add_action( 'woocommerce_after_shop_loop_item', 'my_after_item' );
add_action( 'woocommerce_shop_loop_item_title', 'my_title' );

// Single product
add_action( 'woocommerce_single_product_summary', 'my_summary', 5 );
add_action( 'woocommerce_before_add_to_cart_form', 'my_before_cart' );
add_action( 'woocommerce_after_add_to_cart_button', 'my_after_button' );
add_action( 'woocommerce_product_meta_start', 'my_meta' );

// Cart
add_action( 'woocommerce_before_cart', 'my_before_cart' );
add_action( 'woocommerce_cart_contents', 'my_cart_contents' );
add_action( 'woocommerce_cart_collaterals', 'my_cross_sells' );

// Checkout
add_action( 'woocommerce_before_checkout_form', 'my_before_checkout' );
add_action( 'woocommerce_checkout_process', 'my_validate_checkout' );
add_action( 'woocommerce_checkout_order_processed', 'my_after_order' );
add_action( 'woocommerce_review_order_before_payment', 'my_before_pay' );

// Order lifecycle
add_action( 'woocommerce_payment_complete', 'my_payment_done' );
add_action( 'woocommerce_order_status_changed', 'my_status_change', 10, 4 );
add_action( 'woocommerce_thankyou', 'my_thankyou' );
add_action( 'woocommerce_new_order', 'my_new_order' );

// Account
add_action( 'woocommerce_before_my_account', 'my_account_header' );
add_action( 'woocommerce_account_navigation', 'my_nav' );
add_action( 'woocommerce_register_form', 'my_register_fields' );

// Admin
add_action( 'woocommerce_process_product_meta', 'my_save_meta' );
add_action( 'woocommerce_admin_order_data_after_billing_address', 'my_admin_field' );
```

## Good

```php
// Shop loop
add_action( 'wezone_before_product_loop', 'my_before_loop' );
add_action( 'wezone_after_product_loop_item', 'my_after_item' );
add_action( 'wezone_product_loop_item_title', 'my_title' );

// Single product
add_action( 'wezone_single_product_summary', 'my_summary', 5 );
add_action( 'wezone_before_add_to_cart_form', 'my_before_cart' );
add_action( 'wezone_after_add_to_cart_button', 'my_after_button' );
add_action( 'wezone_product_meta_start', 'my_meta' );

// Cart
add_action( 'wezone_before_cart', 'my_before_cart' );
add_action( 'wezone_cart_contents', 'my_cart_contents' );
add_action( 'wezone_cart_collaterals', 'my_cross_sells' );

// Checkout
add_action( 'wezone_before_checkout_form', 'my_before_checkout' );
add_action( 'wezone_checkout_process', 'my_validate_checkout' );
add_action( 'wezone_checkout_order_processed', 'my_after_order' );
add_action( 'wezone_review_order_before_payment', 'my_before_pay' );

// Order lifecycle
add_action( 'wezone_payment_complete', 'my_payment_done' );
add_action( 'wezone_order_status_changed', 'my_status_change', 10, 4 );
add_action( 'wezone_thankyou', 'my_thankyou' );
add_action( 'wezone_new_order', 'my_new_order' );

// Account
add_action( 'wezone_before_my_account', 'my_account_header' );
add_action( 'wezone_account_navigation', 'my_nav' );

// Admin
add_action( 'wezone_process_product_meta', 'my_save_meta' );
add_action( 'wezone_admin_order_data_after_billing', 'my_admin_field' );
```

## Why

WooCommerce hooks (`woocommerce_*`) không bao giờ được fire trên Wezone Commer.
Callbacks đăng ký vào hooks này sẽ **không bao giờ chạy** — silent failure, không có error log.

Mapping phổ biến nhất:
| WooCommerce Hook | Wezone Hook |
|---|---|
| `woocommerce_before_shop_loop` | `wezone_before_product_loop` |
| `woocommerce_after_shop_loop_item` | `wezone_after_product_loop_item` |
| `woocommerce_single_product_summary` | `wezone_single_product_summary` |
| `woocommerce_before_add_to_cart_form` | `wezone_before_add_to_cart_form` |
| `woocommerce_before_cart` | `wezone_before_cart` |
| `woocommerce_checkout_process` | `wezone_checkout_process` |
| `woocommerce_payment_complete` | `wezone_payment_complete` |
| `woocommerce_order_status_changed` | `wezone_order_status_changed` |
| `woocommerce_thankyou` | `wezone_thankyou` |
| `woocommerce_new_order` | `wezone_new_order` |
