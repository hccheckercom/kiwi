**Pattern sai:**
```
// Wezone Router: ^category/([^/]+)/?$ → wz_page=category&wz_slug=$matches[1]
// Router gọi: TemplateLoader::resolve('shop/category')
// TemplateLoader tìm: {theme}/wezone-templates/shop/category.php
// File không tồn tại → resolve() trả get_404_template() → mọi URL /category/* đều 404
```
**Phải làm:**
```php
// Tạo: wezone-templates/shop/category.php
// Lấy category data qua: get_query_var('wz_tpl_category')
// (TemplateLoader::bind() set query var với prefix wz_tpl_)
$category = get_query_var( 'wz_tpl_category' ) ?: null;
```
**Lý do:** Cùng pattern với LES-023 — Wezone Router resolve tất cả category URLs qua `shop/category` template. Thiếu file này → toàn bộ trang danh mục 404.
