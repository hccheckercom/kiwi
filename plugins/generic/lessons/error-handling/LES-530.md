## Bad
```php
except Exception:\n    pass
```

## Good
```php
except Exception as e:\n    logger.warning(f'Failed: {e}')\n    return None
```

## Why
Silent failures hide bugs and make debugging impossible

## Grep
`except Exception:\s*(pass|return)` — in `**/*.php`
