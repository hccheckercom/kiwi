## Preview

3 product cards nhỏ floating ở phần dưới hero banner. Mỗi card gồm: ảnh sản phẩm (hoặc placeholder) + tên + giá. Background frosted glass, border trắng mờ. Mobile: cards nhỏ (105px) căn giữa. Tablet+: cards lớn hơn (150-165px) dồn sang phải.

## Data Flow

```
Admin Dashboard (Marketing → Banner)
  ↓ saves meta JSON: { "featured_products": [...] }
  ↓
Theme: $hero['meta']['featured_products']
  ↓ fallback nếu admin chưa set:
  wz_get_products(['limit' => 3, 'orderby' => 'created_at', 'order' => 'DESC'])
  ↓
Render cards
```

## Files

| File | Role |

## Code — PHP (Data preparation)

```php
<?php
// Featured products: from banner meta OR fallback query
$hero_products = [];
$first_meta    = $hero_banners[0]['meta'] ?? null;

if ( ! empty( $first_meta['featured_products'] ) ) {
    // Admin đã set sản phẩm cụ thể trong banner meta
    $hero_products = $first_meta['featured_products'];
} elseif ( function_exists( 'wz_get_products' ) ) {
    // Fallback: 3 sản phẩm mới nhất
    $products = wz_get_products( [ 'limit' => 3, 'orderby' => 'created_at', 'order' => 'DESC' ] );
    foreach ( $products as $p ) {
        $hero_products[] = [
            'name'  => $p['name'],
            'price' => number_format( (float) $p['price'], 0, ',', '.' ) . '₫',
            'image' => ! empty( $p['image_id'] ) ? wp_get_attachment_image_url( (int) $p['image_id'], 'medium' ) : '',
            'link'  => home_url( '/product/' . $p['slug'] . '/' ),
        ];
    }
}
?>
```

## Integration Notes

- Cards nằm TRONG `.haven-hero__visual` (position: absolute, bottom)
- Hero cần đủ chiều cao cho cards không che CTA: mobile dùng `padding-bottom: 180px` trên content
- Tablet/desktop: cards dồn sang phải, không cần padding-bottom thêm
- Nếu không có sản phẩm nào (cả meta lẫn DB) → div `.haven-hero__products` không render
- Placeholder icon dùng Material Symbols `image` — luôn có fallback visual
