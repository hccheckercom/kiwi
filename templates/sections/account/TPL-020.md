# Account Dashboard — Welcome + Stats + Recent Orders

Account dashboard page with welcome banner, stats cards, recent orders, and quick action links.

## Features
- Welcome banner with user info
- Stats cards (orders, points, vouchers)
- Recent orders list
- Quick action links
- Support contact card
- Breadcrumb navigation
- Responsive layout

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

if ( ! is_user_logged_in() ) {
    wp_safe_redirect( home_url( '/my-account/login/' ) );
    exit;
}

get_header();

$current_user = wp_get_current_user();
$user_id      = $current_user->ID;
$shop_name    = wz_config( 'shop_name', get_bloginfo( 'name' ) );

set_query_var( 'wz_tpl_account_page', 'dashboard' );
?>

<div class="bg-[var(--wz-surface)] min-h-screen">
<div class="max-w-[1280px] mx-auto px-5 md:px-[20px] py-4">

    <!-- Breadcrumb -->
    <nav aria-label="Breadcrumb" class="pb-4">
        <ol class="flex items-center gap-1 text-xs md:text-sm text-[var(--wz-outline)]">
            <li><a href="<?php echo esc_url( home_url( '/' ) ); ?>" class="hover:text-[var(--wz-primary)] transition">Trang chủ</a></li>
            <li><span class="material-symbols-outlined text-[14px]" aria-hidden="true">chevron_right</span></li>
            <li><a href="<?php echo esc_url( home_url( '/my-account/' ) ); ?>" class="hover:text-[var(--wz-primary)] transition">Tài khoản</a></li>
            <li><span class="material-symbols-outlined text-[14px]" aria-hidden="true">chevron_right</span></li>
            <li class="font-bold text-[var(--wz-primary)]" aria-current="page">Tổng quan</li>
        </ol>
    </nav>

    <div class="flex flex-col lg:flex-row lg:gap-6">
        <?php include get_stylesheet_directory() . '/template-parts/account/sidebar.php'; ?>

        <!-- Main content -->
        <div class="flex-1 min-w-0 space-y-4 mt-4 lg:mt-0">
            <?php include get_stylesheet_directory() . '/template-parts/account/welcome-banner.php'; ?>
            <?php include get_stylesheet_directory() . '/template-parts/account/stats-cards.php'; ?>
            <?php include get_stylesheet_directory() . '/template-parts/account/recent-orders.php'; ?>
            <?php include get_stylesheet_directory() . '/template-parts/account/quick-links.php'; ?>

            <!-- Support -->
            <div class="bg-[var(--wz-surface-cool)] rounded-[var(--wz-radius-card)] p-4">
                <div class="flex items-center gap-3">
                    <span class="material-symbols-outlined text-[24px] text-[var(--wz-primary)]" aria-hidden="true">support_agent</span>
                    <div>
                        <p class="text-sm font-bold text-[var(--wz-on-surface)]">Cần hỗ trợ?</p>
                        <?php $phone = wz_config( 'phone', '' ); ?>
                        <a href="tel:<?php echo esc_attr( preg_replace( '/\s+/', '', $phone ) ); ?>" class="text-sm text-[var(--wz-primary)] font-semibold"><?php echo esc_html( $phone ); ?></a>
                    </div>
                </div>
            </div>
        </div>
    </div>

</div>
</div>

<?php get_footer(); ?>
```

## Usage

Create as `wezone-templates/account/dashboard.php`

## CSS Variables Used
- `--wz-surface` — page background
- `--wz-surface-cool` — card backgrounds
- `--wz-primary` — primary color
- `--wz-on-surface` — text color
- `--wz-outline` — muted text
- `--wz-radius-card` — card border radius

## Required Partials
- `template-parts/account/sidebar.php`
- `template-parts/account/welcome-banner.php`
- `template-parts/account/stats-cards.php`
- `template-parts/account/recent-orders.php`
- `template-parts/account/quick-links.php`
