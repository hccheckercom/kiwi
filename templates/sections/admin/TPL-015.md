## Mô tả

Admin SPA tự động đồng bộ màu sắc với theme đang active thông qua CSS custom properties. PHP đọc `store-config.php` → inject `:root` CSS vars → Tailwind classes nhận màu mới.

## Kiến trúc

```
store-config.php (theme.primary, theme.header_bg)
    ↓ PHP include
AdminPage::get_theme_colors()
    ↓ output
<style>:root{--wz-admin-primary:#00245f;--wz-admin-sidebar:#00245f;...}</style>
    ↓ CSS cascade
Tailwind classes: bg-wz-primary → var(--wz-admin-primary, #6366f1)
```

## Code: AdminPage.php — get_theme_colors()

```php
private function get_theme_colors(): array {
    $option_key = 'wz_design_overrides_' . get_stylesheet();
    $raw        = get_option( $option_key, '' );
    $saved      = ! empty( $raw ) ? json_decode( $raw, true ) : array();
    $saved_theme = $saved['theme'] ?? array();

    $config_theme = array();
    $store_config_file = get_stylesheet_directory() . '/store-config.php';
    if ( file_exists( $store_config_file ) ) {
        $config = include $store_config_file;
        if ( is_array( $config ) && isset( $config['theme'] ) ) {
            $config_theme = $config['theme'];
        }
    }

    $primary   = $saved_theme['primary'] ?? $config_theme['primary'] ?? '#6366f1';
    $header_bg = $saved_theme['header_bg'] ?? $config_theme['header_bg'] ?? '#0F172A';

    return array(
        'primary'       => $primary,
        'primary-hover' => $saved_theme['primary_container'] ?? $config_theme['primary_container'] ?? $this->adjust_brightness( $primary, -15 ),
        'sidebar'       => $header_bg,
        'sidebar-hover' => $this->adjust_brightness( $header_bg, 12 ),
    );
}

private function adjust_brightness( string $hex, int $percent ): string {
    $hex = ltrim( $hex, '#' );
    if ( strlen( $hex ) === 3 ) {
        $hex = $hex[0] . $hex[0] . $hex[1] . $hex[1] . $hex[2] . $hex[2];
    }
    $r = hexdec( substr( $hex, 0, 2 ) );
    $g = hexdec( substr( $hex, 2, 2 ) );
    $b = hexdec( substr( $hex, 4, 2 ) );

    $r = max( 0, min( 255, (int) round( $r + ( $r * $percent / 100 ) ) ) );
    $g = max( 0, min( 255, (int) round( $g + ( $g * $percent / 100 ) ) ) );
    $b = max( 0, min( 255, (int) round( $b + ( $b * $percent / 100 ) ) ) );

    return '#' . str_pad( dechex( $r ), 2, '0', STR_PAD_LEFT )
        . str_pad( dechex( $g ), 2, '0', STR_PAD_LEFT )
        . str_pad( dechex( $b ), 2, '0', STR_PAD_LEFT );
}
```

## Code: render_spa() — inject CSS vars

```php
$theme_colors = $this->get_theme_colors();

// ... wp_localize_script with 'themeColors' => $theme_colors ...

echo '<meta name="theme-color" content="' . esc_attr( $theme_colors['primary'] ) . '">';
echo '<style>:root{';
foreach ( $theme_colors as $key => $value ) {
    echo '--wz-admin-' . esc_attr( $key ) . ':' . esc_attr( $value ) . ';';
}
echo '}</style>';
echo '<div id="wezone-admin-root"></div>';
```

## Ưu tiên đọc màu

1. `wz_design_overrides_{stylesheet}` (WP option — user đã customize qua Design System)
2. `store-config.php` → key `theme.*` (mặc định của theme)
3. Fallback hardcoded (indigo `#6366f1`)

## Bidirectional sync

- Admin → Theme: `DesignAdminPage::ajax_save()` gọi `write_store_config_grouped()` → lưu vào WP option
- Theme → Admin: `AdminPage::get_theme_colors()` đọc store-config.php mỗi lần render

## Yêu cầu theme

Theme chỉ cần có `store-config.php` với:
```php
'theme' => [
    'primary'           => '#00245f',
    'primary_container' => '#1b3a7a',
    'header_bg'         => '#00245f',
],
```
