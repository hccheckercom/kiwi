# Bad Code

```python
@app.post("/auth/login")
async def login(credentials: LoginRequest):
    # BAD: No rate limiting
    # Attacker can brute-force passwords
    user = authenticate(credentials.email, credentials.password)
    return {"token": create_token(user)}

@app.post("/api/search")
async def search(query: str):
    # BAD: No rate limiting
    # Attacker can DoS with expensive queries
    results = expensive_search(query)
    return results
```

# Good Code

```python
from slowapi import Limiter, _rate_limit_exceeded_handler
from slowapi.util import get_remote_address
from slowapi.errors import RateLimitExceeded

limiter = Limiter(key_func=get_remote_address)
app.state.limiter = limiter
app.add_exception_handler(RateLimitExceeded, _rate_limit_exceeded_handler)

@app.post("/auth/login")
@limiter.limit("5/minute")  # 5 attempts per minute per IP
async def login(request: Request, credentials: LoginRequest):
    user = authenticate(credentials.email, credentials.password)
    if not user:
        raise HTTPException(401, "Invalid credentials")
    return {"token": create_token(user)}

@app.post("/api/search")
@limiter.limit("10/minute")
async def search(request: Request, query: str):
    results = expensive_search(query)
    return results

# Per-user rate limiting
@app.post("/api/upload")
@limiter.limit("3/hour", key_func=lambda: get_current_user().id)
async def upload_file(
    file: UploadFile,
    user: User = Depends(get_current_user)
):
    save_file(file, user.id)
    return {"status": "uploaded"}
```

# Why

**Without rate limiting:**
- Brute-force attacks (login, API keys)
- DoS attacks (expensive queries)
- Resource exhaustion (CPU, memory, DB)
- Cost explosion (cloud APIs)

**Attack scenarios:**
```python
# Brute-force login
for password in password_list:
    response = requests.post("/auth/login", json={
        "email": "victim@example.com",
        "password": password
    })
    if response.status_code == 200:
        print(f"Found password: {password}")

# DoS with expensive queries
while True:
    requests.post("/api/search", json={"query": "a" * 10000})
```

**Rate limiting strategies:**
```python
# Fixed window (simple)
@limiter.limit("100/hour")

# Sliding window (more accurate)
@limiter.limit("100/hour", per_method=True)

# Multiple limits
@limiter.limit("5/second;100/minute;1000/hour")

# Different limits per endpoint
@limiter.limit("5/minute")  # Login
@limiter.limit("100/minute")  # Read
@limiter.limit("10/minute")  # Write

# Custom key function
def get_user_tier(request: Request):
    user = get_current_user()
    return f"{user.id}:{user.tier}"

@limiter.limit("1000/hour", key_func=get_user_tier)
```

**Response headers:**
```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1640000000
```
