# Bad Code

```python
@app.post("/users")
async def create_user(email: str, age: int):
    # BAD: No validation
    # Accepts: age=-1, email="not-an-email"
    user = User(email=email, age=age)
    db.add(user)
    return user

@app.post("/upload")
async def upload_file(filename: str, content: str):
    # BAD: Path traversal vulnerability
    # Accepts: filename="../../../etc/passwd"
    with open(f"uploads/{filename}", "w") as f:
        f.write(content)
    return {"status": "uploaded"}
```

# Good Code

```python
from pydantic import BaseModel, EmailStr, Field, validator
import re

class UserCreate(BaseModel):
    email: EmailStr
    age: int = Field(..., ge=0, le=150)
    
    @validator('email')
    def email_must_be_lowercase(cls, v):
        return v.lower()

class FileUpload(BaseModel):
    filename: str = Field(..., regex=r'^[a-zA-Z0-9_\-\.]+$')
    content: str = Field(..., max_length=10_000_000)
    
    @validator('filename')
    def validate_filename(cls, v):
        if '..' in v or '/' in v or '\\' in v:
            raise ValueError('Invalid filename')
        return v

@app.post("/users")
async def create_user(user: UserCreate):
    db_user = User(**user.dict())
    db.add(db_user)
    return db_user

@app.post("/upload")
async def upload_file(file: FileUpload):
    safe_path = Path("uploads") / file.filename
    safe_path.write_text(file.content)
    return {"status": "uploaded"}
```

# Why

**Without validation:**
- **Path traversal:** `../../etc/passwd`
- **Command injection:** `; rm -rf /`
- **XSS:** `<script>alert('xss')</script>`
- **SQL injection:** `' OR 1=1--`
- **DoS:** 10GB file upload

**Real-world attacks:**
```python
# Path traversal
POST /upload
{"filename": "../../../etc/passwd", "content": "hacked"}

# Command injection
POST /process
{"command": "ls; rm -rf /"}

# XSS
POST /comments
{"text": "<script>steal_cookies()</script>"}
```

**Validation layers:**
```python
# 1. Type validation (Pydantic)
age: int  # Must be integer

# 2. Range validation
age: int = Field(..., ge=0, le=150)

# 3. Format validation
email: EmailStr  # Must be valid email

# 4. Pattern validation
filename: str = Field(..., regex=r'^[a-zA-Z0-9_\-\.]+$')

# 5. Custom validation
@validator('password')
def validate_password(cls, v):
    if len(v) < 8:
        raise ValueError('Too short')
    if not re.search(r'[A-Z]', v):
        raise ValueError('Need uppercase')
    return v
```

**Sanitization:**
```python
from bleach import clean

def sanitize_html(text: str) -> str:
    return clean(text, tags=[], strip=True)

def sanitize_filename(filename: str) -> str:
    return re.sub(r'[^a-zA-Z0-9_\-\.]', '', filename)
```
