**Pattern sai:**
```php
// inc/account.php
add_action( 'wp_ajax_wz_change_password', 'wezone_funilux_ajax_change_password' );
function wezone_funilux_ajax_change_password(): void { ... }

// inc/account-profile.php — CÙNG function name, CÙNG hook
add_action( 'wp_ajax_wz_change_password', 'wezone_funilux_ajax_change_password' );
function wezone_funilux_ajax_change_password(): void { ... }
// → PHP Fatal: Cannot redeclare wezone_funilux_ajax_change_password()
```
**Phải làm:**
```php
// Chỉ khai báo function MỘT LẦN trong MỘT file duy nhất
// Nếu cần chia file → dùng function_exists() guard hoặc chỉ đặt ở 1 nơi
// account-profile.php — giữ lại (có re-auth logic đầy đủ)
// account.php — XÓA duplicate
```
**Lý do:** Khi functions.php require_once cả `account.php` lẫn `account-profile.php`, PHP gặp 2 lần `function wezone_funilux_ajax_change_password()` → fatal error "Cannot redeclare" → white screen toàn site. Thường xảy ra khi copy-paste handler giữa các file hoặc refactor tách file mà quên xóa bản gốc.
