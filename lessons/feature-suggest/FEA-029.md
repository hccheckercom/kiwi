**Mô tả:**
Onboarding blueprint yêu cầu mỗi theme có setup wizard 5 bước giúp khách cấu hình shop trong < 10 phút.

**5 bước bắt buộc:**
1. Store Info → shop name, logo, slogan, primary color → auto-generate store-config.php
2. Industry Selection → chọn ngành → auto-apply DNA tokens
3. Demo Data Import → 1-click import 20 products + 5 categories + 3 banners
4. Basic Configuration → shipping, payment, contact (pre-filled defaults)
5. Preview & Complete → preview homepage → success message

**Phải làm:**
```php
class Wezone_Setup_Wizard {
    public function __construct() {
        add_action( 'admin_menu', [ $this, 'register_page' ] );
        add_action( 'wp_ajax_wz_setup_wizard_save', [ $this, 'handle_save' ] );
    }
}
```

**Lý do:**
Khách không rành kỹ thuật cần wizard đơn giản. Setup time > 10 phút = tỷ lệ refund tăng.
