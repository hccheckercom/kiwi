**Mô tả:**
- Product images PHẢI có `onerror` fallback khi ảnh lỗi/404
- Placeholder SVG cho từng context (product, category, avatar, banner)
- Tránh broken image icon hiển thị cho user

**Detect pattern (absence):**
- Template parts thiếu `onerror` hoặc placeholder reference → broken images hiển thị icon lỗi

**Code mẫu:**
```html
<img 
  src="{product_image}" 
  alt="{product_name}"
  onerror="this.onerror=null; this.src='/assets/img/placeholder/product.svg'"
  width="400" height="400"
  loading="lazy"
>
```

**Placeholder images cần có:**
| Context | File | Aspect Ratio |
|---------|------|--------------|
| Product | `/assets/img/placeholder/product.svg` | 1:1 |
| Category | `/assets/img/placeholder/category.svg` | 16:9 |
| Avatar | `/assets/img/placeholder/avatar.svg` | 1:1 |
| Banner | `/assets/img/placeholder/banner.svg` | 21:9 |

**CSS defensive:**
```css
img { max-width: 100%; height: auto; min-height: 1px; }
img[src=""], img:not([src]) { visibility: hidden; }
```

**Lợi ích:** UX chuyên nghiệp, không bao giờ hiển thị broken image icon
