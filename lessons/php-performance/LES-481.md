## Good
```php

    add_action( 'wp_enqueue_scripts', [ $this, 'enqueue_assets' ] );

}

    wp_enqueue_style( 'wz-demo-bar', plugin_dir_url( __FILE__ ) . 'assets/demo-bar.css', [], '1.0.0' );
```
