## Bad
```php
result = subprocess.run(cmd, capture_output=True, text=True, timeout=10)
```

## Good
```php
result = subprocess.run(cmd, capture_output=True, text=True, timeout=10, encoding="utf-8")
```

## Why
Windows default CP1252 encoding for subprocess pipe. Vietnamese UTF-8 text becomes mojibake. kiwi_search missed 100% Vietnamese results due to missing encoding param.

## Grep
`subprocess\.run\([^)]*text\s*=\s*True(?![^)]*encoding\s*=)` — in `**/*.py`
