## Bad
```php
register_rest_route( 'wezone/v1', '/search', [
    'methods' => 'GET',
    'callback' => [ $this, 'search' ],
    'permission_callback' => '__return_true',
] );
```

## Good
```php
register_rest_route( 'wezone/v1', '/search', [
    'methods' => 'GET',
    'callback' => [ $this, 'search' ],
    'permission_callback' => '__return_true',
] );

public function search( $request ) {
    $rate_limit = new ApiRateLimit();
    if ( ! $rate_limit->check( 'search', 60, 100 ) ) {
        return new WP_Error( 'rate_limit', 'Too many requests', [ 'status' => 429 ] );
    }
    // ... search logic
}
```

## Why
Public endpoints (__return_true) are vulnerable to abuse. Must implement rate limiting (IP-based or user-based) to prevent DoS attacks.
