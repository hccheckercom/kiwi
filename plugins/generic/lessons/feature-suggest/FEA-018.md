**Mô tả:**
- Mọi component PHẢI handle trạng thái empty (0 items)
- Hiển thị illustration + message + CTA thay vì blank space
- Flash Sale section: HIDE entirely khi không có active sale

**Detect pattern (absence):**
- Template parts thiếu empty state handling → user thấy blank page hoặc broken layout

**Components cần empty state:**
| Component | Empty Message | CTA |
|-----------|--------------|-----|
| Product Grid | "Chưa có sản phẩm" | Browse other categories |
| Cart | "Giỏ hàng trống" | Shop now |
| Wishlist | "Chưa có SP yêu thích" | Browse |
| Reviews | "Chưa có đánh giá" | Viết đánh giá đầu tiên |
| Search | "Không tìm thấy" | Suggestions + popular |
| Order History | "Chưa có đơn hàng" | Shop now |

**Template mẫu:**
```html
<div class="empty-state">
  <img src="/assets/img/empty/{context}.svg" alt="" width="200" height="160" loading="lazy">
  <h3 class="empty-state__title">{title}</h3>
  <p class="empty-state__desc">{description}</p>
  <a href="{cta_url}" class="btn btn--primary">{cta_text}</a>
</div>
```

**Lợi ích:** UX chuyên nghiệp, user không bị confused khi data trống
