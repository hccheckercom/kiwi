## Pattern

Developer dùng WooCommerce conditional tags trong templates/plugins để check page type.
Các functions này không tồn tại trên Wezone Commer.

## Bad

```php
if ( is_shop() ) {
    // Shop archive page
}

if ( is_product() ) {
    // Single product page
}

if ( is_product_category( 'shoes' ) ) {
    // Product category archive
}

if ( is_cart() ) {
    // Cart page
}

if ( is_checkout() ) {
    // Checkout page
}

if ( is_account_page() ) {
    // My account page
}

if ( is_woocommerce() ) {
    // Any WooCommerce page
}

if ( is_wc_endpoint_url( 'order-received' ) ) {
    // Thank you page
}
```

## Good

```php
if ( wz_is_shop() ) {
    // Shop archive page
}

if ( wz_is_product() ) {
    // Single product page
}

if ( wz_is_product_category( 'shoes' ) ) {
    // Product category archive
}

if ( wz_is_cart() ) {
    // Cart page
}

if ( wz_is_checkout() ) {
    // Checkout page
}

if ( wz_is_account_page() ) {
    // My account page
}

if ( wz_is_wezone_page() ) {
    // Any Wezone Commer page
}

if ( wz_is_endpoint( 'order-received' ) ) {
    // Thank you page
}
```

## Why

WooCommerce conditional functions được register qua `woocommerce.php` plugin file.
Khi WooCommerce không active, gọi bất kỳ function nào → Fatal error.

Đặc biệt nguy hiểm trong theme templates vì:
- Template files load trên mọi page request
- Fatal error = white screen toàn site
- Không có fallback — function literally không tồn tại

Mapping:
| WooCommerce | Wezone |
|---|---|
| `is_shop()` | `wz_is_shop()` |
| `is_product()` | `wz_is_product()` |
| `is_product_category()` | `wz_is_product_category()` |
| `is_product_tag()` | `wz_is_product_tag()` |
| `is_cart()` | `wz_is_cart()` |
| `is_checkout()` | `wz_is_checkout()` |
| `is_account_page()` | `wz_is_account_page()` |
| `is_woocommerce()` | `wz_is_wezone_page()` |
| `is_wc_endpoint_url()` | `wz_is_endpoint()` |
