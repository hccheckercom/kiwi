## Bad
```html
<div class="bg-[var(--wz-surface-pure)] ... max-w-[480px]">
```

## Good
```html
<div class="bg-[var(--wz-surface-pure)] ...">
```

## Why
Trong layout sidebar + content, phần content phải chiếm hết chiều rộng khả dụng. Thêm `max-w-[480px]` khiến form bị thu hẹp — giao diện trông lỗi trên desktop.
