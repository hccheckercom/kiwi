## Bad
```php
add_action('init', [$this, 'boot'], 0); // priority 0 = reserved for core
```

## Good
```php
add_action('init', [$this, 'boot'], 10); // default priority
```

## Why
Priority 0 on init hook — races with wezone-core which also uses priority 0. Undefined execution order.
