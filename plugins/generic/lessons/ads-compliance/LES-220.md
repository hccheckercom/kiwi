## Bad
```tsx
<span className="line-through">350.000đ</span>
<span>150.000đ</span> <span>-57%</span>
// "Giá gốc" never actually sold at that price
```
Inflated reference prices with no basis violate Google "Misleading Pricing" and Meta "Deceptive Practices" policies. This is one of the top reasons for ad account suspension.

## Good
```tsx
// Option 1: Remove strikethrough entirely — show only actual price
<span className="font-bold">150.000đ</span>

// Option 2: If price was genuinely sold at higher price, add proof
<span className="line-through text-gray-400">350.000đ</span>
<span className="font-bold">150.000đ</span>
<span className="text-xs text-gray-500">Giá bán lẻ từng tính năng khi mua riêng</span>
```

## Why
Google requires that strikethrough prices represent a price at which the product was "genuinely offered for sale" for a "reasonable period." Fabricated reference prices to create false urgency/value perception is classified as "deceptive business practice." Meta similarly prohibits "misleading pricing tactics." Fix: either remove fake original prices or provide verifiable justification.
