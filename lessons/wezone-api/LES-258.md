**Pattern sai:** Dùng `$product->get_price()`, `$product->is_in_stock()` — đây là WooCommerce syntax.
WeZone Commer `wz_get_product()` trả về **array**, KHÔNG phải object.

**Code sai:**
```php
$product = wz_get_product( $product_id );
$price = $product->get_price();           // ❌ Fatal error
$in_stock = $product->is_in_stock();      // ❌ Fatal error
$sku = $product->get_sku();               // ❌ Fatal error
```

**Code đúng:**
```php
$product = wz_get_product( $product_id );
$price = $product['price'] ?? 0;
$in_stock = ( $product['stock_status'] ?? '' ) === 'instock';
$sku = $product['sku'] ?? '';
$name = $product['name'] ?? '';
$image = $product['image'] ?? '';
$sale_price = $product['sale_price'] ?? null;
$regular_price = $product['regular_price'] ?? $price;
$average_rating = $product['average_rating'] ?? 0;
```

**Lý do:** WeZone Commer không dùng WooCommerce product objects. `wz_get_product()` query trực tiếp từ `wp_wz_products` table và trả về associative array. Dùng `->` trên array = PHP fatal error ngay lập tức.

**Grep check:** `\$product->(?:get_|is_|has_)` trong themes/ → CRITICAL
