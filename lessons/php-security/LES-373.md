# Bug: Cart Không Giới Hạn Items/Qty — DoS Vector

## Vấn đề

`CartEngine::add()` cho phép thêm vô hạn items và qty vào cart. Attacker spam `/cart/add` với nhiều product_id khác nhau → cart data phình to → DB write chậm → RAM tăng khi decode JSON → tiềm năng DoS.

**Ví dụ lỗi:**
```php
// ❌ SAI — không check max items
public function add( int $product_id, int $qty = 1, ... ): string {
    if ( $qty <= 0 || $product_id <= 0 ) {
        return '';
    }
    $cart = $this->get();
    // ... thêm item trực tiếp, không giới hạn
    $cart['items'][] = array( ... );
}
```

## Fix

```php
// ✅ ĐÚNG — enforce limits
private const MAX_CART_ITEMS = 50;
private const MAX_ITEM_QTY  = 99;

public function add( int $product_id, int $qty = 1, ... ): string {
    if ( $qty <= 0 || $product_id <= 0 ) {
        return '';
    }

    $qty  = min( $qty, self::MAX_ITEM_QTY );
    $cart = $this->get();

    // Check existing item — cap qty
    foreach ( $cart['items'] as &$item ) {
        if ( $item['key'] === $key ) {
            $item['qty'] = min( $item['qty'] + $qty, self::MAX_ITEM_QTY );
            // ...
            return $key;
        }
    }

    // Check max items
    if ( count( $cart['items'] ) >= self::MAX_CART_ITEMS ) {
        return '';
    }

    $cart['items'][] = array( ... );
    // ...
}
```
