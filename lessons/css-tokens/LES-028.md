**Pattern sai:**
```php
<span class="material-symbols-outlined text-[14px]">chevron_right</span>
<span class="material-symbols-outlined text-[16px]">verified</span>
```
**Phải làm:**
```php
<span class="material-symbols-outlined text-[14px]" aria-hidden="true">chevron_right</span>
<span class="material-symbols-outlined text-[16px]" aria-hidden="true">verified</span>
```
**Lý do:** Screen readers đọc icon text content ("chevron_right", "verified") gây noise. Icons decorative (đi kèm text label) phải có `aria-hidden="true"`. Icons standalone (không có text) cần `aria-label` thay vì aria-hidden.
**Grep check:** `material-symbols-outlined` without `aria-hidden` or `aria-label` — mỗi hit cần classify: decorative (cần aria-hidden) vs standalone (cần aria-label) → HIGH
