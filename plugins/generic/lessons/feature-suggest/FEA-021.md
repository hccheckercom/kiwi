**Mô tả:**
- Mọi `<img>` PHẢI có `width` + `height` attributes HOẶC CSS `aspect-ratio`
- Thiếu dimensions → browser không reserve space → CLS khi image load
- Skeleton placeholders phải cùng kích thước với content thật

**Detect pattern (absence):**
- Templates thiếu `width=` + `height=` trên img tags → CLS > 0.05

**Common CLS causes & fixes:**
| Cause | Fix |
|-------|-----|
| Images without dimensions | Add width/height or aspect-ratio |
| Web fonts FOUT | font-display: swap + preload |
| Dynamic content above fold | Reserve space with min-height |
| Sticky header appearing | Reserve space in body padding-top |
| Lazy-loaded content shifting | Skeleton placeholder same size |

**Code mẫu:**
```html
<!-- ĐÚNG: width + height prevents CLS -->
<img src="..." width="400" height="300" alt="..." loading="lazy">

<!-- ĐÚNG: aspect-ratio in CSS -->
<style>.product-card__image { aspect-ratio: 1; }</style>

<!-- SAI: no dimensions = CLS -->
<img src="..." alt="...">
```

**Target:** CLS < 0.05 trên mọi page type

**Lợi ích:** Core Web Vitals pass, Google ranking boost, UX không bị "nhảy layout"
