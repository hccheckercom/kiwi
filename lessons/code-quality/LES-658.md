## Bad
```php
def process():
    import json
    return json.loads(data)
```

## Good
```php
import json

def process():
    return json.loads(data)
```

## Why
Nested imports hide dependencies and add overhead on each call unless intentionally lazy

## Grep
`^\s{4,}(import|from)\s+` — in `**/*.py`
