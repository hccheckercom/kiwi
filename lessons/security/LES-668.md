## Bad
```php
tree = ET.parse(user_uploaded_file)
```

## Good
```php
from defusedxml.ElementTree import parse
tree = parse(user_uploaded_file)
```

## Why
Standard XML parsers are vulnerable to XXE (XML External Entity) attacks
