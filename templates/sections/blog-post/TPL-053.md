# Blog Post — Single Post Template

Single blog post template với prose layout, author bio, và related posts.

## Use Cases
- Single blog post (single.php)
- News article
- Tutorial/guide

## Features
- 📄 Prose layout (readable typography)
- 👤 Author bio box
- 🏷️ Category + tags
- 📅 Published date + reading time
- 💬 Comments section
- 🔗 Social share buttons
- 📰 Related posts grid

## Code

### PHP Template (single.php)

```php
<?php
/**
 * Single Post Template
 */

get_header();

while ( have_posts() ): the_post();
?>

<article class="single-post py-12">
	<div class="max-w-[1280px] mx-auto px-10">
		<!-- Post Header -->
		<header class="max-w-3xl mx-auto mb-10">
			<!-- Category -->
			<?php
			$categories = get_the_category();
			if ( ! empty( $categories ) ):
			?>
				<a href="<?php echo esc_url( get_category_link( $categories[0]->term_id ) ); ?>" class="inline-block px-3 py-1 text-sm font-medium bg-blue-50 text-blue-600 rounded-full hover:bg-blue-100 mb-4">
					<?php echo esc_html( $categories[0]->name ); ?>
				</a>
			<?php endif; ?>

			<!-- Title -->
			<h1 class="text-4xl md:text-5xl font-bold mb-6"><?php the_title(); ?></h1>

			<!-- Meta -->
			<div class="flex items-center gap-4 text-gray-600">
				<div class="flex items-center gap-2">
					<?php echo get_avatar( get_the_author_meta( 'ID' ), 40, '', '', ['class' => 'rounded-full'] ); ?>
					<span class="font-medium"><?php the_author(); ?></span>
				</div>
				<span>•</span>
				<time datetime="<?php echo esc_attr( get_the_date( 'c' ) ); ?>">
					<?php echo get_the_date(); ?>
				</time>
				<span>•</span>
				<span><?php echo wz_reading_time(); ?> phút đọc</span>
			</div>
		</header>

		<!-- Featured Image -->
		<?php if ( has_post_thumbnail() ): ?>
			<div class="max-w-4xl mx-auto mb-10">
				<?php the_post_thumbnail( 'full', [
					'class' => 'w-full rounded-2xl',
					'loading' => 'eager',
				]); ?>
			</div>
		<?php endif; ?>

		<!-- Content -->
		<div class="max-w-3xl mx-auto">
			<div class="prose prose-lg max-w-none mb-10">

			</div>

			<!-- Tags -->
			<?php
			$tags = get_the_tags();
			if ( $tags ):
			?>
				<div class="flex flex-wrap gap-2 mb-10 pb-10 border-b border-gray-200">
					<?php foreach ( $tags as $tag ): ?>
						<a href="<?php echo esc_url( get_tag_link( $tag->term_id ) ); ?>" class="px-3 py-1 text-sm bg-gray-100 text-gray-700 rounded-full hover:bg-gray-200">
							#<?php echo esc_html( $tag->name ); ?>
						</a>
					<?php endforeach; ?>
				</div>
			<?php endif; ?>

			<!-- Author Bio -->
			<div class="bg-gray-50 rounded-2xl p-6 mb-10">
				<div class="flex items-start gap-4">
					<?php echo get_avatar( get_the_author_meta( 'ID' ), 80, '', '', ['class' => 'rounded-full'] ); ?>
					<div>
						<h3 class="font-bold text-lg mb-1"><?php the_author(); ?></h3>
						<p class="text-gray-600 text-sm mb-3"><?php echo esc_html( get_the_author_meta( 'description' ) ?: 'Tác giả tại ' . wz_config('shop_name') ); ?></p>
						<div class="flex gap-3">
							<?php if ( get_the_author_meta( 'facebook' ) ): ?>
								<a href="<?php echo esc_url( get_the_author_meta( 'facebook' ) ); ?>" class="text-gray-400 hover:text-blue-600" target="_blank" rel="noopener">
									<svg class="w-5 h-5" fill="currentColor" viewBox="0 0 24 24"><path d="M24 12.073c0-6.627-5.373-12-12-12s-12 5.373-12 12c0 5.99 4.388 10.954 10.125 11.854v-8.385H7.078v-3.47h3.047V9.43c0-3.007 1.792-4.669 4.533-4.669 1.312 0 2.686.235 2.686.235v2.953H15.83c-1.491 0-1.956.925-1.956 1.874v2.25h3.328l-.532 3.47h-2.796v8.385C19.612 23.027 24 18.062 24 12.073z"/></svg>
								</a>
							<?php endif; ?>
						</div>
					</div>
				</div>
			</div>

			<!-- Share Buttons -->
			<div class="flex items-center gap-4 mb-10 pb-10 border-b border-gray-200">

		<?php
		$related = new WP_Query([
			'post_type' => 'post',
			'posts_per_page' => 3,
			'post__not_in' => [ get_the_ID() ],
			'category__in' => wp_get_post_categories( get_the_ID() ),
			'orderby' => 'rand',

		if ( $related->have_posts() ):
		?>
			<div class="max-w-4xl mx-auto mb-10">

				</div>
			</div>
		<?php endif; ?>

		<!-- Comments -->
		<?php if ( comments_open() || get_comments_number() ): ?>
			<div class="max-w-3xl mx-auto">
				<?php comments_template(); ?>
			</div>
		<?php endif; ?>
	</div>
</article>

<?php
endwhile;
get_footer();
?>
```

### Helper Function (functions.php)

```php
/**
 * Calculate reading time
 */
function wz_reading_time() {
	$content = get_post_field( 'post_content', get_the_ID() );
	$word_count = str_word_count( strip_tags( $content ) );
	$reading_time = ceil( $word_count / 200 ); // 200 words per minute
	return max( 1, $reading_time );
}
```

### Add table of contents
```javascript
// Generate TOC from h2/h3 headings
const content = document.querySelector('.prose');
const headings = content.querySelectorAll('h2, h3');
const toc = document.createElement('nav');
toc.className = 'toc bg-gray-50 rounded-lg p-6 mb-10';
toc.innerHTML = '<h2 class="font-bold mb-4">Nội dung</h2><ul class="space-y-2"></ul>';

headings.forEach((heading, i) => {
	heading.id = `heading-${i}`;
	const li = document.createElement('li');
	li.innerHTML = `<a href="#heading-${i}" class="text-blue-600 hover:underline">${heading.textContent}</a>`;
	toc.querySelector('ul').appendChild(li);
});

content.insertBefore(toc, content.firstChild);
```

### Add progress bar
```javascript
window.addEventListener('scroll', () => {
	const article = document.querySelector('.single-post');
	const scrolled = window.scrollY;
	const height = article.offsetHeight - window.innerHeight;
	const progress = (scrolled / height) * 100;
	document.querySelector('.progress-bar').style.width = `${Math.min(progress, 100)}%`;
});
```
