## Bad
```php
$anim_js = WEZONE_THEME_DIR . '/assets/js/modules/animations.js';
if ( file_exists( $anim_js ) ) {
    wp_enqueue_script( 'wezone-theme-animations', ... );
}
```
Loads on every page — wastes bandwidth on pages that don't use animations.

## Good
```php
if ( is_front_page() || is_singular( 'wz_product' ) ) {
    $anim_js = WEZONE_THEME_DIR . '/assets/js/modules/animations.js';
    if ( file_exists( $anim_js ) ) {
        wp_enqueue_script( 'wezone-theme-animations', ... );
    }
}
```
Or use dynamic import in JS:
```js
if (document.querySelector('[data-animate]')) {
    import('./modules/animations.js');
}
```

## Why
Performance constitution: "KHONG load JS cho features khong co tren page hien tai". Each unnecessary script adds to Total JS budget (80KB homepage target). Conditional loading keeps pages within budget.
