## Use Case

Hero section that must be fullwidth (100vw) even when parent `<main>` or wrapper has `max-width`/`padding` constraints from theme engine or shared design system.

## Structure (PHP)

```php
<?php get_header(); ?>

<!-- Hero OUTSIDE <main> to bypass container constraints -->
<?php get_template_part( 'template-parts/hero/default' ); ?>

<main id="main-content">
    <?php get_template_part( 'template-parts/home/flash-sale' ); ?>
    <!-- other sections -->
</main>

<?php get_footer(); ?>
```

## CSS — Full-Bleed Pattern

```css
html, body {
    overflow-x: hidden;
}

#main-content {
    width: 100%;
    max-width: 100%;
    padding: 0;
    margin: 0;
    overflow-x: hidden;
}

.haven-hero {
    position: relative;
    width: 100vw;
    margin-left: calc(-50vw + 50%);
    overflow: hidden;
    background: #3D2B1F;
}
```

## CSS — Slider Track (Grid Stacking)

```css
.haven-hero__track {
    display: grid;
    grid-template-columns: 100%;
    width: 100%;
    overflow: hidden;
}

.haven-hero__slide {
    grid-area: 1 / 1;
    width: 100%;
    min-width: 0;
    visibility: hidden;
    pointer-events: none;
    transition-property: opacity, transform, filter, clip-path;
    transition-duration: 600ms;
    transition-timing-function: ease-in-out;
}

.haven-hero__slide.is-active {
    visibility: visible;
    pointer-events: auto;
}
```

## CSS — Visual + Products (Mobile-First)

```css
.haven-hero__visual {
    position: relative;
    width: 100%;
    height: calc(var(--hp-height-mobile, 520px) + 120px);
    overflow: hidden;
}

/* Mobile: products strip floating on banner — no background, spaced from edges */
.haven-hero__products {
    position: absolute;
    bottom: 12px;
    left: 12px;
    right: 12px;
    z-index: 3;
    display: flex;
    overflow-x: auto;
    scroll-snap-type: x mandatory;
    -webkit-overflow-scrolling: touch;
    scrollbar-width: none;
    gap: 8px;
    padding: 0;
    background: none;
    box-sizing: border-box;
}

.haven-hero__products::-webkit-scrollbar { display: none; }

.haven-hero__products .haven-hero-card {
    flex: 0 0 calc(33.33% - 6px);
    scroll-snap-align: start;
    min-width: 90px;
    max-width: 120px;
}

/* Desktop: products overlay grid */
@media (min-width: 768px) {
    .haven-hero__visual {
        height: var(--hp-height-desktop, 560px);
    }
    .haven-hero__products {
        display: grid;
        grid-template-columns: repeat(var(--hp-cols, 3), 1fr);
        overflow: visible;
        scroll-snap-type: none;
        bottom: 28px;
        right: 32px;
        left: auto;
        max-width: 520px;
        gap: 12px;
        padding: 0;
        background: transparent;
    }
}
```

## Key Rules

1. **Hero outside `<main>`** — avoids all parent constraints
2. **`width: 100vw` + `margin-left: calc(-50vw + 50%)`** — full-bleed breakout
3. **`overflow-x: hidden` on html/body** — prevents scrollbar from 100vw
4. **Position variants (right/left/bottom) scoped to `@media (min-width: 768px)` only** — mobile always fullwidth
5. **No duplicate CSS blocks** — single source of truth for each selector
6. **Grid stacking for slides** — `grid-area: 1/1` with `grid-template-columns: 100%`
