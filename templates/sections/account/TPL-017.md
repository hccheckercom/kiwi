# Account Sidebar — Responsive Navigation

Dual-layout account navigation: horizontal scrollable tabs on mobile, vertical sidebar on desktop. Includes user avatar, unread notification badge, and logout.

## Features
- Mobile: horizontal scroll tabs with icons
- Desktop: vertical sidebar with user info card
- Active page highlighting
- Unread notification badge
- Material Icons
- Sticky positioning on desktop
- Logout confirmation

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

$current_user = wp_get_current_user();
$avatar_url   = get_avatar_url( $current_user->ID, [ 'size' => 80 ] );
$active_page  = get_query_var( 'wz_tpl_account_page', 'dashboard' );

$menu_items = [
    [ 'slug' => 'dashboard',        'icon' => 'dashboard',           'label' => 'Tổng quan' ],
    [ 'slug' => 'orders',           'icon' => 'receipt_long',        'label' => 'Đơn hàng' ],
    [ 'slug' => 'profile',          'icon' => 'person',              'label' => 'Thông tin cá nhân' ],
    [ 'slug' => 'addresses',        'icon' => 'location_on',         'label' => 'Địa chỉ' ],
    [ 'slug' => 'change-password',  'icon' => 'lock',                'label' => 'Đổi mật khẩu' ],
    [ 'slug' => 'notifications',    'icon' => 'notifications',       'label' => 'Thông báo' ],
    [ 'slug' => 'vouchers',         'icon' => 'confirmation_number', 'label' => 'Mã giảm giá' ],
    [ 'slug' => 'wishlist',         'icon' => 'favorite',            'label' => 'Yêu thích' ],
];

$unread_count = (int) get_user_meta( $current_user->ID, '_wz_unread_notifications', true );
?>

<!-- Mobile: horizontal scroll tabs -->
<div class="lg:hidden overflow-x-auto pb-2 -mx-5 px-5">
    <div class="flex gap-1 min-w-max">
        <?php foreach ( $menu_items as $item ) : ?>
            <a href="<?php echo esc_url( home_url( '/my-account/' . ( $item['slug'] === 'dashboard' ? '' : $item['slug'] . '/' ) ) ); ?>"
               class="flex items-center gap-1.5 px-3 py-2 rounded-[var(--wz-radius)] text-xs font-semibold whitespace-nowrap transition <?php echo $active_page === $item['slug'] ? 'bg-[var(--wz-primary)]/10 text-[var(--wz-primary)]' : 'text-[var(--wz-outline)] hover:bg-[var(--wz-surface-cool)]'; ?>">
                <span class="material-symbols-outlined text-[16px]" aria-hidden="true"><?php echo esc_html( $item['icon'] ); ?></span>
                <?php echo esc_html( $item['label'] ); ?>
                <?php if ( $item['slug'] === 'notifications' && $unread_count > 0 ) : ?>
                    <span class="w-4 h-4 bg-[var(--wz-error)] text-white text-[10px] font-bold rounded-full flex items-center justify-center"><?php echo $unread_count > 9 ? '9+' : $unread_count; ?></span>
                <?php endif; ?>
            </a>
        <?php endforeach; ?>
    </div>
</div>

<!-- Desktop: vertical sidebar -->
<aside class="hidden lg:block w-[280px] shrink-0">
    <div class="bg-[var(--wz-surface-pure)] rounded-[var(--wz-radius-card)] p-5 shadow-[0_4px_24px_rgba(27,58,122,0.08)] sticky top-[120px]">
        <!-- User info -->
        <div class="flex items-center gap-3 pb-4 border-b border-[var(--wz-outline)]/10">
            <img src="<?php echo esc_url( $avatar_url ); ?>" alt="<?php echo esc_attr( $current_user->display_name ); ?>" class="w-12 h-12 rounded-full object-cover">
            <div class="min-w-0">
                <p class="text-sm font-bold text-[var(--wz-on-surface)] truncate"><?php echo esc_html( $current_user->display_name ); ?></p>
                <p class="text-xs text-[var(--wz-outline)] truncate"><?php echo esc_html( $current_user->user_email ); ?></p>
            </div>
        </div>

        <!-- Menu -->
        <nav class="mt-3 space-y-1">
            <?php foreach ( $menu_items as $item ) : ?>
                <a href="<?php echo esc_url( home_url( '/my-account/' . ( $item['slug'] === 'dashboard' ? '' : $item['slug'] . '/' ) ) ); ?>"
                   class="flex items-center gap-3 px-3 py-2.5 rounded-[var(--wz-radius)] text-sm transition <?php echo $active_page === $item['slug'] ? 'bg-[var(--wz-primary)]/10 text-[var(--wz-primary)] font-bold' : 'text-[var(--wz-on-surface)] hover:bg-[var(--wz-surface-cool)]'; ?>">
                    <span class="material-symbols-outlined text-[20px]" aria-hidden="true"><?php echo esc_html( $item['icon'] ); ?></span>
                    <?php echo esc_html( $item['label'] ); ?>
                    <?php if ( $item['slug'] === 'notifications' && $unread_count > 0 ) : ?>
                        <span class="ml-auto w-5 h-5 bg-[var(--wz-error)] text-white text-[10px] font-bold rounded-full flex items-center justify-center"><?php echo $unread_count > 9 ? '9+' : $unread_count; ?></span>
                    <?php endif; ?>
                </a>
            <?php endforeach; ?>

            <div class="border-t border-[var(--wz-outline)]/10 my-2"></div>

            <a href="<?php echo esc_url( wp_logout_url( home_url( '/' ) ) ); ?>"
               class="flex items-center gap-3 px-3 py-2.5 rounded-[var(--wz-radius)] text-sm text-[var(--wz-error)] hover:bg-[var(--wz-error)]/5 transition"
               onclick="return confirm('Bạn có chắc muốn đăng xuất?')">
                <span class="material-symbols-outlined text-[20px]" aria-hidden="true">logout</span>
                Đăng xuất
            </a>
        </nav>
    </div>
</aside>
```

## Usage

```php
// In wezone-templates/account/*.php
<div class="container mx-auto px-5 py-8">
    <div class="flex gap-6">
        <?php get_template_part( 'template-parts/account/sidebar' ); ?>
        <main class="flex-1 min-w-0">
            <!-- Account page content -->
        </main>
    </div>
</div>
```

## CSS Variables Used
- `--wz-primary` — active menu item
- `--wz-surface-pure` — sidebar background
- `--wz-surface-cool` — hover state
- `--wz-outline` — borders and inactive text
- `--wz-on-surface` — text color
- `--wz-error` — logout button and notification badge
- `--wz-radius` — button border radius
- `--wz-radius-card` — card border radius

## Required Query Var
Set `wz_tpl_account_page` query var in account template routing to highlight active page.
