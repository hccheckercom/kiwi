## Bad
```
// Ad says: "Template website miễn phí — tải ngay!"
// Landing page: Shows pricing from 990.000đ, no free option visible
// → Bait-and-switch = immediate ad rejection + account warning

// Ad says: "Thiết kế web chuyên nghiệp 500.000đ"
// Landing page: Lowest price shown is 990.000đ
// → Price mismatch = misleading pricing violation
```
Google and Meta both require that landing page content directly matches ad messaging.

## Good
```
// Ad says: "Template website từ 990.000đ — xem demo ngay"
// Landing page: Shows pricing starting at 990.000đ with demo links
// → Consistent messaging ✓

// Ad says: "Tư vấn miễn phí — không cam kết"
// Landing page: Contact form clearly labeled "Tư vấn miễn phí" with no payment required
// → Promise matches reality ✓
```

## Why
"Destination mismatch" is the #1 reason for Google Ads rejection. The rule is simple: whatever the ad promises, the landing page must deliver immediately and visibly. This includes: (1) price mentioned in ad must be visible on page, (2) offer in ad must be accessible without extra steps, (3) product/service described must match what's shown. Google uses automated crawlers to verify this — cloaking (showing different content to Google vs users) results in permanent account ban.
