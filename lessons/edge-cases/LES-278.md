**Pattern sai:**
```css
body {
  overflow-x: hidden;
}
```

**Phải làm:**
```css
html {
  overflow-x: hidden;
}
```

**Lý do:** `overflow-x: hidden` trên body gây bug trên iOS Safari — content bị thu hẹp, scroll bị kẹt. Phải đặt trên `html` element. Xem thêm LES-045 (Kiwi css-tokens).

**Regression:** Theme wezone-trunganh-v2 đã từng gặp bug này (commit fix: chuyển từ body sang html). CRITICAL vì ảnh hưởng 100% mobile users.
