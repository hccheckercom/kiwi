**Pattern sai:**
```php
<?php foreach ( $products as $product ) : ?>
    <!-- render all products without limit -->
<?php endforeach; ?>
```

**Phải làm:**
```php
<?php
$products = array_slice( $products, 0, 20 );
foreach ( $products as $product ) : ?>
    <!-- max 20 items in horizontal scroll -->
<?php endforeach; ?>
```

**Lý do:** Blueprint quy định flash sale dùng horizontal scroll, max 20 items. Quá nhiều items → scroll quá dài, performance kém trên mobile.
