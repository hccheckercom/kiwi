## Bad
```php
$wpdb->prepare("SELECT * FROM table WHERE name = '$name'");
```

## Good
```php
$wpdb->prepare('SELECT * FROM table WHERE name = %s', $name);
```

## Why
Variable interpolated inside prepare() format string — defeats the purpose of prepared statements.
