## Bad
```php
public function boot(): void {
    if ( ! wezone_is_active() ) {
        return;
    }
    $this->register_hooks();
}
```

## Good
```php
public function boot(): void {
    if ( ! wezone_is_active() ) {
        return;
    }
    if ( function_exists( 'wz_config' ) ) {
        wz_config( 'plugin-name' );
    }
    $this->register_hooks();
}
```

## Why
Plugin entry points must call wz_config() to load store-specific settings. Missing this causes features to use wrong config or fail silently.

## Grep
`public function boot\(\)[^}]*\{(?![^}]*wz_config)` — in `packages/*/src/Plugin.php`
