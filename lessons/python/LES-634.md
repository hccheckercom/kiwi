## Bad
```python
try:
    await websocket.close()
except:
    pass
```

## Good
```python
try:
    await websocket.close()
except Exception:
    pass
```

## Why

Bare `except:` catches `SystemExit`, `KeyboardInterrupt`, and `GeneratorExit` — signals that should propagate. Always catch `Exception` at minimum to allow system signals to work correctly. PEP 8 explicitly warns against bare `except:` clauses.
