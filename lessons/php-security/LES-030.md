**Pattern sai:**
```php
$password = $_POST['password'] ?? '';
```
**Phải làm:**
```php
$password = wp_unslash( $_POST['password'] ?? '' );
```
**Lý do:** WordPress tự động thêm magic quotes vào `$_POST` (qua `wp_magic_quotes()`). Nếu không gọi `wp_unslash()`, password chứa ký tự đặc biệt như `\`, `'`, `"` sẽ bị escape sai — user nhập đúng nhưng hash không khớp khi đăng nhập. Áp dụng cho mọi raw string field: password, token, key — không chỉ text thông thường.
**Grep check:** `\$_POST\['password'\]\s*\?\?` — thiếu `wp_unslash` trong cùng dòng hoặc dòng liền kề → CRITICAL
