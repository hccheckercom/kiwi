## Bad
```tsx
// page-a.tsx
<div className="max-w-7xl mx-auto">

// page-b.tsx
<div className="max-w-screen-xl mx-auto">
```
max-w-7xl = 80rem = 1280px, max-w-screen-xl = 1280px — giống nhau nhưng dùng lẫn lộn gây confusing khi maintain.

## Good
```tsx
// Thống nhất dùng một convention
<div className="max-w-7xl mx-auto">
```

## Why
Dự án phải chọn một container strategy duy nhất. Mixing max-w-7xl và max-w-screen-xl khiến developer khó biết đâu là standard.
