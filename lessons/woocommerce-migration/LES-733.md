## Pattern

Developer dùng WooCommerce utility functions cho formatting, templates, notices, pages.

## Bad

```php
// Price formatting
echo wc_price( 299000 );
$decimals  = wc_get_price_decimals();
$formatted = wc_format_decimal( $number, 2 );
$symbol    = wc_get_currency_symbol( 'VND' );

// Templates
wc_get_template( 'single-product/price.php', array( 'product' => $product ) );
wc_get_template_part( 'content', 'product' );
$html = wc_get_template_html( 'cart/cart-totals.php' );

// Notices
wc_add_notice( 'Đã thêm vào giỏ hàng', 'success' );
wc_print_notices();
if ( wc_has_notice( 'error' ) ) { }
wc_clear_notices();

// Pages
$shop_id     = wc_get_page_id( 'shop' );
$cart_url    = wc_get_page_permalink( 'cart' );
$checkout    = wc_get_page_permalink( 'checkout' );
$endpoint    = wc_get_endpoint_url( 'order-received', $order_id );
$account_url = wc_get_account_endpoint_url( 'orders' );

// Sanitization
$clean = wc_clean( $_POST['field'] );
$text  = wc_sanitize_textarea( $_POST['note'] );

// Misc
$logger = wc_get_logger();
$logger->info( 'message', array( 'source' => 'my-plugin' ) );
wc_enqueue_js( 'jQuery(".btn").on("click", fn);' );
echo wc_get_stock_html( $product );
echo wc_get_rating_html( 4.5, 120 );
```

## Good

```php
// Price formatting
echo wz_format_price( 299000 );
$decimals  = wz_get_price_decimals();
$formatted = number_format( $number, 2 );
$symbol    = wz_get_currency_symbol();

// Templates
wz_get_template( 'product/price.php', array( 'product' => $product ) );
wz_get_template_part( 'content', 'product' );
$html = wz_get_template_html( 'cart/totals.php' );

// Notices
wz_add_notice( 'Đã thêm vào giỏ hàng', 'success' );
wz_print_notices();
if ( wz_has_notice( 'error' ) ) { }
wz_clear_notices();

// Pages
$shop_url    = wz_get_page_url( 'shop' );
$cart_url    = wz_get_page_url( 'cart' );
$checkout    = wz_get_page_url( 'checkout' );
$endpoint    = wz_get_endpoint_url( 'order-received', $order_id );
$account_url = wz_get_account_url( 'orders' );

// Sanitization
$clean = sanitize_text_field( wp_unslash( $_POST['field'] ) );
$text  = sanitize_textarea_field( wp_unslash( $_POST['note'] ) );

// Misc
wz_log( 'info', 'message', 'my-plugin' );
wp_enqueue_script( 'my-inline', ... );
echo wz_get_stock_html( $product );
echo wz_get_rating_html( 4.5, 120 );
```

## Why

Các utility functions này là những function WooCommerce phổ biến nhất — developer dùng hàng ngày.
Gọi bất kỳ function nào trong danh sách → Fatal error: Call to undefined function.

Lưu ý đặc biệt:
- `wc_clean()` chỉ là wrapper cho `sanitize_text_field()` — dùng trực tiếp WordPress function
- `wc_get_page_id()` trả page ID — Wezone dùng URL-based routing, không cần page ID
- `wc_get_template()` tìm trong `yourtheme/woocommerce/` — Wezone dùng `wezone-templates/`
- `wc_enqueue_js()` inject inline JS — dùng `wp_enqueue_script()` hoặc `wp_add_inline_script()`
