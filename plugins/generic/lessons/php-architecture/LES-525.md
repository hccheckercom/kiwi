## Bad
```php
class IconRegistry {
    private static $instance;
    
    public static function instance(): self {
        if ( null === self::$instance ) {
            self::$instance = new self();
        }
        return self::$instance;
    }
}
```

## Good
```php
class IconRegistry {
    private static $instance;
    
    public static function instance(): self {
        if ( null === self::$instance ) {
            self::$instance = new self();
        }
        return self::$instance;
    }
    
    public static function reset_for_testing(): void {
        self::$instance = null;
    }
}
```

## Why
Singletons without reset method are hard to test (state leaks between tests). Add reset_for_testing() method to allow clean slate in unit tests.
