# Landing Hero — High-Converting Landing Page Hero

Landing page hero section với strong CTA, benefits list, và social proof.

## Use Cases
- Landing page (trang 44)
- Campaign page
- Product launch page
- Lead generation page

## Features
- 🎯 Strong headline + subheadline
- 🚀 Primary CTA button (prominent)
- ✅ Key benefits list (3-5 items)
- 📊 Social proof (numbers, testimonials)
- 🖼️ Hero image/video
- 📱 Mobile-optimized layout

## Code

### PHP Template (template-parts/landing/hero.php)

```php
<?php
/**
 * Landing Hero Component
 * 
 * @param array $args Hero configuration
 */

$headline = $args['headline'] ?? 'Giải Pháp Hoàn Hảo Cho Doanh Nghiệp';
$subheadline = $args['subheadline'] ?? 'Tăng doanh thu, tối ưu chi phí, và phát triển bền vững với nền tảng của chúng tôi';
$cta_text = $args['cta_text'] ?? 'Dùng thử miễn phí';
$cta_url = $args['cta_url'] ?? '#';
$benefits = $args['benefits'] ?? [
	'Dễ dàng sử dụng, không cần kỹ thuật',
	'Tích hợp đầy đủ với hệ thống hiện tại',
	'Hỗ trợ 24/7 bằng tiếng Việt',
];
$stats = $args['stats'] ?? [
	['number' => '10,000+', 'label' => 'Khách hàng'],
	['number' => '99.9%', 'label' => 'Uptime'],
	['number' => '4.9/5', 'label' => 'Đánh giá'],
];
$image = $args['image'] ?? get_template_directory_uri() . '/assets/images/hero-landing.png';
?>

<section class="landing-hero bg-gradient-to-br from-blue-50 to-white py-20">
	<div class="max-w-[1280px] mx-auto px-10">
		<div class="grid lg:grid-cols-2 gap-12 items-center">
			<!-- Left: Content -->
			<div>
				<!-- Headline -->
				<h1 class="text-5xl md:text-6xl font-bold mb-6 leading-tight">
					<?php echo esc_html( $headline ); ?>
				</h1>

				<!-- Subheadline -->
				<p class="text-xl text-gray-600 mb-8">
					<?php echo esc_html( $subheadline ); ?>
				</p>

				<!-- Benefits -->
				<ul class="space-y-3 mb-8">
					<?php foreach ( $benefits as $benefit ): ?>
						<li class="flex items-start gap-3">
							<svg class="w-6 h-6 text-green-500 shrink-0 mt-0.5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
								<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M5 13l4 4L19 7" />
							</svg>
							<span class="text-gray-700"><?php echo esc_html( $benefit ); ?></span>
						</li>
					<?php endforeach; ?>
				</ul>

				<!-- CTA -->
				<div class="flex flex-wrap gap-4 mb-8">
					<a 
						href="<?php echo esc_url( $cta_url ); ?>" 
						class="inline-flex items-center gap-2 bg-blue-600 text-white font-semibold px-8 py-4 rounded-lg hover:bg-blue-700 transition-colors shadow-lg hover:shadow-xl"
					>
						<?php echo esc_html( $cta_text ); ?>
						<svg class="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
							<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M13 7l5 5m0 0l-5 5m5-5H6" />
						</svg>
					</a>
					<a 
						href="#features" 
						class="inline-flex items-center gap-2 bg-white text-gray-700 font-semibold px-8 py-4 rounded-lg border-2 border-gray-200 hover:border-gray-300 transition-colors"
					>
						Tìm hiểu thêm
					</a>
				</div>

				<!-- Social Proof Stats -->
				<div class="flex flex-wrap gap-8">
					<?php foreach ( $stats as $stat ): ?>
						<div>
							<div class="text-3xl font-bold text-blue-600"><?php echo esc_html( $stat['number'] ); ?></div>
							<div class="text-sm text-gray-600"><?php echo esc_html( $stat['label'] ); ?></div>
						</div>
					<?php endforeach; ?>
				</div>
			</div>

			<!-- Right: Image -->
			<div class="relative">
				<img 
					src="<?php echo esc_url( $image ); ?>" 
					alt="<?php echo esc_attr( $headline ); ?>"
					class="w-full h-auto rounded-2xl shadow-2xl"
					loading="eager"
				/>
				<!-- Floating badge (optional) -->
				<div class="absolute -bottom-6 -left-6 bg-white rounded-xl shadow-lg p-4 hidden lg:block">
					<div class="flex items-center gap-3">
						<div class="w-12 h-12 bg-green-100 rounded-full flex items-center justify-center">
							<svg class="w-6 h-6 text-green-600" fill="none" stroke="currentColor" viewBox="0 0 24 24">
								<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M5 13l4 4L19 7" />
							</svg>
						</div>
						<div>
							<div class="font-bold">Miễn phí 30 ngày</div>
							<div class="text-sm text-gray-600">Không cần thẻ tín dụng</div>
						</div>
					</div>
				</div>
			</div>
		</div>
	</div>
</section>
```

## Data Sources

**Args array:**
- `headline` — Main headline (H1)
- `subheadline` — Supporting text
- `cta_text` — Primary button text
- `cta_url` — Primary button URL
- `benefits` — Array of benefit strings
- `stats` — Array of [number, label] pairs
- `image` — Hero image URL

## Dependencies
- Tailwind CSS
- wz_component() function

## Responsive Behavior
- Mobile: Stacked layout (content → image)
- Desktop (lg): 2-column grid
- CTA buttons: Stack on mobile, inline on desktop

## Accessibility
- Semantic HTML (h1, ul, li)
- Alt text on hero image
- Focus states on buttons
- Proper heading hierarchy

## Conversion Optimization
- Above-the-fold CTA
- Clear value proposition
- Social proof (stats)
- Benefit-focused (not feature-focused)
- Visual hierarchy (headline → benefits → CTA)
