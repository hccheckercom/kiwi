## Bad
```php
data = eval(row[4]) if row[4] else []
```

## Good
```php
import ast
data = ast.literal_eval(row[4]) if row[4] else []
```

## Why
eval() executes arbitrary code. Use ast.literal_eval() or json.loads() for data parsing
