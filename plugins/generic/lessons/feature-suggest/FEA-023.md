**Mô tả:**
- Mỗi page PHẢI có đúng 1 `<h1>` (page title hoặc product name)
- Headings KHÔNG skip levels (h1 → h2 → h3, KHÔNG h1 → h3)
- Card titles trong section h2 phải là h3 (không phải h2 hoặc h4)

**Detect pattern (presence):**
- File có 2+ `<h1>` tags → vi phạm heading hierarchy, SEO penalty

**Per page type:**
```
HOMEPAGE:
  h1: Shop name (visually hidden hoặc in hero)
  h2: Section titles (Flash Sale, Danh mục, Sản phẩm nổi bật)
  h3: Product names in cards

PRODUCT PAGE:
  h1: Product name
  h2: Mô tả, Đánh giá, Sản phẩm liên quan

CATEGORY PAGE:
  h1: Category name
  h2: Filter section (sr-only), Product grid (sr-only)
  h3: Product names in cards
```

**Rules:**
- Don't use headings for styling — use classes
- Section titles are h2, subsections are h3
- Screen reader users navigate by headings — hierarchy must be logical

**Lợi ích:** SEO heading structure, screen reader navigation, Google understands page structure
