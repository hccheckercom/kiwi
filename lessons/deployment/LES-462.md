## Bad

```python
# Deploy toàn bộ theme directory
kiwi_deploy({
  "path": "themes/sfvn",  # Contains demos/ subdirectory
  "remote_path": "/var/www/wp.wezone.vn/demos/sfvn"
})

# Result: /var/www/wp.wezone.vn/demos/sfvn/demos/demo1/
# URL: wp.wezone.vn/demos/sfvn/demos/demo1/ (duplicate "demos")
```

## Good

```python
# Option 1: Deploy only demos/ content
kiwi_deploy({
  "path": "themes/sfvn/demos",
  "remote_path": "/var/www/wp.wezone.vn/demos/sfvn"
})

# Option 2: Flatten after deploy
ssh root@vps "cd /var/www/wp.wezone.vn/demos/sfvn && mv demos/* . && rmdir demos"

# Result: /var/www/wp.wezone.vn/demos/sfvn/demo1/
# URL: wp.wezone.vn/demos/sfvn/demo1/ (clean)
```

## Why

Khi deploy theme có cấu trúc `themes/sfvn/demos/demo1/`, nếu deploy toàn bộ thư mục `themes/sfvn` vào `/var/www/wp.wezone.vn/demos/sfvn/`, sẽ tạo path lặp: `/demos/sfvn/demos/demo1/` thay vì `/demos/sfvn/demo1/`.

**Impact:**
- URL dài và khó nhớ: `wp.wezone.vn/demos/sfvn/demos/demo1/` thay vì `wp.wezone.vn/demos/sfvn/demo1/`
- Confusing cho user và SEO
- Cần manual fix sau mỗi lần deploy

**Solution:**
1. Deploy chỉ nội dung `demos/` subdirectory
2. Hoặc flatten structure sau khi deploy bằng `mv demos/* . && rmdir demos`
3. Hoặc adjust remote_path để compensate cho nested structure
