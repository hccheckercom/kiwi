**Pattern sai:**
```
// Wezone Router: ^shop/?$ → wz_page=shop → TemplateLoader::resolve('shop/archive')
// TemplateLoader tìm: {theme}/wezone-templates/shop/archive.php
// File không tồn tại → resolve() trả get_404_template() → trang /shop hiển thị 404
// (Tạo shop.php ở root theme KHÔNG có tác dụng — Router bypass WP template hierarchy)
```
**Phải làm:**
```php
// Tạo đúng file: wezone-templates/shop/archive.php
// KHÔNG phải shop.php ở root theme
// TemplateLoader priority: {theme}/wezone-templates/ → plugin templates/
```
**Lý do:** Wezone dùng custom Router + TemplateLoader, bypass hoàn toàn WP template hierarchy. Mọi template phải nằm trong `wezone-templates/` theo đúng path mà Router resolve.
**Grep check:** `wezone-templates/shop/archive\.php` — thiếu = CRITICAL (mọi theme phải có)
