**Pattern sai:**
```php
// file account/orders.php — không có auth check
$orders = wz_get_user_orders( get_current_user_id() );
```
**Phải làm:**
```php
if ( ! is_user_logged_in() ) {
    wp_redirect( wz_get_login_url() );
    exit;
}
```
**Lý do:** User chưa đăng nhập có thể truy cập trang account — IDOR/data leak
**Grep check:** `is_user_logged_in` — thiếu trong bất kỳ file `wezone-templates/account/*.php` → CRITICAL
