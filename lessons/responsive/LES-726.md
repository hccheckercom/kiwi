## Bad
```php
<aside class="w-64 border-r p-4">
  <h3 class="font-bold mb-4">Bộ lọc</h3>
  <?php // filter options ?>
</aside>
```
Sidebar luôn hiện — trên mobile chiếm hết viewport width.

## Good
```php
<aside class="hidden lg:block w-64 border-r p-4">
  <h3 class="font-bold mb-4">Bộ lọc</h3>
  <?php // filter options ?>
</aside>
```

## Why
Sidebar filter trên mobile phải hidden, dùng drawer/bottom-sheet thay thế. Inline sidebar trên 375px viewport = unusable.
