**Pattern sai:**
```php
<div class="grid grid-cols-4 gap-4">
    <?php foreach ( $related as $product ) : ?>
        <!-- all items in static grid — breaks when > 4 -->
    <?php endforeach; ?>
</div>
```

**Phải làm:**
```php
<?php if ( count( $related ) > 4 ) : ?>
<div class="flex overflow-x-auto scroll-snap-x gap-4 pb-4">
<?php else : ?>
<div class="grid grid-cols-2 md:grid-cols-4 gap-4">
<?php endif; ?>
    <?php foreach ( $related as $product ) : ?>
        <!-- slider when > 4, grid when <= 4 -->
    <?php endforeach; ?>
</div>
```

**Lý do:** Blueprint: "Related products: Slider if > 4 items". Static grid với > 4 items sẽ tràn hoặc phải wrap xuống dòng mới — UX kém.
