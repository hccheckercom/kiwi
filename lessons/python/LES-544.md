## Bad
```php
# Global singleton instance
_checkpoint_manager = CheckpointManager()

def create_checkpoint(agent_run_id: int, message: str, options: list[dict]) -> Checkpoint:
    return _checkpoint_manager.create_checkpoint(agent_run_id, message, options)

# Problem: Global mutable state makes testing hard, not thread-safe
```

## Good
```php
from contextvars import ContextVar

_checkpoint_manager: ContextVar[CheckpointManager] = ContextVar('checkpoint_manager')

def get_manager() -> CheckpointManager:
    manager = _checkpoint_manager.get(None)
    if manager is None:
        manager = CheckpointManager()
        _checkpoint_manager.set(manager)
    return manager

def create_checkpoint(agent_run_id: int, message: str, options: list[dict]) -> Checkpoint:
    return get_manager().create_checkpoint(agent_run_id, message, options)
```

## Why
Global mutable singleton (_manager = Manager()) is not thread-safe in async contexts and makes testing difficult (tests share state). Use ContextVar for async-safe context-local state, or dependency injection for testability.
