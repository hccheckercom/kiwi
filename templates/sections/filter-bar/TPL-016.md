# Archive Filter Bar — Dropdown Filters + Sort

Horizontal filter bar with dropdown filters for product attributes (brand, category) and sort options. Mobile-friendly with touch-optimized dropdowns.

## Features
- Dynamic taxonomy filters (pa_brand, pa_category, etc.)
- Sort dropdown (price, popularity, date)
- Active state highlighting
- URL-based filtering (query params)
- Sanitized input handling
- Material Icons for dropdown arrows

## Code

```php
<?php
/**
 * Template Part: Archive — Filter Bar (brand filter + sort).
 */

$filter_attrs = [
    'pa_brand' => __( 'Thương hiệu', 'wezone-haven' ),
];

$sort_options = [
    ''           => __( 'Mặc định', 'wezone-haven' ),
    'price'      => __( 'Giá thấp đến cao', 'wezone-haven' ),
    'price-desc' => __( 'Giá cao đến thấp', 'wezone-haven' ),
    'popularity' => __( 'Bán chạy', 'wezone-haven' ),
    'date'       => __( 'Mới nhất', 'wezone-haven' ),
];

// phpcs:ignore WordPress.Security.NonceVerification.Recommended -- Public archive filter, no state change.
$current_orderby = isset( $_GET['orderby'] ) ? sanitize_text_field( wp_unslash( $_GET['orderby'] ) ) : '';
?>

<section class="wz-filter-bar">
    <?php
    foreach ( $filter_attrs as $attr_slug => $attr_label ) :
        $attr_key = str_replace( 'pa_', '', $attr_slug );
        $taxonomy = 'pa_' . $attr_key;
        $terms    = get_terms(
            [
                'taxonomy'   => $taxonomy,
                'hide_empty' => true,
            ]
        );
        // phpcs:ignore WordPress.Security.NonceVerification.Recommended
        $active_val = isset( $_GET[ 'filter_' . $attr_key ] )
            ? sanitize_text_field( wp_unslash( $_GET[ 'filter_' . $attr_key ] ) )
            : '';

        if ( is_wp_error( $terms ) || empty( $terms ) ) {
            continue;
        }

        $active_label = $attr_label;
        if ( $active_val ) {
            foreach ( $terms as $t ) {
                if ( $t->slug === $active_val ) {
                    $active_label = $t->name;
                    break;
                }
            }
        }

        $trigger_class = 'wz-filter-dropdown__trigger' . ( $active_val ? ' wz-filter-dropdown__trigger--active' : '' );
        ?>
        <div class="wz-filter-dropdown" data-filter="<?php echo esc_attr( $attr_slug ); ?>">
            <button type="button" class="<?php echo esc_attr( $trigger_class ); ?>" aria-expanded="false">
                <span class="wz-filter-dropdown__label"><?php echo esc_html( $attr_label ); ?>:</span>
                <span class="wz-filter-dropdown__value"><?php echo esc_html( $active_label ); ?></span>
                <span aria-hidden="true" class="material-symbols-outlined wz-filter-dropdown__icon">expand_more</span>
            </button>
            <div class="wz-filter-dropdown__panel">
                <a href="<?php echo esc_url( remove_query_arg( 'filter_' . $attr_key ) ); ?>"
                    class="wz-filter-dropdown__option<?php echo ! $active_val ? ' wz-filter-dropdown__option--active' : ''; ?>">
                    <?php esc_html_e( 'Tất cả', 'wezone-haven' ); ?>
                </a>
                <?php foreach ( $terms as $term ) : ?>
                    <a href="<?php echo esc_url( add_query_arg( 'filter_' . $attr_key, $term->slug ) ); ?>"
                        class="wz-filter-dropdown__option<?php echo ( $active_val === $term->slug ) ? ' wz-filter-dropdown__option--active' : ''; ?>">
                        <?php echo esc_html( $term->name ); ?>
                    </a>
                <?php endforeach; ?>
            </div>
        </div>
    <?php endforeach; ?>

    <?php $orderby_trigger_class = 'wz-filter-dropdown__trigger' . ( $current_orderby ? ' wz-filter-dropdown__trigger--active' : '' ); ?>
    <div class="wz-filter-dropdown" data-filter="orderby">
        <button type="button" class="<?php echo esc_attr( $orderby_trigger_class ); ?>" aria-expanded="false">
            <span class="wz-filter-dropdown__label"><?php esc_html_e( 'Sắp xếp:', 'wezone-haven' ); ?></span>
            <span class="wz-filter-dropdown__value">
                <?php echo esc_html( isset( $sort_options[ $current_orderby ] ) ? $sort_options[ $current_orderby ] : $sort_options[''] ); ?>
            </span>
            <span aria-hidden="true" class="material-symbols-outlined wz-filter-dropdown__icon">expand_more</span>
        </button>
        <div class="wz-filter-dropdown__panel">
            <?php foreach ( $sort_options as $value => $label ) : ?>
                <a href="<?php echo esc_url( $value ? add_query_arg( 'orderby', $value ) : remove_query_arg( 'orderby' ) ); ?>"
                    class="wz-filter-dropdown__option<?php echo ( $current_orderby === $value ) ? ' wz-filter-dropdown__option--active' : ''; ?>">
                    <?php echo esc_html( $label ); ?>
                </a>
            <?php endforeach; ?>
        </div>
    </section>
```

## Usage

```php
// In archive-wz_product.php or category template
get_template_part( 'template-parts/archive/filter-bar' );
```

## CSS Variables Used
- `--wz-primary` — active filter highlight
- `--wz-surface-pure` — dropdown panel background
- `--wz-outline` — borders and inactive text
- `--wz-on-surface` — text color

## JavaScript Required
Dropdown toggle logic (click button → toggle panel visibility, close others).
