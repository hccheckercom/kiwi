# Checkout Breadcrumb — Simple Navigation Trail

Simple text-based breadcrumb for checkout flow showing: Home → Cart → Checkout.

## Features
- Text-only breadcrumb (no icons)
- Proper ARIA labels
- Hover underline on links
- Current page marked with aria-current

## Code

```php
<?php
/**
 * Checkout breadcrumb.
 */
$cart_url = function_exists( 'wz_get_cart_url' ) ? wz_get_cart_url() : home_url( '/cart/' );
?>
<nav class="text-sm text-[var(--wz-text-muted)] mb-4" aria-label="Breadcrumb">
    <a href="<?php echo esc_url( home_url( '/' ) ); ?>" class="hover:underline"><?php esc_html_e( 'Trang chủ', 'wezone-haven' ); ?></a>
    <span class="mx-1">/</span>
    <a href="<?php echo esc_url( $cart_url ); ?>" class="hover:underline"><?php esc_html_e( 'Giỏ hàng', 'wezone-haven' ); ?></a>
    <span class="mx-1">/</span>
    <span aria-current="page"><?php esc_html_e( 'Thanh toán', 'wezone-haven' ); ?></span>
</nav>
```

## Usage

```php
// In wezone-templates/checkout/form.php
get_template_part( 'template-parts/checkout/breadcrumb' );
```

## CSS Variables Used
- `--wz-text-muted` — breadcrumb text color

## Variations
- Cart breadcrumb: Home → Cart (remove checkout step)
- Product breadcrumb: Home → Category → Product
