## Bad
```php
$content = file_get_contents( $file_path );
if ( $content ) {
    $data = json_decode( $content, true );
}
```

## Good
```php
if ( ! file_exists( $file_path ) || ! is_readable( $file_path ) ) {
    return new WP_Error( 'file_not_found', 'File not found or not readable' );
}

$size = filesize( $file_path );
if ( $size > 10 * 1024 * 1024 ) { // 10MB limit
    return new WP_Error( 'file_too_large', 'File exceeds 10MB limit' );
}

$content = file_get_contents( $file_path );
if ( false === $content ) {
    return new WP_Error( 'read_failed', 'Failed to read file' );
}
```

## Why
file_get_contents on dynamic paths can cause memory exhaustion (large files) or path traversal. Check file_exists, is_readable, filesize before reading.

## Grep
`file_get_contents\(\s*\$[a-zA-Z_]` — in `**/*.php`
