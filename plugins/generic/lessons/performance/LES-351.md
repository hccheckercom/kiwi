## Bad
```php
private function format_review( \WP_Comment $comment ): array {
    $rating   = (int) get_comment_meta( $comment->comment_ID, 'rating', true );     // query 1
    $verified = (bool) get_comment_meta( $comment->comment_ID, 'verified', true );   // query 2
    $post     = get_post( $comment->comment_post_ID );                                // query 3
    // 20 reviews = 60 queries
```

## Good
```php
// Batch load all comment meta + posts before formatting
$comment_ids = wp_list_pluck( $comments, 'comment_ID' );
$post_ids    = array_unique( wp_list_pluck( $comments, 'comment_post_ID' ) );
update_meta_cache( 'comment', $comment_ids );
_prime_post_caches( $post_ids, false, false );
// Now format_review uses cached data — 0 extra queries
```

## Why
`format_review` fires 3 queries per review. At 20 reviews per page, that's 60 extra queries. Use `update_meta_cache` and `_prime_post_caches` to batch-load before the loop.
