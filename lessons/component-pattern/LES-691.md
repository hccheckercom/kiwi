## Bad
```tsx
// ProductCard.tsx
<button className="bg-blue-600 text-white rounded-md px-4 py-2">Add</button>

// CartButton.tsx
<button className="bg-blue-600 text-white rounded-lg px-4 py-2">Checkout</button>
```
Cùng section nhưng button border-radius khác nhau.

## Good
```tsx
// Thống nhất rounded-lg cho tất cả buttons
<button className="bg-blue-600 text-white rounded-lg px-4 py-2">Add</button>
<button className="bg-blue-600 text-white rounded-lg px-4 py-2">Checkout</button>
```

## Why
Button styling phải consistent trong cùng section/page. Mixed border-radius (md vs lg) khiến UI trông unprofessional và thiếu design system.
