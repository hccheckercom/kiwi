## Bad
```php
response = requests.get(url)
```

## Good
```php
response = requests.get(url, timeout=30)
```

## Why
HTTP requests without timeout can hang indefinitely if server doesn't respond

`requests\.(get|post|put|delete)\([^)]*\)` — in `**/*.py`
