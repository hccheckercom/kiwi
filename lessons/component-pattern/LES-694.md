## Bad
```tsx
// ProductCard.tsx — price dùng text-sm
<span className="text-sm font-bold">199.000đ</span>

// RelatedProduct.tsx — price dùng text-base
<span className="text-base font-bold">199.000đ</span>
```

## Good
```tsx
// Consistent: price luôn dùng text-sm font-bold
<span className="text-sm font-bold">199.000đ</span>
```

## Why
Cùng loại content (price, title, description) phải dùng cùng text size. Inconsistency khiến user khó scan page — typography scale là foundation của visual hierarchy.
