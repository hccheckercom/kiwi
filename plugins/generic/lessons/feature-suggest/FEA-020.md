**Mô tả:**
- Icon-only buttons PHẢI có `aria-label` (screen readers cần text)
- Cart badge dùng `aria-live="polite"` để announce updates
- Modal/drawer dùng `aria-modal="true"` + focus trap
- Touch targets minimum 44×44px

**Detect pattern (absence):**
- Templates thiếu `aria-label` → screen readers đọc "button" không có context

**Patterns cần có:**
```html
<!-- Cart badge -->
<span class="cart-badge" aria-live="polite" aria-atomic="true" 
      aria-label="Giỏ hàng: {count} sản phẩm">{count}</span>

<!-- Icon button -->
<button aria-label="Tìm kiếm"><svg>...</svg></button>

<!-- Breadcrumb -->
<nav aria-label="Breadcrumb">...</nav>

<!-- Modal -->
<div role="dialog" aria-modal="true" aria-labelledby="modal-title">...</div>
```

**Touch target CSS:**
```css
.icon-button { position: relative; width: 24px; height: 24px; }
.icon-button::before { content: ''; position: absolute; inset: -10px; }
```

**Lợi ích:** WCAG 2.1 AA, accessible cho screen readers + mobile users
