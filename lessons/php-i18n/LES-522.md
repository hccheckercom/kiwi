## Bad
```php
return new WP_Error( 'invalid_input', 'Invalid product ID', [ 'status' => 400 ] );
```

## Good
```php
return new WP_Error( 'invalid_input', __( 'Invalid product ID', 'wezone-core' ), [ 'status' => 400 ] );
```

## Why
Hardcoded error messages prevent translation. Wrap all user-facing strings with __() or esc_html__() for i18n support.

## Grep
`return new WP_Error\([^)]*'[^']*'[^)]*\)` — in `**/*.php`
