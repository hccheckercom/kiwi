## Bad
```php
return new WP_REST_Response([
    'user' => $user,  // contains password hash, email, tokens
]);
```

## Good
```php
return new WP_REST_Response([
    'user' => [
        'id' => $user->ID,
        'name' => $user->display_name,
    ],
]);
```

## Why
REST response leaking sensitive data — password hashes, tokens, secrets exposed to frontend.
