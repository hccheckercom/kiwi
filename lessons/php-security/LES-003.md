**Pattern sai:**
```php
<form method="post" action="">
    <!-- các input -->
    <button type="submit">Lưu</button>
</form>
```
**Phải làm:**
```php
<form method="post" action="">
    <?php wp_nonce_field( 'wezone_save_address', 'wezone_address_nonce' ); ?>
    <!-- các input -->
    <button type="submit">Lưu</button>
</form>
```
**Lý do:** CSRF — attacker có thể submit form thay user
**Grep check:** `wp_nonce_field|wp_verify_nonce|check_ajax_referer` — thiếu trong form có method="post" → CRITICAL
