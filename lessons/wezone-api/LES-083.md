## Bad
```php
if ( $response_amount == $order_total ) { // float comparison
    mark_paid();
}
```

## Good
```php
if ( abs($response_amount - $order_total) < 0.01 ) {
    mark_paid();
}
```

## Why
Float comparison with == — 99999.99 vs 99999.990000001 fails. Use epsilon comparison for money.
