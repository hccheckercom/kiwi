## Bad
```php
mod = __import__(user_input)
```

## Good
```php
import importlib; mod = importlib.import_module(name)
```

## Why
__import__ with dynamic input enables arbitrary module loading; importlib is safer and more explicit

## Grep
`__import__\s*\(` — in `**/*.py`
