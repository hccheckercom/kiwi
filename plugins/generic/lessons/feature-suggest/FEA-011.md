**Mô tả:**
- `<a href="#main-content" class="skip-link">` là element focusable đầu tiên trong `<body>`
- Ẩn visually nhưng hiện khi focus (Tab đầu tiên)
- Link đến `<main id="main-content">` — cho phép keyboard users skip navigation

**Detect pattern (absence):**
- `header.php` thiếu `skip-link` hoặc `#main-content` → keyboard users phải Tab qua toàn bộ nav

**Files cần có:**
- `header.php` — `<a href="#main-content" class="skip-link">Bỏ qua đến nội dung chính</a>` ngay sau `<body>`
- `assets/css/main.css` — `.skip-link` positioned absolute, visible on `:focus`

**CSS mẫu:**
```css
.skip-link {
  position: absolute;
  top: -100%;
  left: var(--space-4);
  padding: var(--space-2) var(--space-4);
  background: var(--color-primary);
  color: var(--color-text-inverse);
  z-index: 9999;
}
.skip-link:focus { top: var(--space-2); }
```

**Lợi ích:** WCAG 2.1 AA compliance, keyboard users navigate nhanh hơn
