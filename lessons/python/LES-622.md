## Bad

```python
config_path = base_dir + "\\config\\settings.json"
log_file = "D:\\logs\\kiwi.log"
rel = filepath.replace("/", "\\")
```

## Good

```python
config_path = Path(base_dir) / "config" / "settings.json"
log_file = Path("D:/logs/kiwi.log")
rel = Path(filepath).as_posix()
```

## Why

Hardcoded `\\` path separators break on Linux/macOS. Use `pathlib.Path` with `/` operator or `os.path.join()` for cross-platform compatibility. Even on Windows-only code, `Path` is more readable and handles edge cases (trailing slashes, double separators).
