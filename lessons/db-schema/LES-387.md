## Bad
```php
'slug' => 'varchar(200) NOT NULL',
// indexes:
'idx_slug' => 'slug(100)',  // prefix index, not unique
```

## Good
```php
'slug' => 'varchar(200) NOT NULL',
// indexes:
'UNIQUE idx_slug' => 'slug(100)',
```

## Why
Slugs are used in URLs for routing. Duplicate slugs cause wrong entity loaded (e.g. two tags with same slug — only one shows). UNIQUE constraint prevents data corruption at DB level.
