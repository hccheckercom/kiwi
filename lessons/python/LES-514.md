# Bad Code

```python
def add_item(item, items=[]):
    items.append(item)
    return items

# BUG: Default list is shared across all calls
result1 = add_item(1)  # [1]
result2 = add_item(2)  # [1, 2] - WRONG! Expected [2]
result3 = add_item(3)  # [1, 2, 3] - WRONG! Expected [3]

def create_user(name, roles={}):
    roles['user'] = True
    return {'name': name, 'roles': roles}

# BUG: Default dict is shared
user1 = create_user('Alice')
user2 = create_user('Bob')
# Both users share same roles dict!
```

# Good Code

```python
def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items

result1 = add_item(1)  # [1]
result2 = add_item(2)  # [2] - Correct!
result3 = add_item(3)  # [3] - Correct!

def create_user(name, roles=None):
    if roles is None:
        roles = {}
    roles['user'] = True
    return {'name': name, 'roles': roles}

# Alternative: Use dataclasses with default_factory
from dataclasses import dataclass, field

@dataclass
class User:
    name: str
    roles: dict = field(default_factory=dict)
```

# Why

**Python evaluates default arguments ONCE at function definition time:**
- Mutable defaults (list, dict, set) are shared across all calls
- Each call mutates the same object
- Causes subtle, hard-to-debug bugs

**Real-world impact:**
```python
# Cache implementation - WRONG
def get_cached(key, cache={}):
    if key not in cache:
        cache[key] = expensive_operation(key)
    return cache[key]

# All callers share same cache dict!
# Can leak data between requests in web apps
```

**Detection:**
- Linters catch this: `pylint`, `flake8`, `ruff`
- Type checkers warn: `mypy`, `pyright`
- Tests fail: Assertions fail on second call

**Safe default values:**
- Immutable: `None`, `0`, `""`, `True`, `False`, `()`
- Use `None` and create mutable inside function

**When mutable defaults are OK:**
- Intentional caching (document it!)
- Sentinel values (rare, advanced)

# Grep

```bash
# Find functions with list default
rg 'def\s+\w+\([^)]*=\s*\[\]' --type py

# Find functions with dict default
rg 'def\s+\w+\([^)]*=\s*\{\}' --type py

# Find functions with set default
rg 'def\s+\w+\([^)]*=\s*set\(\)' --type py
```
