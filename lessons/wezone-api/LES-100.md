## Bad
```php
$url = 'https://api.ghn.vn/shiip/public-api/v2/shipping-order/fee';
```

## Good
```php
$url = wz_config('shipping.api_url', 'https://api.ghn.vn/shiip/public-api/v2');
$url .= '/shipping-order/fee';
```

## Why
Shipping API URL hardcoded — can't switch between sandbox and production environments.
