## Bad
```tsx
// app/admin/prompts/PromptFormPage.tsx
<div className="mx-auto max-w-4xl pb-16 md:pb-0">
```
Parent page (prompts/page.tsx) is fullwidth, but sub-page uses max-w-4xl — inconsistent.

## Good
```tsx
// app/admin/prompts/PromptFormPage.tsx
<div className="pb-16 md:pb-0">
```
Sub-page matches parent layout — no width constraint.

## Why
Admin pages trong cùng section phải consistent layout width. Nếu list page fullwidth thì form/detail page cũng phải fullwidth. Inconsistency gây UX kém — user thấy content "nhảy" khi navigate giữa pages.
