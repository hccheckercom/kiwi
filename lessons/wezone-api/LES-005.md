**Pattern sai:**
```php
function wz_get_cart_url(): string {
    return get_permalink( get_option('wz_cart_page_id') );
}
```
**Phải làm:**
```php
if ( ! function_exists( 'wz_get_cart_url' ) ) {
    function wz_get_cart_url(): string {
        return get_permalink( get_option('wz_cart_page_id') );
    }
}
```
**Lý do:** PHP fatal "Cannot redeclare function" nếu wezone-core cũng khai báo hàm này
**Grep check:** `^function wz_` — hàm khai báo global không có `if (!function_exists)` guard → HIGH
