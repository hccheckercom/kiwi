## Bad
```php
if (!$order || (int)($order['user_id'] ?? 0) !== get_current_user_id()) {
    wp_safe_redirect(home_url('/my-account/orders/'));
    exit;
}
```

## Good
```php
if (!$order || (int)($order['customer_id'] ?? $order['user_id'] ?? 0) !== get_current_user_id()) {
    wp_safe_redirect(home_url('/my-account/orders/'));
    exit;
}
```

## Why
`OrderEngine::get()` trả raw DB row với key `customer_id`. Template check `user_id` → key không tồn tại → ownership luôn fail → redirect loop.

## Grep
`$order['user_id']` trong order-detail.php — nếu không có fallback `customer_id` → CRITICAL
