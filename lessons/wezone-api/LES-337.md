## Bad
```php
// WZ_Product.php — has getImages(), getAttributes(), getVariations(), getFeatured()
// WZ_ProductRelations.php — has IDENTICAL getImages(), getAttributes(), getVariations(), getFeatured()
// Any bug fix must be applied to BOTH files
```

## Good
```php
// Option 1: WZ_ProductRelations delegates to WZ_Product
class WZ_ProductRelations {
    private WZ_Product $product;

    public function __construct() {
        $this->product = wz_adapter()->product();
    }

    public function getImages( int $id ): array {
        return $this->product->getImages( $id );
    }
}

// Option 2: Extract shared trait
trait ProductReadTrait {
    abstract protected function getRepo(): ProductRepository;
    // shared methods here
}
```

## Why
WZ_Product and WZ_ProductRelations have 4 identical methods (~80 lines duplicated). BUG-122 fix had to be applied to both. Future changes will diverge silently if not deduplicated.
