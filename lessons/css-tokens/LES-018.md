**Pattern sai:**
```html
<!-- Mobile: grid cố định, không scroll được -->
<div class="grid grid-cols-2 gap-3">
    <?php foreach ( $items as $item ) : ?>
        <div>...</div>
    <?php endforeach; ?>
</div>
```
**Phải làm:**
```html
<!-- Mobile: scroll ngang, desktop mới grid -->
<div class="flex gap-3 overflow-x-auto hide-scrollbar md:grid md:grid-cols-4 md:overflow-visible">
    <?php foreach ( $items as $item ) : ?>
        <div class="min-w-[155px] md:min-w-0 shrink-0 md:shrink">...</div>
    <?php endforeach; ?>
</div>
```
**Lý do:** Mobile 390px không đủ chỗ cho grid 2+ cột product cards. Demo spec dùng horizontal scroll (swipeable) trên mobile, chỉ chuyển grid ở md: breakpoint. Thiếu `min-w-*` + `shrink-0` → card bị co lại thay vì scroll.
**Grep check:** `grid-cols-[234]` trong section product/category mà thiếu `md:grid` prefix hoặc thiếu `overflow-x-auto` → HIGH
