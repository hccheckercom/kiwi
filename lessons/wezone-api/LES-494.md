## Bad
```php
class Plugin {
    public function boot(): void {
        if ( function_exists( 'wz_config' ) ) {
            wz_config( 'wezone-icons' );
        }
        $this->register_hooks();
    }
}
```

## Good
```php
class Plugin {
    public function boot(): void {
        if ( ! function_exists( 'wezone_is_active' ) || ! wezone_is_active() ) {
            return;
        }
        if ( function_exists( 'wz_config' ) ) {
            wz_config( 'wezone-icons' );
        }
        $this->register_hooks();
    }
}
```

## Why
Plugins should not initialize if Wezone Core is inactive or in maintenance mode. Without wezone_is_active() check, plugin registers hooks/REST routes even when system is disabled â†’ unexpected behavior during maintenance. Always guard boot() with wezone_is_active() check.
