## Pattern

Developer dùng WooCommerce REST API endpoints hoặc extend WC REST controllers.

## Bad

```php
// PHP — registering/extending WC REST
class My_API extends WC_REST_Controller {
    protected $namespace = 'wc/v3';
    protected $rest_base = 'custom';
}

// PHP — calling WC REST internally
$url = rest_url( 'wc/v3/products' );
$response = wp_remote_get( $url, array(
    'headers' => array(
        'Authorization' => 'Basic ' . base64_encode( $consumer_key . ':' . $consumer_secret ),
    ),
) );

// JavaScript — calling WC REST
fetch('/wp-json/wc/v3/products?per_page=20', {
    headers: {
        'X-WC-Consumer-Key': consumerKey,
        'X-WC-Consumer-Secret': consumerSecret,
    }
});

// JavaScript — WooCommerce Store API (Blocks)
fetch('/wp-json/wc/store/v1/cart');
fetch('/wp-json/wc/store/v1/checkout');
```

## Good

```php
// PHP — registering Wezone REST
class My_API extends WP_REST_Controller {
    protected $namespace = 'wz/v1';
    protected $rest_base = 'custom';

    public function register_routes(): void {
        register_rest_route( $this->namespace, '/' . $this->rest_base, array(
            'methods'             => 'GET',
            'callback'            => array( $this, 'get_items' ),
            'permission_callback' => array( $this, 'check_permissions' ),
        ) );
    }
}

// PHP — calling Wezone REST internally
$url = rest_url( 'wz/v1/products' );
$response = wp_remote_get( $url, array(
    'headers' => array(
        'X-WP-Nonce' => wp_create_nonce( 'wp_rest' ),
    ),
) );

// JavaScript — calling Wezone REST
fetch('/wp-json/wz/v1/products?per_page=20', {
    headers: {
        'X-WP-Nonce': wzData.nonce,
    }
});

// JavaScript — Wezone Store API
fetch('/wp-json/wz/v1/cart', {
    headers: { 'X-WP-Nonce': wzData.nonce }
});
```

## Why

- WooCommerce REST API (`/wc/v3/`) không được register → 404 response
- `WC_REST_Controller` class không tồn tại → fatal error khi extend
- Consumer key/secret authentication là WooCommerce-specific → không hoạt động
- Wezone dùng standard WordPress REST API với nonce authentication
- Namespace: `wz/v1` thay vì `wc/v3`
