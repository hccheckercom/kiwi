**Pattern sai:**
```php
<?php foreach ( $reviews as $review ) : ?>
    <!-- render ALL reviews at once -->
<?php endforeach; ?>
```

**Phải làm:**
```php
<?php
$per_page = 5;
$visible_reviews = array_slice( $reviews, 0, $per_page );
foreach ( $visible_reviews as $review ) : ?>
    <!-- render first 5 -->
<?php endforeach; ?>

<?php if ( count( $reviews ) > $per_page ) : ?>
<button class="btn-load-more" data-page="1" data-per-page="<?php echo $per_page; ?>">
    Xem thêm đánh giá (<?php echo count( $reviews ) - $per_page; ?>)
</button>
<?php endif; ?>
```

**Lý do:** Blueprint: "Reviews: Load more (5 per batch)". Render 50+ reviews cùng lúc → DOM bloat, slow paint, bad UX trên mobile.
