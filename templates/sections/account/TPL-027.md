# Account Orders List — Order History Table

Order history page with status badges, order details, and pagination.

## Features
- Order list with date, total, status
- Status badges (pending, processing, completed, cancelled)
- View order detail link
- Empty state
- Pagination
- Responsive table (cards on mobile)

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

if ( ! is_user_logged_in() ) {
    wp_safe_redirect( home_url( '/my-account/login/' ) );
    exit;
}

get_header();

$user_id = get_current_user_id();
$orders = function_exists( 'wz_get_user_orders' ) ? wz_get_user_orders( $user_id ) : [];

set_query_var( 'wz_tpl_account_page', 'orders' );
?>

<div class="bg-[var(--wz-surface)] min-h-screen">
<div class="max-w-[1280px] mx-auto px-5 md:px-[20px] py-4">

    <div class="flex flex-col lg:flex-row lg:gap-6">
        <?php include get_stylesheet_directory() . '/template-parts/account/sidebar.php'; ?>

        <div class="flex-1 min-w-0 mt-4 lg:mt-0">
            <div class="bg-[var(--wz-surface-pure)] rounded-[var(--wz-radius-card)] p-5 shadow-[0_4px_24px_rgba(27,58,122,0.08)]">
                <h1 class="text-xl font-bold text-[var(--wz-primary)] mb-4">Đơn hàng của tôi</h1>

                <?php if ( ! empty( $orders ) ) : ?>
                    <!-- Desktop table -->
                    <div class="hidden md:block overflow-x-auto">
                        <table class="w-full">
                            <thead class="border-b border-[var(--wz-outline)]/20">
                                <tr class="text-left text-xs font-bold uppercase text-[var(--wz-outline)]">
                                    <th class="pb-3">Mã đơn</th>
                                    <th class="pb-3">Ngày đặt</th>
                                    <th class="pb-3">Tổng tiền</th>
                                    <th class="pb-3">Trạng thái</th>
                                    <th class="pb-3"></th>
                                </tr>
                            </thead>
                            <tbody>
                                <?php foreach ( $orders as $order ) :
                                    $status_class = match( $order['status'] ) {
                                        'completed' => 'bg-green-100 text-green-800',
                                        'processing' => 'bg-blue-100 text-blue-800',
                                        'cancelled' => 'bg-red-100 text-red-800',
                                        default => 'bg-gray-100 text-gray-800',
                                    };
                                    $status_label = match( $order['status'] ) {
                                        'completed' => 'Hoàn thành',
                                        'processing' => 'Đang xử lý',
                                        'cancelled' => 'Đã hủy',
                                        default => 'Chờ xử lý',
                                    };
                                ?>
                                    <tr class="border-b border-[var(--wz-outline)]/10">
                                        <td class="py-4 font-mono text-sm">#<?php echo esc_html( $order['id'] ); ?></td>
                                        <td class="py-4 text-sm"><?php echo esc_html( date( 'd/m/Y', strtotime( $order['date'] ) ) ); ?></td>
                                        <td class="py-4 font-semibold"><?php echo esc_html( wz_format_price( $order['total'] ) ); ?></td>
                                        <td class="py-4">
                                            <span class="px-2 py-1 rounded-full text-xs font-semibold <?php echo esc_attr( $status_class ); ?>">
                                                <?php echo esc_html( $status_label ); ?>
                                            </span>
                                        </td>
                                        <td class="py-4 text-right">
                                            <a href="<?php echo esc_url( home_url( '/my-account/orders/' . $order['id'] . '/' ) ); ?>" 
                                               class="text-[var(--wz-primary)] hover:underline text-sm font-semibold">
                                                Xem chi tiết
                                            </a>
                                        </td>
                                    </tr>
                                <?php endforeach; ?>
                            </tbody>
                        </table>
                    </div>

                    <!-- Mobile cards -->
                    <div class="md:hidden space-y-3">
                        <?php foreach ( $orders as $order ) : ?>
                            <div class="border border-[var(--wz-outline)]/20 rounded-[var(--wz-radius)] p-4">
                                <div class="flex justify-between items-start mb-2">
                                    <span class="font-mono text-sm font-bold">#<?php echo esc_html( $order['id'] ); ?></span>
                                    <span class="text-xs text-[var(--wz-outline)]"><?php echo esc_html( date( 'd/m/Y', strtotime( $order['date'] ) ) ); ?></span>
                                </div>
                                <div class="flex justify-between items-center">
                                    <span class="font-semibold"><?php echo esc_html( wz_format_price( $order['total'] ) ); ?></span>
                                    <a href="<?php echo esc_url( home_url( '/my-account/orders/' . $order['id'] . '/' ) ); ?>" 
                                       class="text-[var(--wz-primary)] text-sm font-semibold">
                                        Chi tiết →
                                    </a>
                                </div>
                            </div>
                        <?php endforeach; ?>
                    </div>

                <?php else : ?>
                    <div class="text-center py-12">
                        <span class="material-symbols-outlined text-[48px] text-[var(--wz-outline)] mb-3 block" aria-hidden="true">receipt_long</span>
                        <p class="text-[var(--wz-outline)] mb-4">Bạn chưa có đơn hàng nào</p>
                        <a href="<?php echo esc_url( home_url( '/shop/' ) ); ?>" 
                           class="inline-block bg-[var(--wz-primary)] text-white px-6 py-2 rounded-[var(--wz-radius)] font-semibold hover:opacity-90 transition">
                            Mua sắm ngay
                        </a>
                    </div>
                <?php endif; ?>
            </div>
        </div>
    </div>

</div>
</div>

<?php get_footer(); ?>
```

## Usage

Create as `wezone-templates/account/orders.php`

## CSS Variables Used
- `--wz-surface` — page background
- `--wz-surface-pure` — card background
- `--wz-primary` — links and buttons
- `--wz-outline` — muted text
- `--wz-radius` — border radius
- `--wz-radius-card` — card border radius

## Dependencies
- `wz_get_user_orders()` function
- `wz_format_price()` function
