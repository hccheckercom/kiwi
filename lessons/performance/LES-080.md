## Bad
```php
foreach ( $rows as $row ) {
    $wpdb->insert("{$wpdb->prefix}analytics", $row);
}
```

## Good
```php
wz_bulk_insert("{$wpdb->prefix}analytics", $rows, ['%d','%s','%f']);
```

## Why
N+1 DB writes in loop. 100 rows = 100 INSERT queries. Use wz_bulk_insert() for batch operations.
