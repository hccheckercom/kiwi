## Bad
```php
'secret' => "varchar(200) NOT NULL DEFAULT ''",
```

## Good
```php
'secret' => 'varchar(200) NOT NULL',
```

## Why
Webhook secret defaulting to empty string means webhooks created without explicit secret will accept ANY incoming payload without signature verification. Attacker can forge webhook calls. Remove DEFAULT '' to force secret to be set at creation time.
