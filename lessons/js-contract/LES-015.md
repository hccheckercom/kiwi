**Pattern sai:**
```php
<!-- PHP template -->
<form id="wz-login-form" method="post">
    <?php wp_nonce_field('wz_login'); ?>
    <input name="email" type="email" required>
    <button type="submit">Đăng nhập</button>
</form>
```
```js
// main.js — KHÔNG có handler cho #wz-login-form
// Form submit → page reload → 404 hoặc không xử lý
```
