## Pattern

Cart page KHÔNG chứa threshold bar logic:
- `wz_get_threshold_suggestions()` — backend function
- `wz-threshold` — CSS class
- `free.?ship.*progress` — progress bar text
- `shipping.*threshold` — threshold logic

## Why This Matters

Cart threshold bar (progress bar + gợi ý sản phẩm để đạt free ship) tăng 8-12% tỷ lệ khách hàng đạt threshold, tăng AOV trực tiếp.

**Impact:**
- Không có visual feedback → khách không biết còn thiếu bao nhiêu
- Không có gợi ý sản phẩm → khách không biết thêm gì để đạt threshold
- Mất cơ hội upsell tự nhiên nhất trong customer journey

## Bad Example

```php
// page-cart.php — KHÔNG có threshold bar
<main>
    <h1>Giỏ hàng</h1>
    <?php foreach ( $cart_items as $item ) : ?>
        <!-- cart item -->
    <?php endforeach; ?>
    <div class="cart-total">
        Tổng: <?php echo wz_format_price( $cart_total ); ?>
    </div>
</main>
```

## Good Example

```php
// page-cart.php — CÓ threshold bar
<main>
    <h1>Giỏ hàng</h1>
    
    <?php
    // Threshold bar
    get_template_part( 'template-parts/cart/shipping-threshold' );
    ?>
    
    <?php foreach ( $cart_items as $item ) : ?>
        <!-- cart item -->
    <?php endforeach; ?>
</main>
```

```php
// template-parts/cart/shipping-threshold.php
<?php
$cart_total = wz_cart_get_subtotal();
$threshold  = (int) wz_config_get( 'free_shipping_threshold', 500000 );
$remaining  = max( 0, $threshold - $cart_total );
$progress   = min( 100, ( $cart_total / $threshold ) * 100 );
?>

<?php if ( $remaining > 0 ) : ?>
<div class="wz-threshold">
    <p class="wz-threshold__msg">
        Thêm <strong><?php echo wz_format_price( $remaining ); ?></strong> để được
        <span class="wz-threshold__highlight">FREE SHIP</span>
    </p>
    <div class="wz-threshold__bar">
        <div class="wz-threshold__fill" style="width: <?php echo esc_attr( $progress ); ?>%"></div>
    </div>
    <?php
    $suggestions = wz_get_threshold_suggestions( $cart_total, 2 );
    if ( $suggestions ) :
    ?>
    <div class="wz-threshold__suggestions">
        <p>Gợi ý thêm:</p>
        <?php foreach ( $suggestions as $item ) : ?>
        <div class="wz-threshold__item">
            <img src="<?php echo esc_url( $item['image'] ); ?>" width="48" height="48" alt="" />
            <span><?php echo esc_html( $item['name'] ); ?></span>
            <span><?php echo wz_format_price( $item['price'] ); ?></span>
            <button data-add-to-cart="<?php echo esc_attr( $item['id'] ); ?>">+ Thêm</button>
        </div>
        <?php endforeach; ?>
    </div>
    <?php endif; ?>
</div>
<?php else : ?>
<div class="wz-threshold wz-threshold--achieved">
    <p>🎉 Bạn đã được <strong>FREE SHIP</strong>!</p>
</div>
<?php endif; ?>
```

## How to Fix

1. Tạo `template-parts/cart/shipping-threshold.php` với logic trên
2. Thêm `get_template_part( 'template-parts/cart/shipping-threshold' )` vào `page-cart.php`
3. Implement backend function `wz_get_threshold_suggestions()`:

```php
function wz_get_threshold_suggestions( $cart_total, $limit = 2 ) {
    $threshold = (int) wz_config_get( 'free_shipping_threshold', 500000 );
    $remaining = $threshold - $cart_total;
    
    if ( $remaining <= 0 ) return [];
    
    return wz_query_products( [
        'price_min' => $remaining * 0.5,
        'price_max' => $remaining * 1.5,
        'in_stock'  => true,
        'orderby'   => 'sales_count',
        'limit'     => $limit,
    ] );
}
```
