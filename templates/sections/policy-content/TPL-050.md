# Policy Content — GMC-Compliant Policy Page Template

Full-featured policy page template với prose layout, sections, và GMC compliance.

## Use Cases
- Chính sách hoàn tiền (refund policy)
- Chính sách bảo mật (privacy policy)
- Điều khoản sử dụng (terms of service)
- Chính sách vận chuyển (shipping policy)
- Chính sách đổi trả (return policy)

## Features
- 📄 Prose layout (max-w-3xl, readable typography)
- 📋 Numbered sections với H2/H3 hierarchy
- ✅ Lists (ul, ol) cho điều kiện/quy trình
- 📅 Last updated date
- 🔗 Contact info integration từ wz_config()
- GMC-compliant structure

## Data Sources

All data from `wz_config()`:
- `shop_name` — Tên shop
- `email` — Email hỗ trợ
- `phone_display` — SĐT có format
- `address` — Địa chỉ đầy đủ
- `business_hours` — Giờ làm việc (optional, default: "8:00 - 22:00 (Thứ 2 - Chủ nhật)")

## Structure Pattern

```
H1: Policy Title
├─ Last updated date
├─ Section 1: Introduction
├─ Section 2: Main Content
│  ├─ H3: Subsection
│  ├─ Lists (ul/ol)
│  └─ Strong emphasis
├─ Section N-1: Contact Info
└─ Section N: Policy Changes
```

## Dependencies
- Tailwind CSS prose classes
- wz_config() function
- PHP date() function

## Responsive Behavior
- max-w-[1280px] container
- prose prose-lg for readable typography
- Responsive padding (px-10)

## Accessibility
- Semantic HTML (h1, h2, h3, section, ul, ol)
- Strong emphasis for important terms
- Links with hover states
- Readable font sizes (prose-lg)

## GMC Compliance
✅ Specific conditions (not vague)
✅ Clear timeframes (7/14/30 days)
✅ Contact information visible
✅ Last updated date
✅ Refund process explained step-by-step
✅ Exclusions clearly stated

## Customization

### Change prose width

