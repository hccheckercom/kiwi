## Bad
```php
// wz-shims.php chỉ tạo URL, không đăng ký route
'url' => home_url( '/category/' . $row['slug'] . '/' ),
// Nhưng không có add_rewrite_rule cho ^category/([^/]+)/?$
```

## Good
```php
if ( ! class_exists( 'WeZone\\Core\\ThemeEngine\\Router' ) ) {
    add_action( 'init', function () {
        add_rewrite_rule( '^category/([^/]+)/?$', 'index.php?wz_page=category&wz_slug=$matches[1]', 'top' );
        if ( ! get_option( 'wz_shim_category_rewrite_flushed' ) ) {
            flush_rewrite_rules( false );
            update_option( 'wz_shim_category_rewrite_flushed', 1 );
        }
    }, 5 );
}
```

## Why
Theme tạo URL `/category/slug/` nhưng không đảm bảo WordPress có route cho URL đó. Khi wezone-core Router không active (dev, staging), URL trở thành dead link.
