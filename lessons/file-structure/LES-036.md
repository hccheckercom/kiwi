**Required OG tags:**
```
meta name="description"
og:title
og:type
og:url
og:site_name
og:locale (vi_VN)
```
**Pattern sai:**
```php
// inc/seo.php chỉ output meta description, thiếu OG tags
function wezone_theme_seo_meta() {
    echo '<meta name="description" content="...">';
    // thiếu og:title, og:type, og:url, og:site_name, og:locale
}
```
**Phải làm:**
```php
function wezone_theme_seo_meta() {
    $title = wp_get_document_title();
    $url   = esc_url( home_url( $_SERVER['REQUEST_URI'] ?? '/' ) );
    $name  = wz_config( 'store.name', get_bloginfo('name') );
    echo '<meta name="description" content="' . esc_attr( $desc ) . '">';
    echo '<meta property="og:title" content="' . esc_attr( $title ) . '">';
    echo '<meta property="og:type" content="website">';
    echo '<meta property="og:url" content="' . $url . '">';
    echo '<meta property="og:site_name" content="' . esc_attr( $name ) . '">';
    echo '<meta property="og:locale" content="vi_VN">';
}
```
**Lý do:** Share link lên Facebook/Zalo không có OG tags → hiển thị title/image sai hoặc rỗng. Mọi theme ecommerce cần đầy đủ OG.
**Grep check:** `og:title` absent trong output của seo.php → HIGH
