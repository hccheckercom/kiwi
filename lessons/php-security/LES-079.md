## Bad
```php
$wpdb->insert('orders', $data);  // missing prefix
```

## Good
```php
$wpdb->insert("{$wpdb->prefix}orders", $data);
```

## Why
DB operation without $wpdb->prefix — breaks on multisite where tables have different prefixes.
