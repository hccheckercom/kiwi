## Pattern

Developer quen dùng `WC()` singleton để truy cập cart, session, checkout, countries, customer.
Trên Wezone Commer, WooCommerce không được cài — gọi `WC()` sẽ fatal error.

## Bad

```php
// Cart
$count = WC()->cart->get_cart_contents_count();
$total = WC()->cart->get_cart_total();
$items = WC()->cart->get_cart();
WC()->cart->add_to_cart( $product_id, $qty );
WC()->cart->empty_cart();

// Session
$val = WC()->session->get( 'my_key' );
WC()->session->set( 'my_key', $value );

// Checkout
$fields = WC()->checkout()->get_checkout_fields();

// Countries
$countries = WC()->countries->get_countries();
$states    = WC()->countries->get_states( 'VN' );

// Customer
$country = WC()->customer->get_billing_country();

// Payment
$gateways = WC()->payment_gateways()->get_available_payment_gateways();

// Shipping
$methods = WC()->shipping()->get_shipping_methods();
```

## Good

```php
// Cart
$count = wz_get_cart_count();
$total = wz_get_cart_total();
$items = wz_get_cart_items();
wz_add_to_cart( $product_id, $qty );
wz_empty_cart();

// Session
$val = wz_session_get( 'my_key' );
wz_session_set( 'my_key', $value );

// Checkout
$fields = wz_get_checkout_fields();

// Countries
$countries = wz_get_countries();
$states    = wz_get_states( 'VN' );

// Customer
$country = wz_get_customer_billing_country();

// Payment
$gateways = wz_get_payment_gateways();

// Shipping
$methods = wz_get_shipping_methods();
```

## Why

WooCommerce không được cài trên hệ thống Wezone Commer. Gọi `WC()` sẽ gây:
- Fatal error: Call to undefined function WC()
- Hoặc nếu có stub: trả về null → null pointer exception

Mapping đầy đủ WC() → wz_*:
| WC() pattern | Wezone equivalent |
|---|---|
| `WC()->cart->get_cart()` | `wz_get_cart_items()` |
| `WC()->cart->get_cart_contents_count()` | `wz_get_cart_count()` |
| `WC()->cart->get_cart_total()` | `wz_get_cart_total()` |
| `WC()->cart->add_to_cart()` | `wz_add_to_cart()` |
| `WC()->cart->empty_cart()` | `wz_empty_cart()` |
| `WC()->cart->calculate_totals()` | `wz_recalculate_cart()` |
| `WC()->session->get()` | `wz_session_get()` |
| `WC()->session->set()` | `wz_session_set()` |
| `WC()->checkout()` | `wz_checkout()` |
| `WC()->countries->get_countries()` | `wz_get_countries()` |
| `WC()->countries->get_states()` | `wz_get_states()` |
| `WC()->customer` | `wz_get_current_customer()` |
| `WC()->payment_gateways()` | `wz_get_payment_gateways()` |
| `WC()->shipping()` | `wz_get_shipping_methods()` |
| `WC()->mailer()` | `wz_mailer()` |
