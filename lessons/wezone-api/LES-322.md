## Pattern

Admin creates flash sale with start/end time. Backend doesn't validate: (1) end_time > start_time, (2) start_time not in the past, (3) duration reasonable (not 30 days). Invalid time ranges cause: sale never activates, sale runs forever, or frontend shows negative countdown.

## Bug

```php
// SAI — end trước start = sale không bao giờ active
public function create_flash_sale( WP_REST_Request $request ): WP_REST_Response {
    $start = sanitize_text_field( $request->get_param( 'start_time' ) );
    $end   = sanitize_text_field( $request->get_param( 'end_time' ) );

    // Không validate! end < start → frontend hiển thị countdown âm
    $wpdb->insert( $table, array(
        'start_time' => $start,
        'end_time'   => $end,
        'status'     => 'scheduled',
    ) );
}
```

## Fix

```php
public function create_flash_sale( WP_REST_Request $request ): WP_REST_Response {
    $start_str = sanitize_text_field( $request->get_param( 'start_time' ) );
    $end_str   = sanitize_text_field( $request->get_param( 'end_time' ) );

    $start_ts = strtotime( $start_str );
    $end_ts   = strtotime( $end_str );
    $now      = time();

    if ( ! $start_ts || ! $end_ts ) {
        return new WP_REST_Response( array( 'error' => 'Thời gian không hợp lệ' ), 422 );
    }

    if ( $end_ts <= $start_ts ) {
        return new WP_REST_Response( array( 'error' => 'Thời gian kết thúc phải sau thời gian bắt đầu' ), 422 );
    }

    if ( $start_ts < $now - 300 ) { // 5 min grace
        return new WP_REST_Response( array( 'error' => 'Thời gian bắt đầu không được ở quá khứ' ), 422 );
    }

    $max_duration = 7 * DAY_IN_SECONDS;
    if ( ( $end_ts - $start_ts ) > $max_duration ) {
        return new WP_REST_Response( array( 'error' => 'Flash sale không được kéo dài quá 7 ngày' ), 422 );
    }

    // Check overlap with existing active sales
    $overlap = $wpdb->get_var( $wpdb->prepare(
        "SELECT COUNT(*) FROM {$table}
         WHERE status != 'cancelled'
         AND start_time < %s AND end_time > %s",
        $end_str, $start_str
    ) );

    if ( $overlap ) {
        return new WP_REST_Response( array( 'error' => 'Trùng thời gian với flash sale khác đang chạy' ), 409 );
    }

    $wpdb->insert( $table, array(
        'start_time' => gmdate( 'Y-m-d H:i:s', $start_ts ),
        'end_time'   => gmdate( 'Y-m-d H:i:s', $end_ts ),
        'status'     => 'scheduled',
    ) );

    return new WP_REST_Response( array( 'id' => $wpdb->insert_id ), 201 );
}
```

## Áp dụng

Mọi time-range input (flash sale, voucher expiry, campaign schedule) PHẢI validate: (1) end > start, (2) start not past, (3) duration within business limit, (4) no overlap with existing active entries.
