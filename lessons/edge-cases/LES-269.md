**Pattern sai:**
```php
<?php printf( esc_html__( 'Đã bán %s', 'theme' ), number_format( $sold_count ) ); ?>
```

**Phải làm:**
```php
<?php printf( esc_html__( 'Đã bán %s', 'theme' ), wz_abbreviate_number( $sold_count, 'sold' ) ); ?>
```

**Quy tắc abbreviation (từ edge-cases.md):**
- Sold > 9,999 → "10K+"
- Stock > 999 → "999+"
- Reviews > 9,999 → "9.9K+"
- Cart badge > 99 → "99+"
- Price > 999,999,999 → "1.2 tỷ"
