**Pattern sai:**
```php
esc_html_e( 'Tìm kiếm', '{theme-slug}' );
function {theme_slug}_assets(): void {}
wp_enqueue_style( '{theme-slug}-main', ... );
```
**Phải làm:**
```php
esc_html_e( 'Tìm kiếm', '{{theme-slug}}' );
function {{theme_slug}}_assets(): void {}
wp_enqueue_style( '{{theme-slug}}-main', ... );
```
**Lý do:** Blueprint dùng double-brace `{{...}}` để phân biệt placeholder cần thay thế vs text thường. Single-brace không được nhận diện bởi script thay thế khi clone theme.
**Grep check:** `(?<!\{)\{theme[-_][^}]*\}(?!\})` — single-brace placeholder → HIGH
