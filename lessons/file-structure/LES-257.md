**Pattern sai:** Code theme mà KHÔNG đọc `03-MUST-READ.md` trước → dẫn đến:
1. Dùng BEM custom classes (`.haven-header`, `.haven-shop__grid`) thay vì Tailwind utilities
2. Code HTML thủ công cho trust-badges, star-rating thay vì `wz_component()`
3. Gọi optional plugin functions (`wz_wishlist_button()`) mà không guard `wezone_is_active()`
4. Thiếu JSON-LD schema, canonical URL, OG tags (GMC/SEO compliance)

**Phải làm:**
```php
// 1. Dùng wz_component() thay vì code HTML thủ công
wz_component( 'trust-badges' );
wz_component( 'star-rating', [ 'rating' => $product['average_rating'] ?? 0 ] );

// 2. Guard optional plugins
if ( wezone_is_active('wezone-wishlist') ) {
    wz_wishlist_button( $product_id );
}

// 3. Dùng Tailwind utilities, KHÔNG BEM custom class
// ĐÚNG: <div class="grid grid-cols-2 md:grid-cols-4 gap-6">
// SAI:  <div class="haven-shop__grid">

// 4. JSON-LD + canonical trong inc/seo.php
```

**Lý do:** Blueprint rule #13 cấm BEM class (không có CSS → invisible). Rule #15 bắt buộc guard. `wz_component()` system có 16 components sẵn — code lại = duplicate + miss updates. GMC/SEO là bắt buộc trước bàn giao.

**Checklist trước khi code theme:**
1. Đọc `03-MUST-READ.md` — đặc biệt Phần 2 (luật bất biến) + Phần 2D (icons/components)
2. Đọc `_COMPONENT-REFERENCE.md` — biết component nào đã có
3. Đọc `11-GMC-COMPLIANCE.md` — biết SEO requirements
4. Đọc `12-SEO-GEO-COMPLIANCE.md` — JSON-LD + meta tags

**Grep check:** `wz_component\(` absence in template-parts → HIGH
