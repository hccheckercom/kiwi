## Bad
```php
<input type="text" name="billing_name" class="w-full border rounded px-3 py-2" placeholder="Họ tên">
```
Input không có id + label[for=] hoặc aria-label — screen reader không biết field này là gì.

## Good
```php
<label for="billing_name" class="text-sm font-medium">Họ tên</label>
<input type="text" id="billing_name" name="billing_name" class="w-full border rounded px-3 py-2" aria-label="Họ tên">
```

## Why
WCAG 1.3.1 + 4.1.2: form inputs phải có accessible name. Placeholder không phải label — nó biến mất khi user bắt đầu nhập.
