# Hero Product Cards — Admin Configurable

Glassmorphism product cards overlaid on hero banner with full admin control: source, count, card style, position, animation, hover effects, columns, ratio, badges.

## Settings (20+ options via wp_options `wz_banner_hero_products`)

| Setting | Type | Values |
|---------|------|--------|
| enabled | bool | on/off |
| source | enum | latest, featured, sale, best_selling, random, manual |
| count | int | 1-12 |
| card_style | enum | compact, card, minimal, overlay |
| position | enum | right, bottom, left, overlay |
| image_ratio | enum | 1:1, 4:3, 3:4, 16:9 |
| columns_mobile | int | 1-3 |
| columns_desktop | int | 1-6 |
| show_price | bool | |
| show_badge | bool | |
| badge_text | string | |
| show_sale_percent | bool | |
| max_name_lines | int | 1-3 |
| card_bg | color | |
| card_radius | int | 0-32px |
| animation | enum | none, fade-up, fade-in, slide-left, slide-right, zoom-in, bounce, flip |
| animation_delay | int | 0-500ms (between items) |
| animation_duration | int | 100-1500ms |
| hover_effect | enum | none, zoom-in, zoom-out, shine, float, glow, shadow-lift, border-pulse |

## PHP (banner-helpers.php)

```php
function wz_get_hero_product_settings(): array {
    $defaults = array(
        'enabled' => true, 'source' => 'latest', 'count' => 3,
        'category_id' => 0, 'card_style' => 'compact', 'position' => 'right',
        'show_price' => true, 'show_badge' => false, 'badge_text' => '',
        'image_ratio' => '1:1', 'columns_mobile' => 1, 'columns_desktop' => 3,
        'animation' => 'fade-up', 'animation_delay' => 100, 'animation_duration' => 400,
        'hover_effect' => 'zoom-in', 'max_name_lines' => 2,
        'show_sale_percent' => false, 'card_bg' => '', 'card_radius' => 12,
    );
    $saved = get_option( 'wz_banner_hero_products', array() );
    return array_merge( $defaults, is_array( $saved ) ? $saved : array() );
}
```

## Template (hero/default.php — render function)

```php
<div class="haven-hero__products haven-hero__products--<?php echo esc_attr( $card_style ); ?>"
    style="--hp-cols-mobile:<?php echo esc_attr( $cols_mobile ); ?>;--hp-cols-desktop:<?php echo esc_attr( $cols_desktop ); ?>;--hp-img-ratio:<?php echo esc_attr( $ratio_pad ); ?>;">
    <?php foreach ( $products as $idx => $product ) :
        $card_styles = [];
        if ( $card_bg ) $card_styles[] = 'background:' . esc_attr( $card_bg );
        if ( $card_radius ) $card_styles[] = 'border-radius:' . $card_radius . 'px';
        $card_styles[] = '--anim-delay:' . ($anim_delay * $idx) . 'ms';
        $card_styles[] = '--anim-duration:' . $anim_dur . 'ms';
        $card_styles[] = '--hp-line-clamp:' . $max_lines;
    ?>
        <a href="<?php echo esc_url( $product['link'] ); ?>"
            class="haven-hero-card haven-hero-card--<?php echo esc_attr( $card_style ); ?> haven-hero-card--hover-<?php echo esc_attr( $hover_effect ); ?>"
            style="<?php echo implode( ';', $card_styles ); ?>"
            data-anim="<?php echo esc_attr( $animation ); ?>">
            <div class="haven-hero-card__img-wrap">
                <img src="..." alt="..." loading="lazy" />
            </div>
            <div class="haven-hero-card__info">
                <span class="haven-hero-card__name">...</span>
                <span class="haven-hero-card__price">...</span>
            </div>
        </a>
    <?php endforeach; ?>
</div>
```

## JS (IntersectionObserver — progressive enhancement)

```js
var heroCards = document.querySelectorAll('.haven-hero-card[data-anim]');
heroCards.forEach(function (card) {
    if (card.getAttribute('data-anim') !== 'none') {
        card.classList.add('hp-anim-ready');
    }
});
var observer = new IntersectionObserver(function (entries) {
    entries.forEach(function (entry) {
        if (entry.isIntersecting) {
            entry.target.classList.add('is-visible');
            observer.unobserve(entry.target);
        }
    });
}, { threshold: 0.1 });
heroCards.forEach(function (card) {
    if (card.classList.contains('hp-anim-ready')) observer.observe(card);
});
```
