## Bad
```ts
export async function POST(req: Request) {
  const { prompt } = await req.json()
  const image = await generateImage(prompt)
  // Save directly to public storage
  await uploadToS3(image, `generated/${Date.now()}.png`)
  return Response.json({ url: image.url })
}
```
User-provided prompt → AI generates image → served publicly without review. Could generate inappropriate, copyrighted, or harmful content.

## Good
```ts
export async function POST(req: Request) {
  const { prompt } = await req.json()

  // Input moderation
  const isSafe = await moderatePrompt(prompt)
  if (!isSafe) return Response.json({ error: 'Prompt violates content policy' }, { status: 400 })

  const image = await generateImage(prompt)

  // Store as draft, not public
  await supabase.from('generated_images').insert({
    prompt, url: image.url, status: 'pending_review', user_id: userId
  })

  return Response.json({ url: image.url, status: 'pending_review' })
}
```

## Why
AI image generators can produce inappropriate content even from seemingly innocent prompts. Without moderation: (1) inappropriate images on your platform, (2) legal liability, (3) brand damage. Always moderate input prompts AND review outputs before public display.
