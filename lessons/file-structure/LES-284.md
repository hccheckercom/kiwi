## Bad
```php
// Homepage structured data only outputs Store schema — no WebSite + SearchAction
function wezone_theme_structured_data(): void {
    if ( ! is_front_page() ) return;
    $ld = [
        '@context' => 'https://schema.org',
        '@type'    => 'Store',
        'name'     => $shop_name,
        'url'      => $site_url,
    ];
    // Missing WebSite schema entirely
}
```

## Good
```php
function wezone_theme_structured_data(): void {
    if ( ! is_front_page() ) return;

    // Store schema
    $ld_store = [
        '@context' => 'https://schema.org',
        '@type'    => 'Store',
        'name'     => $shop_name,
        'url'      => $site_url,
    ];

    // WebSite + SearchAction — enables Google sitelinks search box
    $ld_website = [
        '@context'        => 'https://schema.org',
        '@type'           => 'WebSite',
        'name'            => $shop_name,
        'url'             => $site_url,
        'potentialAction' => [
            '@type'       => 'SearchAction',
            'target'      => $site_url . '/tim-kiem/?q={search_term_string}',
            'query-input' => 'required name=search_term_string',
        ],
    ];

    printf('<script type="application/ld+json">%s</script>', wp_json_encode($ld_store, JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES));
    printf('<script type="application/ld+json">%s</script>', wp_json_encode($ld_website, JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES));
}
```

## Why
Blueprint `seo.md` requires WebSite + SearchAction schema on homepage. Without it, Google won't show the sitelinks search box in SERPs — a significant CTR loss for branded queries.
