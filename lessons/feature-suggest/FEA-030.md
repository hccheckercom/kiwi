**Mô tả:**
Onboarding blueprint yêu cầu REST API endpoint để wizard gọi khi import demo data.

**Endpoint spec:**
- Route: `POST /wp-json/wezone/v1/import-demo`
- Params: `package` (full | minimal | products-only | settings-only)
- Import order: categories → products → pages → menus → settings
- Permission: `manage_options`
- Response: `{ products: N, categories: N, pages: N, images: N, settings: bool }`

**Phải làm:**
```php
add_action( 'rest_api_init', function () {
    register_rest_route( 'wezone/v1', '/import-demo', [
        'methods'             => 'POST',
        'callback'            => 'wz_handle_import_demo',
        'permission_callback' => function () {
            return current_user_can( 'manage_options' );
        },
    ] );
} );
```

**Lý do:**
Wizard step 3 cần API backend để import data. Thiếu endpoint = wizard không hoạt động.
