**Pattern sai:**
```php
namespace WeZone\Core\ThemeEngine;

class Router {
    public function add_rewrite_rules(): void {
        // flush_rewrite_rules() → PHP tìm trong namespace hiện tại → undefined
        flush_rewrite_rules( false );
    }
}
```
**Phải làm:**
```php
namespace WeZone\Core\ThemeEngine;

class Router {
    public function add_rewrite_rules(): void {
        \flush_rewrite_rules( false );
    }
}
```
**Lý do:** Trong namespaced class, gọi WordPress global function mà không có `\` prefix → PHP resolve thành `WeZone\Core\ThemeEngine\flush_rewrite_rules()` → "Call to undefined function". Unit test fail, deploy bị block. Áp dụng cho TẤT CẢ WordPress functions gọi trong namespace: `\add_rewrite_rule()`, `\get_option()`, `\update_option()`, `\flush_rewrite_rules()`, v.v.
