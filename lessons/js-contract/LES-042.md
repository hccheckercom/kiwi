**Pattern sai:**
```javascript
// JS tự tính total
const total = subtotal - discount + shipping;
// Nếu discount > subtotal → total < 0 → hiển thị "-50.000₫"
document.querySelector('.total').textContent = formatPrice(total);
```
**Phải làm:**
```javascript
// Dùng total_raw từ cart response — Core đảm bảo >= 0
fetch('/wp-json/wezone/v1/cart')
    .then(r => r.json())
    .then(data => {
        document.querySelector('.total').textContent = data.total; // formatted
        // Hoặc dùng data.total_raw cho calculation
    });
```
**Lý do:** CartEngine có logic `max(0, subtotal - discount + shipping)` — total KHÔNG BAO GIỜ âm. Nhưng nếu theme JS tự tính lại từ subtotal/discount/shipping riêng lẻ, có thể ra số âm khi coupon lớn hơn subtotal. Luôn dùng `total_raw` từ response thay vì tự tính.
**Grep check:** `subtotal.*discount\|subtotal\s*-\s*` trong JS — nếu tự tính total thay vì dùng response.total_raw → HIGH
