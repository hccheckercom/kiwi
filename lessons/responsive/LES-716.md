## Bad
```tsx
<div className="grid grid-cols-3 gap-4">
  {products.map(p => <ProductCard key={p.id} {...p} />)}
</div>
```
3-column grid on all screens — cards become tiny on mobile.

## Good
```tsx
<div className="grid grid-cols-1 gap-4 md:grid-cols-2 lg:grid-cols-3">
  {products.map(p => <ProductCard key={p.id} {...p} />)}
</div>
```
Single column on mobile, progressive enhancement for larger screens.

## Why
Mobile viewport (375px) divided by 3 = 125px per card — unreadable. Product grids, card layouts, and dashboards must start at 1 column and scale up.
