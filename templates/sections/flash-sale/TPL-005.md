## Code — JS

```js
(function () {
    var end = <?php echo (int) $end_time; ?> * 1000;
    function pad(n) { return n < 10 ? '0' + n : '' + n; }
    function tick() {
        var diff = Math.max(0, end - Date.now());
        var h = Math.floor(diff / 3600000);
        var m = Math.floor((diff % 3600000) / 60000);
        var s = Math.floor((diff % 60000) / 1000);
        var eh = document.getElementById('haven-countdown-h');
        var em = document.getElementById('haven-countdown-m');
        var es = document.getElementById('haven-countdown-s');
        if (eh) eh.textContent = pad(h);
        if (em) em.textContent = pad(m);
        if (es) es.textContent = pad(s);
        if (diff > 0) setTimeout(tick, 1000);
    }
    tick();
}());
```

## Usage Notes

- Nếu plugin `wezone-flash-sale` active → delegate render cho plugin
- Fallback: query products có `_sale_price` meta
- Countdown end time cần dynamic (từ plugin hoặc option)
- Badge đỏ (#c0392b) — khác với badge đen của featured products
