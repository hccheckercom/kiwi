**Pattern sai:**
```php
$shipping_free = $subtotal >= 2000000;
$shipping_cost = $shipping_free ? 0 : 50000;
echo wz_format_price( 100000 ); // express fee
```
**Phải làm:**
```php
$free_ship_threshold = (float) wz_config( 'shipping.free_ship_threshold', 2000000 );
$default_fee         = (float) wz_config( 'shipping.fees.default', 50000 );
$shipping_free       = $free_ship_threshold <= 0 || $subtotal >= $free_ship_threshold;
$shipping_cost       = $shipping_free ? 0 : $default_fee;
echo wz_format_price( (float) wz_config( 'shipping.fees.express', 100000 ) );
```
**Lý do:** Mỗi shop có threshold/fee khác nhau. Hardcode = sai khi clone theme sang shop mới. `store-config.php` là single source of truth cho business values.
**Grep check:** `>= \d{5,}|? 0 : \d{4,}|wz_format_price\(\s*\d` — hardcoded number trong logic shipping/pricing → HIGH
