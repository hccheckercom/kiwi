**Pattern sai:**
```php
// inc/seo.php không output JSON-LD trên front page
// hoặc output JSON invalid (thiếu @context, @type sai)
function wezone_theme_structured_data() {
    if ( ! is_front_page() ) return;
    echo '<script type="application/ld+json">';
    echo '{"name": "Shop"}'; // thiếu @context, @type
    echo '</script>';
}
```
**Phải làm:**
```php
function wezone_theme_structured_data() {
    if ( ! is_front_page() ) return;
    $data = [
        '@context' => 'https://schema.org',
        '@type'    => 'Store',
        'name'     => wz_config( 'store.name', get_bloginfo('name') ),
        'url'      => home_url('/'),
        'telephone' => wz_config( 'store.phone', '' ),
    ];
    echo '<script type="application/ld+json">' . wp_json_encode( $data, JSON_UNESCAPED_UNICODE ) . '</script>';
}
```
**Lý do:** Google Search Console báo "Missing field" nếu thiếu @context/@type. Rich results không hiển thị → mất CTR từ search.
**Grep check:** `application/ld+json` absent trên front page output, hoặc JSON thiếu `"@type":"Store"` → HIGH
