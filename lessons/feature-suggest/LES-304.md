## Pattern

Product page KHÔNG chứa bundle offer logic:
- `wz_get_product_bundle()` — backend function
- `wz_get_frequently_bought()` — backend function
- `bundle-offer` — template part
- `mua kèm` — Vietnamese text
- `bought together` — English text

## Why This Matters

Bundle offers ("Mua kèm giảm giá", "Frequently bought together") tăng 5-10% AOV bằng cách gợi ý sản phẩm bổ sung với discount.

**Impact:**
- Mất cơ hội cross-sell tự nhiên nhất (khi khách đang xem sản phẩm)
- Không có incentive để mua nhiều items cùng lúc
- Competitor có bundle → khách chuyển sang đó

## Bad Example

```php
// single-product.php — KHÔNG có bundle offer
<main>
    <div class="product-images">...</div>
    <div class="product-info">...</div>
    <div class="product-description">...</div>
    
    <!-- Chỉ có related products, không có bundle -->
    <?php get_template_part( 'template-parts/single-product/related-products' ); ?>
</main>
```

## Good Example

```php
// single-product.php — CÓ bundle offer
<main>
    <div class="product-images">...</div>
    <div class="product-info">
        <?php get_template_part( 'template-parts/product/bundle-offer', null, [ 'product_id' => $product_id ] ); ?>
    </div>
    <div class="product-description">...</div>
    
    <?php get_template_part( 'template-parts/single-product/related-products' ); ?>
</main>
```

```php
// template-parts/product/bundle-offer.php
<?php
$product_id = $args['product_id'] ?? get_the_ID();
$bundle     = wz_get_product_bundle( $product_id );

if ( ! $bundle ) {
    // Fallback: frequently bought together
    $bundle_products = wz_get_frequently_bought( $product_id, 3 );
    if ( empty( $bundle_products ) ) return;
    
    $bundle = [
        'id'       => 'auto_' . $product_id,
        'name'     => 'Mua kèm giảm thêm 5%',

        'discount' => 5,
    ];
}

$bundle_products = wz_get_products_by_ids( $bundle['products'] );
?>
<div class="wz-bundle" data-bundle-id="<?php echo esc_attr( $bundle['id'] ); ?>">
    <h3 class="wz-bundle__title"><?php echo esc_html( $bundle['name'] ); ?></h3>
    <div class="wz-bundle__items">
        <?php foreach ( $bundle_products as $bp ) : ?>
            <?php if ( $bp['id'] === $product_id ) continue; ?>

                <img src="<?php echo esc_url( $bp['image'] ); ?>" width="60" height="60" alt="" />

            </label>

        <span>Giá gốc: <del data-bundle-original></del></span>
```

## How to Fix

1. Tạo `template-parts/product/bundle-offer.php` với UI trên
2. Thêm vào product page (trong `.product-info` section)
3. Implement backend functions:

```php
// Priority: manual curated → co-occurrence data → complementary category
function wz_get_frequently_bought( $product_id, $limit = 3 ) {
    // 1. Manual curated
    $manual = wz_get_product_meta( $product_id, '_wz_bought_together_ids' );
    if ( ! empty( $manual ) ) {
        return wz_get_products_by_ids( $manual, $limit );
    }
    
    // 2. Co-occurrence trong orders (90 ngày)
    global $wpdb;
    $table = $wpdb->prefix . 'wz_order_items';
    $results = $wpdb->get_col( $wpdb->prepare(
        "SELECT oi2.product_id
         FROM {$table} oi1
         JOIN {$table} oi2 ON oi1.order_id = oi2.order_id
         WHERE oi1.product_id = %d
           AND oi2.product_id != %d
           AND oi1.created_at > DATE_SUB(NOW(), INTERVAL 90 DAY)
         GROUP BY oi2.product_id
         ORDER BY COUNT(*) DESC
         LIMIT %d",
        $product_id, $product_id, $limit
    ) );
    
    if ( empty( $results ) ) {
        // 3. Fallback: complementary by category
        return wz_get_complementary_by_category( $product_id, $limit );
    }
    
    return wz_get_products_by_ids( $results, $limit );
}
```

4. Add JS để tính tổng bundle real-time khi user check/uncheck items
