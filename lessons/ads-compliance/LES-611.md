## Bad
```php
// header.php — loads tracker without consent
<?php wp_head(); ?>
<script src="analytics.js"></script>
```
Tracker sets cookies immediately without user consent. Violates GDPR Article 6, CCPA 1798.140, Google Ads EU User Consent Policy.

## Good
```javascript
// wz-consent.js — consent banner with localStorage persistence
(function() {
  var CONSENT_KEY = 'wz_consent';
  var CONSENT_VERSION = '1.0';
  
  function getConsent() {
    try {
      var stored = localStorage.getItem(CONSENT_KEY);
      if (!stored) return null;
      var data = JSON.parse(stored);
      if (data.version !== CONSENT_VERSION) return null;
      return data;
    } catch(e) {
      return null;
    }
  }
  
  function setConsent(accepted) {
    var data = {
      accepted: accepted,
      version: CONSENT_VERSION,
      timestamp: Date.now()
    };
    localStorage.setItem(CONSENT_KEY, JSON.stringify(data));
    return data;
  }
  
  function showBanner() {
    // Show fixed-bottom banner with accept/reject buttons
    // Only load tracker after user accepts
  }
  
  // Respect Do Not Track
  if (navigator.doNotTrack === '1' || window.doNotTrack === '1') {
    window.wzConsentGranted = false;
    return;
  }
  
  var consent = getConsent();
  if (consent === null) {
    showBanner();
  } else if (consent.accepted) {
    window.wzConsentGranted = true;
    loadTracker();
  }
})();
```

```javascript
// wz-tracker.js — check consent before tracking
(function(){
  function checkConsent() {
    return window.wzConsentGranted === true;
  }
  
  if (!checkConsent()) {
    var checkInterval = setInterval(function() {
      if (checkConsent()) {
        clearInterval(checkInterval);
        initTracker();
      }
    }, 100);
    return;
  }
  
  initTracker();
})();
```

```php
// Plugin.php — load consent script before tracker
add_action( 'wp_enqueue_scripts', function () {
  wp_enqueue_script(
    'wezone-consent',
    PLUGIN_URL . 'assets/js/wz-consent.js',
    array(),
    VERSION,
    true
  );
  
  wp_enqueue_script(
    'wezone-tracker',
    PLUGIN_URL . 'assets/js/wz-tracker.js',
    array( 'wezone-consent' ),
    VERSION,
    true
  );
} );
```

## Why
**GDPR (EU):** Article 6 requires explicit consent before processing personal data. Cookies (visitor_id, session_id) are personal data. No consent = €20M fine or 4% revenue.

**CCPA (California):** Section 1798.140 requires opt-out mechanism. No opt-out = $2,500-$7,500 per violation.

**Google Ads EU User Consent Policy:** Requires consent banner for EU traffic. No banner = account suspension during verification.

**Implementation requirements:**
1. Consent banner with accept/reject buttons
2. localStorage persistence with version tracking
3. Do Not Track browser setting respect
4. Tracker only loads after consent granted
5. Privacy Policy page with GDPR/CCPA rights
6. Opt-out settings page
7. IP anonymization (mask last octet/segment)
8. 90-day auto-delete for analytics data

**Related violations:**
- Full IP storage without anonymization (GDPR Article 32)
- Missing Privacy Policy content
- Missing opt-out mechanism
- No data export/delete request handling
