**Mô tả:**
- Tất cả interactive elements (buttons, links, inputs) PHẢI có visible focus indicator
- Dùng `:focus-visible` (không phải `:focus`) để chỉ hiện khi keyboard navigation
- Outline 2px solid primary color, offset 2px

**Detect pattern (absence):**
- CSS files thiếu `:focus-visible` rule → keyboard users không thấy element nào đang focus

**CSS mẫu:**
```css
:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
  border-radius: var(--radius-sm);
}
:focus:not(:focus-visible) {
  outline: none;
}
```

**Lợi ích:** WCAG 2.1 AA — keyboard users luôn biết element nào đang active
