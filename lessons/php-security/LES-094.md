## Bad
```php
try {
    $order = $engine->create($data);
} catch (Exception $e) {
    // empty
}
```

## Good
```php
try {
    $order = $engine->create($data);
} catch (Exception $e) {
    wz_log('Order creation failed: ' . $e->getMessage(), 'error');
    throw $e;
}
```

## Why
Empty catch block swallows exception — bug is hidden, no error logged, impossible to debug in production.
