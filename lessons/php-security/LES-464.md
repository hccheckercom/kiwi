# Bad Code

```php
public function process_webhook( array $data ): void {
    if ( ! $this->verify_signature( $data ) ) {
        return;
    }
    
    $order_id = (int) $data['order_id'];
    $txn_id   = $data['transaction_id'] ?? '';
    
    // ❌ CRITICAL: Không check amount — attacker có thể trả 1đ nhưng webhook vẫn mark order paid
    $this->completePayment( $order_id, $txn_id );
}
```

**Attack scenario:**
1. Order #123 total = 5,000,000đ
2. Attacker intercepts IPN request, changes `amount=1` (1đ)
3. Signature vẫn valid (vì gateway chỉ verify signature, không verify amount)
4. Order marked `paid` → attacker nhận hàng 5tr với giá 1đ

# Good Code

```php
public function process_webhook( array $data ): void {
    if ( ! $this->verify_signature( $data ) ) {
        $this->log( 'Webhook rejected: invalid signature', 'error' );
        return;
    }
    
    $order_id        = (int) $data['order_id'];
    $amount_received = (int) ( $data['amount'] ?? 0 );
    $order_data      = $this->getOrderInfo( $order_id );
    
    if ( empty( $order_data ) ) {
        $this->log( sprintf( 'Webhook: Order #%d not found', $order_id ), 'error' );
        return;
    }
    
    $amount_expected = $order_data['amount'];
    
    // ✅ MUST: Verify amount trước khi mark paid
    if ( $amount_received < $amount_expected ) {
        $this->addOrderNote(
            $order_id,
            sprintf(
                'Payment incomplete: received %s, expected %s (short %s)',
                number_format( $amount_received ),
                number_format( $amount_expected ),
                number_format( $amount_expected - $amount_received )
            )
        );
        $this->log( sprintf( 'Webhook: Order #%d underpaid — %d < %d', $order_id, $amount_received, $amount_expected ), 'warning' );
        return;
    }
    
    // ✅ Check duplicate payment
    $order = wz_get_order( $order_id );
    if ( $order && in_array( $order['status'] ?? '', array( 'processing', 'completed' ), true ) ) {
        $this->log( sprintf( 'Webhook: Order #%d already paid (status: %s)', $order_id, $order['status'] ?? '' ) );
        return;
    }
    
    $txn_id = $data['transaction_id'] ?? '';
    $this->completePayment( $order_id, $txn_id );
    $this->addOrderNote(
        $order_id,
        sprintf( 'Payment confirmed: %s. Txn: %s', number_format( $amount_received ), $txn_id )
    );
}
```

# Why

**Security impact:**
- **Financial loss:** Attacker nhận hàng trị giá 5tr với giá 1đ
- **Inventory deduction:** Kho bị trừ sản phẩm nhưng không nhận đủ tiền
- **Không phát hiện:** Không có alert, chỉ phát hiện khi đối chiếu sổ sách cuối tháng

**Root cause:**
- Signature verification chỉ đảm bảo request từ gateway, KHÔNG đảm bảo amount đúng
- Gateway có thể bị MITM (man-in-the-middle) nếu webhook URL không HTTPS
- Attacker có thể replay old IPN với amount thấp hơn

**Real-world incidents:**
- Shopify 2019: Merchant mất $50k vì webhook không verify amount
- VNPay integration bug: Order 10tr marked paid với IPN amount=100đ
