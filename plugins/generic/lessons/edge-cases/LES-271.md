**Pattern hiện tại:**
```php
<span class="material-symbols-outlined text-6xl">shopping_cart</span>
```

**Blueprint yêu cầu:**
```php
<img src="<?php echo get_template_directory_uri(); ?>/assets/images/empty/cart.svg"
     alt="" width="200" height="160" loading="lazy">
```

**Lý do:** SVG illustrations tạo cảm giác friendly hơn icon đơn giản. Tuy nhiên đây là SUGGEST — icon vẫn functional.
