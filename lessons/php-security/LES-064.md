**Pattern sai:**
```php
// Handler registered nhưng không verify nonce
public function get_districts(): void {
    $province_id = sanitize_text_field( wp_unslash( $_POST['parent_id'] ?? '' ) );
    // ... xử lý trực tiếp, CSRF possible
}
```
**Phải làm:**
```php
public function get_districts(): void {
    check_ajax_referer( 'wz_address_lookup', 'nonce' );
    $province_id = absint( wp_unslash( $_POST['parent_id'] ?? 0 ) );
    // ...
}
```
**Lý do:** Khác LES-003 (form thiếu nonce field) — ở đây form CÓ nonce nhưng server KHÔNG verify. Attacker có thể forge request. Thêm: dùng `absint()` thay `sanitize_text_field()` cho numeric IDs dùng trong URL construction để tránh path injection.
**Grep check:** `wp_ajax_` registered → tìm function body → thiếu `check_ajax_referer` → CRITICAL
