## Bad
```tsx
const ProductList = ({ filters }: { filters: FilterOptions }) => {
  useEffect(() => {
    fetchProducts(filters)
  }, [filters]) // filters is a new object every render → infinite loop
}
```

## Good
```tsx
const ProductList = ({ filters }: { filters: FilterOptions }) => {
  const stableFilters = useMemo(() => filters, [filters.category, filters.sort, filters.page])
  useEffect(() => {
    fetchProducts(stableFilters)
  }, [stableFilters])
}
```
Or use individual primitive values:
```tsx
useEffect(() => {
  fetchProducts({ category, sort, page })
}, [category, sort, page])
```

## Why
Objects/arrays create new references on every render. useEffect compares by reference, not value — so it fires every render, causing an infinite fetch loop. Destructure to primitives or memoize the dependency.
