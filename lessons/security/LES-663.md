## Bad
```php
DEBUG = True
```

## Good
```php
DEBUG = os.environ.get('DEBUG', 'False').lower() == 'true'
```

## Why
DEBUG=True exposes stack traces, internal paths, and sensitive data to users

## Grep
`DEBUG\s*=\s*True` — in `**/*.py`
