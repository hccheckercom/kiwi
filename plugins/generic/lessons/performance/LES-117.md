# LES-117: get_avatar_url() N+1

## Problem

`get_avatar_url()` trong customer list loop:
- Mỗi call trigger `get_avatar_data` filter chain
- Có thể gọi `get_user_by()` nếu pass user ID
- Compute MD5 hash mỗi lần
- Với 20 customers/page → 20 function calls + filter chains

## Fix

Compute Gravatar URL trực tiếp từ email (đã có sẵn từ batch query):

```php
// ĐÚNG — direct computation, no WP overhead
$hash = md5( strtolower( trim( $user['user_email'] ) ) );
$avatar_url = "https://secure.gravatar.com/avatar/{$hash}?s=64&d=mm&r=g";

// SAI — N+1 with filter overhead
$avatar_url = get_avatar_url( $uid, array( 'size' => 64 ) );
```

## Impact

- Eliminates N filter chain calls per page
- No DB queries for avatar (email already loaded in batch)
- Time: ~50ms → ~1ms for 20 customers

## Detection

Tìm `get_avatar_url` bên trong `foreach` hoặc `array_map` trong REST controller.
