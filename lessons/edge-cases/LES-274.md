**Pattern sai:**
```php
function wz_placeholder_image() {
    return get_template_directory_uri() . '/assets/images/placeholder.jpg';
}
```

**Phải làm:**
```php
function wz_placeholder_image( string $context = 'product' ): string {
    return get_template_directory_uri() . "/assets/images/placeholder-{$context}.svg";
}
```

**Lý do:** SVG placeholder nhẹ hơn, scale tốt hơn, và có thể style bằng CSS. Blueprint yêu cầu placeholder khác nhau theo context: product (1:1), category (16:9), banner (21:9).
