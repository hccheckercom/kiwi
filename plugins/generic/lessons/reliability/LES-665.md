## Bad
```php
except ValueError:
    raise
```

## Good
```php
except ValueError as e:
    logger.error("Validation failed: %s", e)
    raise
```

## Why
Bare raise without context makes debugging harder. Consider logging before re-raise

`^\s*raise$` — in `**/*.py`
