**Pattern sai:**
```php
// update_profile() — không fire hook sau update
wp_update_user( $user_data );
update_user_meta( $user_id, 'wezone_phone', $phone );
return new WP_REST_Response( array( 'success' => true ), 200 );
```
**Phải làm:**
```php
wp_update_user( $user_data );
update_user_meta( $user_id, 'wezone_phone', $phone );
do_action( 'wezone/customer/profile_updated', $user_id );
return new WP_REST_Response( array( 'success' => true ), 200 );
```
**Lý do:** Listeners (cache invalidation, email, sync) cần biết khi data thay đổi — không fire hook = không có cách nào react
**Grep check:** File có `wp_update_user` hoặc `update_user_meta` nhưng không có `do_action('wezone/` → HIGH
