**Mô tả:**
- 5 mẫu thẻ voucher đẹp lấy cảm hứng từ Shopee, Gift Voucher in ấn, Sale Banner, Luxury Dark, Modern Minimal
- Admin chọn mẫu tại `wp-admin/admin.php?page=wezone-voucher-template` — preview trực quan, click chọn, save qua REST API
- Mẫu được áp dụng toàn site qua `get_option('wz_voucher_card_template')` (giá trị 1-5)

**Detect pattern (absence):**
- `template-parts/voucher/card.php` thiếu `$template === 4` hoặc `$template === 5` → chỉ có 3 mẫu cũ
- `VoucherTemplateAdminPage.php` thiếu `preview_luxury` hoặc `preview_minimal` → admin UI chưa có 5 mẫu

**5 mẫu thiết kế:**
- Template 1 — Shopee Card: scalloped edge, header gradient, dashed divider, code + nút ở dưới
- Template 2 — Gift Voucher: ngang rộng, trái tối + phải trắng, khung vuông discount, perforated divider
- Template 3 — Sale Banner: nền accent/vàng, ribbon SALE góc trái, discount lớn, footer primary
- Template 4 — Luxury Dark: nền đen, dải vàng gradient, badge tròn discount, italic "Voucher"
- Template 5 — Modern Minimal: trắng sạch, border-l accent, typography đơn giản, code pill

**Files cần có:**
- `themes/wezone-funilux/template-parts/voucher/card.php` — 5 template blocks
- `packages/wezone-core/src/AdminUI/VoucherTemplateAdminPage.php` — 5 preview methods + grid layout
- `packages/wezone-core/src/AdminUI/CouponController.php` — `max(1, min(5, $template))`

**Lợi ích:** Admin kiểm soát visual branding thẻ voucher phù hợp từng phong cách shop (trẻ trung, sang trọng, đơn giản)
