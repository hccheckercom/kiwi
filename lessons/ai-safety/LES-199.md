## Bad
```ts
const prompt = `Analyze this customer order:
  Name: ${order.customer_name}
  Email: ${order.customer_email}
  Phone: ${order.customer_phone}
  Address: ${order.shipping_address}
  Card: ${order.payment_method}`

await anthropic.messages.create({ messages: [{ role: 'user', content: prompt }] })
```
Customer PII sent to third-party AI API — potential data breach, GDPR violation.

## Good
```ts
const prompt = `Analyze this order pattern:
  Order value: ${order.total}
  Category: ${order.category}
  Region: ${order.region}
  Payment type: ${order.payment_type}`
// Only send aggregated/anonymized data — no PII
```

## Why
Data sent to AI APIs may be logged, used for training (depending on plan), or exposed in a breach. Never send: names, emails, phones, addresses, payment details, passwords, API keys, or health data. Anonymize or aggregate before sending.
