**Mô tả:**
- Hiển thị thẻ voucher dạng card có thể thu thập ở homepage, product page, cart
- User click "Thu thập" → lưu vào account (`wz_user_vouchers` table) → hiển thị trong `/my-account/vouchers/`
- Admin chọn 1 trong 3 mẫu thiết kế thẻ (Classic/Bold/Minimal) áp dụng toàn site
- Mã vẫn public — collect chỉ là bookmark/shortcut, không restrict usage

**Detect pattern (absence):**
- `inc/cart.php` hoặc `inc/ajax.php` thiếu `wz_collect_voucher` → chưa có collect handler
- `assets/js/main.js` thiếu `wz-collect-btn` → chưa có JS handler
- `inc/wz-shims.php` có `wz_get_user_vouchers` trả về `[]` → stub chưa implement

**Files cần có:**
- `template-parts/voucher/card.php` — 3 mẫu thẻ + collect button
- `template-parts/voucher/section.php` — section wrapper fetch vouchers từ DB
- `inc/cart.php` — AJAX handler `wz_collect_voucher` (INSERT IGNORE vào `wz_user_vouchers`)
- `inc/wz-shims.php` — `wz_get_user_vouchers()` JOIN `wz_user_vouchers` + `wz_coupons`
- `assets/js/main.js` — event delegation trên `.wz-collect-btn[data-coupon-id]`

**Plugin cần có:**
- DB table `wz_user_vouchers` (user_id, coupon_id, UNIQUE idx_user_coupon)
- REST endpoint `POST /wezone/v1/coupons/{id}/collect`
- REST endpoint `GET|PUT /wezone/v1/settings/voucher-template` (option `wz_voucher_card_template`)

**Lợi ích:** Tăng engagement, user có động lực thu thập mã trước khi mua, admin kiểm soát visual branding thẻ voucher
