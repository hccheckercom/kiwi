## Pattern

Recommendation files dùng `'orderby' => 'rand'` thay vì smart algorithm (personalized, bestsellers, complementary).

## Why This Matters

Random recommendations không tận dụng data → low conversion. Smart algorithm (purchase history, bestsellers, co-occurrence) tăng 2-4% conversion.

**Impact:**
- Gợi ý sản phẩm không relevant → khách bỏ qua
- Waste prime real estate (thank-you page, cart page)
- Không học từ behavior data → không cải thiện theo thời gian

## Bad Example

```php
// thankyou/recommended.php — dùng random
$products = wz_get_products( [
    'limit'   => 6,
    'orderby' => 'rand', // ❌ Random — không personalized
] );
```

```php
// cart cross-sell — dùng random
$cross_sell = wz_get_products( [
    'limit'   => 4,
    'orderby' => 'rand', // ❌ Random
] );
```

## Good Example

### Thank-you page — personalized based on purchase history

```php
// thankyou/recommended.php
$customer_id = wz_get_current_customer_id();
$order_id    = $args['order_id'] ?? 0;

// Exclude sản phẩm vừa mua trong order này
$order_items = wz_get_order_items( $order_id );
$exclude_ids = array_column( $order_items, 'product_id' );

// Personalized recommendations
$products = wz_get_personalized_recommendations( $customer_id, 6, $exclude_ids );

// Fallback nếu không đủ data
if ( count( $products ) < 6 ) {
    $bestsellers = wz_get_bestsellers( 6 - count( $products ), $exclude_ids );
    $products = array_merge( $products, $bestsellers );
}
```

### Cart cross-sell — complementary products

```php
// cart cross-sell
$cart_items      = wz_get_cart_items();
$cart_categories = [];
$cart_ids        = [];

foreach ( $cart_items as $item ) {
    $product = wz_get_product( $item['product_id'] );
    $cart_categories = array_merge( $cart_categories, $product['category_ids'] ?? [] );
    $cart_ids[] = $item['product_id'];
}

$cart_categories = array_unique( $cart_categories );

// Complementary products (khác category, bổ sung)
$cross_sell = wz_get_complementary_products( $cart_categories, 4, $cart_ids );

// Fallback: bestsellers
if ( count( $cross_sell ) < 4 ) {
    $bestsellers = wz_get_bestsellers( 4 - count( $cross_sell ), $cart_ids );
    $cross_sell = array_merge( $cross_sell, $bestsellers );
}
```

## Smart Algorithm Implementation

```php
// Personalized based on purchase history
function wz_get_personalized_recommendations( $customer_id, $limit = 4, $exclude = [] ) {
    // Lấy categories từ purchase history (90 ngày)
    $purchased_categories = wz_get_customer_purchased_categories( $customer_id, 90 );
    $purchased_ids        = wz_get_customer_purchased_product_ids( $customer_id, 90 );
    
    $exclude = array_merge( $exclude, $purchased_ids );
    
    if ( empty( $purchased_categories ) ) {
        return wz_get_bestsellers( $limit, $exclude );
    }
    
    return wz_query_products( [
        'category_ids' => $purchased_categories,
        'exclude'      => $exclude,
        'in_stock'     => true,
        'orderby'      => 'sales_count', // ✅ Bestsellers trong categories đã mua
        'limit'        => $limit,
    ] );
}

// Complementary products (khác category, bổ sung)
function wz_get_complementary_products( $cart_categories, $limit = 4, $exclude = [] ) {
    // Mapping: category → complementary categories
    $complementary_map = wz_config_get( 'complementary_categories', [] );
    
    $target_categories = [];
    foreach ( $cart_categories as $cat_id ) {
        if ( isset( $complementary_map[ $cat_id ] ) ) {
            $target_categories = array_merge( $target_categories, $complementary_map[ $cat_id ] );
        }
    }
    
    $target_categories = array_unique( $target_categories );
    
    if ( empty( $target_categories ) ) {
        // Fallback: any category NOT in cart
        return wz_query_products( [
            'exclude_categories' => $cart_categories,
            'exclude'            => $exclude,
            'in_stock'           => true,
            'orderby'            => 'sales_count',
            'limit'              => $limit,
        ] );
    }
    
    return wz_query_products( [
        'category_ids' => $target_categories,
        'exclude'      => $exclude,
        'in_stock'     => true,
        'orderby'      => 'sales_count',
        'limit'        => $limit,
    ] );
}
```

## Priority Rules (Blueprint)

Khi có nhiều recommendation types, ưu tiên:
1. **Manual curated** (admin chọn) — luôn ưu tiên cao nhất
2. **Frequently bought together** — data-driven, high conversion
3. **Complementary** (khác category, bổ sung) — tăng basket diversity
4. **Related** (cùng category) — safe fallback
5. **Trending/Bestsellers** — cold-start fallback

**KHÔNG BAO GIỜ dùng random** — luôn có ít nhất bestsellers fallback.

## How to Fix

1. Xác định context: thank-you / cart / product page / checkout
2. Chọn algorithm phù hợp:
   - Thank-you → personalized (purchase history)
   - Cart → complementary (khác category)
   - Product page → related (same category + price range)
   - Checkout → last-chance (bestsellers)
3. Luôn có fallback chain: smart algorithm → bestsellers
4. Luôn exclude: sản phẩm hiện tại, sản phẩm trong cart, sản phẩm đã mua
5. Luôn filter `in_stock = true`
