## Bad
```php
public function backup_database( string $date ): ?string {
    $file = $this->backup_dir . "/db_{$date}.sql";
    $handle = fopen( $file, 'w' ); // No disk space check â€” crash if disk full
    // ... write data
}
```

## Good
```php
public function backup_database( string $date ): ?string {
    $file = $this->backup_dir . "/db_{$date}.sql";
    $required = 1024 * 1024 * 100; // 100MB estimate
    if ( disk_free_space( $this->backup_dir ) < $required ) {
        return null; // Not enough space
    }
    $handle = fopen( $file, 'w' );
    // ... write data
}
```

## Why
Writing large files (backups, exports) without checking disk space causes fatal errors when disk is full. Check disk_free_space() before writing.
