## Bad
```tsx
// app/admin/settings/Sidebar.tsx — w-64
<aside className="w-64 border-r">

// app/admin/products/Sidebar.tsx — w-72
<aside className="w-72 border-r">
```
Hai section admin dùng sidebar width khác nhau — UX không nhất quán.

## Good
```tsx
// Cả hai dùng cùng width
<aside className="w-64 border-r">
```

## Why
Admin sidebar phải consistent width trên toàn bộ sections. User navigate giữa các section không nên thấy sidebar "nhảy" width.
