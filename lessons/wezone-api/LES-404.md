# Bug: wz_coupon_engine() Dùng `new` Thay Vì Container::make() — Inconsistent DI

## Vấn đề

`wz_cart()` và `wz_order_engine()` dùng `Container::make()` để resolve dependencies. Nhưng `wz_coupon_engine()` dùng `new CouponEngine()` trực tiếp → nếu CouponEngine cần dependencies sau này, code này sẽ break.

**Impact:** Maintainability, future-proofing.

## Bad

```php
// ❌ SAI — inconsistent với cart/order engine
function wz_coupon_engine(): CouponEngine {
    static $engine = null;
    if ( $engine === null ) {
        $engine = new CouponEngine(); // ← hardcoded instantiation
    }
    return $engine;
}
```

## Good

```php
// ✅ ĐÚNG — consistent DI pattern
function wz_coupon_engine(): CouponEngine {
    static $engine = null;
    if ( $engine === null ) {
        try {
            $engine = Container::getInstance()->make( CouponEngine::class );
        } catch ( \Throwable $e ) {
            wp_die( esc_html( 'WeZone: CouponEngine không thể khởi tạo — ' . $e->getMessage() ) );
        }
    }
    return $engine;
}
```
