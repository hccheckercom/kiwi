## Bad
```tsx
// BottomNav.tsx — hiện trên mọi màn hình
<nav className="fixed bottom-0 w-full bg-white border-t">
```

## Good
```tsx
<nav className="fixed bottom-0 w-full bg-white border-t md:hidden">
```

## Why
Bottom navigation chỉ dành cho mobile. Trên desktop đã có top nav/sidebar. Hiện cả hai gây thừa UI và che content.
