## Pattern

Developer dùng WooCommerce shortcodes trong templates hoặc content.

## Bad

```php
// In template files
echo do_shortcode( '[products limit="4" columns="4" category="shoes"]' );
echo do_shortcode( '[recent_products limit="8"]' );
echo do_shortcode( '[sale_products limit="4"]' );
echo do_shortcode( '[best_selling_products limit="4"]' );
echo do_shortcode( '[featured_products limit="4"]' );
echo do_shortcode( '[product_page id="123"]' );
echo do_shortcode( '[add_to_cart id="123"]' );

// Page content (stored in DB)
// [woocommerce_cart]
// [woocommerce_checkout]
// [woocommerce_my_account]
// [woocommerce_order_tracking]
```

## Good

```php
// In template files — dùng template functions
$products = wz_get_products( array( 'limit' => 4, 'category_id' => $cat_id ) );
wz_get_template( 'loop/product-grid.php', array(
    'products' => $products,
    'columns'  => 4,
) );

// Recent products
$recent = wz_get_products( array( 'limit' => 8, 'orderby' => 'date' ) );

// Sale products
$sale = wz_get_products( array( 'limit' => 4, 'on_sale' => true ) );

// Best selling
$best = wz_get_products( array( 'limit' => 4, 'orderby' => 'sales' ) );

// Featured
$featured = wz_get_products( array( 'limit' => 4, 'featured' => true ) );

// Single product embed
wz_get_template( 'product/embed.php', array( 'product' => wz_get_product( 123 ) ) );

// Add to cart button
wz_add_to_cart_button( 123 );

// Cart/Checkout/Account — handled by Wezone Router, không cần shortcode
// URL: /cart, /checkout, /account — auto-routed
```

## Why

WooCommerce shortcodes được register bởi `WC_Shortcodes` class — không tồn tại trên Wezone.
- `do_shortcode('[products ...]')` → output rỗng (shortcode không registered)
- Cart/Checkout/Account pages: Wezone dùng Router-based routing, không cần shortcode
- Product queries: dùng `wz_get_products()` + template system cho full control

Lưu ý: Nếu content trong DB chứa WooCommerce shortcodes (từ migration), cần cleanup:
```sql
UPDATE wp_posts SET post_content = REPLACE(post_content, '[woocommerce_cart]', '')
WHERE post_content LIKE '%[woocommerce_%';
```
