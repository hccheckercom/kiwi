**Mô tả:**
- Mỗi public page PHẢI có `<link rel="canonical">` pointing to preferred URL
- Cart, Checkout, Account, Search, Thank-you pages PHẢI có `<meta name="robots" content="noindex, nofollow">`
- Paginated categories: canonical points to page 1 (NOT self-referencing)

**Detect pattern (absence):**
- `header.php` hoặc `inc/seo.php` thiếu `rel="canonical"` → Google index duplicate URLs

**Canonical rules:**
| Scenario | Canonical |
|----------|-----------|
| Product page | `/san-pham/{slug}/` |
| Category page 1 | `/danh-muc/{slug}/` |
| Category page 2+ | `/danh-muc/{slug}/` (page 1) |
| Filtered results | Base category URL (no params) |
| HTTP vs HTTPS | Always HTTPS |
| Trailing slash | Always include |

**Noindex pages:**
```html
<!-- Cart, Checkout, Account, Search, Thank-you -->
<meta name="robots" content="noindex, nofollow">
```

**Lợi ích:** Tránh duplicate content penalty, Google index đúng URL preferred
