**Pattern sai (PHP):**
```php
<span class="cart-badge"><?php echo esc_html( $cart_count ); ?></span>
```

**Pattern sai (JS):**
```js
badge.textContent = count;
```

**Phải làm (PHP):**
```php
<span class="cart-badge"><?php echo esc_html( $cart_count > 99 ? '99+' : $cart_count ); ?></span>
```

**Phải làm (JS):**
```js
badge.textContent = count > 99 ? '99+' : count;
```

**Lý do:** Badge > 2 chữ số sẽ tràn khỏi circle/pill shape, phá layout header.
