## Bad
```php
$ld = [
    '@context' => 'https://schema.org',
    '@type'    => 'Store',
    'name'     => $shop_name,
    'url'      => $site_url,
    'telephone' => $phone,
    'address'  => [...],
    // Missing: logo, sameAs (social profiles)
];
```

## Good
```php
$logo = wz_config( 'logo_url', '' );
$socials = array_filter([
    wz_config( 'facebook', '' ),
    wz_config( 'instagram', '' ),
    wz_config( 'tiktok', '' ),
    wz_config( 'youtube', '' ),
]);

$ld = [
    '@context' => 'https://schema.org',
    '@type'    => 'Store',
    'name'     => $shop_name,
    'url'      => $site_url,
    'telephone' => $phone,
    'address'  => [...],
];

if ( $logo ) {
    $ld['logo'] = $logo;
}
if ( ! empty( $socials ) ) {
    $ld['sameAs'] = array_values( $socials );
}
```

## Why
Blueprint `seo.md` Organization schema requires `logo` and `sameAs`. Without `logo`, Google Knowledge Panel won't display the brand logo. Without `sameAs`, Google can't link social profiles to the business entity — weakening brand authority signals.
