# Search Results Page — Query + Filters + Grid

Search results page with query display, filter sidebar, product grid, and pagination.

## Features
- Search query display with result count
- Filter sidebar (reuse archive filters)
- Product grid (reuse product cards)
- Pagination
- Empty state with suggestions
- Breadcrumb navigation

## Code

```php
<?php
get_header();

$search_query = get_search_query();
$results = function_exists( 'wz_search_products' ) ? wz_search_products( $search_query ) : [];
$total = count( $results );
?>

<main class="search-page">
    <!-- Breadcrumb -->
    <nav class="breadcrumb">
        <a href="<?php echo esc_url( home_url( '/' ) ); ?>">Trang chủ</a>
        <span>/</span>
        <span>Tìm kiếm</span>
    </nav>

    <!-- Search header -->
    <div class="search-header">
        <h1>Kết quả tìm kiếm: "<?php echo esc_html( $search_query ); ?>"</h1>
        <p class="search-count"><?php echo esc_html( sprintf( '%d sản phẩm', $total ) ); ?></p>
    </div>

    <?php if ( ! empty( $results ) ) : ?>
        <div class="search-layout">
            <!-- Sidebar filters -->
            <aside class="search-sidebar">
                <?php get_template_part( 'template-parts/search/sidebar-filters' ); ?>
            </aside>

            <!-- Results grid -->
            <div class="search-results">
                <div class="product-grid">
                    <?php foreach ( $results as $product ) : ?>
                        <?php get_template_part( 'template-parts/product/card', null, [
                            'product_id' => $product['id'],
                            'context' => 'search',
                        ] ); ?>
                    <?php endforeach; ?>
                </div>

                <!-- Pagination -->
                <?php if ( $total > 20 ) : ?>
                    <div class="pagination">
                        <?php echo paginate_links(); ?>
                    </div>
                <?php endif; ?>
            </div>
        </div>

    <?php else : ?>
        <!-- Empty state -->
        <div class="search-empty">
            <span class="material-symbols-outlined">search_off</span>
            <h2>Không tìm thấy kết quả</h2>
            <p>Không có sản phẩm nào phù hợp với từ khóa "<?php echo esc_html( $search_query ); ?>"</p>
            <div class="search-suggestions">
                <p>Gợi ý:</p>
                <ul>
                    <li>Kiểm tra lại chính tả từ khóa</li>
                    <li>Thử từ khóa khác hoặc tổng quát hơn</li>
                    <li>Duyệt qua <a href="<?php echo esc_url( home_url( '/shop/' ) ); ?>">danh mục sản phẩm</a></li>
                </ul>
            </div>
        </div>
    <?php endif; ?>
</main>

<?php get_footer(); ?>
```

## Usage

Create as `search.php` in theme root.

## CSS Classes
- `.search-page` — page wrapper
- `.search-empty` — empty state
- `.product-grid` — reuse from archive

## Dependencies
- `wz_search_products()` function
- Product card component
- Filter sidebar component
