# Brand Grid — Brand/Manufacturer Showcase

Brand/manufacturer logo grid với hover effects và links.

## Use Cases
- Brand showcase page (trang 47)
- Partner/vendor page
- "Shop by brand" section
- Footer brand logos

## Features
- 🏢 Logo grid responsive
- 🔗 Clickable brand links
- ✨ Hover scale effect
- 📱 Mobile-optimized (2-3 cols)
- 🖼️ Grayscale → color on hover
- 🔍 SEO-friendly brand names

## Code

### PHP Template (page-thuong-hieu.php)

```php
<?php
/**
 * Template Name: Thương Hiệu
 */

get_header();
?>

<main class="brands-page py-12">
	<div class="max-w-[1280px] mx-auto px-10">
		<!-- Page Header -->
		<div class="text-center mb-12">
			<h1 class="text-4xl font-bold mb-4">Thương Hiệu Đối Tác</h1>
			<p class="text-gray-600 max-w-2xl mx-auto">
				Chúng tôi hợp tác với các thương hiệu hàng đầu để mang đến sản phẩm chất lượng cao nhất
			</p>
		</div>

		<?php
		// Get brands from taxonomy or custom post type
		$brands = get_terms([
			'taxonomy' => 'wz_brand',
			'hide_empty' => true,
			'orderby' => 'name',
			'order' => 'ASC',
		]);

		if ( ! empty( $brands ) && ! is_wp_error( $brands ) ):
		?>
			<!-- Brand Grid -->
			<div class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 xl:grid-cols-5 gap-6">
				<?php foreach ( $brands as $brand ): 
					$brand_logo = get_term_meta( $brand->term_id, 'logo', true );
					$brand_url = get_term_link( $brand );
				?>
					<a 
						href="<?php echo esc_url( $brand_url ); ?>" 
						class="brand-card bg-white rounded-lg border border-gray-200 p-6 flex items-center justify-center hover:shadow-lg hover:border-blue-500 transition-all group"
						aria-label="<?php echo esc_attr( $brand->name ); ?>"
					>
						<?php if ( $brand_logo ): ?>
							<img 
								src="<?php echo esc_url( $brand_logo ); ?>" 
								alt="<?php echo esc_attr( $brand->name ); ?>"
								class="max-w-full h-auto max-h-20 object-contain grayscale group-hover:grayscale-0 group-hover:scale-110 transition-all duration-300"
								loading="lazy"
							/>
						<?php else: ?>
							<span class="text-lg font-bold text-gray-700 group-hover:text-blue-600 transition-colors">
								<?php echo esc_html( $brand->name ); ?>
							</span>
						<?php endif; ?>
					</a>
				<?php endforeach; ?>
			</div>

			<!-- Brand Count -->
			<div class="text-center mt-12">
				<p class="text-gray-600">
					Hợp tác với <strong class="text-blue-600"><?php echo count( $brands ); ?>+</strong> thương hiệu uy tín
				</p>
			</div>

		<?php else: ?>
			<!-- Empty State -->
			<div class="text-center py-20">
				<svg class="w-20 h-20 text-gray-300 mx-auto mb-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
					<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M19 21V5a2 2 0 00-2-2H7a2 2 0 00-2 2v16m14 0h2m-2 0h-5m-9 0H3m2 0h5M9 7h1m-1 4h1m4-4h1m-1 4h1m-5 10v-5a1 1 0 011-1h2a1 1 0 011 1v5m-4 0h4" />
				</svg>
				<h2 class="text-2xl font-bold text-gray-900 mb-2">Chưa có thương hiệu</h2>
				<p class="text-gray-600">Danh sách thương hiệu sẽ được cập nhật sớm</p>
			</div>
		<?php endif; ?>
	</div>
</main>

<?php get_footer(); ?>
```

```php
		</a>
	<?php endforeach; ?>
</div>
```

## Data Sources

**Option 1: Taxonomy** (recommended)
- `wz_brand` taxonomy
- Term meta: `logo` (image URL)
- `get_terms()` to fetch all brands

**Option 2: Custom Post Type**
```php
$brands = new WP_Query([
	'post_type' => 'wz_brand',
	'posts_per_page' => -1,
	'orderby' => 'title',
	'order' => 'ASC',
]);
```

**Option 3: Static array** (for small sites)

## Dependencies
- Tailwind CSS
- Brand taxonomy or CPT
- Brand logos (SVG recommended)

- Semantic HTML (a, img)

- Semantic links to brand archives
- Structured brand count

## Customization

### Add brand description on hover
```php
<a href="..." class="brand-card group relative">
	<img src="..." alt="..." />
	<div class="absolute inset-0 bg-black/80 opacity-0 group-hover:opacity-100 transition-opacity flex items-center justify-center p-4">
		<p class="text-white text-sm text-center"><?php echo esc_html( $brand->description ); ?></p>
	</div>
</a>
```

### Add product count badge
```php
<?php $product_count = wz_get_brand_product_count( $brand->term_id ); ?>
<div class="absolute top-2 right-2 bg-blue-600 text-white text-xs px-2 py-1 rounded-full">
	<?php echo esc_html( $product_count ); ?> SP
</div>
```

### Alphabetical sections
```php
<?php
$brands_by_letter = [];
foreach ( $brands as $brand ) {
	$first_letter = mb_strtoupper( mb_substr( $brand->name, 0, 1 ) );
	$brands_by_letter[$first_letter][] = $brand;
}
ksort( $brands_by_letter );

foreach ( $brands_by_letter as $letter => $letter_brands ):
?>
	<div class="mb-10">
		<h2 class="text-2xl font-bold mb-6 pb-2 border-b"><?php echo esc_html( $letter ); ?></h2>
		<div class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 xl:grid-cols-5 gap-6">
			<!-- Brand cards -->
		</div>
	</div>
<?php endforeach; ?>
```
