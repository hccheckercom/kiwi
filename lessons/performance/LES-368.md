# Bug: Cache Key Thiếu Version/Namespace

## Vấn đề

Cache keys không có version/namespace → deploy mới không invalidate cache cũ → user nhận stale data.

**Ví dụ lỗi:**
```php
// ❌ SAI — key không có version
wp_cache_delete( "wz_product_{$product_id}", 'wezone' );
delete_transient( "wz_product_data_{$product_id}" );

// ❌ SAI — 2 keys khác nhau cho cùng data
wp_cache_set( "wz_product_{$id}", $data, 'wezone' );
delete_transient( "wz_product_data_{$id}" ); // key khác!
```

**Hậu quả:**
- Deploy code mới → cache cũ vẫn còn → logic mới đọc data cũ → fatal error
- Thay đổi schema → cached object không compatible → unserialize warning
- Không có cách purge selective → phải flush toàn bộ cache

## Root Cause

1. **Không có version trong key** → không thể invalidate by version
2. **Key naming không consistent** → object cache vs transient dùng key khác nhau
3. **Không có namespace hierarchy** → không thể purge by prefix

## Fix

### 1. Thêm version vào mọi cache key

```php
// ✅ ĐÚNG — key có version
define( 'WZ_CACHE_VERSION', '2' ); // bump khi schema thay đổi

wp_cache_set( 
    "wz_product_v{WZ_CACHE_VERSION}_{$id}", 
    $data, 
    'wezone' 
);

delete_transient( "wz_product_v{WZ_CACHE_VERSION}_{$id}" );
```

### 2. Dùng helper function để enforce naming

```php
// ✅ ĐÚNG — centralized key builder
function wz_cache_key( string $type, $id, array $params = [] ): string {
    $version = WZ_CACHE_VERSION;
    $suffix  = $params ? '_' . md5( serialize( $params ) ) : '';
    return "wz_{$type}_v{$version}_{$id}{$suffix}";
}

// Usage
$key = wz_cache_key( 'product', $product_id );
wp_cache_set( $key, $data, 'wezone' );
delete_transient( $key ); // same key!
```

### 3. Invalidate by version on deploy

```php
// ✅ ĐÚNG — auto-bump version on deploy
function wz_maybe_bump_cache_version(): void {
    $hash_file = WP_PLUGIN_DIR . '/wezone-core/wezone-core.php';
    $current   = md5_file( $hash_file );
    $stored    = get_option( 'wz_deploy_hash', '' );
    
    if ( $current !== $stored ) {
        update_option( 'wz_deploy_hash', $current, true );
        // Bump version → old keys become unreachable
        update_option( 'wz_cache_version', time(), true );
        if ( ! defined( 'WZ_CACHE_VERSION' ) ) {
            define( 'WZ_CACHE_VERSION', (string) get_option( 'wz_cache_version', '1' ) );
        }
    }
}
add_action( 'plugins_loaded', 'wz_maybe_bump_cache_version', 5 );
```

## Verification

```bash
# Grep tất cả cache keys
rg 'wp_cache_(set|add|get|delete)\s*\(\s*["\']wz_' --type php

# Check có version không
rg 'wp_cache_\w+\s*\(\s*["\'][^"\']+_v\d+_' --type php

# Check transient keys
rg 'delete_transient\s*\(\s*["\']wz_' --type php
```
