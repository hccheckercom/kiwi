**Mô tả:**
- Custom fonts PHẢI có `font-display: swap` — tránh invisible text (FOIT)
- Critical CSS (above-fold) nên inline trong `<head>` (< 5KB)
- Full CSS load non-blocking: `<link rel="preload" as="style">`

**Detect pattern (absence):**
- CSS files có `@font-face` nhưng thiếu `font-display: swap` → text invisible trong 3s đầu

**Code mẫu:**
```css
@font-face {
  font-family: 'BrandFont';
  src: url('brand.woff2') format('woff2');
  font-display: swap;
  font-weight: 400 700;
  unicode-range: U+0000-024F, U+1E00-1EFF;
}
```

**Anti-patterns (CẤM):**
- `@import` in CSS (blocks rendering chain)
- Render-blocking JS in `<head>` without `defer`
- Inline `<style>` > 5KB
- CSS without `font-display` on custom fonts

**Lợi ích:** FCP nhanh hơn 1-2s, text luôn visible ngay khi page load
