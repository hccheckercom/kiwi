**Pattern sai:** Theme tạo folder `assets/css/partials/` với BEM classes thủ công thay vì dùng Tailwind pipeline.

**Cấu trúc SAI (Haven):**
```
assets/css/partials/_layout.css      ← BEM: .haven-header, .haven-trust__grid
assets/css/partials/_components.css  ← BEM: .haven-card, .haven-badge--sale
assets/css/partials/_responsive.css  ← @media max-width (vi phạm mobile-first)
assets/css/main.css                  ← @import partials thủ công
```

**Cấu trúc ĐÚNG:**
```
src/main.css              ← @tailwind base; @tailwind components; @tailwind utilities;
tailwind.config.js        ← colors: var(--wz-*), screens: 768/1024
package.json              ← "build:css": "postcss src/main.css -o assets/css/tailwind.css"
assets/css/tailwind.css   ← Compiled output (KHÔNG sửa trực tiếp)
```

**Phải làm:**
1. Tạo `src/main.css` với @tailwind directives
2. Tạo `tailwind.config.js` — colors chỉ dùng `var(--wz-*)`
3. Tạo `package.json` với script `build:css`
4. Dùng Tailwind utility classes trong PHP: `class="flex items-center gap-4 bg-primary"`
5. KHÔNG tạo `assets/css/partials/` — mọi styling qua Tailwind

**Lý do:** BEM classes không có CSS definition trong Tailwind output → elements render nhưng invisible (không màu, không spacing, không layout). Blueprint Rule #13 cấm tuyệt đối BEM custom class.

**Grep check:** `assets/css/partials/` hoặc `@import.*partials` trong themes/ → CRITICAL
