**Pattern sai:**
```php
// functions.php không có include inc/wz-shims.php
require_once get_template_directory() . '/inc/setup.php';
// wz_*() gọi ở template → PHP fatal undefined function
```
**Phải làm:**
```php
require_once get_template_directory() . '/inc/wz-shims.php'; // PHẢI ĐẦU TIÊN
require_once get_template_directory() . '/inc/seo.php';
require_once get_template_directory() . '/inc/setup.php';
require_once get_template_directory() . '/inc/cart.php';
```
**Lý do:** wz-shims.php phải load trước — nó khai báo wz_*() dùng ở mọi template
**Grep check:** `inc/wz-shims\.php` — thiếu trong functions.php → CRITICAL
