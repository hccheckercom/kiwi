**Pattern sai:**
```js
cartContainer.innerHTML = response.data.html;
itemEl.innerHTML = response.data.product_name;
```
**Phải làm:**
```js
// Nếu server trả về text data → dùng textContent
itemEl.textContent = response.data.product_name;
// Nếu cần render HTML: verify server đã wp_kses_post() trước khi return
```
**Lý do:** innerHTML với data từ server có thể execute script nếu server không escape đúng
**Grep check:** `\.innerHTML\s*=` — mỗi hit phải verify data source đã được sanitize → HIGH
