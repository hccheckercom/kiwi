## Bad
```php
namespace WeZone\Core;
class OrderEngine {
    public function create(array $data) {
```

## Good
```php
declare(strict_types=1);
namespace WeZone\Core;
class OrderEngine {
    public function create(array $data): int {
```

## Why
Namespaced file without declare(strict_types=1) — silent type coercion bugs. String '0' treated as falsy int.
