## Bad

```python
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent))
from memory.db import get_connection
```

## Good

```python
# Use relative imports within package
from ..memory.db import get_connection

# Or use proper package structure with __init__.py
# and install with: pip install -e .
```

## Why

`sys.path.insert(0, ...)` pollutes the module search path, can shadow stdlib modules, and makes the import order fragile. It's a common workaround in Kiwi's codebase but should be replaced with proper package structure using `__init__.py` and relative imports. For CLI entry points, use `python -m package.module`.
