## Bad
```php
foreach ( $ids as $id ) {
    $product = wz_get_product( $id ); // 1 query per iteration
}
```

## Good
```php
$products = wz_get_products_by_ids( $ids ); // 1 query total
```

## Why
N+1 DB queries — wz_get_product() in loop fires 1 query per item. 50 products = 50 queries. Use batch function.
