## Bad
```php
// Theme loads GA4 + Facebook Pixel immediately
// No consent banner, no consent check
// Tracking fires on first visit without user permission
```
Violates Google Consent Mode v2 (required since March 2024) and Meta's Limited Data Use policy. Ads may be rejected or account restricted.

## Good
```php
// inc/cookie-consent.php
function wz_cookie_consent_banner() {
    if ( isset( $_COOKIE['wz_consent'] ) ) return;
    ?>
    <div id="wz-cookie-consent" class="fixed bottom-0 inset-x-0 z-50 p-4 bg-white shadow-lg border-t">
        <p>Chúng tôi sử dụng cookie để phân tích và quảng cáo.
           <a href="/chinh-sach-bao-mat">Chính sách</a></p>
        <button onclick="wzAcceptConsent()">Đồng ý</button>
        <button onclick="wzDeclineConsent()">Từ chối</button>
    </div>
    <script>
    function wzAcceptConsent() {
        document.cookie = 'wz_consent=' + JSON.stringify({analytics:true,marketing:true}) + ';path=/;max-age=31536000';
        gtag('consent', 'update', { analytics_storage: 'granted', ad_storage: 'granted' });
        document.getElementById('wz-cookie-consent').remove();
    }
    function wzDeclineConsent() {
        document.cookie = 'wz_consent=' + JSON.stringify({analytics:false,marketing:false}) + ';path=/;max-age=31536000';
        document.getElementById('wz-cookie-consent').remove();
    }
    </script>
    <?php
}
add_action( 'wp_footer', 'wz_cookie_consent_banner' );
```

## Why
Google Consent Mode v2 (mandatory March 2024) requires: (1) consent banner with genuine choice, (2) default deny for analytics_storage + ad_storage, (3) update consent state on user action. Without it: Google Ads rejects landing pages, Meta restricts ad accounts, GDPR fines up to 4% revenue for EU-targeting. Blueprint Section 7 specifies full consent integration.
