**Pattern sai:**
```php
<li class="font-bold text-[var(--wz-error)]">Thanh toán thất bại</li>
```
**Phải làm:**
```php
<li class="font-bold text-[var(--wz-error)]" aria-current="page">Thanh toán thất bại</li>
```
**Lý do:** `aria-current="page"` cho screen readers biết đây là trang hiện tại trong breadcrumb trail. Thiếu attribute này khiến user dùng assistive technology không phân biệt được item nào là active. Áp dụng cho MỌI breadcrumb `<nav aria-label="Breadcrumb">` — item cuối cùng (không phải link) luôn cần `aria-current="page"`.
**Grep check:** `aria-label="Breadcrumb"` trong file → verify item cuối có `aria-current="page"` — nếu thiếu → HIGH
