## Bad
```php
$cache_key = "wz_loyalty_tier_{$user_id}";
wp_cache_set( $cache_key, $tier ); // group = 'default' -> collision risk
```

## Good
```php
$cache_key = "wz_loyalty_tier_{$user_id}";
wp_cache_set( $cache_key, $tier, 'wezone-loyalty', HOUR_IN_SECONDS );
```

## Why
wp_cache_set() without group uses 'default' group. Key collision with other plugins (e.g. 'user_123' could be loyalty tier or user meta). Must use unique group.

## Grep
`wp_cache_set\s*\([^,]+,\s*[^,]+\s*\)` — in `**/*.php`
