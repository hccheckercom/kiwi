## Bad
```tsx
// Section A — gap-4
<div className="grid gap-4">

// Section B — gap-6
<div className="grid gap-6">
```

## Good
```tsx
// Consistent spacing scale
<div className="grid gap-4">
```

## Why
Sibling sections nên dùng cùng spacing scale. gap-4 (16px) cho cards, gap-2 (8px) cho compact lists. Mixing gap-4/gap-6/gap-8 random gây visual noise.
