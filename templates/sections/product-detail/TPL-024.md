# Product Image Gallery — Lightbox Slider

Product image gallery with thumbnail navigation and lightbox zoom.

## Features
- Main image display
- Thumbnail navigation
- Lightbox/modal on click
- Keyboard navigation (arrow keys)
- Mobile swipe support
- Fallback icon if no images

## Code

```php
<?php
$images = $product['gallery'] ?? [];
$main_image = $images[0] ?? null;
?>

<div class="product-gallery">
    <div class="product-gallery__main">
        <?php if ( $main_image ) : ?>
            <img src="<?php echo esc_url( $main_image['url'] ); ?>" 
                 alt="<?php echo esc_attr( $product['name'] ); ?>"
                 class="product-gallery__main-img"
                 data-lightbox="product">
        <?php else : ?>
            <div class="product-gallery__placeholder">
                <span class="material-symbols-outlined">image</span>
            </div>
        <?php endif; ?>
    </div>

    <?php if ( count( $images ) > 1 ) : ?>
        <div class="product-gallery__thumbs">
            <?php foreach ( $images as $index => $img ) : ?>
                <button type="button" 
                        class="product-gallery__thumb <?php echo 0 === $index ? 'active' : ''; ?>"
                        data-index="<?php echo esc_attr( (string) $index ); ?>">
                    <img src="<?php echo esc_url( $img['thumb'] ?? $img['url'] ); ?>" 
                         alt="<?php echo esc_attr( sprintf( 'Image %d', $index + 1 ) ); ?>">
                </button>
            <?php endforeach; ?>
        </div>
    <?php endif; ?>
</div>

<!-- Lightbox modal -->
<div id="product-lightbox" class="product-lightbox hidden">
    <button class="product-lightbox__close" aria-label="Close">
        <span class="material-symbols-outlined">close</span>
    </button>
    <button class="product-lightbox__prev" aria-label="Previous">
        <span class="material-symbols-outlined">chevron_left</span>
    </button>
    <button class="product-lightbox__next" aria-label="Next">
        <span class="material-symbols-outlined">chevron_right</span>
    </button>
    <img id="product-lightbox-img" src="" alt="">
</div>
```

## Usage

```php
// In single-wz_product.php
get_template_part( 'template-parts/product/gallery', null, [
    'product' => $product,
] );
```

## JavaScript Required
- Thumbnail click → update main image
- Main image click → open lightbox
- Lightbox navigation (prev/next, keyboard, close)
- Mobile swipe gestures
