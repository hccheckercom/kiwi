## Preview

Grid danh mục dạng icon tròn + tên bên dưới. Mobile 4 cột, tablet 6, desktop 8. Mỗi item có border nhẹ, hover đổi border color + shadow. Dùng Material Symbols icons, fallback sang thumbnail nếu có.

## Files

| File | Role |
|------|------|
| template-parts/home/categories.php | Template chính |

## Code — PHP

```php
<?php
$icon_map = array(
    'phong-khach'  => 'weekend',
    'phong-ngu'    => 'bed',
    'ban-an'       => 'dining',
    'van-phong'    => 'desk',
    'trang-tri'    => 'light_mode',
    'den-chieu'    => 'lamp',
    'luu-tru'      => 'shelves',
    'ngoai-troi'   => 'deck',
    'default'      => 'category',
);

$categories = get_terms( array(
    'taxonomy'   => 'product_cat',
    'hide_empty' => true,
    'number'     => 8,
    'orderby'    => 'count',
    'order'      => 'DESC',
    'exclude'    => get_option( 'default_product_cat', 0 ),
) );

if ( is_wp_error( $categories ) || empty( $categories ) ) {
    $categories = array(
        (object) array( 'term_id' => 0, 'slug' => 'phong-khach', 'name' => 'Phòng khách', 'link' => home_url( '/phong-khach/' ) ),
        (object) array( 'term_id' => 0, 'slug' => 'phong-ngu',   'name' => 'Phòng ngủ',   'link' => home_url( '/phong-ngu/' ) ),
        // ... more static fallbacks
    );
}
?>

<section style="padding:40px 24px;max-width:1280px;margin:0 auto;">
    <div class="haven-section-header">
        <p class="haven-eyebrow"><?php esc_html_e( 'Khám phá', 'wezone-haven' ); ?></p>
        <h2 class="haven-heading"><?php esc_html_e( 'Danh mục', 'wezone-haven' ); ?></h2>
    </div>

    <div style="display:grid;grid-template-columns:repeat(4,1fr);gap:12px;" class="haven-cat-grid">
        <?php foreach ( $categories as $cat ) :
            $slug = $cat->slug;
            $icon = $icon_map[ $slug ] ?? $icon_map['default'];
            $thumb_id = isset( $cat->term_id ) && $cat->term_id ? get_term_meta( $cat->term_id, 'thumbnail_id', true ) : 0;
            $link = isset( $cat->link ) ? $cat->link : get_term_link( $cat );
        ?>
            <a href="<?php echo esc_url( $link ); ?>"
                style="display:flex;flex-direction:column;align-items:center;gap:10px;padding:20px 8px;text-decoration:none;border:1px solid var(--wz-border,#cdc6b7);background:var(--wz-surface,#f4f3f3);border-radius:8px;transition:border-color 0.2s,box-shadow 0.2s;text-align:center;"
                onmouseover="this.style.borderColor='var(--wz-primary)';this.style.boxShadow='0 4px 16px rgba(81,70,31,0.1)'"
                onmouseout="this.style.borderColor='var(--wz-border)';this.style.boxShadow='none'">

                <?php if ( $thumb_id ) : ?>
                    <?php echo wp_get_attachment_image( (int) $thumb_id, 'thumbnail', false, array( 'style' => 'width:48px;height:48px;object-fit:cover;border-radius:50%;' ) ); ?>
                <?php else : ?>
                    <div style="width:52px;height:52px;border-radius:50%;background:var(--wz-surface,#F5F0E8);display:flex;align-items:center;justify-content:center;">
                        <span class="material-symbols-outlined" aria-hidden="true" style="font-size:26px;color:var(--wz-primary,#8B7355);"><?php echo esc_html( $icon ); ?></span>
                    </div>
                <?php endif; ?>

                <span style="font-size:12px;font-weight:500;color:var(--wz-text,#3D2B1F);line-height:1.3;">
                    <?php echo esc_html( $cat->name ); ?>
                </span>
            </a>
        <?php endforeach; ?>
    </div>
</section>

<style>
@media (min-width: 768px)  { .haven-cat-grid { grid-template-columns: repeat(6, 1fr) !important; } }
@media (min-width: 1024px) { .haven-cat-grid { grid-template-columns: repeat(8, 1fr) !important; } }
</style>
```

## Usage Notes

- Icon map cần customize theo ngành hàng (furniture, beauty, food...)
- Fallback static categories khi chưa có data thật
- Dùng `product_cat` taxonomy — đổi nếu dùng custom taxonomy
- Thumbnail từ term meta `thumbnail_id`
