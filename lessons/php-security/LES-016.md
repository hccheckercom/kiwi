**Pattern sai:**
```php
// returns.php — lấy order từ GET param, không check ownership
$order_id = (int) ( $_GET['order_id'] ?? 0 );
$order    = wz_get_order( $order_id );
// → user A submit return request cho đơn hàng của user B
```
**Phải làm:**
```php
$order_id = (int) ( $_GET['order_id'] ?? 0 );
$order    = $order_id ? wz_get_order( $order_id ) : null;
if ( $order && (int) ( $order['user_id'] ?? 0 ) !== $user_id ) {
    $order = null;
}
```
**Lý do:** IDOR — bất kỳ order page nào nhận order_id từ URL ($_GET) hoặc form ($_POST) đều phải verify `$order['user_id'] === get_current_user_id()`. Áp dụng cho: order-detail, returns, tracking, order-cancel, refund request.
**Grep check:** `\$_GET\['order_id'\]|\$_POST\['order_id'\]` + thiếu `user_id.*get_current_user_id` trong cùng file → CRITICAL
