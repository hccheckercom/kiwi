# Cart Page — Item List + Order Summary

Shopping cart page with product list, quantity controls, order summary sidebar, and support contact.

## Features
- Product list with images
- Quantity +/- controls
- Remove item button
- Order summary with subtotal/shipping/total
- Free shipping threshold message
- Support contact card (phone + Zalo)
- Empty cart state
- Breadcrumb navigation

## Code

```php
<?php
defined( 'ABSPATH' ) || exit;

get_header();

$cart_items    = function_exists( 'wz_get_cart_items' ) ? wz_get_cart_items() : [];
$cart_count    = count( $cart_items );
$is_empty      = empty( $cart_items );
$shop_url      = function_exists( 'wz_get_shop_url' ) ? wz_get_shop_url() : home_url( '/shop/' );
$checkout_url  = function_exists( 'wz_get_checkout_url' ) ? wz_get_checkout_url() : home_url( '/checkout/' );
$phone         = function_exists( 'wz_config' ) ? wz_config( 'phone', '' ) : '';
$zalo_url      = function_exists( 'wz_config' ) ? wz_config( 'zalo_url', '' ) : '';
$free_threshold = function_exists( 'wz_config' ) ? (int) wz_config( 'shipping.free_ship_threshold', 5000000 ) : 5000000;
?>

<main id="main-content" class="haven-cart">

    <!-- Breadcrumb -->
    <nav class="haven-shop__breadcrumb" aria-label="<?php esc_attr_e( 'Breadcrumb', 'wezone-haven' ); ?>">
        <a href="<?php echo esc_url( home_url( '/' ) ); ?>"><?php esc_html_e( 'Trang chủ', 'wezone-haven' ); ?></a>
        <span aria-hidden="true">/</span>
        <span aria-current="page"><?php esc_html_e( 'Giỏ hàng', 'wezone-haven' ); ?></span>
    </nav>

    <?php if ( ! $is_empty ) : ?>

        <h1 class="haven-cart__title">
            <?php esc_html_e( 'Giỏ hàng', 'wezone-haven' ); ?>
            <span class="haven-cart__count">(<?php echo esc_html( (string) $cart_count ); ?>)</span>
        </h1>

        <div class="haven-cart__layout">
            <!-- Product List -->
            <div class="haven-cart__items">
                <?php foreach ( $cart_items as $key => $item ) :
                    $product_id = (int) ( $item['product_id'] ?? 0 );
                    $name       = $item['name'] ?? get_the_title( $product_id );
                    $price      = (float) ( $item['price'] ?? 0 );
                    $qty        = (int) ( $item['quantity'] ?? 1 );
                    $subtotal   = $price * $qty;
                    $image_id   = (int) ( $item['image_id'] ?? get_post_thumbnail_id( $product_id ) );
                    $permalink  = home_url( '/product/' . ( $item['slug'] ?? '' ) . '/' );
                ?>
                    <div class="haven-cart__item" data-key="<?php echo esc_attr( $key ); ?>">
                        <div class="haven-cart__item-img">
                            <?php if ( $image_id ) : ?>
                                <?php echo wp_get_attachment_image( $image_id, 'thumbnail', false, array( 'class' => 'haven-cart__item-photo' ) ); ?>
                            <?php else : ?>
                                <span class="material-symbols-outlined" aria-hidden="true">chair</span>
                            <?php endif; ?>
                        </div>
                        <div class="haven-cart__item-info">
                            <a href="<?php echo esc_url( $permalink ); ?>" class="haven-cart__item-name"><?php echo esc_html( $name ); ?></a>
                            <span class="haven-cart__item-price">
                                <?php echo function_exists( 'wz_format_price' ) ? esc_html( wz_format_price( $price ) ) : esc_html( number_format( $price, 0, ',', '.' ) ) . 'đ'; ?>
                            </span>
                            <div class="haven-cart__item-qty">
                                <button class="haven-cart__qty-btn" data-action="decrease" aria-label="<?php esc_attr_e( 'Giảm', 'wezone-haven' ); ?>" <?php echo $qty <= 1 ? 'disabled' : ''; ?>>
                                    <span class="material-symbols-outlined" aria-hidden="true">remove</span>
                                </button>
                                <span class="haven-cart__qty-val"><?php echo esc_html( (string) $qty ); ?></span>
                                <button class="haven-cart__qty-btn" data-action="increase" aria-label="<?php esc_attr_e( 'Tăng', 'wezone-haven' ); ?>">
                                    <span class="material-symbols-outlined" aria-hidden="true">add</span>
                                </button>
                            </div>
                        </div>
                        <div class="haven-cart__item-right">
                            <span class="haven-cart__item-subtotal">
                                <?php echo function_exists( 'wz_format_price' ) ? esc_html( wz_format_price( $subtotal ) ) : esc_html( number_format( $subtotal, 0, ',', '.' ) ) . 'đ'; ?>
                            </span>
                            <button class="haven-cart__item-remove" aria-label="<?php esc_attr_e( 'Xóa', 'wezone-haven' ); ?>">
                                <span class="material-symbols-outlined" aria-hidden="true">close</span>
                            </button>
                        </div>
                    </div>
                <?php endforeach; ?>
            </div>

            <!-- Sidebar -->
            <aside class="haven-cart__sidebar">
                <div class="haven-cart__summary">
                    <h3 class="haven-cart__summary-title"><?php esc_html_e( 'Tóm tắt đơn hàng', 'wezone-haven' ); ?></h3>
                    <?php
                    $subtotal_all = 0;
                    foreach ( $cart_items as $item ) {
                        $subtotal_all += (float) ( $item['price'] ?? 0 ) * (int) ( $item['quantity'] ?? 1 );
                    }
                    $shipping = $subtotal_all >= $free_threshold ? 0 : 30000;
                    $total    = $subtotal_all + $shipping;
                    ?>
                    <div class="haven-cart__summary-row">
                        <span><?php esc_html_e( 'Tạm tính', 'wezone-haven' ); ?></span>
                        <span><?php echo function_exists( 'wz_format_price' ) ? esc_html( wz_format_price( $subtotal_all ) ) : esc_html( number_format( $subtotal_all, 0, ',', '.' ) ) . 'đ'; ?></span>
                    </div>
                    <div class="haven-cart__summary-row">
                        <span><?php esc_html_e( 'Vận chuyển', 'wezone-haven' ); ?></span>
                        <span><?php echo 0 === $shipping ? esc_html__( 'Miễn phí', 'wezone-haven' ) : esc_html( number_format( $shipping, 0, ',', '.' ) . 'đ' ); ?></span>
                    </div>
                    <div class="haven-cart__summary-row haven-cart__summary-row--total">
                        <span><?php esc_html_e( 'Tổng cộng', 'wezone-haven' ); ?></span>
                        <span><?php echo function_exists( 'wz_format_price' ) ? esc_html( wz_format_price( $total ) ) : esc_html( number_format( $total, 0, ',', '.' ) ) . 'đ'; ?></span>
                    </div>
                    <a href="<?php echo esc_url( $checkout_url ); ?>" class="haven-cart__checkout-btn">
                        <?php esc_html_e( 'Tiến hành thanh toán', 'wezone-haven' ); ?>
                    </a>
                    <p class="haven-cart__free-note">
                        <?php echo esc_html( sprintf( __( 'Miễn phí vận chuyển đơn từ %s', 'wezone-haven' ), number_format( $free_threshold, 0, ',', '.' ) . 'đ' ) ); ?>
                    </p>
                </div>

                <div class="haven-cart__support">
                    <span class="material-symbols-outlined" aria-hidden="true">support_agent</span>
                    <div>
                        <strong><?php esc_html_e( 'Cần hỗ trợ?', 'wezone-haven' ); ?></strong>
                        <?php if ( $phone ) : ?>
                            <a href="tel:<?php echo esc_attr( preg_replace( '/\s+/', '', $phone ) ); ?>"><?php echo esc_html( $phone ); ?></a>
                        <?php endif; ?>
                        <?php if ( $zalo_url ) : ?>
                            <a href="<?php echo esc_url( $zalo_url ); ?>" target="_blank" rel="noopener noreferrer"><?php esc_html_e( 'Chat Zalo', 'wezone-haven' ); ?></a>
                        <?php endif; ?>
                        <span class="haven-cart__support-hours"><?php esc_html_e( '8:00 - 21:00 hàng ngày', 'wezone-haven' ); ?></span>
                    </div>
                </div>
            </aside>
        </div>

    <?php else : ?>

        <div class="haven-shop__empty">
            <span class="material-symbols-outlined" aria-hidden="true">shopping_cart</span>
            <h1><?php esc_html_e( 'Giỏ hàng trống', 'wezone-haven' ); ?></h1>
            <p><?php esc_html_e( 'Hãy khám phá các sản phẩm nội thất tuyệt vời của chúng tôi.', 'wezone-haven' ); ?></p>
            <a href="<?php echo esc_url( $shop_url ); ?>" class="haven-shop__empty-cta">
                <?php esc_html_e( 'Tiếp tục mua sắm', 'wezone-haven' ); ?>
            </a>
        </div>

    <?php endif; ?>

</main>

<?php get_footer(); ?>
```

## Usage

Create as `wezone-templates/cart/cart.php`

## CSS Classes
- `.haven-cart__*` — cart-specific BEM classes
- Material Icons for UI elements

## JavaScript Required
- Quantity update AJAX
- Remove item AJAX
- Update cart totals
