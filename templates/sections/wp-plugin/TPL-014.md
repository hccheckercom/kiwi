## Mô tả

Plugin standalone inject một floating bar vào `wp_footer` của bất kỳ theme nào đang active. Không cần sửa theme. Chỉ hiển thị khi domain match hoặc có query param `?demo-bar=1`.

**Pattern tái dụng cho:** affiliate bar, announcement bar, survey bar, cookie consent bar.

## Code

```php
<?php
/**
 * Plugin Name: Wezone Demo Bar
 * Description: Floating CTA bar hiển thị trên demo sites — tự động inject vào mọi theme.
 * Version: 1.0.0
 */

defined( 'ABSPATH' ) || exit;

class Wezone_Demo_Bar {

    private const API_URL  = 'https://wezone.vn/api/wordpress/themes';
    private const ZALO_URL = 'https://zalo.me/wezone';
    private const SITE_URL = 'https://wezone.vn';

    public function boot(): void {
        if ( ! $this->should_show() ) return;
        add_action( 'wp_footer', [ $this, 'render' ] );
    }

    private function should_show(): bool {
        $host = $_SERVER['HTTP_HOST'] ?? '';
        return str_contains( $host, 'wp.wezone.vn' ) || isset( $_GET['demo-bar'] );
    }

    private function get_slug(): string {
        $option = get_option( 'wezone_demo_slug', '' );
        if ( $option ) return sanitize_key( $option );

        $text_domain = wp_get_theme()->get( 'TextDomain' );
        if ( $text_domain && str_starts_with( $text_domain, 'wezone-' ) ) {
            return sanitize_key( substr( $text_domain, strlen( 'wezone-' ) ) );
        }
        return '';
    }

    private function get_theme_data( string $slug ): ?array {
        $cache_key = 'wezone_demo_bar_' . $slug;
        $cached    = get_transient( $cache_key );
        if ( $cached !== false ) return $cached ?: null;

        $response = wp_remote_get(
            self::API_URL . '?slug=' . rawurlencode( $slug ),
            [ 'timeout' => 5 ]
        );

        if ( is_wp_error( $response ) || wp_remote_retrieve_response_code( $response ) !== 200 ) {
            set_transient( $cache_key, [], HOUR_IN_SECONDS );
            return null;
        }

        $data = json_decode( wp_remote_retrieve_body( $response ), true );
        if ( ! is_array( $data ) || empty( $data['name'] ) ) {
            set_transient( $cache_key, [], HOUR_IN_SECONDS );
            return null;
        }

        set_transient( $cache_key, $data, HOUR_IN_SECONDS );
        return $data;
    }

    public function render(): void {
        $slug = $this->get_slug();
        if ( ! $slug ) return;

        $data = $this->get_theme_data( $slug );
        if ( ! $data ) return;

        $name       = esc_html( $data['name'] ?? '' );
        $industry   = esc_html( $data['industry'] ?? '' );
        $price      = esc_html( $data['price'] ?? 'Từ 1.700.000đ' );
        $accent     = esc_attr( $data['accent'] ?? '#e3970d' );
        $detail_url = esc_url( self::SITE_URL . '/wordpress/' . $slug );
        $zalo_url   = esc_url( self::ZALO_URL );
        ?>
        <style>
        #wz-demo-bar {
            position: fixed; left: 0; right: 0; z-index: 99999;
            background: #fff; border-top: 3px solid <?php echo $accent; ?>;
            box-shadow: 0 -2px 12px rgba(0,0,0,.12);
            display: flex; align-items: center; gap: 12px; padding: 10px 20px;
            font-family: 'Be Vietnam Pro', sans-serif; font-size: 14px;
        }
        @media (min-width: 768px) {
            #wz-demo-bar { bottom: 0; top: auto; }
            body { padding-bottom: 60px !important; }
        }
        @media (max-width: 767px) {
            #wz-demo-bar { top: 0; bottom: auto; flex-wrap: wrap; padding: 8px 12px; font-size: 13px; }
            body { padding-top: 52px !important; }
        }
        #wz-demo-bar .wz-bar-info { display: flex; align-items: center; gap: 8px; flex: 1; min-width: 0; }
        #wz-demo-bar .wz-bar-dot { width: 10px; height: 10px; border-radius: 50%; background: <?php echo $accent; ?>; flex-shrink: 0; }
        #wz-demo-bar .wz-bar-name { font-weight: 700; color: #0f172a; white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }
        #wz-demo-bar .wz-bar-industry { color: #94a3b8; font-size: 12px; white-space: nowrap; }
        #wz-demo-bar .wz-bar-price { font-weight: 700; color: <?php echo $accent; ?>; white-space: nowrap; }
        #wz-demo-bar .wz-bar-actions { display: flex; align-items: center; gap: 8px; flex-shrink: 0; }
        #wz-demo-bar .wz-bar-back { color: #64748b; text-decoration: none; font-size: 13px; white-space: nowrap; }
        #wz-demo-bar .wz-bar-back:hover { color: #0f172a; }
        #wz-demo-bar .wz-bar-cta {
            background: <?php echo $accent; ?>; color: #fff;
            padding: 8px 18px; border-radius: 8px; font-weight: 700;
            text-decoration: none; white-space: nowrap; transition: opacity .15s;
        }
        #wz-demo-bar .wz-bar-cta:hover { opacity: .85; }
        </style>
        <div id="wz-demo-bar">
            <div class="wz-bar-info">
                <span class="wz-bar-dot"></span>
                <span class="wz-bar-name"><?php echo $name; ?></span>
                <span class="wz-bar-industry"><?php echo $industry; ?></span>
            </div>
            <span class="wz-bar-price"><?php echo $price; ?></span>
            <div class="wz-bar-actions">
                <a href="<?php echo $detail_url; ?>" class="wz-bar-back" target="_blank">← Chi tiết</a>
                <a href="<?php echo $zalo_url; ?>" class="wz-bar-cta" target="_blank" rel="noreferrer">Mua ngay</a>
            </div>
        </div>
        <?php
    }
}

( new Wezone_Demo_Bar() )->boot();
```

## Điểm quan trọng

- **Detect slug:** ưu tiên `get_option('wezone_demo_slug')` → fallback parse TextDomain (`wezone-funilux` → `funilux`)
- **Cache:** `set_transient()` 1h — tránh HTTP request mỗi pageview
- **Conditional display:** chỉ show trên `wp.wezone.vn` hoặc `?demo-bar=1` — không ảnh hưởng site thật
- **Body padding:** JS-free — dùng CSS `!important` để push content xuống/lên
- **API source:** `https://wezone.vn/api/wordpress/themes?slug=X` — quản lý giá/tên tập trung tại Next.js
