## Bad
```php
logger.info(f"Login: user={user}, password={password}")
```

## Good
```php
logger.info(f"Login: user={user}, password=***")
```

## Why
Logging sensitive data exposes it in log files, monitoring, and error tracking

## Grep
`(log|print).*password` — in `**/*.py`
