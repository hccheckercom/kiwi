## Bad
```tsx
{ num: '4.9★', label: 'Đánh giá' }
<span>4.9★ Đánh giá tốt</span>
```
Displaying a rating without source platform, review count, or verification link violates Google "Misrepresentation" policy and Meta "Misleading Claims."

## Good
```tsx
// Option 1: Link to verifiable source
<a href="https://g.page/wezone-vn/review">4.9★ trên Google Maps (23 đánh giá)</a>

// Option 2: Replace with verifiable metric
{ num: '200+', label: 'Khách hàng' }

// Option 3: Schema markup with proper aggregateRating
{ '@type': 'AggregateRating', ratingValue: '4.9', reviewCount: '23', bestRating: '5' }
```

## Why
Unverified ratings are considered "fabricated social proof" by both Google and Meta. The rating must be: (1) from a named third-party platform, (2) include sample size, (3) link to verification. Self-reported ratings without these elements trigger manual review and potential suspension.
