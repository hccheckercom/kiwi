**Required keys:**
```
name (string, non-empty)
theme.primary (string, hex color /^#[0-9A-Fa-f]{3,6}$/)
seo.site_url (string)
shipping.free_ship_threshold (numeric, > 0)
shipping.fees.default (numeric)
phone (string)
email (string)
tagline (string)
```
**Pattern sai:**
```php
// store-config.php thiếu key 'seo' hoặc 'theme.primary' không phải hex
return [
    'name' => 'Shop Name',
    'theme' => [ 'primary' => 'red' ], // ← không phải hex
    // thiếu 'seo', 'shipping', 'phone', 'email'
];
```
**Phải làm:**
```php
return [
    'name'     => 'Trung Anh Mobile',
    'tagline'  => 'Điện thoại chính hãng',
    'theme'    => [ 'primary' => '#e63946' ],
    'seo'      => [ 'site_url' => 'https://trunganh.vn' ],
    'shipping' => [ 'free_ship_threshold' => 2000000, 'fees' => [ 'default' => 50000 ] ],
    'phone'    => '0123456789',
    'email'    => 'info@trunganh.vn',
];
```
**Lý do:** CSS tokens đọc `theme.primary` để generate `--wz-primary`. Thiếu key → token rỗng → UI mất màu. Thiếu seo/shipping → fatal khi template gọi `wz_config('seo.site_url')`.
**Grep check:** Validate store-config.php trả array có đủ keys + đúng type → HIGH
