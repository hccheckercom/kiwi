**Mô tả:**
Blueprint yêu cầu skeleton loading (pulse animation) hiển thị trong khi data đang fetch. Hiện tại CSS có `.skeleton` class nhưng không có PHP template component để render skeleton cards.

**Implementation gợi ý:**
```php
<!-- template-parts/components/skeleton-card.php -->
<div class="skeleton-card animate-pulse">
    <div class="aspect-square bg-gray-200 rounded-lg"></div>
    <div class="mt-3 space-y-2">
        <div class="h-4 bg-gray-200 rounded w-3/4"></div>
        <div class="h-4 bg-gray-200 rounded w-1/2"></div>
        <div class="h-5 bg-gray-200 rounded w-1/3"></div>
    </div>
</div>
```

**Sử dụng:** Render skeleton cards trước khi AJAX load products, hoặc dùng `<template>` tag cho JS-driven loading.
