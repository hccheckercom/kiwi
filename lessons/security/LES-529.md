## Bad
```php
subprocess.run(['cmd'], check=True)
```

## Good
```php
subprocess.run(['cmd'], check=True, timeout=30)
```

## Why
Subprocess without timeout can hang indefinitely

## Grep
`subprocess\.(run|call|Popen)(?!.*timeout)` — in `**/*.php`
