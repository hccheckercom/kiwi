## Bad
```php
import sys
from pathlib import Path

KIWI_DIR = Path(__file__).parent.parent
sys.path.insert(0, str(KIWI_DIR))

from scanner.cli import scan_theme
```

## Good
```php
import sys
from pathlib import Path

# Add parent to path for relative imports
KIWI_DIR = Path(__file__).parent.parent
if str(KIWI_DIR) not in sys.path:
    sys.path.insert(0, str(KIWI_DIR))

from scanner.cli import scan_theme
```

## Why
Calling sys.path.insert(0, path) multiple times (e.g., when module is imported multiple times) adds duplicate entries to sys.path. This wastes memory and slows down import resolution. Check if path already exists before inserting.

## Grep
`sys\.path\.insert\(0,.*\)(?!.*if.*not in sys\.path)` — in `**/*.py`
