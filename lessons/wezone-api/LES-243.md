# LES-243: Hero Banner Hardcoded — Phải Load Từ Plugin

## Pattern sai

```php
// template-parts/hero/default.php — hardcode nội dung
$hero_img = WEZONE_THEME_URL . '/assets/images/hero-banner.jpg';
?>
<section>
    <img src="<?php echo esc_url( $hero_img ); ?>" />
    <h1>Bộ sưu tập Tối Giản 2025</h1>
    <p>Mô tả hardcode...</p>
    <a href="/shop/">Xem bộ sưu tập</a>
</section>
```

## Pattern đúng

```php
// inc/banner-helpers.php — shim khi plugin chưa active
if ( ! function_exists( 'wz_get_banners' ) ) :
function wz_get_banners( string $position ): array {
    global $wpdb;
    $table = $wpdb->prefix . 'wz_banners';
    $now = current_time( 'mysql' );
    $results = $wpdb->get_results(
        $wpdb->prepare(
            "SELECT * FROM {$table}
            WHERE position = %s AND status = 'active'
            AND (start_date IS NULL OR start_date <= %s)
            AND (end_date IS NULL OR end_date >= %s)
            ORDER BY sort_order ASC",
            $position, $now, $now
        ), ARRAY_A
    );
    // ... decode meta, resolve references
    return $results ?: [];
}
endif;

// template-parts/hero/default.php — dynamic từ plugin
$hero_banners = function_exists( 'wz_get_banners' ) ? wz_get_banners( 'home_hero' ) : [];
$hero = ! empty( $hero_banners ) ? $hero_banners[0] : null;

if ( $hero ) {
    $hero_img   = $hero['image_url'] ?? '';
    $hero_title = $hero['title'] ?? '';
    $hero_cta   = $hero['cta_text'] ?? 'Mua ngay';
    $hero_link  = $hero['link_url'] ?? $shop_url;
}
// Fallback static image nếu admin chưa config
```

## Lý do

Banner Marketing plugin (`wezone-core`) cho phép admin quản lý banner từ dashboard:
- Thay đổi ảnh, title, CTA, link mà không cần sửa code
- Hẹn giờ hiển thị (start_date / end_date)
- Chọn layout (hero đơn, slider, 2 mini + 1 lớn)
- Gắn sản phẩm/danh mục reference

Hardcode = admin bất lực, mỗi lần đổi banner phải nhờ dev.

## Grep check

```
# Tìm hero template không có wz_get_banners
grep -rL "wz_get_banners" themes/*/template-parts/hero/
```

## Checklist

- [ ] `inc/banner-helpers.php` tồn tại (shim function)
- [ ] `functions.php` require `banner-helpers.php`
- [ ] Hero template gọi `wz_get_banners('home_hero')`
- [ ] Có fallback khi admin chưa tạo banner
- [ ] Product cards đọc từ `$hero['meta']['featured_products']` hoặc fallback `wz_get_products()`
