**Pattern sai:**
```php
// wezone-templates/product/single.php
$product_id = (int) ( $_GET['product_id'] ?? 0 );
$product = wz_get_product( $product_id );
// → product_id luôn = 0 vì Router không dùng query string
```
**Phải làm:**
```php
// TemplateLoader::set('product_id', 42) → bind vào query var 'wz_tpl_product_id'
$product_id = (int) get_query_var( 'wz_tpl_product_id', 0 );
$product = wz_get_product( $product_id );
```
**Lý do:** Wezone Router + TemplateLoader bind data vào WordPress query vars với prefix `wz_tpl_`. Template KHÔNG nhận data qua superglobals ($_GET, $_POST, $_REQUEST). Dùng superglobals → data luôn empty → page blank hoặc hiển thị sai.
**Grep check:** `\$_GET\['product_id'\]|\$_GET\['category'\]|\$_GET\['slug'\]` trong wezone-templates/ — phải dùng `get_query_var('wz_tpl_*')` thay thế → HIGH
