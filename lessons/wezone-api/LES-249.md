## Problem

When proxying WordPress Multisite through nginx to serve under a different URL path (e.g., `wezone.vn/wordpress/[slug]/demo` → `wp.wezone.vn/[slug]/`), using `fastcgi_pass` directly causes multiple failures:

1. `try_files $uri $uri/ /index.php` — fallback triggers internal redirect to `location /` which proxies to Next.js instead of PHP-FPM
2. Even without try_files, `SCRIPT_FILENAME`, `REQUEST_URI`, `SCRIPT_NAME` all need manual overrides
3. WordPress Multisite uses `HTTP_HOST` + `REQUEST_URI` to identify the blog — getting either wrong = 302 redirect to main site or 404

## Solution

Use `proxy_pass` to the existing wp.wezone.vn server block (which already handles multisite correctly), then rewrite URLs in the response:

```nginx
location ~ ^/wordpress/([^/]+)/demo(/.*)?$ {
    set $wp_slug $1;
    set $wp_path $2;

    rewrite ^/wordpress/([^/]+)/demo/?$ /$1/ break;
    rewrite ^/wordpress/([^/]+)/demo(/.+)$ /$1$2 break;

    proxy_pass https://127.0.0.1;
    proxy_set_header Host wp.wezone.vn;
    proxy_set_header Accept-Encoding "";  # Required for sub_filter
    proxy_set_header X-Forwarded-Host wezone.vn;
    proxy_ssl_verify off;
    proxy_ssl_server_name on;

    # Rewrite redirect headers
    proxy_redirect https://wp.wezone.vn/$wp_slug/ https://wezone.vn/wordpress/$wp_slug/demo/;

    # Rewrite URLs in response body
    sub_filter_once off;
    sub_filter_types text/html text/css application/javascript;
    sub_filter 'https://wp.wezone.vn/$wp_slug' 'https://wezone.vn/wordpress/$wp_slug/demo';
}
```

Key points:
- `Accept-Encoding ""` disables gzip from upstream — required for sub_filter to work
- `proxy_redirect` rewrites Location headers (301/302)
- `sub_filter` rewrites URLs in HTML/CSS/JS body
- WordPress runs normally on its original domain — no wp-config.php patches needed

## Why This Matters

Direct fastcgi_pass requires replicating all of WordPress Multisite's routing logic in nginx config. The proxy approach lets WordPress handle its own routing while nginx only does URL translation.
