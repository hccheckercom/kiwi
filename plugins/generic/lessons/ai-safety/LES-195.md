## Bad
```ts
const response = await anthropic.messages.create({
  model: 'claude-sonnet-4-6',
  messages: [{ role: 'user', content: prompt }]
  // No max_tokens → defaults to model maximum (8192 output tokens)
})
```
Each call costs up to $0.12 with no limit. A loop or retry bug can burn $100+ in minutes.

## Good
```ts
const response = await anthropic.messages.create({
  model: 'claude-sonnet-4-6',
  max_tokens: 1024, // Cap output to what's actually needed
  messages: [{ role: 'user', content: prompt }]
})

// Rate limit per user
const { success } = await rateLimiter.check(userId, { max: 10, window: '1h' })
if (!success) throw new Error('AI usage limit reached')
```

## Why
AI API costs scale with token usage. Without `max_tokens`, a single malformed prompt can generate maximum-length responses. Without rate limiting, a user (or bot) can trigger thousands of calls. Always set: max_tokens, per-user rate limits, and monthly budget alerts.
