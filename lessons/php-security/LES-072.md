## Bad
```php
register_rest_route( 'wezone/v1', '/login', [
    'methods' => 'POST',
    'permission_callback' => '__return_true',
    'callback' => 'handle_login',
]);
```

## Good
```php
register_rest_route( 'wezone/v1', '/login', [
    'methods' => 'POST',
    'permission_callback' => '__return_true',
    'callback' => 'handle_login',
]);
// In handle_login():
ApiRateLimit::check( 'login', 5, 60 ); // 5 attempts per 60s
```

## Why
Public endpoint without rate limiting — brute-force login, DDoS, or resource exhaustion attacks.
