## Bad
```php
fetch('/api/data').then(r => r.json()).then(handle);
// No .catch() — network error silently breaks UI
```

## Good
```php
fetch('/api/data')
  .then(r => r.json())
  .then(handle)
  .catch(err => {
    console.error('Request failed:', err);
    showErrorToast('Không thể tải dữ liệu');
  });
```

## Why
fetch() without .catch() — network failure, timeout, or server error silently breaks UI. User sees nothing happen.
