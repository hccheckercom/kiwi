## Bad
```php
$hour = (int) get_option( 'backup_hour', 3 );
$ts = strtotime( "today {$hour}:00:00" ); // If $hour = "3; rm -rf /", command injection
```

## Good
```php
$hour = (int) get_option( 'backup_hour', 3 );
$hour = max( 0, min( 23, $hour ) ); // Clamp to 0-23
$ts = strtotime( sprintf( 'today %02d:00:00', $hour ) );
```

## Why
strtotime() can execute shell commands if input contains semicolons or backticks. User input (get_option, $_GET, $_POST) must be validated before passing to strtotime().

## Grep
`strtotime\([^)]*get_option\(|strtotime\([^)]*\$_GET|strtotime\([^)]*\$_POST` — in `**/*.php`
