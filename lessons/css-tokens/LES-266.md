## Bad
```php
<div style="background:#fff;padding:16px;">
```
White background hardcoded — invisible in dark mode (white on dark-gray).

## Good
```php
<div style="background:var(--color-surface,#fff);padding:16px;">
```
Uses semantic token with fallback — auto-adapts to dark mode.

## Why
Dark mode constitution rule #3: "NEVER use white/black directly — use `var(--color-surface)` / `var(--color-text)`". Hardcoded `#fff` backgrounds become invisible white boxes in dark mode. Always use semantic tokens: `--color-surface`, `--color-bg`, `--color-surface-raised`.
