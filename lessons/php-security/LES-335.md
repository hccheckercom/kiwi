**Pattern sai:**
```php
// change_password() — không return fresh nonce
wp_update_user( array( 'ID' => $user->ID, 'user_pass' => $new_password ) );
do_action( 'wezone/customer/password_changed', $user->ID );
return new WP_REST_Response( array( 'success' => true ), 200 );
```
**Phải làm:**
```php
wp_update_user( array( 'ID' => $user->ID, 'user_pass' => $new_password ) );
do_action( 'wezone/customer/password_changed', $user->ID );
return new WP_REST_Response(
    array(
        'success' => true,
        'nonce'   => wp_create_nonce( 'wp_rest' ),
    ),
    200
);
```
**Lý do:** Password change có thể invalidate session/nonce — frontend cần fresh nonce để tiếp tục gọi REST API
**Grep check:** REST API password endpoint không có `wp_create_nonce` trong response → HIGH
