## Bad
```php
ip = request.headers.get('X-Forwarded-For', '').split(',')[0]
```

## Good
```php
ip = request.remote_addr  # or use trusted proxy config
```

## Why
X-Forwarded-For header can be spoofed by clients. Only trust behind a known proxy
