## Bad
```php
register_rest_route( 'wezone/v1', '/search', [
    'args' => [
        'q' => [ 'type' => 'string' ],  // no sanitize_callback
    ],
]);
```

## Good
```php
register_rest_route( 'wezone/v1', '/search', [
    'args' => [
        'q' => [
            'type' => 'string',
            'sanitize_callback' => 'sanitize_text_field',
            'validate_callback' => function( $v ) { return ! empty( $v ); },
        ],
    ],
]);
```

## Why
REST route args without sanitize_callback — raw user input reaches handler. Always sanitize at boundary.
