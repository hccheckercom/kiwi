**Pattern sai:**
```php
// wz-shims.php khai báo:
function wz_get_categories( int $parent_id = 0 ): array { ... }

// Nhưng template gọi:
$cats = wz_get_categories( ['limit' => 4, 'parent' => 0] );
// → PHP TypeError: Argument #1 must be int, array given
```
**Phải làm:**
```php
// Option A: Sửa shim cho nhận array options
function wz_get_categories( array $args = [] ): array {
    $defaults = ['parent' => 0, 'limit' => 0];
    $args = wp_parse_args( $args, $defaults );
    // ...
}

// Option B: Sửa caller cho đúng signature
$cats = wz_get_categories( 0 ); // rồi slice kết quả
```
**Lý do:** Signature mismatch giữa shim và caller → TypeError tại runtime
**Grep check:** So sánh signature trong `wz-shims.php` với cách gọi trong templates — mismatch = CRITICAL
