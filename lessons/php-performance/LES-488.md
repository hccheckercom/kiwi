## Bad
```php
<script src="https://www.googletagmanager.com/gtag/js?id=<?php echo esc_attr( $ga_id ); ?>"></script>
// No async â†’ blocks parser until script downloads + executes
```

## Good
```php
<script async src="https://www.googletagmanager.com/gtag/js?id=<?php echo esc_attr( $ga_id ); ?>"></script>
// async â†’ downloads in parallel, executes when ready
```

## Why
Tracking scripts without async/defer block HTML parsing and delay First Contentful Paint. GTM/GA/FB Pixel should load asynchronously. Only GTM noscript iframe needs to be synchronous.

## Grep
`<script(?!.*async).*googletagmanager|<script(?!.*async).*fbevents|<script(?!.*async).*analytics\.tiktok` — in `**/*.php`
