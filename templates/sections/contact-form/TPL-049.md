# Contact Form — Multi-Channel Contact Page

Contact page với 5 kênh liên hệ (hotline, email, Zalo, Facebook, địa chỉ) trong card grid layout.

## Use Cases
- Trang liên hệ GMC-compliant (yêu cầu bắt buộc)
- Multi-channel support page
- Customer service hub

## Features
- 📞 Hotline card với tel: link
- 📧 Email card với mailto: link
- 💬 Zalo card với zalo.me link
- 📘 Facebook card với social link
- 📍 Address card với giờ làm việc
- Responsive grid (1 col mobile, 2 cols desktop)
- Icon-first design với background color coding

## Data Sources

All data from `wz_config()`:
- `shop_name` — Tên shop
- `phone` — SĐT không format (0393331334)
- `phone_display` — SĐT có format (0393 331 334)
- `email` — Email hỗ trợ
- `facebook_url` — Facebook page URL
- `facebook_display` — Facebook display name
- `address` — Địa chỉ đầy đủ
- `business_hours` — Giờ làm việc (optional)

## Dependencies
- Tailwind CSS
- wz_config() function
- SVG icons inline

## Responsive Behavior
- Mobile: 1 column stack
- Desktop (md:): 2 columns grid
- Address card: full width on all breakpoints

## Accessibility
- Semantic HTML (h1, h3, p, a)
- tel: and mailto: links
- target="_blank" với rel="noreferrer" cho external links
- Color contrast compliant

## GMC Compliance
✅ Địa chỉ thực hiển thị rõ ràng
✅ Nhiều kênh liên hệ (phone, email, social)
✅ Giờ làm việc được ghi rõ
✅ Link từ footer (cần verify)
