## Bad
```php
re.match(r'(a+)+b', user_input)
```

## Good
```php
re.match(r'a+b', user_input)  # Remove nested quantifiers
```

## Why
Nested quantifiers cause exponential backtracking — ReDoS on untrusted input

## Grep
`re\.(match|search).*\(\.[*+]\)+` — in `**/*.py`
