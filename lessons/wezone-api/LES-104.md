## Bad
```php
$path = ABSPATH . 'wp-content/plugins/wezone-core/templates/';
```

## Good
```php
$path = WP_PLUGIN_DIR . '/wezone-core/templates/';
```

## Why
Hardcoded wp-content/plugins/ path — breaks when WP_CONTENT_DIR is customized.
