## Vấn đề

Khi dùng **cả** `wp_localize_script()` lẫn `wp_add_inline_script()` để truyền cùng dữ liệu (nonce, ajaxUrl, currentVals):

1. **Tăng payload size** — cùng data gửi 2 lần trong HTML
2. **Tăng attack surface** — 2 global variables chứa nonce thay vì 1
3. **Maintenance burden** — phải sync 2 nơi khi thay đổi

## Sai

```php
// Cách 1: wp_localize_script
wp_localize_script( 'wz-design-admin', 'wzDesign', array(
    'ajaxUrl'  => admin_url( 'admin-ajax.php' ),
    'nonceSave' => wp_create_nonce( 'wz_save_design_tokens' ),
) );

// Cách 2: wp_add_inline_script — CÙNG DATA
wp_add_inline_script( 'wz-design-admin',
    'window.wzDesignInline = ' . wp_json_encode( array(
        'ajaxUrl'  => admin_url( 'admin-ajax.php' ),
        'nonceSave' => wp_create_nonce( 'wz_save_design_tokens' ),
    ) ) . ';',
    'before'
);
```

## Đúng

```php
// Chỉ dùng 1 phương pháp — wp_localize_script đủ
wp_localize_script( 'wz-design-admin', 'wzDesign', array(
    'ajaxUrl'      => admin_url( 'admin-ajax.php' ),
    'nonceSave'    => wp_create_nonce( 'wz_save_design_tokens' ),
    'nonceReset'   => wp_create_nonce( 'wz_reset_design_tokens' ),
    'nonceHistory' => wp_create_nonce( 'wz_get_design_history' ),
    'nonceRestore' => wp_create_nonce( 'wz_restore_design_snapshot' ),
    'currentVals'  => $this->get_current_values(),
    'presets'      => $this->get_presets(),
) );
// KHÔNG cần wp_add_inline_script cho cùng data
```
