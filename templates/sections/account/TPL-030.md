# Account Address Book — Add/Edit/Delete

Address management page with list, add/edit forms, and delete functionality.

## Features
- Address list with default badge
- Add new address button
- Edit/delete actions per address
- Set default address
- Empty state
- Modal or inline form

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

if ( ! is_user_logged_in() ) {
    wp_safe_redirect( home_url( '/my-account/login/' ) );
    exit;
}

get_header();

$user_id = get_current_user_id();
$addresses = function_exists( 'wz_get_user_addresses' ) ? wz_get_user_addresses( $user_id ) : [];

set_query_var( 'wz_tpl_account_page', 'addresses' );
?>

<div class="bg-[var(--wz-surface)] min-h-screen">
<div class="max-w-[1280px] mx-auto px-5 md:px-[20px] py-4">

    <div class="flex flex-col lg:flex-row lg:gap-6">
        <?php include get_stylesheet_directory() . '/template-parts/account/sidebar.php'; ?>

        <div class="flex-1 min-w-0 mt-4 lg:mt-0">
            <div class="bg-[var(--wz-surface-pure)] rounded-[var(--wz-radius-card)] p-5 shadow-[0_4px_24px_rgba(27,58,122,0.08)]">
                
                <div class="flex items-center justify-between mb-6">
                    <h1 class="text-xl font-bold text-[var(--wz-primary)]">Sổ địa chỉ</h1>
                    <button id="add-address-btn" class="flex items-center gap-2 bg-[var(--wz-primary)] text-white px-4 py-2 rounded-[var(--wz-radius)] text-sm font-semibold hover:opacity-90 transition">
                        <span class="material-symbols-outlined text-[18px]" aria-hidden="true">add</span>
                        Thêm địa chỉ
                    </button>
                </div>

                <?php if ( ! empty( $addresses ) ) : ?>
                    <div class="grid grid-cols-1 md:grid-cols-2 gap-4">
                        <?php foreach ( $addresses as $addr ) : ?>
                            <div class="border border-[var(--wz-outline)]/20 rounded-[var(--wz-radius)] p-4 relative">
                                <?php if ( $addr['is_default'] ) : ?>
                                    <span class="absolute top-3 right-3 bg-[var(--wz-primary)]/10 text-[var(--wz-primary)] text-xs font-bold px-2 py-1 rounded">
                                        Mặc định
                                    </span>
                                <?php endif; ?>

                                <p class="font-bold text-[var(--wz-on-surface)] mb-1"><?php echo esc_html( $addr['name'] ); ?></p>
                                <p class="text-sm text-[var(--wz-outline)] mb-1"><?php echo esc_html( $addr['phone'] ); ?></p>
                                <p class="text-sm text-[var(--wz-on-surface)] mb-3"><?php echo esc_html( $addr['address'] ); ?></p>

                                <div class="flex gap-2">
                                    <button class="edit-address-btn text-[var(--wz-primary)] text-sm font-semibold hover:underline" data-id="<?php echo esc_attr( (string) $addr['id'] ); ?>">
                                        Sửa
                                    </button>
                                    <?php if ( ! $addr['is_default'] ) : ?>
                                        <button class="delete-address-btn text-[var(--wz-error)] text-sm font-semibold hover:underline" data-id="<?php echo esc_attr( (string) $addr['id'] ); ?>">
                                            Xóa
                                        </button>
                                        <button class="set-default-btn text-[var(--wz-outline)] text-sm font-semibold hover:underline" data-id="<?php echo esc_attr( (string) $addr['id'] ); ?>">
                                            Đặt mặc định
                                        </button>
                                    <?php endif; ?>
                                </div>
                            </div>
                        <?php endforeach; ?>
                    </div>

                <?php else : ?>
                    <div class="text-center py-12">
                        <span class="material-symbols-outlined text-[48px] text-[var(--wz-outline)] mb-3 block" aria-hidden="true">location_on</span>
                        <p class="text-[var(--wz-outline)] mb-4">Bạn chưa có địa chỉ nào</p>
                        <button id="add-first-address-btn" class="inline-block bg-[var(--wz-primary)] text-white px-6 py-2 rounded-[var(--wz-radius)] font-semibold hover:opacity-90 transition">
                            Thêm địa chỉ đầu tiên
                        </button>
                    </div>
                <?php endif; ?>
            </div>
        </div>
    </div>

</div>
</div>

<!-- Address form modal (rendered by JS) -->
<div id="address-modal" class="hidden fixed inset-0 bg-black/50 flex items-center justify-center z-50">
    <div class="bg-white rounded-[var(--wz-radius-card)] p-6 max-w-md w-full mx-4">
        <h2 id="modal-title" class="text-lg font-bold text-[var(--wz-primary)] mb-4">Thêm địa chỉ mới</h2>
        <form id="address-form" class="space-y-4">
            <!-- Form fields rendered by JS -->
        </form>
    </div>
</div>

<?php get_footer(); ?>
```

## Usage

Create as `wezone-templates/account/addresses.php`

## JavaScript Required
- Add address modal
- Edit address (populate form)
- Delete address (confirm + AJAX)
- Set default address (AJAX)
- Form validation + submit

## Dependencies
- `wz_get_user_addresses()` function
