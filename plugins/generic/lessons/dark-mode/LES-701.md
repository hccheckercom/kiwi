## Bad
```tsx
<input className="ring-1 ring-gray-300 focus:ring-blue-500 rounded-md">
```
Dark mode: ring-gray-300 invisible, focus state khó nhận biết.

## Good
```tsx
<input className="ring-1 ring-gray-300 dark:ring-gray-600 focus:ring-blue-500 dark:focus:ring-blue-400 rounded-md">
```

## Why
Ring/outline là visual feedback quan trọng cho focus states. Nếu invisible trong dark mode, user không biết element nào đang active — accessibility issue.
