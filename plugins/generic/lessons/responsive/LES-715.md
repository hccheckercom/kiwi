## Bad
```tsx
<div className="w-[800px] mx-auto">
  <ProductTable />
</div>
```
Fixed 800px width — causes horizontal scroll on any screen < 800px.

## Good
```tsx
<div className="w-full max-w-[800px] mx-auto">
  <ProductTable />
</div>
```
Full width on mobile, capped at 800px on larger screens.

## Why
Fixed pixel widths > 400px will overflow on mobile (375px viewport). Always use `w-full` with `max-w-*` for responsive containers. Exception: modals/dialogs that have their own scroll context.
