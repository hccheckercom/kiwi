**Pattern sai:**
```php
// front-page.php — gọi trực tiếp, fatal nếu plugin chưa active
$shop_url = wz_get_shop_url();
echo wz_format_price( $price );
```
**Phải làm:**
```php
$shop_url = function_exists( 'wz_get_shop_url' ) ? wz_get_shop_url() : home_url( '/shop/' );
$fmt = function_exists( 'wz_format_price' ) ? 'wz_format_price' : 'number_format';
echo esc_html( $fmt( $price ) );
```
**Lý do:** Khi wezone-commer plugin chưa active (dev local, staging, hoặc plugin deactivated), gọi wz_*() trực tiếp gây fatal error. Khác với LES-005 (khai báo function), đây là lỗi **gọi** function ở template layer — cần fallback value hợp lý thay vì chỉ guard.
**Grep check:** `wz_get_shop_url\(|wz_get_myaccount_url\(|wz_get_cart_url\(|wz_format_price\(` — mỗi hit phải có `function_exists` trong cùng scope → CRITICAL
