## Bad
```php
<img src="<?php echo esc_url( $slide['image'] ); ?>"
    alt="<?php echo esc_attr( $slide['alt'] ); ?>"
    loading="eager"
    style="width:100%;height:100%;object-fit:cover;" />
```

## Good
```php
<img src="<?php echo esc_url( $slide['image'] ); ?>"
    alt="<?php echo esc_attr( $slide['alt'] ); ?>"
    loading="eager" fetchpriority="high"
    width="1440" height="600"
    style="width:100%;height:100%;object-fit:cover;" />
```

## Why
The hero image is typically the Largest Contentful Paint (LCP) element. Without `fetchpriority="high"`, the browser may deprioritize it behind other resources. Combined with `loading="eager"`, this tells the browser to fetch the hero image with highest priority, directly improving LCP scores. Target: LCP < 2.0s.
