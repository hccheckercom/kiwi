## Bad
```php
hashlib.md5(password.encode()).hexdigest()
```

## Good
```php
hashlib.sha256(password.encode()).hexdigest()
```

## Why
MD5 and SHA1 are cryptographically broken. Use SHA256+ for security-sensitive hashing
