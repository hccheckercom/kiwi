## Bad
```php
<img src="<?php echo esc_url( $banner['image'] ); ?>"
  class="w-full h-full object-cover" loading="eager" />
```
Has `loading="eager"` but missing `fetchpriority="high"` — browser won't prioritize this over other eager resources.

## Good
```php
<img src="<?php echo esc_url( $banner['image'] ); ?>"
  class="w-full h-full object-cover"
  loading="eager" fetchpriority="high"
  width="1440" height="600" />
```

## Why
`loading="eager"` only prevents lazy-loading — it doesn't boost priority. `fetchpriority="high"` tells the browser to fetch this image before other resources of the same type. For LCP elements, both are required per performance.md constitution.
