## Pattern

Developer query trực tiếp WooCommerce database tables hoặc dùng WC meta keys.

## Bad

```php
global $wpdb;

// WooCommerce HPOS tables (WC 8.0+)
$orders = $wpdb->get_results(
    "SELECT * FROM {$wpdb->prefix}wc_orders WHERE status = 'wc-completed'"
);

$items = $wpdb->get_results(
    "SELECT * FROM {$wpdb->prefix}wc_order_items WHERE order_id = {$order_id}"
);

// WooCommerce sessions
$wpdb->query( "DELETE FROM {$wpdb->prefix}woocommerce_sessions WHERE session_expiry < UNIX_TIMESTAMP()" );

// WooCommerce meta keys (legacy CPT storage)
$price = get_post_meta( $product_id, '_price', true );
$sku   = get_post_meta( $product_id, '_sku', true );
$stock = get_post_meta( $product_id, '_stock', true );
$sale  = get_post_meta( $product_id, '_sale_price', true );
$reg   = get_post_meta( $product_id, '_regular_price', true );

// WooCommerce taxonomies
$terms = wp_get_post_terms( $product_id, 'product_cat' );
$tags  = wp_get_post_terms( $product_id, 'product_tag' );

// WooCommerce options
$currency = get_option( 'woocommerce_currency' );
$weight   = get_option( 'woocommerce_weight_unit' );
$country  = get_option( 'woocommerce_default_country' );
```

## Good

```php
global $wpdb;

// Wezone orders table
$orders = $wpdb->get_results( $wpdb->prepare(
    "SELECT * FROM {$wpdb->prefix}wz_orders WHERE status = %s",
    'completed'
) );

$items = $wpdb->get_results( $wpdb->prepare(
    "SELECT * FROM {$wpdb->prefix}wz_order_items WHERE order_id = %d",
    $order_id
) );

// Wezone sessions — dùng WordPress transients hoặc wz_sessions
delete_expired_transients();

// Wezone product data — stored in wz_products table, NOT post_meta
$product = wz_get_product( $product_id );
$price   = $product['price'];
$sku     = $product['sku'];
$stock   = $product['stock_qty'];
$sale    = $product['sale_price'];
$reg     = $product['regular_price'];

// Wezone categories — custom table, NOT taxonomy
$categories = wz_get_product_categories( $product_id );
$tags       = wz_get_product_tags( $product_id );

// Wezone options
$currency = wz_get_option( 'currency' );
$weight   = wz_get_option( 'weight_unit' );
$country  = wz_get_option( 'default_country' );
```

## Why

Wezone Commer kiến trúc DB khác hoàn toàn WooCommerce:
- **Products**: stored trong `wz_products` table (NOT post_meta)
- **Orders**: stored trong `wz_orders` + `wz_order_items` (NOT `wc_orders`)
- **Categories**: stored trong `wz_product_categories` (NOT WordPress taxonomy `product_cat`)
- **Sessions**: dùng WordPress transients hoặc custom `wz_sessions`
- **Options**: prefix `wz_` trong wp_options (NOT `woocommerce_*`)

Query WC tables → empty results hoặc SQL error (table doesn't exist).
Query WC meta keys → empty results (meta keys không tồn tại).
