## Bad
```php
register_rest_route( 'wezone/v1', '/cart/add', [
    'methods' => WP_REST_Server::CREATABLE,
    'callback' => 'add_to_cart',
    // No nonce verification
]);
```

## Good
```php
// In callback:
if ( ! wp_verify_nonce( $request->get_header('X-WP-Nonce'), 'wp_rest' ) ) {
    return new WP_Error( 'rest_forbidden', 'Invalid nonce', ['status' => 403] );
}
```

## Why
Write REST route without nonce — CSRF attack can trigger actions on behalf of logged-in user.
