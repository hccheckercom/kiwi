# Checkout Form — Multi-Section Layout

Complete checkout page with address, shipping, payment sections and sticky order summary sidebar.

## Features
- Multi-section form (address, items, shipping, voucher, payment)
- Sticky order summary sidebar
- Loading/empty states
- Nonce security
- Responsive 2-column layout (mobile stacks)
- Material Icons
- Tailwind utility classes

## Code

```php

?>
```

## Usage

Create as `wezone-templates/checkout/form.php`

## CSS Variables Used
- `--wz-primary` — primary color
- `--wz-on-surface` — text color
- `--wz-on-surface-variant` — muted text
- `--wz-error` — error state
- `--wz-surface-container-high` — borders

## JavaScript Required
- Load cart items via AJAX
- Form validation
- Submit checkout
- Update order summary
