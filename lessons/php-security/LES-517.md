## Bad
```php
class SettingsAdmin {
    public function save_settings(): void {
        update_option( 'wz_settings', $_POST['settings'] );
    }
}
```

## Good
```php
class SettingsAdmin {
    public function save_settings(): void {
        if ( ! current_user_can( 'manage_options' ) ) {
            wp_die( 'Unauthorized' );
        }
        check_admin_referer( 'wz_save_settings' );
        update_option( 'wz_settings', $_POST['settings'] );
    }
}
```

## Why
Admin handlers without capability checks allow any logged-in user to access admin functions. Must check manage_options or equivalent.
