## Bad
```php
public static function instance(): self {
    self::$instance = new self();
    return self::$instance;
}
```

## Good
```php
public static function instance(): self {
    if ( null === self::$instance ) {
        self::$instance = new self();
    }
    return self::$instance;
}
```

## Why
Singleton without null check can create multiple instances in concurrent requests. Always check if instance exists before creating.
