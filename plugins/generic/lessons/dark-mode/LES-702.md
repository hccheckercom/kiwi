## Bad
```tsx
<div style={{ backgroundColor: '#ffffff', color: '#000000' }}>
  Content
</div>
```
Inline styles không respond to dark mode class — luôn white/black.

## Good
```tsx
<div className="bg-white dark:bg-gray-900 text-gray-900 dark:text-white">
  Content
</div>
```

## Why
Inline styles override Tailwind dark: variants. Hardcoded white/black trong style={} sẽ không thay đổi khi toggle dark mode, gây inconsistency.
