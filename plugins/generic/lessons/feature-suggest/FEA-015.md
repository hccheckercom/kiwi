**Mô tả:**
- Font sizes dùng `clamp()` để scale mượt giữa mobile → desktop
- KHÔNG dùng fixed px cho headings — phải responsive
- Vietnamese body text PHẢI ≥ 15px (characters phức tạp hơn Latin)
- Line-height body PHẢI ≥ 1.5 (dấu tiếng Việt cần vertical space)

**Detect pattern (absence):**
- CSS thiếu `clamp(` hoặc `--text-base` tokens → font sizes cứng, không responsive

**Token scale mẫu:**
```css
:root {
  --text-xs: clamp(0.6875rem, 0.65rem + 0.1vw, 0.75rem);
  --text-sm: clamp(0.8125rem, 0.77rem + 0.15vw, 0.875rem);
  --text-base: clamp(0.9375rem, 0.89rem + 0.2vw, 1rem);
  --text-lg: clamp(1.0625rem, 1rem + 0.25vw, 1.125rem);
  --text-xl: clamp(1.25rem, 1.15rem + 0.4vw, 1.375rem);
  --text-2xl: clamp(1.5rem, 1.3rem + 0.7vw, 1.75rem);
  --text-3xl: clamp(1.75rem, 1.45rem + 1.2vw, 2.25rem);
  --text-4xl: clamp(2rem, 1.5rem + 2vw, 3rem);
}
```

**Vietnamese rules:**
- KHÔNG dùng `text-transform: uppercase` cho Vietnamese — dấu bị cắt
- Prefer fonts có Vietnamese support: Inter, Noto Sans, Be Vietnam Pro

**Lợi ích:** Typography mượt mọi viewport, đọc dễ trên mobile lẫn desktop
