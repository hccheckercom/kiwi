## Bad
```tsx
// Website sells templates/services but:
// - No /refund page exists
// - Footer has no link to refund policy
// - Terms page doesn't mention refund conditions
```
Google Ads requires: "Clearly state your refund and cancellation policies." Meta requires: "Transparent business practices including return policies."

## Good
```tsx
// Footer links:
<Link href="/refund">Chính sách hoàn tiền</Link>

// Refund page covers:
// 1. Conditions for refund (technical failure, product mismatch)
// 2. Non-refundable cases (downloaded files, aesthetic preference)
// 3. Process and timeline (email → 24h response → 3-5 days refund)
// 4. Contact method for refund requests
```

## Why
Missing refund policy is a top-5 reason for Google Ads landing page rejection. Both platforms require that users can easily find refund/return information before purchasing. The policy must be: (1) accessible from any page (footer link), (2) clear about conditions, (3) specific about process and timeline. Digital products can state "no refund after download" but must still have a policy page explaining this.
