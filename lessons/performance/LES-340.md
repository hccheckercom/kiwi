## Bad
```php
public function getCategories(): array {
    global $wpdb;
    $rows = $wpdb->get_results(
        "SELECT id, name, slug, parent_id FROM {$wpdb->prefix}wz_categories ORDER BY name ASC",
        ARRAY_A
    );
    // fires every call — no caching
}
```

## Good
```php
public function getCategories(): array {
    $cache_key = 'wz_all_categories';
    $cached    = wp_cache_get( $cache_key, 'wezone' );
    if ( $cached !== false ) {
        return $cached;
    }

    global $wpdb;
    $rows = $wpdb->get_results(
        "SELECT id, name, slug, parent_id FROM {$wpdb->prefix}wz_categories ORDER BY name ASC",
        ARRAY_A
    );
    // ... process rows ...

    wp_cache_set( $cache_key, $rows, 'wezone', 3600 );
    return $rows;
}
```

## Why
Category list rarely changes but is queried on every page with category rendering (homepage, archive, sidebar). A 1-hour object cache eliminates redundant queries. Invalidate on category create/update/delete.
