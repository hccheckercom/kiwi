## Bad
```php
'description' => sprintf( 'Thanh toan don hang #%s', $order['order_number'] ),
```

## Good
```php
'description' => sprintf(
    /* translators: %s: order number */
    __( 'Thanh toán đơn hàng #%s', 'wezone-core' ),
    $order['order_number']
),
```

## Why
Payment gateway description is visible in payment provider dashboard and customer receipts. Hardcoded Vietnamese without `__()` prevents future localization and uses ASCII-only characters ("Thanh toan" instead of "Thanh toán").
