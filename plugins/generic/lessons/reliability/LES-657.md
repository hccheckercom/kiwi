## Bad
```php
def process(items=[]):
    items.append(x)
```

## Good
```php
def process(items=None):
    items = items or []
```

## Why
Mutable defaults are shared across calls, causing unexpected state bugs

`def \w+\([^)]*=\s*[\[\{]` — in `**/*.py`
