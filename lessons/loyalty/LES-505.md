## Bad
```php
public function calculate_tier( int $points ): string {
    if ( $points >= 5000 ) return 'diamond';
    if ( $points >= 2000 ) return 'gold';
    if ( $points >= 500 ) return 'silver';
    return 'bronze';
    // Hardcoded thresholds - cannot change without code deploy
}
```

## Good
```php
public function calculate_tier( int $points ): string {
    $tier_diamond = (int) get_option( 'loyalty_tier_diamond', 5000 );
    $tier_gold    = (int) get_option( 'loyalty_tier_gold', 2000 );
    $tier_silver  = (int) get_option( 'loyalty_tier_silver', 500 );
    
    return match ( true ) {
        $points >= $tier_diamond => 'diamond',
        $points >= $tier_gold    => 'gold',
        $points >= $tier_silver  => 'silver',
        default                  => 'bronze',
    };
}
```

## Why
Hardcoded tier thresholds (500, 2000, 5000) require code changes to adjust. Business needs to test different thresholds for conversion optimization. Must use get_option() for runtime configuration.
