**Pattern sai:**
```php
$qty = (int) $item['quantity'];
// CartEngine lưu key 'qty', không phải 'quantity'
// → $item['quantity'] = null → (int) null = 0
// → subtotal = price * 0 = 0đ, UI hiển thị qty=0
```
**Phải làm:**
```php
$qty = (int) ( $item['qty'] ?? $item['quantity'] ?? 1 );
```
**Lý do:** CartEngine (`wezone-core`) lưu item với key `'qty'`, nhưng dev quen WooCommerce convention dùng `'quantity'`. Kết quả: qty=0, subtotal=0đ, cart/checkout hiển thị sai hoàn toàn. Header badge đúng vì dùng `getCount()` (đọc `'qty'` internally).
**Grep check:** `\$item\['quantity'\]` trong cart/checkout templates — nếu data source là CartEngine → CRITICAL
