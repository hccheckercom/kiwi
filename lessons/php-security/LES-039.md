**Pattern sai:**
```javascript
// JS gọi cart API không gửi nonce
fetch('/wp-json/wezone/v1/cart/add', {
    method: 'POST',
    body: JSON.stringify({ product_id: 5, qty: 1 })
});
// → CartController::check_nonce() return false → 403 Forbidden
```
**Phải làm:**
```javascript
fetch('/wp-json/wezone/v1/cart/add', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-WP-Nonce': wzTheme.nonce  // từ wp_localize_script
    },
    body: JSON.stringify({ product_id: 5, qty: 1 })
});
```
**Lý do:** CartController verify nonce trên mọi write endpoint (add, remove, update, clear, checkout). Thiếu header → 403 → user click "Thêm vào giỏ" nhưng không có gì xảy ra. Liên quan LES-014 (wzTheme property not localized).
**Grep check:** `fetch.*cart\|XMLHttpRequest.*cart` trong JS — thiếu `X-WP-Nonce` hoặc `wzTheme.nonce` → CRITICAL
