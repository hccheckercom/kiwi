## Bad
```php
class Cache {
    private static array $store = [];
    public static function set($k, $v) { self::$store[$k] = $v; }
}
```

## Good
```php
class Cache {
    private array $store = [];
    public function set(string $k, $v): void { $this->store[$k] = $v; }
}
```

## Why
Mutable static property — state leaks between PHPUnit tests. Test A's cache pollutes Test B.
