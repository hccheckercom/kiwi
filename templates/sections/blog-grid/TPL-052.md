# Blog Grid — Post Archive with Pagination

Blog archive page với card grid layout, featured images, và pagination.

## Use Cases
- Blog homepage (home.php)
- Category archive
- Tag archive
- Author archive

## Features
- 📰 Post cards với featured image
- 📅 Published date + author
- 📝 Excerpt (150 characters)
- 🏷️ Category badges
- 📄 Pagination (prev/next + numbers)
- 🔍 SEO-friendly (proper heading hierarchy)

## Code

### PHP Template (home.php or archive.php)

```php
<?php
/**
 * Blog Archive Template
 */

get_header();
?>

<main class="blog-archive py-12">
	<div class="max-w-[1280px] mx-auto px-10">
		<!-- Archive Header -->
		<div class="mb-10">
			<?php if ( is_category() ): ?>
				<h1 class="text-4xl font-bold mb-2"><?php single_cat_title(); ?></h1>
				<?php if ( category_description() ): ?>
					<p class="text-gray-600"><?php echo wp_kses_post( category_description() ); ?></p>
				<?php endif; ?>
			<?php elseif ( is_tag() ): ?>
				<h1 class="text-4xl font-bold mb-2">Tag: <?php single_tag_title(); ?></h1>
			<?php elseif ( is_author() ): ?>
				<h1 class="text-4xl font-bold mb-2">Bài viết của <?php the_author(); ?></h1>
			<?php else: ?>
				<h1 class="text-4xl font-bold mb-2">Blog</h1>
				<p class="text-gray-600">Tin tức, hướng dẫn và cập nhật mới nhất</p>
			<?php endif; ?>
		</div>

		<?php if ( have_posts() ): ?>
			<!-- Post Grid -->
			<div class="grid md:grid-cols-2 lg:grid-cols-3 gap-6 mb-10">
				<?php while ( have_posts() ): the_post(); ?>
					<article class="bg-white rounded-lg border border-gray-200 overflow-hidden hover:shadow-lg transition-shadow">
						<!-- Featured Image -->
						<?php if ( has_post_thumbnail() ): ?>
							<a href="<?php the_permalink(); ?>" class="block aspect-[16/9] overflow-hidden">
								<?php the_post_thumbnail( 'medium_large', [
									'class' => 'w-full h-full object-cover hover:scale-105 transition-transform duration-300',
									'loading' => 'lazy',
								]); ?>
							</a>
						<?php else: ?>
							<a href="<?php the_permalink(); ?>" class="block aspect-[16/9] bg-gray-100 flex items-center justify-center">
								<svg class="w-16 h-16 text-gray-300" fill="none" stroke="currentColor" viewBox="0 0 24 24">
									<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M4 16l4.586-4.586a2 2 0 012.828 0L16 16m-2-2l1.586-1.586a2 2 0 012.828 0L20 14m-6-6h.01M6 20h12a2 2 0 002-2V6a2 2 0 00-2-2H6a2 2 0 00-2 2v12a2 2 0 002 2z" />
								</svg>
							</a>
						<?php endif; ?>

						<!-- Content -->
						<div class="p-6">
							<!-- Category -->
							<?php
							$categories = get_the_category();
							if ( ! empty( $categories ) ):
							?>
								<div class="mb-3">
									<a href="<?php echo esc_url( get_category_link( $categories[0]->term_id ) ); ?>" class="inline-block px-3 py-1 text-xs font-medium bg-blue-50 text-blue-600 rounded-full hover:bg-blue-100">
										<?php echo esc_html( $categories[0]->name ); ?>
									</a>
								</div>
							<?php endif; ?>

							<!-- Title -->
							<h2 class="text-xl font-bold mb-2 line-clamp-2">
								<a href="<?php the_permalink(); ?>" class="hover:text-blue-600 transition-colors">
									<?php the_title(); ?>
								</a>
							</h2>

							<!-- Excerpt -->
							<p class="text-gray-600 text-sm mb-4 line-clamp-3">
								<?php echo wp_trim_words( get_the_excerpt(), 20, '...' ); ?>
							</p>

							<!-- Meta -->
							<div class="flex items-center text-xs text-gray-500">
								<time datetime="<?php echo esc_attr( get_the_date( 'c' ) ); ?>">
									<?php echo get_the_date(); ?>
								</time>
								<span class="mx-2">•</span>
								<span><?php echo esc_html( get_the_author() ); ?></span>
							</div>
						</div>
					</article>
				<?php endwhile; ?>
			</div>

			<!-- Pagination -->
			<?php
			$pagination = paginate_links([
				'prev_text' => '<svg class="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24"><path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M15 19l-7-7 7-7" /></svg>',
				'next_text' => '<svg class="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24"><path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M9 5l7 7-7 7" /></svg>',
				'type' => 'array',
			]);

			if ( $pagination ):
			?>
				<nav class="flex justify-center items-center gap-2" aria-label="Pagination">
					<?php foreach ( $pagination as $page ): ?>
						<?php echo str_replace( 'page-numbers', 'page-numbers px-4 py-2 rounded-lg border border-gray-200 hover:bg-gray-50 transition-colors', $page ); ?>
					<?php endforeach; ?>
				</nav>
			<?php endif; ?>

		<?php else: ?>
			<!-- Empty State -->
			<div class="text-center py-20">
				<svg class="w-20 h-20 text-gray-300 mx-auto mb-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
					<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M9 12h6m-6 4h6m2 5H7a2 2 0 01-2-2V5a2 2 0 012-2h5.586a1 1 0 01.707.293l5.414 5.414a1 1 0 01.293.707V19a2 2 0 01-2 2z" />
				</svg>
				<h2 class="text-2xl font-bold text-gray-900 mb-2">Chưa có bài viết</h2>
				<p class="text-gray-600">Hãy quay lại sau để đọc nội dung mới nhất</p>
			</div>
		<?php endif; ?>
	</div>
</main>

<?php get_footer(); ?>
```

## Data Sources

**WordPress Loop:**
- `have_posts()` / `the_post()` — Standard loop
- `the_title()` — Post title
- `the_permalink()` — Post URL
- `the_post_thumbnail()` — Featured image
- `get_the_excerpt()` — Post excerpt
- `get_the_date()` — Published date
- `get_the_author()` — Author name
- `get_the_category()` — Post categories

## Dependencies
- Tailwind CSS
- WordPress loop functions
- Featured image support (add_theme_support('post-thumbnails'))

## Responsive Behavior
- Mobile: 1 column
- Tablet (md): 2 columns
- Desktop (lg): 3 columns
- Featured image: 16:9 aspect ratio

## Accessibility
- ✅ Semantic HTML (article, time, nav)
- ✅ datetime attribute on time element
- ✅ aria-label on pagination nav
- ✅ Proper heading hierarchy (h1 → h2)
- ✅ Alt text from featured image

## SEO Features
- Proper heading hierarchy
- Semantic HTML5 elements

- Category/tag archive titles
- Meta descriptions from excerpts

## Customization

### Change posts per page
```php
// In functions.php
function wz_blog_posts_per_page( $query ) {
	if ( ! is_admin() && $query->is_main_query() && is_home() ) {
		$query->set( 'posts_per_page', 9 );
	}
}
add_action( 'pre_get_posts', 'wz_blog_posts_per_page' );
```

### Add sidebar
```php
<div class="grid lg:grid-cols-[1fr_300px] gap-8">
	<div>
		<!-- Post grid here -->
	</div>
	<aside>
		<?php get_sidebar(); ?>
	</aside>
</div>
```

### Change grid columns
```php
<!-- 4 columns on desktop -->
<div class="grid md:grid-cols-2 lg:grid-cols-4 gap-6">
```

### Add load more button
```javascript
// Replace pagination with AJAX load more
document.querySelector('.load-more').addEventListener('click', async () => {
	const page = parseInt(this.dataset.page) + 1;
	const response = await fetch(`/wp-json/wp/v2/posts?page=${page}&per_page=9`);
	const posts = await response.json();
	// Append posts to grid
});
```
