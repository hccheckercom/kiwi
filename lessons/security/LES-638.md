## Bad
```python
API_KEY = "sk_live_abc123def456"
```

## Good
```python
API_KEY = os.environ.get("API_KEY")
```

## Why
Hardcoded credentials in source are exposed via version control

## Grep
```bash
rg '(password|secret|api_key|token)\s*=\s*["'"'"'][^"'"'"']{8,}' --type py
```
