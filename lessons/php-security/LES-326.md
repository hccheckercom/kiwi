## Pattern

Admin session lasts 48h (WordPress default). If session is hijacked or admin leaves computer unlocked, attacker can: change payment keys, export customer data, delete all orders — without any additional verification. No password re-confirmation or 2FA for destructive/sensitive actions.

## Bug

```php
// SAI — đổi payment secret key chỉ cần session cookie
public function update_payment_settings( WP_REST_Request $request ): WP_REST_Response {
    $secret = sanitize_text_field( $request->get_param( 'vnpay_hash_secret' ) );
    update_option( 'wezone_admin_vnpay_hash_secret', $secret, false );
    return new WP_REST_Response( array( 'success' => true ) );
}
```

## Fix

```php
/**
 * Sudo mode: admin phải nhập lại password cho sensitive actions.
 * Token valid 15 phút sau khi confirm.
 */
public function verify_sudo_mode( WP_REST_Request $request ): WP_REST_Response {
    $password = $request->get_param( 'password' );
    $user     = wp_get_current_user();

    if ( ! wp_check_password( $password, $user->user_pass, $user->ID ) ) {
        return new WP_REST_Response( array( 'error' => 'Mật khẩu không đúng' ), 403 );
    }

    // Set sudo token — expires in 15 minutes
    set_transient( 'wz_sudo_' . $user->ID, wp_hash( $user->user_pass . time() ), 900 );

    return new WP_REST_Response( array( 'sudo' => true, 'expires_in' => 900 ) );
}

private function require_sudo(): bool {
    $user_id = get_current_user_id();
    return (bool) get_transient( 'wz_sudo_' . $user_id );
}

// Sensitive action — requires sudo mode
public function update_payment_settings( WP_REST_Request $request ): WP_REST_Response {
    if ( ! $this->require_sudo() ) {
        return new WP_REST_Response(
            array( 'error' => 'sudo_required', 'message' => 'Vui lòng xác nhận mật khẩu để thực hiện thao tác này' ),
            403
        );
    }

    $secret = sanitize_text_field( $request->get_param( 'vnpay_hash_secret' ) );
    update_option( 'wezone_admin_vnpay_hash_secret', $secret, false );

    wz_log( 'settings.payment_updated', array(
        'admin_id' => get_current_user_id(),
        'field'    => 'vnpay_hash_secret',
    ) );

    return new WP_REST_Response( array( 'success' => true ) );
}
```

## Áp dụng

Sensitive actions cần sudo mode (re-auth): (1) thay đổi payment keys/secrets, (2) export customer PII, (3) bulk delete > 10 items, (4) thay đổi admin email/password, (5) disable security features. Sudo token valid 15 phút — sau đó phải confirm lại.
