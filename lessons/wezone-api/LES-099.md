## Bad
```php
$response = wp_remote_get($url);
$body = wp_remote_retrieve_body($response);
```

## Good
```php
$response = wp_remote_get($url, ['timeout' => 10]);
if ( is_wp_error($response) ) {
    wz_log('API request failed: ' . $response->get_error_message(), 'error');
    return [];
}
$body = wp_remote_retrieve_body($response);
```

## Why
wp_remote_* response not checked for WP_Error — timeout/DNS failure causes fatal when accessing body.
