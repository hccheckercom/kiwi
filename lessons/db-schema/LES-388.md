## Bad
```php
// wz_order_items
'product_id'   => 'bigint(20) unsigned NOT NULL',
'variation_id' => 'bigint(20) unsigned DEFAULT NULL',
// indexes: only idx_order — no idx_product or idx_variation
```

## Good
```php
'product_id'   => 'bigint(20) unsigned NOT NULL',
'variation_id' => 'bigint(20) unsigned DEFAULT NULL',
// indexes:
'idx_order'     => 'order_id',
'idx_product'   => 'product_id',
'idx_variation' => 'variation_id',
```

## Why
Querying "which orders contain product X" without index on product_id requires full table scan. With 100K+ order items, this query takes seconds instead of milliseconds.
