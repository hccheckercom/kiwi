**Pattern sai:**
```php
<img src="<?php echo esc_url( $img ); ?>" alt="<?php echo esc_attr( $name ); ?>">
```

**Phải làm:**
```php
<img src="<?php echo esc_url( $img ); ?>" alt="<?php echo esc_attr( $name ); ?>"
     onerror="this.onerror=null;this.src='<?php echo esc_url( get_template_directory_uri() . '/assets/images/placeholder-product.svg' ); ?>'">
```

**Lý do:** Khi ảnh CDN lỗi hoặc URL sai, user thấy broken icon — UX rất tệ. Mọi `<img>` trong template-parts phải có `onerror` fallback về placeholder SVG phù hợp context (product 1:1, category 16:9, banner 21:9).
