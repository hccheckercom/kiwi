## Bad
```php
update_order_status($order_id, 'cancelled');
```

## Good
```php
wz_log("Order #{$order_id} status: processing → cancelled by user #{$user_id}");
update_order_status($order_id, 'cancelled');
```

## Why
Order status change without audit log — no trail when customer disputes. Always log who/when/why.
