# Bad Code

```python
counter = 0

def increment():
    global counter
    counter += 1
    return counter

# BAD: Hard to test, not thread-safe
```

# Good Code

```python
class Counter:
    def __init__(self):
        self.value = 0
    
    def increment(self):
        self.value += 1
        return self.value

# Or use function parameters
def increment(counter: int) -> int:
    return counter + 1
```

# Why

**Global mutation issues:**
- Not thread-safe
- Hard to test
- Hidden dependencies
- State pollution between tests
