## Bad
```php
p = lesson.get('pattern', '')
if p:
    existing_patterns.add(p.strip())  # AttributeError if p is a list
```

## Good
```php
p = lesson.get('pattern', '')
if p:
    if isinstance(p, list):
        for item in p:
            existing_patterns.add(str(item).strip())
    else:
        existing_patterns.add(p.strip())
```

## Why
YAML frontmatter `pattern:` field can be a list (multi-line array) or a string. Calling `.strip()` directly on a list raises AttributeError. Always check `isinstance(p, list)` before calling string methods.
