## Bad
```php
useEffect(() => {
  loadCheckpoints()
  const interval = setInterval(loadCheckpoints, 5000)
  return () => clearInterval(interval)
}, [])

// Problem: loadCheckpoints is async but not tracked
// If component unmounts during fetch, setState on unmounted component
```

## Good
```php
useEffect(() => {
  let isMounted = true
  
  async function poll() {
    if (!isMounted) return
    await loadCheckpoints()
  }
  
  poll()
  const interval = setInterval(poll, 5000)
  
  return () => {
    isMounted = false
    clearInterval(interval)
  }
}, [])
```

## Why
When polling with setInterval calls async functions, component may unmount during fetch. Without isMounted guard, setState on unmounted component causes memory leak and React warning. Always track mount state and check before setState in async callbacks.
