## Preview

Card sản phẩm đơn lẻ, dùng trong mọi grid. Ảnh vuông trên (với brand/sale badge góc trái), info dưới (title 2-line clamp + price). Nút add-to-cart tròn góc phải dưới ảnh (chỉ hiện khi in stock). BEM classes cho dễ style override.

## Files

| File | Role |
|------|------|
| template-parts/product/card.php | Component template |
| assets/css/partials/_shop.css | CSS cho .wz-product-card* |

## Code — PHP

```php
<?php
$product_id = get_the_ID();
$wz_product = function_exists( 'wz_get_product' ) ? wz_get_product( $product_id ) : null;

if ( empty( $wz_product ) ) {
    return;
}

$brand       = '';
$brand_terms = wp_get_post_terms( $product_id, 'product_cat', [ 'number' => 1 ] );
if ( ! is_wp_error( $brand_terms ) && ! empty( $brand_terms ) ) {
    $brand = strtoupper( $brand_terms[0]->name );
}

$pa_brand = get_post_meta( $product_id, 'attribute_pa_brand', true );
if ( $pa_brand ) {
    $brand = strtoupper( $pa_brand );
}

$regular_price = (float) ( $wz_product['regular_price'] ?? $wz_product['price'] ?? 0 );
$sale_price    = (float) ( $wz_product['sale_price'] ?: ( $wz_product['price'] ?? 0 ) );
$is_on_sale    = $regular_price > 0 && $sale_price > 0 && $sale_price < $regular_price;

$badge = $is_on_sale ? __( 'Giảm giá', 'wezone-haven' ) : '';

if ( ! function_exists( 'wz_format_price' ) ) {
    function wz_format_price( float $price ): string {
        return number_format( $price, 0, ',', '.' ) . '₫';
    }
}

$cart_url        = function_exists( 'wz_get_cart_url' ) ? wz_get_cart_url() : '#';
$add_to_cart_url = add_query_arg( [ 'wz_add_to_cart' => $product_id ], $cart_url );
$in_stock        = ( $wz_product['stock_status'] ?? '' ) === 'instock' || get_post_meta( $product_id, '_stock_status', true ) === 'instock';
?>

<div class="wz-product-card">
    <a href="<?php echo esc_url( get_permalink( $product_id ) ); ?>" class="wz-product-card__link">
        <div class="wz-product-card__image">
            <?php if ( $brand ) : ?>
                <span class="wz-product-card__badge"><?php echo esc_html( $brand ); ?></span>
            <?php elseif ( $badge ) : ?>
                <span class="wz-product-card__badge wz-product-card__badge--sale"><?php echo esc_html( $badge ); ?></span>
            <?php endif; ?>
            <?php echo get_the_post_thumbnail( $product_id, 'wezone-card', [ 'class' => 'wz-product-card__img' ] ); ?>
        </div>
        <div class="wz-product-card__body">
            <h3 class="wz-product-card__title"><?php echo esc_html( get_the_title( $product_id ) ); ?></h3>
            <div class="wz-product-card__footer">
                <div>
                    <?php if ( $is_on_sale ) : ?>
                        <span class="wz-product-card__price-old"><?php echo esc_html( wz_format_price( $regular_price ) ); ?></span>
                        <span class="wz-product-card__price"><?php echo esc_html( wz_format_price( $sale_price ) ); ?></span>
                    <?php else : ?>
                        <span class="wz-product-card__price"><?php echo esc_html( wz_format_price( $sale_price ) ); ?></span>
                    <?php endif; ?>
                </div>
            </div>
        </div>
    </a>
    <?php if ( $in_stock ) : ?>
        <a href="<?php echo esc_url( $add_to_cart_url ); ?>"
            class="wz-product-card__cart"
            data-quantity="1"
            data-product_id="<?php echo esc_attr( $product_id ); ?>"
            aria-label="<?php esc_attr_e( 'Thêm vào giỏ', 'wezone-haven' ); ?>"
            rel="nofollow">
            <span aria-hidden="true" class="material-symbols-outlined">shopping_cart</span>
        </a>
    <?php endif; ?>
</div>
```

## Usage Notes

- Dùng trong WP_Query loop: `while ( $products->have_posts() ) : $products->the_post(); get_template_part( 'template-parts/product/card' ); endwhile;`
- Cần `wz_get_product()` từ wezone-core plugin
- Fallback `wz_format_price()` nếu plugin chưa active
- BEM classes cho phép override CSS dễ dàng per-theme
- Cart button chỉ hiện khi `in_stock`
