**Triệu chứng:**
- Trang chủ hiển thị icon danh mục đúng tên + đúng icon
- Click vào bất kỳ danh mục nào → chuyển về `/shop/` thay vì `/category/bec-tuoi/`
- WordPress nhận `/?cat=bec-tuoi`, không match rewrite rule nào → fallback `/shop/`
**Root cause:**
- `wz-shims.php` function `wz_get_product_categories()`, `wz_get_category_by_slug()`, `wz_get_category()` đều tạo URL bằng `home_url( '/?cat=' . $row['slug'] )`
- Router dùng pretty permalink `/category/{slug}/`, không dùng query param
**Pattern sai:**
```php
'url' => home_url( '/?cat=' . $row['slug'] ),
```
**Phải làm:**
```php
'url' => home_url( '/category/' . $row['slug'] . '/' ),
```
**Lý do:** Wezone Router dùng `add_rewrite_rule` để map `/category/{slug}/` → template. Query param `?cat=` không có rule nào handle → WordPress fallback về shop page hoặc 404.
**Grep check:** `home_url.*\?cat=` trong `inc/wz-shims.php` → nếu có → CRITICAL
**Ảnh hưởng:** Mọi theme dùng wz-shims — tất cả link danh mục trên trang chủ, breadcrumb, sidebar filter đều trỏ sai URL
