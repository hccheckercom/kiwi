## Bad

```python
_cache = {}

def get_data(key):
    global _cache
    if key not in _cache:
        _cache[key] = fetch_from_db(key)
    return _cache[key]
```

## Good

```python
class DataCache:
    def __init__(self):
        self._cache = {}

    def get(self, key):
        if key not in self._cache:
            self._cache[key] = fetch_from_db(key)
        return self._cache[key]
```

## Why

`global` makes functions depend on hidden shared state, breaking testability and making concurrency unsafe. Use module-level constants (immutable), class instances, or function parameters instead. Kiwi's own codebase uses `global` for cache variables — these should be refactored to class-based caches.

## Grep

```bash
rg '^\s*global\s+\w' --type py -g '!test_*.py'
```
