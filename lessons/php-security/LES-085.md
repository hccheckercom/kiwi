## Bad
```php
$return_url = $_GET['redirect_to'];
wp_redirect($return_url);
```

## Good
```php
$return_url = wp_validate_redirect(
    esc_url_raw($_GET['redirect_to'] ?? ''),
    home_url('/')
);
wp_safe_redirect($return_url);
```

## Why
Payment redirect URL not validated — open redirect to phishing site after checkout.
