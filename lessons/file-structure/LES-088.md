## Bad
```php
use WeZone\Core\Services\OldPaymentGateway; // class was removed
```

## Good
```php
use WeZone\Core\Services\PaymentGateway; // verify class exists
```

## Why
use statement imports non-existent class — fatal 'Class not found' when autoloader tries to load it.
