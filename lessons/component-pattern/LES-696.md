## Bad
```php
<!-- cart/support-card.php -->
<a href="#" class="bg-blue-600 text-white rounded-md px-4 py-2">Liên hệ</a>

<!-- checkout/order-summary.php -->
<button class="bg-blue-600 text-white rounded-lg px-4 py-2">Đặt hàng</button>
```
Cùng theme nhưng button border-radius khác nhau.

## Good
```php
<!-- Thống nhất rounded-lg cho tất cả buttons -->
<a href="#" class="bg-blue-600 text-white rounded-lg px-4 py-2">Liên hệ</a>
<button class="bg-blue-600 text-white rounded-lg px-4 py-2">Đặt hàng</button>
```

## Why
Button styling phải consistent trong toàn bộ theme. Mixed border-radius gây cảm giác theme chưa hoàn thiện.
