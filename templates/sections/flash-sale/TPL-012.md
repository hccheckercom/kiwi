Body: title 2 dòng + giá mới bold primary + giá cũ gạch ngang + "Tiết kiệm Xđ" xanh lá + progress bar gradient + social proof. Nút "Săn deal" gradient full-width.

<script>
(function () {
    var end = <?php echo (int) $end_time; ?> * 1000;
    function pad(n) { return n < 10 ? '0' + n : '' + n; }
    function tick() {
        var diff = Math.max(0, end - Date.now());
        var h = Math.floor(diff / 3600000);
        var m = Math.floor((diff % 3600000) / 60000);
        var s = Math.floor((diff % 60000) / 1000);
        var eh = document.getElementById('haven-cd-h');
        var em = document.getElementById('haven-cd-m');
        var es = document.getElementById('haven-cd-s');
        if (eh) eh.textContent = pad(h);
        if (em) em.textContent = pad(m);
        if (es) es.textContent = pad(s);
        var timers = document.querySelectorAll('.haven-flash__card-timer-val');
        timers.forEach(function(el) { el.textContent = pad(h) + ':' + pad(m) + ':' + pad(s); });
        if (diff > 0) setTimeout(tick, 1000);
    }
    tick();
}());
</script>

## Usage Notes

- Nếu plugin `wezone-flash-sale` active → delegate render cho plugin
- Fallback: query products có `_sale_price` meta
- `$badges`, `$sold_percents`, `$viewers` là demo data — production nên lấy từ plugin/meta
- Badge cam (#d17d5d) cho urgency tags, badge đỏ (--wz-error) cho discount
- Progress bar gradient: primary-fixed → primary (vàng → nâu đậm)
- Social proof text đỏ italic khi `$sold_pct >= 85` (urgency threshold)
- Glassmorphism countdown box cần backdrop-filter support (Safari prefix tự động)
- So với TPL-005 (basic): thêm progress bar, dual badges, per-card timer, savings text, CTA button
