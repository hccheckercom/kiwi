**Pattern sai:**
```php
$order_id = absint( $_GET['order_id'] ?? 0 );
$order = wz_get_order( $order_id ); // không check ownership
```
**Phải làm:**
```php
$order_id = absint( $_GET['order_id'] ?? 0 );
$order    = wz_get_order( $order_id );
$customer = $order ? wz_get_order_customer( $order_id ) : null;
if ( ! $order || ! $customer || $customer['id'] !== get_current_user_id() ) {
    wp_die( esc_html__( 'Không có quyền xem đơn hàng này.', 'wezone' ) );
}
```
**Lý do:** IDOR — user A xem đơn của user B bằng cách đổi order_id trên URL
**Grep check:** `get_current_user_id|wz_get_order_customer` — thiếu trong order-detail.php → CRITICAL
