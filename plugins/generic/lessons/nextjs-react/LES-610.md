# Bad — Unhandled Promise

```typescript
async function fetchData() {
  await fetch('/api/data'); // No error handling
}
```

# Good — Wrapped in try/catch

```typescript
async function fetchData() {
  try {
    await fetch('/api/data');
  } catch (error) {
    console.error('Failed to fetch data:', error);
  }
}
```

# Why

Unhandled promise rejections can crash the app or leave it in inconsistent state. Always wrap `await` in try/catch or chain `.catch()`.
