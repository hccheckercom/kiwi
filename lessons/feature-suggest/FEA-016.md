**Mô tả:**
- Mỗi trang PHẢI có JSON-LD structured data phù hợp page type
- Product page: `@type: Product` với offers, rating, brand
- All pages: `@type: BreadcrumbList`
- Homepage: `@type: Organization` + `@type: WebSite` (SearchAction)

**Detect pattern (absence):**
- `inc/seo.php` hoặc `header.php` thiếu `application/ld+json` → Google không hiểu structured data

**Schema types cần có:**
| Page | Schema Type |
|------|-------------|
| Homepage | Organization + WebSite + SearchAction |
| Product | Product + Offer + AggregateRating |
| Category | BreadcrumbList |
| Blog | Article |
| FAQ | FAQPage |
| Store (physical) | LocalBusiness |

**Code mẫu (Product):**
```php
function wz_product_schema($product) {
    return json_encode([
        '@context' => 'https://schema.org',
        '@type' => 'Product',
        'name' => $product['name'],
        'image' => $product['images'],
        'offers' => [
            '@type' => 'Offer',
            'priceCurrency' => 'VND',
            'price' => $product['price'],
            'availability' => 'https://schema.org/InStock',
        ],
    ], JSON_UNESCAPED_UNICODE);
}
```

**Lợi ích:** Rich snippets trên Google (sao, giá, tình trạng), CTR tăng 20-30%
