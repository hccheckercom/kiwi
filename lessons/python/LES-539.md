# Bad Code

```python
def process_order(status):
    if status == "pending":
        send_notification()
    elif status == "completed":
        archive_order()
    # Typo: "complted" won't match
```

# Good Code

```python
from enum import Enum

class OrderStatus(Enum):
    PENDING = "pending"
    COMPLETED = "completed"
    CANCELLED = "cancelled"

def process_order(status: OrderStatus):
    if status == OrderStatus.PENDING:
        send_notification()
    elif status == OrderStatus.COMPLETED:
        archive_order()
```

# Why

**Enums provide:**
- Type safety (IDE autocomplete)
- Typo prevention
- Centralized values
- Easy refactoring
