## Pattern

Admin product form accepts price as text input. Backend saves without validating: negative prices, zero for non-free products, NaN from empty string, prices with wrong decimal format. Corrupted prices break cart calculations, checkout, and analytics.

## Scan

```yaml
pattern: 'price.*\$request->get_param|get_param.*price'
scope: "**/AdminUI/**/*.php|**/Product*/**/*.php"
exclude_line: "absint|abs\(|max\(.*0|floatval.*>|filter_var.*FLOAT|validate_price"
message: "Product price from request without numeric validation"
```

## Bug

```php
// SAI — price = "" → 0.0, price = "-100" → -100, price = "abc" → 0
$price = (float) $request->get_param( 'price' );
$sale_price = (float) $request->get_param( 'sale_price' );

$wpdb->update( $table, array(
    'price'      => $price,      // Có thể negative!
    'sale_price' => $sale_price, // Có thể > price!
) );
```

## Fix

```php
$price      = $request->get_param( 'price' );
$sale_price = $request->get_param( 'sale_price' );

// Validate price
if ( ! is_numeric( $price ) || (float) $price < 0 ) {
    return new WP_REST_Response(
        array( 'error' => 'invalid_price', 'message' => 'Giá sản phẩm phải là số >= 0' ),
        422
    );
}

$price_float = round( (float) $price, 0 ); // VND không có decimal

// Validate sale_price nếu có
$sale_float = null;
if ( '' !== $sale_price && null !== $sale_price ) {
    if ( ! is_numeric( $sale_price ) || (float) $sale_price < 0 ) {
        return new WP_REST_Response(
            array( 'error' => 'invalid_sale_price', 'message' => 'Giá khuyến mãi phải là số >= 0' ),
            422
        );
    }
    $sale_float = round( (float) $sale_price, 0 );
    if ( $sale_float >= $price_float ) {
        return new WP_REST_Response(
            array( 'error' => 'sale_price_too_high', 'message' => 'Giá khuyến mãi phải nhỏ hơn giá gốc' ),
            422
        );
    }
}
```

## Áp dụng

Mọi numeric field từ admin form (price, stock_qty, weight, dimensions) PHẢI validate: (1) is_numeric, (2) range check (>= 0), (3) business logic (sale < regular). Return 422 với message rõ ràng — KHÔNG silent coerce.
