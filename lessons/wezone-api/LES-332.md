  type: "presence"

  scope: "**/*.php"

**Pattern sai:**
```php
// update_address() — dùng legacy billing_/shipping_ meta keys
update_user_meta( $user_id, 'billing_first_name', $first_name );
update_user_meta( $user_id, 'shipping_address_1', $address );
```
**Phải làm:**
```php
// Wezone Commer convention: wezone_ prefix
update_user_meta( $user_id, 'wezone_billing_first_name', $first_name );
update_user_meta( $user_id, 'wezone_shipping_address_1', $address );
```
**Lý do:** Wezone Commer không dùng WooCommerce — meta keys phải dùng `wezone_` prefix để tránh conflict và theo convention
**Grep check:** `update_user_meta.*billing_|update_user_meta.*shipping_` (không có `wezone_`) → HIGH
