## Bad
```tsx
// components/analytics/GoogleAnalytics.tsx exists with gtag
// components/analytics/MetaPixel.tsx exists with fbq
// BUT privacy/page.tsx only mentions: Supabase, SePay, Resend
// No mention of GA4 or Meta Pixel tracking
```
Both Google Ads and Meta Ads require that your privacy policy explicitly discloses all tracking technologies used on the website.

## Good
```tsx
// In privacy/page.tsx:
<h2>Công cụ phân tích và quảng cáo</h2>
<ul>
  <li><strong>Google Analytics 4 (GA4):</strong> Thu thập dữ liệu ẩn danh về hành vi truy cập.
    Opt-out: <a href="https://tools.google.com/dlpage/gaoptout">GA Opt-out Add-on</a></li>
  <li><strong>Meta Pixel:</strong> Thu thập dữ liệu tương tác để tối ưu quảng cáo.
    Quản lý: <a href="https://www.facebook.com/adpreferences">Ad Preferences</a></li>
</ul>
```

## Why
Google Ads requires: "Your website must have a privacy policy that discloses how you collect, use, and share user data, including any tracking technologies." Meta requires: "Advertisers must have a privacy policy that clearly explains data collection practices." Missing disclosure = landing page policy violation = ad rejection or account restriction.
