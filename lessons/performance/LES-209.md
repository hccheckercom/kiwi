## Bad
```tsx
<img src={product.image} alt={product.name} className="rounded" />
```
Browser doesn't know dimensions until image loads → content shifts down → poor CLS score.

## Good
```tsx
<Image src={product.image} alt={product.name} width={300} height={300} className="rounded" />
```
Or with fill mode for responsive images:
```tsx
<div className="relative aspect-square">
  <Image src={product.image} alt={product.name} fill className="object-cover rounded" />
</div>
```

## Why
Images without dimensions cause Cumulative Layout Shift (CLS) — a Core Web Vital. The browser reserves 0px until the image loads, then pushes content down. Use Next.js Image component with explicit width/height or fill + aspect-ratio container.
