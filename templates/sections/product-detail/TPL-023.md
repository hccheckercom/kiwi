# Related Products Section

Display related products in a grid below product details.

## Features
- Fetches related products via `wz_get_related_products()`
- Responsive grid (2 cols mobile, 4 cols desktop)
- Reuses product card component
- Empty state handling (returns early if no related products)

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

$related = function_exists( 'wz_get_related_products' ) ? wz_get_related_products( $product_id, 4 ) : [];
$related = array_filter( $related );

if ( empty( $related ) ) { return; }
?>

<section class="mt-10 md:mt-16">
    <h2 class="text-xl md:text-[28px] font-bold text-[var(--wz-primary)] mb-6">Sản phẩm liên quan</h2>
    <div class="grid grid-cols-2 md:grid-cols-4 gap-3 md:gap-4">
        <?php foreach ( $related as $rel ) :
            if ( ! $rel ) { continue; }
            get_template_part( 'template-parts/product/card', null, [
                'product_id' => $rel['id'],
                'context'    => 'grid',
            ] );
        endforeach; ?>
    </div>
</section>
```

## Usage

```php
// In single-wz_product.php or product detail template
get_template_part( 'template-parts/single-product/related', null, [
    'product_id' => $product_id,
] );
```

## CSS Variables Used
- `--wz-primary` — heading color

## Dependencies
- `wz_get_related_products()` function
- Product card component at `template-parts/product/card.php`
