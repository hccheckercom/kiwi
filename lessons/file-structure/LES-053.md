**Pattern sai:**
```php
<!-- ; ? thay vì ; ?> — thiếu dấu > đóng PHP tag -->
<span style="font-variation-settings: 'FILL' <?php echo $done ? '1' : '0'; ? aria-hidden="true">
<!-- PHP Parse error: syntax error, unexpected token "?" → white screen of death -->
```
**Phải làm:**
```php
<span style="font-variation-settings: 'FILL' <?php echo $done ? '1' : '0'; ?>" aria-hidden="true">
```
**Lý do:** Khi viết inline PHP echo trong HTML attributes phức tạp (đặc biệt style + aria), dễ bị typo mất `>` ở closing tag `?>`. PHP parser thấy `?` đứng một mình → Parse error → critical error toàn page. Khó phát hiện vì IDE không highlight lỗi này trong .php template files.
**Grep check:** `; \?[^>]` hoặc `; \? ` trong *.php — closing tag thiếu `>` → CRITICAL
