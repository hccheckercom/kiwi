## Bad
```python
path = path.replace("~", "/home/user")
path = path.replace("~", "C:/Users/Windows")
```

## Good
```python
import os
path = os.path.expanduser(path)
# or
from pathlib import Path
path = str(Path(path).expanduser())
```

## Why

`.replace("~", ...)` is fragile — it only works for the hardcoded username and OS. `os.path.expanduser()` handles tilde expansion correctly across all platforms and user accounts.
