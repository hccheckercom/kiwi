**Pattern sai:**
CSS chỉ có `.line-clamp-1`, `.line-clamp-2`, `.line-clamp-3` — thiếu `.line-clamp-4`.

**Phải làm:**
```css
.line-clamp-4 {
  display: -webkit-box;
  -webkit-line-clamp: 4;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
```

**Lý do:** Blueprint yêu cầu review text preview giới hạn 4 dòng + "Xem thêm". Thiếu utility này → review text tràn layout.
