# Bad Code

```python
def get_db():
    db = SessionLocal()
    return db  # BAD: Connection never closed

@app.get("/users")
async def get_users(db: Session = Depends(get_db)):
    users = db.query(User).all()
    return users
# Database connection leaks after each request
```

# Good Code

```python
def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

@app.get("/users")
async def get_users(db: Session = Depends(get_db)):
    users = db.query(User).all()
    return users
# Connection automatically closed after request
```

# Why

**Without cleanup:**
- Connection pool exhausted
- "Too many connections" errors
- Memory leaks
- Database server overload

**Real-world impact:**
```python
# After 100 requests without cleanup:
# - 100 open connections
# - Connection pool exhausted
# - New requests fail with "connection timeout"
# - Requires server restart
```

**Dependency lifecycle:**
```python
# Generator pattern (recommended)
def get_db():
    db = SessionLocal()
    try:
        yield db  # Request processing happens here
    finally:
        db.close()  # Always runs, even on exception

# Context manager pattern
from contextlib import contextmanager

@contextmanager
def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

**Async database:**
```python
async def get_async_db():
    async with AsyncSessionLocal() as session:
        yield session
    # Automatically closed by context manager

@app.get("/users")
async def get_users(db: AsyncSession = Depends(get_async_db)):
    result = await db.execute(select(User))
    return result.scalars().all()
```
