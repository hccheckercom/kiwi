**Pattern sai:**
```css
@media (max-width: 768px) {
    .header { padding: 0.5rem; }
}
```
**Phải làm:**
```css
/* mobile styles by default, then override at breakpoint */
.header { padding: 0.5rem; }
@media (min-width: 768px) {
    .header { padding: 1rem; }
}
```
**Lý do:** Mobile-first = default CSS cho mobile, min-width cho larger. max-width = desktop-first
**Grep check:** `@media[^{]*max-width` — bất kỳ hit nào → HIGH
