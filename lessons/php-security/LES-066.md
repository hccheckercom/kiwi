**Pattern sai:**
```php
// wezone-loyalty.php (entry point)
define( 'WEZONE_LOYALTY_VERSION', '1.0.0' );
define( 'WEZONE_LOYALTY_DIR', plugin_dir_path( __FILE__ ) );
define( 'WEZONE_LOYALTY_URL', plugin_dir_url( __FILE__ ) );
// → WordPress multisite load plugin nhiều lần → "Constant already defined" warning x100
```
**Phải làm:**
```php
if ( ! defined( 'WEZONE_LOYALTY_VERSION' ) ) {
    define( 'WEZONE_LOYALTY_VERSION', '1.0.0' );
    define( 'WEZONE_LOYALTY_FILE', __FILE__ );
    define( 'WEZONE_LOYALTY_DIR', plugin_dir_path( __FILE__ ) );
    define( 'WEZONE_LOYALTY_URL', plugin_dir_url( __FILE__ ) );
}
```
**Lý do:** Trong WordPress Multisite, plugin entry file có thể được load nhiều lần (network activate + per-site). Nếu `define()` không wrap trong `defined()` guard → PHP Warning "Constant already defined" lặp lại hàng trăm lần trong deploy log, gây noise che mất lỗi thật. Áp dụng cho MỌI plugin entry point có `define()`.
**Grep check:** `grep -rn "function wezone_funilux_" inc/ | sort` — nếu cùng function name xuất hiện 2+ lần → CRITICAL
