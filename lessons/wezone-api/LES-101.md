## Bad
```php
function calculate_shipping($items) {
    $total_weight = array_sum(array_column($items, 'weight'));
    return $this->get_rate($total_weight);
}
```

## Good
```php
function calculate_shipping(array $items): float {
    $total_weight = array_sum(array_column($items, 'weight'));
    if ($total_weight <= 0) {
        wz_log('Invalid weight for shipping calculation', 'warning');
        return (float) wz_config('shipping.fees.default', 30000);
    }
    return $this->get_rate($total_weight);
}
```

## Why
Shipping rate calculation without weight/dimensions validation — carrier API returns error or fee=0.
