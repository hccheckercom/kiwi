## Bad
```php
$ld_product['offers'] = [
    '@type'         => 'Offer',
    'price'         => $price,
    'priceCurrency' => 'VND',
    'availability'  => 'https://schema.org/InStock',
    'url'           => get_permalink(),
];
```

## Good
```php
$ld_product['offers'] = [
    '@type'           => 'Offer',
    'price'           => $price,
    'priceCurrency'   => 'VND',
    'availability'    => $in_stock ? 'https://schema.org/InStock' : 'https://schema.org/OutOfStock',
    'itemCondition'   => 'https://schema.org/NewCondition',
    'priceValidUntil' => date( 'Y-m-d', strtotime( '+1 year' ) ),
    'url'             => get_permalink(),
    'shippingDetails' => [
        '@type'               => 'OfferShippingDetails',
        'shippingDestination' => [ '@type' => 'DefinedRegion', 'addressCountry' => 'VN' ],
    ],
    'hasMerchantReturnPolicy' => [
        '@type'                => 'MerchantReturnPolicy',
        'applicableCountry'    => 'VN',
        'returnPolicyCategory' => 'https://schema.org/MerchantReturnFiniteReturnWindow',
        'merchantReturnDays'   => (int) wz_config( 'return_days', 30 ),
        'returnMethod'         => 'https://schema.org/ReturnByMail',
    ],
];
// Also add brand at Product level:
$ld_product['brand'] = [ '@type' => 'Brand', 'name' => wz_config( 'shop_name', get_bloginfo('name') ) ];
```

## Why
GMC v4.1 (2026) yêu cầu:
- `priceValidUntil` — effectively required cho rich results (M10-008)
- `itemCondition` — 1 trong 4 trường Automatic Item Updates (M10-007)
- `brand` — required cho hầu hết categories (M10-006)
- `MerchantReturnPolicy` — Google weight return policy trong Shopping ranking (M10-009)
- `OfferShippingDetails` — cải thiện Shopping listing visibility

Thiếu các fields này → sản phẩm không eligible cho rich results, Shopping snippets bị hạn chế, ranking thấp hơn đối thủ.

## Grep check
Tìm `application/ld+json` trong inc/seo.php → nếu có nhưng KHÔNG chứa `priceValidUntil` hoặc `MerchantReturnPolicy` → HIGH
