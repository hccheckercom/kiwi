## Bad
```php
error_log('Order created: ' . $order_id);
```

## Good
```php
wz_log('Order created', 'info', ['order_id' => $order_id]);
```

## Why
error_log() not structured — can't filter, search, or aggregate. Use wz_log() for structured logging.
