## Bad
```python
if key_path.startswith("~"):
    key_path = key_path.replace("~", "C:/Users/Windows")
```

## Good
```python
import os
if key_path.startswith("~"):
    key_path = os.path.expanduser(key_path)
```

## Why

Hardcoding `C:/Users/Username` breaks the code for any other user or OS. `os.path.expanduser("~")` resolves the home directory correctly on Windows, macOS, and Linux. Always use `os.path.expanduser()` or `Path.home()` for tilde expansion.
