  type: "presence"

**Pattern sai:**
```php
$cart_count = WC()->cart->get_cart_contents_count();
$product    = wc_get_product( $product_id );
```
**Phải làm:**
```php
$cart_count = wz_get_cart_count();
$product    = wz_get_product( $product_id );
```
**Lý do:** WooCommerce không được cài — fatal error tại runtime
**Grep check:** `wc_|WC_|WC\(\)|woocommerce_` — bất kỳ hit nào → CRITICAL
