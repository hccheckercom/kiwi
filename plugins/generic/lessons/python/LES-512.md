# Bad Code

```python
async def fetch_user(user_id: int) -> User:
    return await db.get_user(user_id)

async def process_users(user_ids: List[int]):
    users = []
    for user_id in user_ids:
        # BUG: Not awaited - returns coroutine object, not User
        user = fetch_user(user_id)
        users.append(user)
    return users

async def send_notification(user_id: int, message: str):
    user = await fetch_user(user_id)
    # BUG: Not awaited - notification never sent
    notify_service.send(user.email, message)
```

# Good Code

```python
async def fetch_user(user_id: int) -> User:
    return await db.get_user(user_id)

async def process_users(user_ids: List[int]):
    users = []
    for user_id in user_ids:
        user = await fetch_user(user_id)
        users.append(user)
    return users

# Better: Parallel execution
async def process_users_parallel(user_ids: List[int]):
    tasks = [fetch_user(user_id) for user_id in user_ids]
    return await asyncio.gather(*tasks)

async def send_notification(user_id: int, message: str):
    user = await fetch_user(user_id)
    await notify_service.send(user.email, message)
```

# Why

**Not awaiting async functions causes:**
- **Silent failures:** Function never executes
- **Coroutine warnings:** `RuntimeWarning: coroutine was never awaited`
- **Wrong return type:** Returns coroutine object instead of result
- **Resource leaks:** Connections never closed
- **Race conditions:** Operations complete in wrong order

**Common mistakes:**
```python
# Forgot await
user = fetch_user(123)  # Returns <coroutine object>

# Awaited non-async function
result = await sync_function()  # TypeError

# Mixed sync/async in loop
for item in items:
    await process(item)  # Sequential, slow
# Should use: await asyncio.gather(*[process(i) for i in items])
```

**Detection:**
- Python warns: `RuntimeWarning: coroutine 'func' was never awaited`
- Type checkers catch: `Incompatible types in assignment`
- Tests fail: Assertions on coroutine objects

**Best practices:**
- Always `await` async functions
- Use `asyncio.gather()` for parallel execution
- Use `asyncio.create_task()` for fire-and-forget
- Never mix sync/async without `asyncio.to_thread()`
