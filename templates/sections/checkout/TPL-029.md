# Order Summary Card — Sticky Sidebar

Sticky order summary card for checkout/cart pages showing items, subtotal, shipping, and total.

## Features
- Sticky positioning on desktop
- Item list with quantities
- Subtotal/shipping/total breakdown
- Free shipping threshold indicator
- Responsive (full width on mobile)

## Code

```php
<?php
$cart_items = $cart_items ?? [];
$subtotal = 0;
foreach ( $cart_items as $item ) {
    $subtotal += ( $item['price'] ?? 0 ) * ( $item['quantity'] ?? 1 );
}
$free_threshold = wz_config( 'shipping.free_ship_threshold', 5000000 );
$shipping = $subtotal >= $free_threshold ? 0 : 30000;
$total = $subtotal + $shipping;
?>

<div class="order-summary">
    <h3 class="order-summary__title">Tóm tắt đơn hàng</h3>

    <!-- Items -->
    <div class="order-summary__items">
        <?php foreach ( $cart_items as $item ) : ?>
            <div class="order-summary__item">
                <span class="order-summary__item-name">
                    <?php echo esc_html( $item['name'] ); ?>
                    <span class="order-summary__item-qty">×<?php echo esc_html( (string) $item['quantity'] ); ?></span>
                </span>
                <span class="order-summary__item-price">
                    <?php echo esc_html( wz_format_price( $item['price'] * $item['quantity'] ) ); ?>
                </span>
            </div>
        <?php endforeach; ?>
    </div>

    <!-- Totals -->
    <div class="order-summary__totals">
        <div class="order-summary__row">
            <span>Tạm tính</span>
            <span><?php echo esc_html( wz_format_price( $subtotal ) ); ?></span>
        </div>
        <div class="order-summary__row">
            <span>Vận chuyển</span>
            <span><?php echo 0 === $shipping ? 'Miễn phí' : esc_html( wz_format_price( $shipping ) ); ?></span>
        </div>
        <div class="order-summary__row order-summary__row--total">
            <span>Tổng cộng</span>
            <span><?php echo esc_html( wz_format_price( $total ) ); ?></span>
        </div>
    </div>

    <!-- Free shipping indicator -->
    <?php if ( $subtotal < $free_threshold ) : ?>
        <div class="order-summary__free-ship">
            <span class="material-symbols-outlined">local_shipping</span>
            <p>Mua thêm <?php echo esc_html( wz_format_price( $free_threshold - $subtotal ) ); ?> để được miễn phí vận chuyển</p>
        </div>
    <?php endif; ?>
</div>
```

## Usage

```php
// In checkout or cart template
get_template_part( 'template-parts/checkout/order-summary', null, [
    'cart_items' => $cart_items,
] );
```

## CSS Classes
- `.order-summary` — card wrapper (add `position: sticky; top: 120px;` on desktop)
- `.order-summary__row--total` — bold total row

## Dependencies
- `wz_format_price()` function
- `wz_config()` for free shipping threshold
