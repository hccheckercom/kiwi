# Archive Sidebar — Category + Price Filters

Sticky sidebar with collapsible filter groups: categories, price range, and dynamic product attributes.

## Features
- Sticky positioning (desktop only)
- Collapsible filter groups
- Category list with counts
- Price range inputs + quick chips
- Dynamic attribute filters (checkboxes)
- Active state highlighting
- Material Icons

## Code

```php
<?php
declare(strict_types=1);

if ( ! defined( 'ABSPATH' ) ) { exit; }

$categories   = $categories ?? [];
$current_slug = $current_slug ?? '';
?>

<?php $wz_filter_cat_id = isset( $category ) && ! empty( $category['id'] ) ? (int) $category['id'] : 0; ?>
<aside class="hidden lg:block w-[260px] shrink-0 sticky top-[120px] self-start max-h-[calc(100vh-140px)] overflow-y-auto" id="archive-sidebar" data-category-id="<?php echo esc_attr( (string) $wz_filter_cat_id ); ?>">
    <div class="bg-[var(--wz-surface-pure)] rounded-[var(--wz-radius-card)] p-5 shadow-[0_4px_24px_rgba(27,58,122,0.08)]">

        <!-- Categories -->
        <div class="wz-filter-group" data-group="category">
            <button type="button" class="wz-filter-toggle flex items-center justify-between w-full pb-3 border-b border-[var(--wz-outline)]/20">
                <span class="text-[11px] font-bold uppercase tracking-wide text-[var(--wz-primary)]">Danh mục</span>
                <span class="material-symbols-outlined text-[18px] text-[var(--wz-primary)] transition-transform duration-200" aria-hidden="true">expand_more</span>
            </button>
            <div class="wz-filter-body pt-3 space-y-1">
                <?php foreach ( $categories as $cat ) :
                    $slug = $cat['slug'] ?? '';
                    $name = $cat['name'] ?? '';
                    $count = $cat['count'] ?? 0;
                    $is_active = $slug === $current_slug;
                ?>
                    <a href="<?php echo esc_url( home_url( '/category/' . $slug . '/' ) ); ?>"
                       class="flex items-center justify-between py-1.5 px-2 rounded text-sm transition-colors <?php echo $is_active ? 'border-l-2 border-[var(--wz-accent)] font-semibold text-[var(--wz-primary)] bg-[var(--wz-surface-cool)]' : 'text-[var(--wz-on-surface)] hover:bg-[var(--wz-surface-cool)] hover:text-[var(--wz-primary)]'; ?>"
                       data-filter="category" data-value="<?php echo esc_attr( $slug ); ?>">
                        <span><?php echo esc_html( $name ); ?></span>
                        <span class="text-xs text-[var(--wz-outline)]"><?php echo esc_html( (string) $count ); ?></span>
                    </a>
                <?php endforeach; ?>
            </div>
        </div>

        <!-- Price range -->
        <div class="wz-filter-group mt-5" data-group="price">
            <button type="button" class="wz-filter-toggle flex items-center justify-between w-full pb-3 border-b border-[var(--wz-outline)]/20">
                <span class="text-[11px] font-bold uppercase tracking-wide text-[var(--wz-primary)]">Khoảng giá</span>
                <span class="material-symbols-outlined text-[18px] text-[var(--wz-primary)] transition-transform duration-200" aria-hidden="true">expand_more</span>
            </button>
            <div class="wz-filter-body pt-3">
                <div class="flex gap-2 mb-3">
                    <input type="number" name="price_min" placeholder="Từ" min="0"
                           class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-3 py-2 text-xs bg-[var(--wz-surface-cool)] placeholder:text-[var(--wz-outline)] focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                    <input type="number" name="price_max" placeholder="Đến" min="0"
                           class="w-full border border-[var(--wz-outline)]/40 rounded-[var(--wz-radius)] px-3 py-2 text-xs bg-[var(--wz-surface-cool)] placeholder:text-[var(--wz-outline)] focus:ring-2 focus:ring-[var(--wz-primary)] focus:outline-none">
                </div>
                <div class="flex flex-wrap gap-1.5">
                    <button type="button" class="wz-price-chip px-3 py-1.5 rounded-full text-xs font-semibold bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] hover:bg-[var(--wz-primary)]/10 transition-colors" data-min="0" data-max="2000000">Dưới 2tr</button>
                    <button type="button" class="wz-price-chip px-3 py-1.5 rounded-full text-xs font-semibold bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] hover:bg-[var(--wz-primary)]/10 transition-colors" data-min="2000000" data-max="5000000">2-5tr</button>
                    <button type="button" class="wz-price-chip px-3 py-1.5 rounded-full text-xs font-semibold bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] hover:bg-[var(--wz-primary)]/10 transition-colors" data-min="5000000" data-max="10000000">5-10tr</button>
                    <button type="button" class="wz-price-chip px-3 py-1.5 rounded-full text-xs font-semibold bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] hover:bg-[var(--wz-primary)]/10 transition-colors" data-min="10000000" data-max="20000000">10-20tr</button>
                    <button type="button" class="wz-price-chip px-3 py-1.5 rounded-full text-xs font-semibold bg-[var(--wz-surface-cool)] text-[var(--wz-primary)] hover:bg-[var(--wz-primary)]/10 transition-colors" data-min="20000000" data-max="0">Trên 20tr</button>
                </div>
            </div>
        </div>

        <!-- Dynamic attributes -->
        <?php
        $cat_id_for_filter = isset( $category ) && ! empty( $category['id'] ) ? (int) $category['id'] : 0;
        $filter_attrs = function_exists( 'wz_get_filter_attributes' ) ? wz_get_filter_attributes( $cat_id_for_filter ) : [];
        foreach ( $filter_attrs as $attr_name => $attr_values ) :
            $group_key = sanitize_title( $attr_name );
        ?>
        <div class="wz-filter-group mt-5" data-group="<?php echo esc_attr( $group_key ); ?>">
            <button type="button" class="wz-filter-toggle flex items-center justify-between w-full pb-3 border-b border-[var(--wz-outline)]/20">
                <span class="text-[11px] font-bold uppercase tracking-wide text-[var(--wz-primary)]"><?php echo esc_html( $attr_name ); ?></span>
                <span class="material-symbols-outlined text-[18px] text-[var(--wz-primary)] transition-transform duration-200" aria-hidden="true">expand_more</span>
            </button>
            <div class="wz-filter-body pt-3 space-y-2">
                <?php foreach ( $attr_values as $val ) : ?>
                    <label class="flex items-center gap-2 text-sm text-[var(--wz-on-surface)] cursor-pointer hover:text-[var(--wz-primary)] hover:bg-[var(--wz-surface-cool)] rounded px-1 -mx-1 transition-colors">
                        <input type="checkbox" name="<?php echo esc_attr( $group_key ); ?>[]" value="<?php echo esc_attr( sanitize_title( $val ) ); ?>"
                               class="w-4 h-4 rounded accent-[var(--wz-primary)] border-[var(--wz-outline)]/30 focus:ring-[var(--wz-primary)]">
                        <?php echo esc_html( $val ); ?>
                    </label>
                <?php endforeach; ?>
            </div>
        </div>
        <?php endforeach; ?>

    </div>
</aside>
```

## Usage

```php
// In archive-wz_product.php or category template
get_template_part( 'template-parts/archive/sidebar-filters', null, [
    'categories' => $categories,
    'current_slug' => $current_slug,
    'category' => $category,
] );
```

## CSS Variables Used
- `--wz-surface-pure` — sidebar background
- `--wz-primary` — active states
- `--wz-accent` — active border
- `--wz-on-surface` — text
- `--wz-outline` — muted text
- `--wz-surface-cool` — hover states
- `--wz-radius` — input border radius
- `--wz-radius-card` — card border radius

## JavaScript Required
- Toggle filter groups (expand/collapse)
- Price chip click → fill inputs
- Apply filters via AJAX or URL params
