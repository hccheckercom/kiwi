## Bad

```python
def add_item(item, items=[]):
    items.append(item)
    return items

def merge_config(overrides, base={}):
    base.update(overrides)
    return base
```

## Good

```python
def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items

def merge_config(overrides, base=None):
    if base is None:
        base = {}
    base.update(overrides)
    return base
```

## Why

Mutable default arguments are shared across all calls. The list/dict is created once at function definition time, not at each call. This causes data to "leak" between calls — a notorious Python gotcha.

## Grep

```bash
rg 'def\s+\w+\s*\([^)]*=\s*\[\]' --type py
rg 'def\s+\w+\s*\([^)]*=\s*\{\}' --type py
```
