## Bad
```tsx
<p>Triển khai trong 24h</p>
<p>Bàn giao website trong 24 giờ</p>
{ num: '24h', label: 'Ra store' }
```
Absolute time guarantee violates Google Ads "Misleading Claims" and Meta "Unsubstantiated Claims" policies. Account can be suspended immediately.

## Good
```tsx
<p>Triển khai nhanh từ 24h*</p>
<p className="text-xs">*Áp dụng cho template có sẵn. Dự án custom 3-7 ngày làm việc.</p>
{ num: 'từ 24h', label: 'Ra store' }
```
Use qualifier ("từ", "thường", "lên đến") + disclaimer with conditions.

## Why
Google Ads policy explicitly prohibits "false or unrealistic claims about delivery timelines." A guarantee without conditions is considered misrepresentation because external factors (client delays, complexity) can prevent delivery. This triggers the 3-strike system: 1st violation = 3-day hold, 2nd = 7-day hold, 3rd = full suspension.
