## Bad
```php
f = open('file.txt')\ndata = f.read()
```

## Good
```php
with open('file.txt') as f:\n    data = f.read()
```

## Why
Unclosed file handles cause resource leaks and lock files

## Grep
`open\([^)]+\)(?!\s*as\s)` — in `**/*.php`
