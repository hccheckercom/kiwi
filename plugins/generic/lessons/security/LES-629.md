## Bad
```python
examples = eval(row[4]) if row[4] else []
config = eval(user_input)
```

## Good
```python
import json
examples = json.loads(row[4]) if row[4] else []

import ast
config = ast.literal_eval(user_input)
```

## Why

`eval()` executes arbitrary Python code. If the input comes from a database, file, API, or user, an attacker can inject malicious code like `__import__('os').system('rm -rf /')`. Use `json.loads()` for JSON data or `ast.literal_eval()` for Python literals (strings, numbers, lists, dicts, booleans, None).
