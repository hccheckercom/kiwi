## Bad
```javascript
// cart.js — adds item but no tracking
document.querySelector('.btn-add-cart').addEventListener('click', function() {
    fetch('/wp-json/wezone/v1/cart/add', { body: JSON.stringify(item) });
    showToast('Đã thêm vào giỏ');
});
```
Cart additions happen but GA4 never knows — intent stage of funnel is unmeasured.

## Good
```javascript
// tracking.js
document.addEventListener('wz:cart:added', function(e) {
    const item = e.detail;
    dataLayer.push({ ecommerce: null });
    dataLayer.push({
        event: 'add_to_cart',
        ecommerce: {
            currency: 'VND',
            value: item.price * item.quantity,
            items: [{
                item_id: item.sku,
                item_name: item.name,
                item_category: item.category,
                price: item.price,
                quantity: item.quantity
            }]
        }
    });
});
```

## Why
`add_to_cart` is the key intent signal for ecommerce. Without it: (1) GA4 funnel shows 0% intent rate, (2) Google Ads cannot optimize for add-to-cart conversions, (3) cart abandonment rate cannot be calculated. Blueprint target: 8-12% add-to-cart rate — unmeasurable without this event.
