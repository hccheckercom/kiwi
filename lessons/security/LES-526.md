## Bad
```php
subprocess.run(cmd, shell=True)
```

## Good
```php
subprocess.run(cmd.split(), shell=False)
```

## Why
shell=True with user input enables command injection attacks

## Grep
`subprocess\.(run|call|Popen).*shell=True` — in `**/*.php`
