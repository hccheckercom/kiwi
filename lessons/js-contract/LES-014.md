**Pattern sai:**
```php
// functions.php localize:
wp_localize_script( 'main', 'wzTheme', [
    'ajaxUrl' => admin_url('admin-ajax.php'),
    'nonce'   => wp_create_nonce('wz_nonce'),
]);

// main.js dùng:
window.location.href = wzTheme.loginUrl; // undefined → navigates to "undefined"
```
**Phải làm:**
```php
wp_localize_script( 'main', 'wzTheme', [
    'ajaxUrl'  => admin_url('admin-ajax.php'),
    'nonce'    => wp_create_nonce('wz_nonce'),
    'loginUrl' => wp_login_url( get_permalink() ),
    'cartUrl'  => wz_get_cart_url(),
]);
```
**Lý do:** JS dùng property chưa được localize → undefined → broken navigation/logic
**Grep check:** `wzTheme\.\w+` trong *.js → cross-check với `wp_localize_script.*wzTheme` trong PHP — thiếu property = HIGH
