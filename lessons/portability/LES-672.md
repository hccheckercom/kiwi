## Bad
```php
DB_HOST = "192.168.1.100"
```

## Good
```php
DB_HOST = os.environ.get("DB_HOST", "localhost")
```

## Why
Hardcoded IPs break across environments. Use environment variables or config files

## Grep
`["']\d+\.\d+\.\d+\.\d+["']` — in `**/*.py`
