## Bad
```php
file_put_contents($path, $content);
```

## Good
```php
global $wp_filesystem;
if ( ! $wp_filesystem ) {
    require_once ABSPATH . 'wp-admin/includes/file.php';
    WP_Filesystem();
}
$wp_filesystem->put_contents($path, $content);
```

## Why
file_get/put_contents bypasses WP_Filesystem — permission issues on hosts with restricted file access.
