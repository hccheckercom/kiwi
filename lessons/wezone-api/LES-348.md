## Vấn đề

`wp_send_json_success()` và `wp_send_json_error()` luôn gọi `wp_die()` ở cuối → code sau chúng không bao giờ thực thi.

Viết `return;` sau `wp_send_json_*()`:
- Là **dead code** — tạo false impression rằng function có thể tiếp tục
- Che giấu lỗi logic — developer có thể nghĩ function sẽ return bình thường

## Sai

```php
public function ajax_save(): void {
    // ...
    wp_send_json_success( array( 'message' => 'OK' ) );
    return; // ← dead code — wp_send_json_success() đã gọi wp_die()
}
```

## Đúng

```php
public function ajax_save(): void {
    // ...
    wp_send_json_success( array( 'message' => 'OK' ) );
    // Không cần return — execution đã dừng
}
```
