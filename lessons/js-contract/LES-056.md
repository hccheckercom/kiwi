**Pattern sai:**
```php
// addresses.php — có HTML buttons với class wz-edit-addr, wz-delete-addr, wz-default-addr
<button type="button" class="wz-edit-addr ...">Chỉnh sửa</button>
<button type="button" class="wz-delete-addr ...">Xóa</button>
```
```js
// main.js — KHÔNG có event listener nào cho các class này
// Click buttons → không có gì xảy ra
```
**Phải làm:**
```js
// Thêm handler trong main.js IIFE, dùng event delegation
document.addEventListener('click', function (e) {
  var editBtn = e.target.closest('.wz-edit-addr');
  if (!editBtn) return;
  var card = editBtn.closest('[data-id]');
  // populate form từ data attributes của card
  openForm({ id: card.dataset.id, name: card.dataset.name, ... });
});
// PHP card phải có đủ data-* attributes để JS đọc:
// data-id, data-name, data-phone, data-province, data-district, data-ward, data-detail, data-default
```
**Lý do:** Dev viết HTML/PHP template với buttons nhưng quên viết JS handler tương ứng. Buttons hoàn toàn không hoạt động — silent failure, không có error.
**Grep check:** `class="wz-edit-addr\|class="wz-delete-addr\|class="wz-default-addr` trong PHP mà không có `.wz-edit-addr` trong JS → HIGH
**Phải làm:**
```js
const loginForm = document.getElementById('wz-login-form');
if (loginForm) {
    loginForm.addEventListener('submit', async (e) => {
        e.preventDefault();
        const formData = new FormData(loginForm);
        formData.append('action', 'wz_login');
        const res = await fetch(wzTheme.ajaxUrl, { method: 'POST', body: formData });
        // handle response...
    });
}
```
**Lý do:** Form PHP render nhưng JS không handle → form không hoạt động hoặc reload page
**Grep check:** `id="wz-[^"]*form"` trong *.php → cross-check với `getElementById.*wz-.*form` trong *.js — thiếu handler = HIGH
