**Required keys trong response:**
```
count (int) — tổng qty tất cả items
items (array) — mỗi item phải có: key, product_id, name, qty, price, subtotal
subtotal (string) — formatted price
shipping (string) — formatted shipping text
total (string) — formatted total
subtotal_raw (float) — raw number cho JS calculation
total_raw (float) — raw number cho JS calculation
```
**Pattern sai:**
```php
function wz_shim_cart_response( array $cart ): array {
    return [
        'items' => $cart['items'],
        'total' => wz_format_price( $cart['total'] ),
        // thiếu count, subtotal, shipping, subtotal_raw, total_raw
    ];
}
```
**Phải làm:**
```php
function wz_shim_cart_response( array $cart ): array {
    $count = array_sum( array_column( $cart['items'], 'qty' ) );
    return [
        'count'        => $count,
        'items'        => $cart['items'],
        'subtotal'     => wz_format_price( $cart['subtotal'] ),
        'shipping'     => $count > 0 ? wz_format_price( $shipping_fee ) : 'Miễn phí',
        'total'        => wz_format_price( $cart['total'] ),
        'subtotal_raw' => (float) $cart['subtotal'],
        'total_raw'    => (float) $cart['total'],
    ];
}
```
**Lý do:** JS frontend đọc `response.count` cho badge, `response.subtotal_raw` cho calculation. Thiếu key → `undefined` → NaN trong UI, cart badge hiển thị "NaN", mini-cart blank.
**Grep check:** `wz_shim_cart_response` return array — verify có đủ 7 keys → HIGH
