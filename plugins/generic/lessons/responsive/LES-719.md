## Bad
```tsx
// AI demo tool tạo layout mobile-only
<body className="bg-white md:hidden">
  <main>...</main>
</body>
```
Toàn bộ page bị hidden trên desktop.

## Good
```tsx
<body className="bg-white">
  <main>...</main>
</body>
```

## Why
md:hidden trên body/main container ẩn toàn bộ nội dung trên desktop. Chỉ dùng md:hidden cho mobile-only components (bottom nav, mobile menu), không cho page containers.
