## Bad
```php
$theme_slug = wp_get_theme()->get_stylesheet();
return update_option( "wz_icon_overrides_{$theme_slug}", $data );
```

## Good
```php
$theme_slug = sanitize_key( wp_get_theme()->get_stylesheet() );
return update_option( "wz_icon_overrides_{$theme_slug}", $data );
```

## Why
wp_get_theme()->get_stylesheet() returns user-controlled theme directory name. Without sanitization, attackers can create themes with malicious names containing SQL injection chars or path traversal. Always sanitize_key() before interpolating into option names.

## Grep
`(?:get|update|delete)_option\s*\(\s*["\'][\w_]+\{\$(?:theme_slug|stylesheet|template)` — in `**/*.php`
