## Bad
```tsx
<p className="text-gray-900">Product description here</p>
<span className="text-gray-600">Last updated: May 2026</span>
```
Dark text on dark background = invisible content.

## Good
```tsx
<p className="text-gray-900 dark:text-gray-100">Product description here</p>
<span className="text-gray-600 dark:text-gray-400">Last updated: May 2026</span>
```

## Why
`text-gray-900` (#111827) on `dark:bg-gray-900` (#111827) = 1:1 contrast — literally invisible. Text must invert in dark mode. WCAG 1.4.3 requires minimum 4.5:1 contrast ratio for normal text.
