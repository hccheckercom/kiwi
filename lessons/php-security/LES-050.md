## Bad
```php
$query = new WP_Query( $args );
foreach ( $query->posts as $post ) {
    // use $post
}
// Missing wp_reset_postdata()
```

## Good
```php
$query = new WP_Query( $args );
if ( $query->have_posts() ) {
    while ( $query->have_posts() ) {
        $query->the_post();
        // use the_title(), the_content(), etc.
    }
    wp_reset_postdata();
}
```

## Why
WP_Query without wp_reset_postdata() corrupts the global $post — subsequent template tags return wrong data.
