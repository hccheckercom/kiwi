## Bad
```php
f = tempfile.NamedTemporaryFile()
```

## Good
```php
with tempfile.NamedTemporaryFile() as f:
```

## Why
Temp files without context manager may not be cleaned up on exception

## Grep
`tempfile\.\w+\(` — in `**/*.py`
