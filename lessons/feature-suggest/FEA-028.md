**Mô tả:**
Onboarding blueprint yêu cầu mỗi theme phải có folder `theme-demo-data/` chứa dữ liệu mẫu để khách import 1-click.

**Cấu trúc bắt buộc:**
```
theme-demo-data/
├── products.json    (20 sản phẩm: name, price, desc, images, categories)
├── categories.json  (5 danh mục: name, slug, image, parent)
├── pages.json       (static pages: about, contact, policy)
├── menus.json       (navigation menus)
├── banners.json     (hero + promo banners)
├── settings.json    (theme settings: colors, fonts, layout)
└── README.md        (import instructions)
```

**Lý do:**
Khách mua theme cần import demo data trong < 30 giây. Thiếu package = khách phải tự nhập liệu → tỷ lệ bỏ cuộc cao.
