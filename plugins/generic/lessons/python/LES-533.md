# Bad Code

```python
# CRITICAL: Hardcoded credentials
DATABASE_URL = "postgresql://admin:MyP@ssw0rd@localhost/db"
API_KEY = "sk_live_abc123xyz789"
SECRET_KEY = "super-secret-key-12345"

def connect_db():
    return psycopg2.connect(
        host="localhost",
        user="admin",
        password="MyP@ssw0rd",  # Hardcoded
        database="production"
    )

def call_api():
    headers = {
        "Authorization": "Bearer sk_live_abc123xyz789"  # Hardcoded
    }
    return requests.get(API_URL, headers=headers)
```

# Good Code

```python
import os
from dotenv import load_dotenv

load_dotenv()

# GOOD: Load from environment
DATABASE_URL = os.getenv("DATABASE_URL")
API_KEY = os.getenv("API_KEY")
SECRET_KEY = os.getenv("SECRET_KEY")

def connect_db():
    return psycopg2.connect(os.getenv("DATABASE_URL"))

def call_api():
    headers = {
        "Authorization": f"Bearer {os.getenv('API_KEY')}"
    }
    return requests.get(API_URL, headers=headers)

# Or use config management
from config import settings

DATABASE_URL = settings.database_url
API_KEY = settings.api_key
```

# Why

**Hardcoded credentials risks:**
- **Git history:** Credentials in commits forever
- **Public repos:** Exposed to internet
- **Logs:** Credentials in error messages
- **Rotation:** Can't change without code deploy
- **Compliance:** Violates security standards

**Real-world breaches:**
```python
# Committed to GitHub
AWS_ACCESS_KEY = "AKIAIOSFODNN7EXAMPLE"
AWS_SECRET_KEY = "wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY"

# Bots scan GitHub for these patterns
# Account compromised within minutes
# $10,000+ AWS bill from crypto mining
```

**Secure credential management:**
```python
# .env file (not committed)
DATABASE_URL=postgresql://user:pass@localhost/db
API_KEY=sk_live_abc123xyz789
SECRET_KEY=super-secret-key-12345

# .gitignore
.env
.env.local
secrets/

# Load in code
from dotenv import load_dotenv
load_dotenv()

db_url = os.getenv("DATABASE_URL")
```

**Production secrets:**
- AWS Secrets Manager
- HashiCorp Vault
- Azure Key Vault
- Google Secret Manager
- Kubernetes Secrets

**Detection:**
```bash
# Scan for secrets before commit
git secrets --scan
truffleHog --regex --entropy=False .
```
