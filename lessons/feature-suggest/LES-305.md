## Pattern

File `related-products.php` gọi `wz_get_products()` hoặc `wz_query_products()` NHƯNG không có `price_min`, `price_max`, hoặc `price_range` trong cùng file.

## Why This Matters

Related products phải trong cùng price range (±30%) để relevant. Gợi ý áo 50k khi khách đang xem áo 500k → low CTR, waste impression.

**Impact:**
- Gợi ý sản phẩm quá đắt → khách không mua
- Gợi ý sản phẩm quá rẻ → downgrade AOV
- Low CTR → waste prime real estate trên product page

## Bad Example

```php
// related-products.php — KHÔNG filter price range
$product    = wz_get_product( $product_id );
$categories = $product['category_ids'];

$related = wz_get_products( [
    'category_ids' => $categories,
    'exclude'      => [ $product_id ],
    'limit'        => 8,
] );
// → Có thể trả về sản phẩm 50k và 500k cùng lúc
```

## Good Example

```php
// related-products.php — CÓ filter price range
$product    = wz_get_product( $product_id );
$categories = $product['category_ids'];
$price      = $product['price'];

// Price range ±30%
$min_price = $price * 0.7;
$max_price = $price * 1.3;

$related = wz_get_products( [
    'category_ids' => $categories,
    'price_min'    => $min_price,
    'price_max'    => $max_price,
    'exclude'      => [ $product_id ],
    'in_stock'     => true,
    'orderby'      => 'sales_count',
    'limit'        => 8,
] );

// Fallback nếu không đủ 8 items
if ( count( $related ) < 8 ) {
    $bestsellers = wz_get_bestsellers( 8 - count( $related ), [ $product_id ] );
    $related = array_merge( $related, $bestsellers );
}
```

## Priority Order (Blueprint)

```php
function wz_get_recommendations_related( $product_id, $limit = 8 ) {
    // 1. Manual curated (admin chọn tay)
    $manual = wz_get_product_meta( $product_id, '_wz_related_ids' );
    if ( ! empty( $manual ) ) {
        return wz_get_products_by_ids( $manual, $limit );
    }
    
    // 2. Same category + price range (±30%)
    $product    = wz_get_product( $product_id );
    $categories = $product['category_ids'];
    $price      = $product['price'];
    
    $results = wz_query_products( [
        'category_ids' => $categories,
        'price_min'    => $price * 0.7,
        'price_max'    => $price * 1.3,
        'exclude'      => [ $product_id ],
        'in_stock'     => true,
        'orderby'      => 'sales_count',
        'limit'        => $limit,
    ] );
    
    if ( count( $results ) >= $limit ) return $results;
    
    // 3. Same tags (nếu không đủ)
    $tags = $product['tag_ids'] ?? [];
    if ( ! empty( $tags ) ) {
        $tag_results = wz_query_products( [
            'tag_ids'  => $tags,
            'exclude'  => array_merge( [ $product_id ], array_column( $results, 'id' ) ),
            'in_stock' => true,
            'limit'    => $limit - count( $results ),
        ] );
        $results = array_merge( $results, $tag_results );
    }
    
    if ( count( $results ) >= $limit ) return $results;
    
    // 4. Fallback: bestsellers
    $bestsellers = wz_get_bestsellers( $limit - count( $results ), array_column( $results, 'id' ) );
    return array_merge( $results, $bestsellers );
}
```

## How to Fix

1. Đọc price của sản phẩm hiện tại
2. Thêm `price_min` và `price_max` vào query (±30%)
3. Thêm fallback chain: same category+price → same tags → bestsellers
4. Luôn filter `in_stock = true`
5. Luôn exclude sản phẩm hiện tại + sản phẩm đã có trong cart
