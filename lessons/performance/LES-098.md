## Bad
```php
$response = wp_remote_get('https://api.shipping.vn/rates');
```

## Good
```php
$response = wp_remote_get('https://api.shipping.vn/rates', [
    'timeout' => 10,
]);
```

## Why
wp_remote_get without timeout — default is 5s but can block PHP worker if API is slow. Set explicit timeout.
