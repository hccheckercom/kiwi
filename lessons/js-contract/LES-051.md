## Bad
```php
const msg = 'Thêm vào giỏ hàng thành công';
alert(msg);
```

## Good
```php
const msg = wzTheme.i18n.addedToCart || 'Added to cart';
showToast(msg);
```

## Why
Hardcoded Vietnamese in JS — can't rebrand when cloning theme for English/other language clients.
