**Pattern sai:** Dùng `is_singular('product')` hoặc `'post_type' => 'product'` — đây là WooCommerce CPT slug.
WeZone Commer dùng CPT slug `wz_product`.

**Code sai:**
```php
// ❌ WooCommerce slug
if ( is_singular( 'product' ) ) { ... }
if ( get_post_type() === 'product' ) { ... }
$args = [ 'post_type' => 'product' ];
```

**Code đúng:**
```php
// ✅ WeZone Commer slug
if ( is_singular( 'wz_product' ) ) { ... }
if ( get_post_type() === 'wz_product' ) { ... }
$args = [ 'post_type' => 'wz_product' ];
```

**Lý do:** WeZone Commer đăng ký CPT `wz_product` (không phải `product`). Dùng slug sai → `is_singular()` luôn trả false → JSON-LD không render, template conditions fail, SEO schema thiếu. Không fatal error nhưng logic hoàn toàn sai.

**Grep check:** `is_singular\(['"]product['"]\)` trong themes/ → CRITICAL
