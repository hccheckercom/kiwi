## Bad
```tsx
<button className="bg-blue-600 text-white rounded-lg px-4 py-2">
  Submit
</button>
```
Không có focus indicator — keyboard user không biết button nào đang active.

## Good
```tsx
<button className="bg-blue-600 text-white rounded-lg px-4 py-2 focus-visible:ring-2 focus-visible:ring-blue-500 focus-visible:ring-offset-2">
  Submit
</button>
```

## Why
WCAG 2.4.7 yêu cầu visible focus indicator cho mọi interactive element. Thiếu focus-visible khiến keyboard-only users không thể navigate page.
