# Bug: wezone_is_active() Gọi Trực Tiếp Không function_exists() Guard

## Vấn đề

`wezone_is_active()` được define trong `Loader.php` hoặc `shared/src/Helpers.php`. Khi plugin boot() gọi `wezone_is_active()` trực tiếp mà Loader chưa kịp load → **Fatal error: Call to undefined function**.

**Ví dụ lỗi:**
```php
// ❌ SAI — fatal nếu Loader chưa define function
public function boot(): void {
    wz_config( 'wezone-banner' );
    if ( ! wezone_is_active( 'banner' ) ) {  // ← FATAL
        return;
    }
}
```

## Fix

```php
// ✅ ĐÚNG — guard giống Analytics\Plugin.php
public function boot(): void {
    if ( function_exists( 'wz_config' ) ) {
        wz_config( 'wezone-banner' );
    }
    if ( function_exists( 'wezone_is_active' ) && ! wezone_is_active( 'banner' ) ) {
        return;
    }
}
```
