## Bad
```tsx
<input className="placeholder-gray-400" placeholder="Tìm kiếm...">
```
Dark mode: placeholder-gray-400 có thể quá tối trên dark input background.

## Good
```tsx
<input className="placeholder-gray-400 dark:placeholder-gray-500" placeholder="Tìm kiếm...">
```

## Why
Placeholder text cần đủ contrast để readable nhưng không quá sáng để confuse với actual input value. Dark mode cần adjust riêng.
