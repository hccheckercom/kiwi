# Bug: wz_apply_coupon() Không Forward user_id → Bypass Per-User Coupon Limit

## Vấn đề

`wz_validate_coupon()` nhận `$user_id` để kiểm tra giới hạn sử dụng coupon theo user. Nhưng `wz_apply_coupon()` gọi `$engine->validate($code, $subtotal)` **không truyền `$user_id`** → coupon per-user limit bị bypass.

**Impact:** User dùng coupon vượt giới hạn cho phép → mất tiền cho shop owner.

## Bad

```php
// ❌ SAI — thiếu $user_id → per-user limit không check
function wz_apply_coupon( string $code, float $subtotal ): float {
    $engine = wz_coupon_engine();
    $error  = $engine->validate( $code, $subtotal ); // ← thiếu user_id!
    if ( $error !== '' ) {
        return -1.0;
    }
    ...
}
```

## Good

```php
// ✅ ĐÚNG — forward user_id
function wz_apply_coupon( string $code, float $subtotal, int $user_id = 0 ): float {
    $engine  = wz_coupon_engine();
    $uid     = $user_id > 0 ? $user_id : get_current_user_id();
    $error   = $engine->validate( $code, $subtotal, $uid );
    if ( $error !== '' ) {
        return -1.0;
    }
    $coupon_row = $engine->getByCode( $code );
    if ( $coupon_row === null ) {
        return -1.0;
    }
    return $engine->calculateDiscount( $code, $subtotal, $coupon_row );
}
```
